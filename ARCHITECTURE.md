## 每部独立文献的文件体例

每一部独立文献对应 **Excel 一行 + 三份 Markdown**，构成"四件套"：

| 文件 | 命名规则 | 内容 |
|---|---|---|
| Excel 行 | 见多级索引 | 元信息：名称、原名、年代、语言、编号、md 路径等 |
| `XX_原文.md` | `经名_原文.md` | 原始语言落盘（巴利/梵/汉/藏）+ 汉译对照，存真第一 |
| `XX_解释_传统.md` | `经名_解释_传统.md` | 传统佛教术语体系解读，含名称还原、体系定位、术语小考 |
| `XX_解释_现代.md` | `经名_解释_现代.md` | 全程现代中文熟词，术语沉入脚注，流畅如现代文章 |

### 解释_传统.md 固定章节结构

1. **名称还原解释** — 拆解原名词根，还原字面本义
2. **这部经讲了什么（大白话）** — 内容概述
3. **体系定位** — 在佛教知识体系中的位置（核心硬教义 / 大众实践指南 / 戒律制度 / 后期哲学论辩等）
4. **术语小考** — 关键术语的字面本义、汉译方式（音译/意译）、古今理解落差
5. **为什么它重要**

### 解释_现代.md 写作原则

- 正文**不出现**令人陌生的音译术语（涅槃、般若、三昧等）
- 一切专名以大白话表达，如"四条圣者的真理"而非"四圣谛"
- 原始术语以 Markdown 脚注形式保留：`[^1]: 即四圣谛，巴利语 cattāri ariyasaccāni`
- 结构不强制，保持行文流畅，像一篇现代文章

## 多级索引体系

### 顶层总表

`佛教佛法文献古籍大全.xlsx` 按七个历史分期收录 181 条文献，13 个字段：

| 字段 | 说明 |
|---|---|
| 序号 | 1–181，全局唯一 |
| 分期 | I–VII |
| 估计年代 | 原典成立年代 |
| 文献名称 | 中文通用名 |
| 原名异名 | 巴利/梵/藏原名 |
| 原始语言 | |
| 所属传统 | 南传/北传/藏传等 |
| 类型 | 经/律/论/史传/注疏等 |
| 重要译本及译者 | 含年代 |
| 现存语言版本 | |
| 收录编号 | 大正藏T/PTS/Toh.等 |
| 内容简介 | |
| 备注 | |

### 条目级多级索引

每个顶层条目可向下展开为独立的 `XX_明细.xlsx`，采用**灵活分层**结构：

| 字段 | 说明 |
|---|---|
| 节点ID | 如 `001.3.5.11.1`，体现层级 |
| 父节点ID | 串联成树 |
| 层级 | 第几层（不固定上限） |
| 节点类型 | `分类节点`（容器）/ `独立文献` |
| 名称(中文) | 名称列按层级缩进显示 |
| 原名/异名 | |
| 估计年代 | |
| 原始语言 | |
| 收录编号 | |
| 篇幅类型 | 短经·全录 / 长经·节录核心+名句 |
| 原文档案 | 相对路径，仅独立文献行填写 |
| 解释档案 | 相对路径，仅独立文献行填写 |
| 备注 | |

## 原文落盘规范

- **短经**：巴利/梵/汉/藏原文全录 + 汉译全文对照
- **长经**：录开篇总纲、核心名句、结尾，并注明"全文见 SuttaCentral/CBETA"
- 巴利原文使用拉丁转写（含变音符号，如 ā ī ū ṭ ḍ ṇ ṃ ḷ）
- 梵文同上；汉文直接录原文；藏文录威利转写或藏文原字

## 贡献规范

- 新增文献请先在对应的 `XX_明细.xlsx` 中添加节点行，再创建对应 md 文件
- 文件命名统一使用中文，不用拼音缩写
- 中间生成脚本（Python 等）在产物验证通过后删除，不提交入库
- Commit message 格式：`feat: 新增 XX经 原文+解释` / `fix: 修正 XX经 术语注释`
