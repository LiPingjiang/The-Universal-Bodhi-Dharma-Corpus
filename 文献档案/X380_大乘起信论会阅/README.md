## 学术解读

### 一、文献学信息

《大乘起信论会阅》是清代彭际清（1740-1796）所撰。彭际清，字允初，号知归子，清代著名居士佛学家，精通儒释道三家，著有《居士传》《善女人传》《华严念佛三昧论》等。此疏以"会阅"为名，取"会通诸家，阅尽精微"之意，是清代《大乘起信论》注疏中较有特色的一部。

彭际清出身江西名门，早年学儒，后归心佛教，尤其推崇净土宗。他虽为居士，但佛学造诣深厚，与当时高僧如省庵、彻悟等交往密切。此疏作于其晚年，融会儒释，以净土宗立场重新解释《起信论》。

### 二、历史背景

清代佛教以居士佛学为一大特色，彭际清是其中的代表人物。清代中期，佛教宗派衰落，但居士佛学蓬勃发展。彭际清以儒者的身份深入佛学，试图以儒家"心学"来会通佛教"心性"思想。此疏正是在这一背景下产生的，体现了清代"儒释合一"的学术思潮。

### 三、思想史定位

彭际清此疏的特色在于以"儒释会通"和"会归净土"为两大主线。他认为，《起信论》的"心真如门"即是儒家所说的"本心""良知"，"心生灭门"即是"人欲""习气"。彭际清进一步将"信"解释为对阿弥陀佛和西方净土的信，将《起信论》完全纳入净土宗的框架。这一解释虽然偏离了《起信论》的原意，但反映了清代居士佛学的独特风貌，对后来近代佛教的转型有间接影响。

## 白话解读

### 核心内容

《大乘起信论会阅》是清代大居士彭际清对《大乘起信论》的会通性注释。彭际清是儒生出身，后来学佛，所以他用儒家的语言来解释佛教。

彭际清说：《起信论》的"心真如门"，就是儒家王阳明说的"良知"——每个人心中本有的道德光明；"心生灭门"就是"人欲"——被私欲遮蔽的状态。修行的目的，就是让"良知"重新显现。

同时，彭际清又把《起信论》和净土宗联系起来："信"就是信阿弥陀佛，"愿"就是愿生西方，"行"就是念佛。这样，一部教理论书就变成了净土修行的指南。

### 现代启示

1. **跨文化理解**：彭际清以儒解佛，启示我们：不同文化传统之间可以相互理解、相互借鉴。在全球化时代，这种跨文化的能力尤为重要。

2. **居士佛学**：彭际清作为居士，其佛学造诣不逊于出家人。这启示我们：学佛不必出家，在家同样可以深入研究。

3. **会通精神**："会阅"的精神——广泛阅读、融会贯通，对今天的终身学习有重要启示。

### 阅读建议

- 彭际清的疏解较为自由，适合有一定儒学和佛学基础的读者
- 可配合《居士传》《华严念佛三昧论》一同阅读，了解其整体思想
- 注意其"儒释会通"的特点，理解清代居士佛学的独特视角

3. "会阅"的广博与会归的专一，如何平衡？
