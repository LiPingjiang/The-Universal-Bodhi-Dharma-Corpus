# 华严金师子章 · 白話解讀

## 這是什麼書？

《華嚴金師子章》是法藏法師用金獅子做比喻，給武則天講解華嚴宗道理的記錄。「師子」就是「獅子」的古寫。這是華嚴宗最通俗的入門書——用一個生動的比喻，把最難懂的哲學講得清清楚楚。

## 核心內容

### 1. 金獅子的比喻

金 = 理（本質、真如）
獅子 = 事（現象、形象）

金做成獅子，獅子就是金。兩者不即不離、圓融無礙。

### 2. 十玄門的金獅子版

法藏用金獅子逐一解釋十玄門，使抽象法門變得具體可感。

### 3. 六相圓融的金獅子版

總相——整個獅子
別相——眼耳鼻舌身各部分
同相——同屬一獅子
異相——各部分不同
成相——合力構成獅子
壞相——各守自性

### 4. 五教的金獅子版

小教——只見獅子不見金
始教——知金與獅子有別
終教——知金即獅子
頓教——悟金獅雙亡
圓教——見金獅重重無盡

## 現代啟示

1. **比喻的力量**：用比喻講道理是最有效的教學方法之一。
2. **本質與現象**：金和獅子的關係就是本質和現象的關係。
3. **整體與部分**：六相圓融說明整體和部分不對立而圓融。
4. **認識的層次**：五教說明同一事物的認識可以有多個層次。
