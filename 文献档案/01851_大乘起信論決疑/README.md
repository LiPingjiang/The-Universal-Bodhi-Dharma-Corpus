# 大乘起信論決疑

## 學術解讀

### 一、文獻學信息

《大乘起信論決疑》是唐代吉藏（549-623）所撰。吉藏，俗姓安，祖籍安息（今伊朗），生於金陵（今江蘇南京），爲隋代三論宗創始人，世稱"嘉祥大師"。此疏以"決疑"爲名，取"決斷疑惑，疏通滯礙"之意，是三論宗對《大乘起信論》進行系統批判和重新詮釋的重要著作。

吉藏一生精通三論（《中論》《百論》《十二門論》），善於以"破邪顯正"的方式闡發中觀思想。著有《三論玄義》《中論疏》《百論疏》《法華經義疏》等。此疏作於隋代，是吉藏以三論宗"中觀"思想來審視《起信論》的代表作。

### 二、歷史背景

隋代是中國佛教統一南北、整合各宗的重要時期。吉藏作爲三論宗大師，面對當時流行的《起信論》如來藏思想，認爲其某些表述容易讓人產生"實體化"的誤解——即將"真如""一心"理解爲某種實體或本體。三論宗主張"緣起性空"，反對任何實體化的傾向。此疏正是在這一背景下產生，旨在以"中觀"的"空"思想來"決斷"《起信論》可能引發的"疑惑"。

### 三、思想史定位

吉藏此疏的核心立場是"以空決疑"。他認爲，《起信論》的"一心二門"雖然善巧，但學人容易將"心真如"理解爲"實體"，將"心生滅"理解爲"現象"，從而陷入"體用二元"的誤區。吉藏以三論宗的"八不中道"來重新解釋："心真如"不是"有"，也不是"無"，而是超越有無的"空"；"心生滅"不是"實"，也不是"虛"，而是緣起的假名。"一心"不是某種"本體"，而是對"緣起性空"的另一種表述。這一解釋對後來理解三論宗與如來藏思想的關係有重要參考價值。

## 白話解讀

### 核心內容

《大乘起信論決疑》是隋代三論宗大師吉藏對《大乘起信論》的批判性詮釋。吉藏大師是三論宗的創始人，精通"中觀"思想。

吉藏大師說：《起信論》講"一心二門"很好，但很多人讀完後會誤解。他們以爲"心真如"是一個"東西"——像桌子、椅子一樣真實存在的"本體"；"心生滅"是另一個"東西"——像影子一樣虛幻的"現象"。這樣就變成了"兩個東西"，陷入了"實體化"的誤區。

吉藏大師用"八不中道"來糾正："心真如"不是"有"，也不是"無"；不是"常"，也不是"斷"。它超越了一切概念，只能用"空"來描述。"一心"不是某種"本體"，而是對"緣起性空"的方便說法。

### 現代啓示

1. **避免實體化**：現代人常常把抽象概念當成實體（如把"市場""國家"當成有意志的存在）。吉藏的"空"思想啓示我們：概念是工具，不是實體。

2. **超越二元對立**：吉藏強調"離於二邊"（不執着於有、無、常、斷等對立），啓示我們：面對複雜問題，不要陷入非此即彼的思維。

3. **批判性思維**：吉藏對《起信論》持批判態度，但不是否定。這種"批判性繼承"的態度，對今天的學習和研究都有啓示。

### 閱讀建議

- 吉藏的疏解需要三論宗基礎，建議先讀《三論玄義》
- 可對比曇延以三論宗解《起信論》的方式，理解三論宗解釋傳統的延續
- 注意吉藏"以空決疑"的批判立場，理解中觀思想與如來藏思想的差異
