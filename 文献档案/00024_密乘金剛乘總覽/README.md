## 密乘源流骨架

密乘（Vajrayāna，"金剛乘"，又稱 Mantrayāna "真言乘"、Tantrayāna "怛特羅乘"）是印度佛教發展的第三階段，在大乘佛教（Mahāyāna）的基礎上，融合了印度密教（Tantra）的修行方法，形成了以灌頂（abhiṣeka）、本尊（iṣṭadevatā）、咒語（mantra）、手印（mudrā）、壇城（maṇḍala）爲核心的修行體系：

```
        密乘（Vajrayāna）发展脉络
        ├── 事续（Kriyā-tantra）— 以外在仪式为主，如《大日经》
        │       ├── 《苏悉地经》（Susiddhikara-sūtra）
        │       └── 《金刚顶经》（Vajraśekhara-sūtra）部分
        ├── 行续（Caryā-tantra）— 外在仪式与内在修行并重
        │       └── 《大日经》（Mahāvairocana-sūtra）
        ├── 瑜伽续（Yoga-tantra）— 以内在修行为主
        │       └── 《金刚顶经》（Vajraśekhara-sūtra）
        └── 无上瑜伽续（Anuttara-yoga-tantra）— 最高密续
                ├── 父续（Pitṛ-tantra）— 以方便（upāya）为主
                │       ├── 《密集金刚续》（Guhyasamāja-tantra）
                │       └── 《喜金刚续》（Hevajra-tantra）
                ├── 母续（Mātṛ-tantra）— 以智慧（prajñā）为主
                │       ├── 《胜乐轮续》（Cakrasaṃvara-tantra）
                │       └── 《时轮续》（Kālacakra-tantra）
                └── 不二续（Advaya-tantra）
                        └── 《时轮续》（部分分类）
```

密乘的核心特色在於：以"果位道"（phala-yāna）爲修行方法，即將成佛的果位（buddhahood）作爲修行的起點和方法，通過觀想本尊（iṣṭadevatā，"所選擇的神明"）、持誦咒語（mantra）、結手印（mudrā）、觀想壇城（maṇḍala）等方法，在此生此世快速成就佛果。這與顯教（Sūtrayāna）的"因位道"（hetu-yāna）——通過積累福德和智慧，經歷三大阿僧祇劫才能成佛——形成鮮明對比。

密乘在印度約公元7–12世紀達到鼎盛，隨後隨着伊斯蘭教的入侵而在印度本土衰落，但通過西藏（藏傳佛教）、中國（漢傳密教）、日本（東密、臺密）等地的傳播，至今仍是重要的佛教傳統。

## 關鍵概念速查

| 概念 | 梵語 | 簡釋 |
|------|------|------|
| 金剛乘 | Vajrayāna | 密乘，以金剛（堅不可摧）爲喻的修行乘 |
| 灌頂 | abhiṣeka | 密乘入門儀式，傳授修法權限 |
| 本尊 | iṣṭadevatā | 修行者所選擇觀想的佛菩薩 |
| 咒語 | mantra | 具有神聖力量的音節或語句 |
| 手印 | mudrā | 具有象徵意義的手勢 |
| 壇城 | maṇḍala | 本尊及其眷屬的宮殿圖，修行的象徵宇宙 |
| 續 | tantra | 密乘經典，字面義"連續""脈絡" |
| 無上瑜伽 | Anuttara-yoga | 最高級別的密續，包含生起次第與圓滿次第 |
| 生起次第 | utpatti-krama | 觀想本尊的修行階段 |
| 圓滿次第 | sampanna-krama | 修煉氣脈明點的修行階段 |
