## 關於本目錄

本目錄收錄與**佛教第二次結集**（巴利語：Dutiya Saṅgīti，梵語：Dutiya Saṅgīti）直接相關的原始文獻與解讀資料。第二次結集是佛陀入滅約一百年後，由長老耶舍（Yasa Kākaṇḍakaputta）發起、在毗舍離（Vesālī）舉行的經典結集，是佛教史上第一次因戒律爭議而召開的大規模僧團會議，也是導致上座部與大衆部根本分裂的直接導火索。此次結集裁定"十事"全部非法，確立了戒律的嚴格標準，同時也開啓了佛教部派分裂的歷史進程。

本目錄中每部文獻均提供**三件套**：
- **原文** — 巴利語或漢文原始文本節錄，含甲譯（白話意譯）與乙譯（嚴謹對照）
- **學術解讀** — 文獻學、歷史學、思想史角度的專業分析
- **白話解讀** — 面向普通讀者的通俗講解，佛教術語均作白話化翻譯並附腳註

## 文獻列表

### 1. 律藏·毗舍離犍度（Cullavagga XII）

> 巴利律藏《小品》第十二章，記載第二次結集最權威的原始資料。全犍度敘述結構清晰：起因（耶舍目睹十事）→ 奔走召集 → 七百人結集 → 逐條裁定十事 → 結集圓滿。是現存關於這一歷史事件最詳盡、最權威的第一手文獻。

### 2. 十事非法

> "十事"（dasa vatthūni）是毗舍離跋耆族比丘奉行的十項被保守派長老認爲違律的做法，是導致第二次結集召開、進而引發上座部與大衆部分裂的直接導火索。七百長老逐一裁定十事均屬"非法"（adhamma）。

## 關鍵人物

- **耶舍**（Yasa Kākaṇḍakaputta）：發起結集的長老，來自拘舍彌（Kosambī），是第二次結集的核心推動者
- **離婆多**（Revata）：德高望重的長老，被推舉主持裁定十事，其權威性是結集合法性的關鍵
- **薩婆迦彌**（Sabbakāmī）：據傳是當時在世最年長的比丘，曾親從阿難受戒，其證詞具有歷史連接意義
- **跋耆族比丘**（Vajjiputtaka bhikkhū）：毗舍離的比丘羣體，十事的奉行者，結集後另立大衆部

## 關鍵概念

- **十事**（dasa 原文.md)
- **七百人結集**（Sattasatikā）：正統派召集七百比丘裁定十事的結集，即第二次結集
- **大結集**（Mahāsaṅgīti）：毗耆離比丘不服裁決後另行召集的萬人大會，大衆部由此得名
- **根本分裂**（mūlabheda）：上座部與大衆部的分裂，被視爲佛教部派史的起點
