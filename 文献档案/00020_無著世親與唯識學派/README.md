## 概述

唯識學派（Vijñānavāda，"唯識論"；Yogācāra，"瑜伽行派"）是大乘佛教的兩大哲學傳統之一（另一爲中觀學派）。無著（Asaṅga，約4世紀）和世親（Vasubandhu，約4-5世紀）兄弟是唯識學派的奠基人。唯識學派的核心主張是：一切現象都是識（vijñāna）的變現，沒有獨立於識之外的外境（bāhya-artha）。

唯識學派對佛教心理學、認識論和修行論有深遠影響，是中國法相宗（玄奘、窺基）和藏傳佛教的重要思想基礎。

## 核心概念

- **無著**（Asaṅga）：唯識學派奠基人，約4世紀，著有《攝大乘論》、《瑜伽師地論》
- **世親**（Vasubandhu）：無著之弟，著有《唯識二十論》、《唯識三十頌》
- **唯識**（vijñaptimātratā）：一切現象都只是識的變現
- **阿賴耶識**（ālaya-vijñāna）："藏識"，儲存一切業種子的根本識
- **三性**（tri-svabhāva）：遍計所執性、依他起性、圓成實性
- **種子**（bīja）：儲存在阿賴耶識中的業力潛能
- **轉依**（āśraya-parāvṛtti）：修行的目標，轉化阿賴耶識
