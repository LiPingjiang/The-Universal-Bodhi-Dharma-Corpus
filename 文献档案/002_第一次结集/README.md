## 关于本目录

本目录收录与**佛教第一次结集**（梵语：Saṅgīti，巴利语：Saṅgāyana）直接相关的原始文献与解读资料。第一次结集是佛陀入灭后，由大迦叶（Mahākassapa）主持、在王舍城（Rājagaha）附近的七叶窟（Sattapaṇṇi-guhā）举行的经典结集，是佛教史上最早、最重要的一次经典整理活动。此次结集确立了经藏（Sutta-piṭaka）与律藏（Vinaya-piṭaka）的基本框架，奠定了后世佛教三藏传承的基础。

本目录中每部文献均提供**三件套**：
- **原文** — 巴利语或汉文原始文本节录，含甲译（白话意译）与乙译（严谨对照）
- **学术解读** — 文献学、历史学、思想史角度的专业分析
- **白话解读** — 面向普通读者的通俗讲解，佛教术语均作白话化翻译并附脚注

## 第一次结集的历史意义

第一次结集发生在佛陀涅槃（约公元前5世纪）后的第一个雨安居期间，地点在王舍城附近的七叶窟。参加结集的是五百位阿罗汉（已彻底断除烦恼的圣者），由大迦叶主持。结集的核心成果是：

1. **经藏的诵出**：阿难诵出佛陀生前所说的教法（Dhamma），形成经藏（Sutta-piṭaka）的雏形。
2. **律藏的诵出**：优波离诵出佛陀制定的戒律（Vinaya），形成律藏（Vinaya-piṭaka）的雏形。
3. **经典的确认**：通过五百阿罗汉的共同认证，确立了"如是我闻"（Evaṃ me sutaṃ）的经典开头格式，保证了传承的可靠性。

第一次结集不仅是佛教经典传承的起点，也确立了佛教僧团内部以"结集"（Saṅgīti）方式整理与确认经典的制度，为后世第二次结集（毘舍离结集）、第三次结集（华氏城结集）乃至第四次结集（阿育王时代、迦腻色迦王时代）奠定了基础。
