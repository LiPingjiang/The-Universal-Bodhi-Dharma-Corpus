## 學術解讀

### 一、文獻學信息

《大乘起信論裂網疏》是明代憨山德清（1546-1623）所撰。憨山大師爲明代四大高僧之一，精通華嚴、禪宗、淨土，著述宏富。此疏以"裂網"爲名，取"裂破疑網"之意，旨在破除學人對《大乘起信論》的種種疑惑和誤解。

《大乘起信論》相傳爲馬鳴菩薩造，真諦三藏譯，是漢傳佛教影響最深遠的論書之一。全論以"一心二門"（心真如門、心生滅門）爲綱，系統闡述如來藏緣起思想。

### 二、歷史背景

明代中後期，佛教復興運動興起，憨山德清與紫柏真可、雲棲祩宏、藕益智旭並稱"明末四大高僧"。憨山大師一生坎坷，曾因牽涉政治事件被流放嶺南，但始終堅持弘法。此疏作於其晚年，融會禪宗與華嚴思想，以"裂網"破斥當時學界對《起信論》的諸多誤解，特別是對"一心二門"結構的疑惑。

### 三、思想史定位

憨山此疏的特色在於以禪宗"明心見性"的立場來疏解《起信論》。他強調"心真如門"即是禪宗所證的"本心"，"心生滅門"即是"妄念"。憨山認爲，學人之所以被"網"所困，是因爲將"二門"看作兩個獨立的東西，不知二門本是一心，生滅即真如、真如即生滅。這一解釋對後來禪宗與教宗的融合有重要推動作用。

## 白話解讀

### 核心內容

《大乘起信論裂網疏》是憨山德清大師對《大乘起信論》的註解。"裂網"就是撕破迷惑之網的意思。憨山大師發現，很多人讀《起信論》時會被各種概念困住，就像被網纏住一樣。

《起信論》的核心是"一心二門"：我們的心有兩個面向。一個是"心真如門"——心的本來面目，清淨、光明、不生不滅；另一個是"心生滅門"——我們平常感受到的喜怒哀樂、念頭來去。憨山大師說，這兩個門不是兩個心，而是同一個心的兩個表現。就像水有平靜的一面，也有波浪的一面，但都是水。

### 現代啓示

1. **情緒管理**：當我們被負面情緒困擾時，可以回想"心真如門"——情緒的深層是清淨的心。不是壓抑情緒，而是看到情緒背後的本質。

2. **自我覺察**：現代人常常陷入思維反芻（不斷回想過去或擔憂未來）。"一心二門"提醒我們：念頭來來去去，但覺察念頭的那個"心"本身是不動的。

3. **壓力應對**：工作壓力、生活焦慮就像"生滅門"中的波浪。認識到"波浪即是水"，可以在動盪中保持內心的穩定。

### 閱讀建議

- 憨山大師的文風較爲自由，常引禪宗公案，建議有一定佛學基礎者閱讀
- 可配合《大乘起信論》原文和憨山《觀老莊影響論》一同研讀
- 注意憨山以禪解教的特點，不要將其與純粹義理派的解釋混淆
