## 佛教因明源流骨架

佛教因明（Hetuvidyā，"因的學問"，即關於推理與論證的學問）是印度佛教哲學中最具系統性的邏輯認識論傳統。其發展脈絡可分爲古因明與新因明兩大階段：

```
        佛教因明发展脉络
        ├── 古因明（旧因明）
        │       ├── 足目（Akṣapāda Gautama）《正理经》— 印度逻辑总源
        │       ├── 弥勒（Maitreya）《瑜伽师地论》— 佛教因明初建
        │       └── 无著（Asaṅga）《显扬圣教论》— 五支作法
        ├── 新因明（陈那体系）
        │       ├── 陈那（Dignāga，约480–540）
        │       │       ├── 《集量论》（Pramāṇasamuccaya）— 新因明奠基
        │       │       ├── 《因明正理门论》（Nyāyamukha）— 三支作法
        │       │       └── 《观所缘缘论》（Ālambanaparīkṣā）— 认识论
        │       └── 法称（Dharmakīrti，约600–660）
        │               ├── 《量评释》（Pramāṇavārttika）— 因明集大成
        │               ├── 《量决择》（Pramāṇaviniścaya）— 简明纲要
        │               └── 《正理一滴》（Nyāyabindu）— 入门要典
        └── 汉传因明
                ├── 玄奘（602–664）— 传入中国，翻译《因明入正理论》
                └── 窥基（632–682）— 《因明大疏》，汉传因明集大成
```

陳那（Dignāga）是佛教因明史上的革命性人物。他將古因明的五支作法（宗、因、喻、合、結）簡化爲三支作法（宗、因、喻），並建立了以現量（pratyakṣa，直接知覺）和比量（anumāna，推理）爲核心的認識論體系。法稱（Dharmakīrti）在陳那基礎上進一步深化，將因明與佛教哲學（特別是唯識學）緊密結合，成爲印度佛教哲學史上最重要的邏輯學家之一。

漢傳因明以玄奘從印度帶回的《因明入正理論》（商羯羅主著）和《因明正理門論》（陳那著）爲核心，經窺基系統註疏，形成了獨具特色的漢傳因明傳統。藏傳因明則以法稱七論爲核心，至今仍是藏傳佛教寺院教育的必修科目。

## 關鍵概念速查

| 概念 | 梵語 | 簡釋 |
|------|------|------|
| 因明 | Hetuvidyā | 關於推理與論證的學問，佛教邏輯學 |
| 現量 | pratyakṣa | 直接知覺，不經推理的直接認識 |
| 比量 | anumāna | 推理，通過因（理由）推知宗（結論） |
| 三支作法 | tri-rūpa-liṅga | 宗（命題）、因（理由）、喻（例證）三支論式 |
| 宗 | pakṣa | 論式的命題，待證的結論 |
| 因 | hetu | 論式的理由，支持宗的根據 |
| 喻 | dṛṣṭānta | 論式的例證，同喻與異喻 |
| 九句因 | nava-padārtha | 陳那建立的因的九種形式，判斷因的有效性 |
| 量 | pramāṇa | 有效的認識手段，現量與比量 |
| 自比量 | svārthānumāna | 爲自己推理，內心的推理過程 |
| 他比量 | parārthānumāna | 爲他人論證，公開的論式表達 |
