## 名称还原与解释

**大藏经**（Tripitaka / Da Zang Jing）：将佛教一切经/律/论汇集成一套完整的"全集"——是人类历史上最庞大的宗教文献编纂工程之一。

| 碛砂藏 | 1231-1322（宋元） | |

### 大正藏的结构

大正藏（Taisho Shinshu Daizokyo）编号体系（T#）是全球佛教学术的标准：

| 卷号 | 内容 |
|------|------|
| 1-21 | 经集（阿含/般若/法华/华严/宝积/大集/涅槃等） |
| 22-24 | 律部 |
| 25-32 | 论部（印度论典） |
| 33-43 | 经疏部（中国注疏） |
| 44-48 | 论疏/诸宗部 |
| 49-52 | 史传部/事汇部 |
| 53-54 | 外教/目录部 |
| 55 | 目录部（经录） |
| 56-84 | 续藏（日本撰述等） |
| 85 | 敦煌写本/古逸部 |

### 汉文大藏经的历史意义

1. **保存了大量在印度已失传的文献**：大毗婆沙论、大智度论、十住毗婆沙论等重要论典只存汉译。
2. **世界最早的大规模印刷工程**：开宝藏（971年）是人类历史上最早的大规模印刷出版项目之一。
3. **文化交流的纪念碑**：从安世高（148年）到义净（713年）——500多年间——无数翻译家把梵文佛典翻译成汉文——这是古代世界最大规模的跨文化翻译运动。
4. **CBETA革命**：电子化大藏经让全球任何人都可以免费检索全部汉文佛典——彻底民主化了佛教研究。

---

## 大白话总结

汉文大藏经是人类文明史上最庞大的翻译-编纂工程之一——它把数千部印度佛教经典翻译成汉文、汇编成一套"全集"——再加上中国人自己写的注释和著作——最终形成了一座包含约10万页文字的巨大知识宝库。

从971年第一部刻本大藏经到今天的CBETA电子版——汉文大藏经经历了千年的演进。对于现代研究者来说，大正藏（1924-1934年日本编纂）仍然是全球标准——几乎所有学术论文引用汉传佛教文献时都使用"T"加编号的格式。
