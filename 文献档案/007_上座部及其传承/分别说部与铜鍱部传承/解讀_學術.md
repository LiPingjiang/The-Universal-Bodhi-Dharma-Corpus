## 一、历史背景：从分别说部到铜鍱部

### 1.1 部派分裂的历史脉络

分别说部（Vibhajyavāda）是上座部系的重要分支，其历史可以追溯到佛灭后百年的第二次结集（吠舍离结集）。

**历史脉络**：

1. **原始佛教**（佛在世至佛灭后百年）：统一僧团
2. **第二次结集**（佛灭后百年）：大众部与上座部分裂
3. **上座部分裂**：说一切有部、分别说部等分出
4. **分别说部分支**：铜鍱部、法藏部、化地部等六支
5. **铜鍱部南传**：摩哂陀传教斯里兰卡，形成南传佛教

**历史意义**：

分别说部的形成标志着佛教从「统一」走向「多元」，从「口传」走向「文字」，从「印度」走向「世界」。

### 1.2 分别说部的名称与主张

**名称分析**：

- **Vibhajyavāda** = Vibhajya（分别）+ vāda（说、主张）
- **含义**：「分别说者」，即分别说「有」与「无」，非一概而论

**核心主张**：

分别说部的核心主张是「分别说」——即分别说「有」与「无」，非如说一切有部「一切说有」。

| 问题 | 分别说部 | 说一切有部 | 大众部 |
|------|----------|------------|--------|
| 过去法 | 无（已灭） | 有（法体恒有） | 无 |
| 现在法 | 有（正起） | 有（法体恒有） | 有 |
| 未来法 | 无（未生） | 有（法体恒有） | 无 |
| 无为法 | 有 | 有 | 有 |

**哲学意义**：

分别说部的「分别说」立场是一种「中道」立场：既不如说一切有部「一切说有」（堕入常见），也不如大众部「一切说无」（堕入断见）。这种立场在后来的佛教发展中具有重要意义。

### 1.3 铜鍱部的形成

铜鍱部（Theravāda）是分别说部的一支，后来成为南传佛教的正统。

**名称分析**：

- **Theravāda** = Thera（长老）+ vāda（说、主张）
- **含义**：「长老部」，即坚持长老传统者
- **汉译**：「铜鍱部」或「上座部」

**形成过程**：

1. **分别说部分支**：铜鍱部从分别说部分出
2. **南传斯里兰卡**：摩哂陀传教斯里兰卡，建立铜鍱部传统
3. **巴利三藏结集**：在斯里兰卡结集巴利三藏，形成完整传承
4. **东南亚传播**：从斯里兰卡传播到缅甸、泰国、柬埔寨、老挝

## 二、摩哂陀传教：佛教南传的历史事件

### 2.1 阿育王的佛教政策

阿育王（Aśoka，约公元前304-232年）是印度历史上最重要的佛教护法王之一。

**阿育王的佛教政策**：

1. **皈依佛教**：征服羯陵伽（Kalinga）后，因战争惨烈而皈依佛教
2. **弘扬佛法**：派遣使者到各地传教，包括斯里兰卡、希腊、埃及等
3. **建立石柱**：建立法敕石柱，刻上佛教教义
4. **支持僧团**：供养僧团，建立寺院

**派遣使者**：

| 使者 | 目的地 | 结果 |
|------|--------|------|
| 摩哂陀（Mahinda） | 斯里兰卡 | 成功，建立南传佛教 |
| 须那迦（Soṇa） & 郁多罗（Uttara） | 金地（Suvarṇabhūmi，今缅甸） | 成功，建立缅甸佛教 |
| 末阐提（Majjhantika） | 罽宾（Kashmir） & 乾陀罗（Gandhāra） | 成功 |
| 摩诃勒弃多（Mahārakkhita） | 臾那世界（Yona，希腊） | 记载不详 |
| 摩诃昙无德（Mahādeva） | 摩诃勒吒（Mahāraṭṭha） | 成功 |

### 2.2 摩哂陀传教的历史意义

摩哂陀（Mahinda）传教斯里兰卡是佛教南传的里程碑事件。

**历史意义**：

1. **佛教国际化**：佛教从印度传播到斯里兰卡，开始国际化进程
2. **巴利语保存**：巴利语在斯里兰卡保存，成为现存最完整的佛教语言
3. **三藏结集**：巴利三藏于斯里兰卡结集，成为南传佛教的根本经典
4. **僧团建立**：在斯里兰卡建立僧团，使佛教制度化

**传教过程分析**：

1. **时机选择**：阿育王统一印度后，社会稳定，适合传教
2. **人员选择**：摩哂陀是阿育王子，有王室背景，易于与斯里兰卡王室接触
3. **地点选择**：斯里兰卡地理位置重要，是印度洋贸易枢纽
4. **方法选择**：先接触王室，再推广民间，自上而下

### 2.3 僧伽蜜多与比丘尼传承

僧伽蜜多（Saṅghamittā）是摩哂陀的妹妹，她传教斯里兰卡的历史意义同样重要。

**历史意义**：

1. **比丘尼传承**：建立斯里兰卡的比丘尼传承，使女性可以正式出家
2. **菩提树移植**：将菩提树枝移植到斯里兰卡，象征佛教传承
3. **女性地位**：提高佛教中女性的地位，促进性别平等

**现代反思**：

- 斯里兰卡的比丘尼传承后来中断，现代由泰国、台湾等地重新引入
- 僧伽蜜多的历史表明，佛教从一开始就重视女性修行
- 但后来的发展中，女性地位有所下降，这是需要反思的

## 三、巴利三藏结集：佛教经典的保存

### 3.1 第四次结集（斯里兰卡）

巴利三藏的结集是佛教历史上第四次结集（按南传佛教传统），也是最重要的结集之一。

**结集背景**：

- **时间**：佛灭后约400年（约公元前1世纪）
- **地点**：斯里兰卡摩罗伐陀岩寺（Mālādeva cave）
- **主持**：伐多伽摩尼王（Vaṭṭagāmaṇi Abhaya）
- **原因**：此前三藏口耳相传，因战乱恐失传，故书于贝叶

**结集意义**：

1. **文字化**：三藏从口传变为文字，保存更可靠
2. **标准化**：统一三藏内容，减少传承差异
3. **制度化**：建立结集制度，为后来的结集提供模式

### 3.2 巴利语的选择

巴利语（Pāli）是佛教历史上最重要的语言之一，也是现存最完整的佛教语言。

**巴利语特点**：

| 特点 | 说明 |
|------|------|
| 来源 | 摩揭陀语（Māgadhī）的一种方言 |
| 性质 | 不是摩揭陀语本身，而是相近的方言 |
| 名称 | Pāli（圣典语） |
| 现状 | 不是任何人的母语，仅为宗教语言 |
| 文献 | 巴利三藏、注释书、史书 |

**学术争议**：

- **传统观点**：巴利语就是佛所说的摩揭陀语
- **现代观点**：巴利语是摩揭陀语的一种方言，与佛所说的语言相近但不完全相同
- **最新研究**：巴利语可能是一种「混合语」，包含多种方言成分

### 3.3 巴利三藏的结构

巴利三藏是佛教历史上最完整的经典之一，其结构如下：

**律藏（Vinaya-piṭaka）**：

| 部 | 内容 |
|----|------|
| 经分别（Sutta-vibhaṅga） | 比丘戒、比丘尼戒 |
| 犍度（Khandhaka） | 僧团制度 |
| 附随（Parivāra） | 律藏附录 |

**经藏（Sutta-piṭaka）**：

| 部 | 内容 | 经文数量 |

| 长部（Dīgha-nikāya） | 长篇经典 | 34经 |
| 中部（Majjhima-nikāya） | 中篇经典 | 152经 |

| 增支部（Aṅguttara-nikāya） | 数字分类 | 11集 |

**论藏（Abhidhamma-piṭaka）**：

## 四、铜鍱部哲学体系

### 4.1 分别说立场的哲学分析

铜鍱部（分别说部）的「分别说」立场是一种精密的哲学立场，既不同于说一切有部的「一切有」，也不同于大众部的「一切无」。

**哲学分析**：

| 问题 | 铜鍱部 | 说一切有部 | 大众部 |
|------|--------|------------|--------|
| 存在论 | 现在有，过去未来无 | 三世实有 | 现在实有，过去未来无 |
| 变化论 | 法体刹那生灭 | 法体恒有，作用起灭 | 法体刹那生灭 |
| 因果论 | 刹那生灭相续 | 法体恒有，作用起灭 | 业力相续 |
| 认识论 | 识缘现在法 | 识缘三世法 | 识缘现在法 |

**哲学意义**：

铜鍱部的立场是一种「中道」立场：
- 不如说一切有部「一切有」（堕入常见）
- 不如大众部「一切无」（堕入断见）
- 而是「分别有」：现在有，过去未来无

这种立场与后来的大乘「中观」有相通之处，但铜鍱部更强调「现在」的真实性。

### 4.2 阿毗达摩七部论

铜鍱部的阿毗达摩七部论是佛教哲学史上最精密的体系之一。

**七部论哲学分析**：

| 论名 | 哲学主题 | 现代对应 |
|------|----------|----------|
| 法集论 | 存在论 | 本体论 |
| 分别论 | 解释学 | 注释学 |
| 界论 | 认识论 | 认知科学 |
| 人施设论 | 人格论 | 人格心理学 |
| 论事 | 批判论 | 批判哲学 |
| 双论 | 逻辑论 | 逻辑学 |
| 发趣论 | 因果论 | 因果分析 |

**发趣论二十四缘**：

发趣论的二十四缘是佛教因果论最精密的分析，远超说一切有部的四缘说。

**二十四缘分类**：

| 类别 | 缘 | 功能 |
|------|----|------|
| 基本缘 | 因缘、所缘缘、增上缘 | 主要条件 |
| 时间缘 | 无间缘、等无间缘、前生缘、后生缘 | 时间关系 |
| 空间缘 | 俱生缘、相互缘、依止缘 | 空间关系 |
| 修行缘 | 数数修习缘、业缘、异熟缘 | 修行关系 |
| 生理缘 | 食缘、根缘 | 生理关系 |
| 禅定缘 | 禅缘、道缘 | 禅定关系 |
| 逻辑缘 | 相应缘、不相应缘 | 逻辑关系 |
| 存在缘 | 有缘、无有缘、离去缘、不离去缘 | 存在关系 |

### 4.3 与说一切有部的比较

铜鍱部与说一切有部是上座部系的两大分支，两者的比较是佛教研究的重要课题。

**核心差异**：

| 问题 | 铜鍱部 | 说一切有部 |
|------|--------|------------|
| 语言 | 巴利语 | 梵语 |
| 三藏 | 巴利三藏 | 梵语三藏（已失传） |
| 三世观 | 现在实有 | 三世实有 |
| 法体观 | 刹那生灭 | 法体恒有 |
| 缘说 | 二十四缘 | 四缘 |
| 论书 | 七部论 | 六足论、发智论、大毗婆沙论 |
| 现存状况 | 现存（南传佛教） | 已消失（仅存汉译） |

**历史命运**：

- 说一切有部在印度兴盛，但后随印度佛教灭亡而消失
- 铜鍱部在斯里兰卡保存，传播到东南亚，至今兴盛
- 两者的比较研究是佛教研究的重要课题

## 五、南传佛教的现代传承

### 5.1 南传佛教的分布

南传佛教（Theravāda）是现存最古老的佛教传统之一，主要分布在东南亚。

**分布国家**：

| 国家 | 历史 | 特点 |
|------|------|------|
| 斯里兰卡 | 佛灭后200年传入 | 南传佛教发源地，巴利三藏保存地 |
| 缅甸 | 佛灭后约500年传入 | 禅修传统发达，内观（Vipassanā）流行 |
| 泰国 | 佛灭后约1000年传入 | 王室支持，僧团制度化 |
| 柬埔寨 | 佛灭后约1000年传入 | 吴哥窟佛教艺术 |
| 老挝 | 佛灭后约1000年传入 | 与泰国传统相近 |

### 5.2 南传佛教的特点

南传佛教有以下特点：

1. **保守传统**：坚持巴利三藏为唯一权威，不接受大乘经典
2. **重视禅修**：强调内观（Vipassanā）和禅定（Samatha）
3. **僧团中心**：以僧团为佛教中心，重视戒律
4. **出家传统**：重视出家修行，但在家信众也受重视
5. **巴利语传统**：以巴利语为宗教语言，学习巴利语是重要修行

### 5.3 南传佛教的现代挑战

南传佛教在现代社会面临一些挑战：

1. **现代化**：如何适应现代社会的价值观和生活方式
2. **世俗化**：如何在世俗化社会中保持宗教影响力
3. **性别平等**：如何提高女性在佛教中的地位
4. **与其他传统对话**：如何与大乘、藏传佛教对话
5. **西方传播**：如何向西方传播，适应西方文化
