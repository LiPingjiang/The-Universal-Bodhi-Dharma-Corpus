## 上座部系源流骨架

根本分裂后，上座部系（Sthavira）坚持传统、严守戒律，后分出多个重要分支，影响深远：

```
        上座部系（Sthavira，长老一系）
        ├── 说一切有部（Sarvāstivāda）→ 三世实有，七十五法
        │       ├── 六足一身论（阿毗达摩体系）
        │       ├── 大毗婆沙论（百科全书）
        │       └── 经量部（Sautrāntika）→ 只信经，不信论
        ├── 犊子部（Vātsīputrīya）→ 补特伽罗实有论
        │       ├── 法上部（Dharmottarīya）
        │       ├── 贤胄部（Bhadrayānīya）
        │       └── 正量部（Saṃmitīya）
        ├── 化地部（Mahīśāsaka）→ 五分律
        ├── 法藏部（Dharmaguptaka）→ 四分律（汉传主流律）
        ├── 分别说部（Vibhajjavāda）→ 逐一分别
        │       └── 铜鍱部（Tāmraparṇīya）→ 南传上座部
        │               └── 觉音论师 → 清净道论
        └── 雪山部（Haimavata）等
```
