# 大乘起信論會閱

## 學術解讀

### 一、文獻學信息

《大乘起信論會閱》是清代彭際清（1740-1796）所撰。彭際清，字允初，號知歸子，清代著名居士佛學家，精通儒釋道三家，著有《居士傳》《善女人傳》《華嚴唸佛三昧論》等。此疏以"會閱"爲名，取"會通諸家，閱盡精微"之意，是清代《大乘起信論》註疏中較有特色的一部。

彭際清出身江西名門，早年學儒，後歸心佛教，尤其推崇淨土宗。他雖爲居士，但佛學造詣深厚，與當時高僧如省庵、徹悟等交往密切。此疏作於其晚年，融會儒釋，以淨土宗立場重新解釋《起信論》。

### 二、歷史背景

清代佛教以居士佛學爲一大特色，彭際清是其中的代表人物。清代中期，佛教宗派衰落，但居士佛學蓬勃發展。彭際清以儒者的身份深入佛學，試圖以儒家"心學"來會通佛教"心性"思想。此疏正是在這一背景下產生的，體現了清代"儒釋合一"的學術思潮。

### 三、思想史定位

彭際清此疏的特色在於以"儒釋會通"和"會歸淨土"爲兩大主線。他認爲，《起信論》的"心真如門"即是儒家所說的"本心""良知"，"心生滅門"即是"人慾""習氣"。彭際清進一步將"信"解釋爲對阿彌陀佛和西方淨土的信，將《起信論》完全納入淨土宗的框架。這一解釋雖然偏離了《起信論》的原意，但反映了清代居士佛學的獨特風貌，對後來近代佛教的轉型有間接影響。

## 白話解讀

### 核心內容

《大乘起信論會閱》是清代大居士彭際清對《大乘起信論》的會通性註釋。彭際清是儒生出身，後來學佛，所以他用儒家的語言來解釋佛教。

彭際清說：《起信論》的"心真如門"，就是儒家王陽明說的"良知"——每個人心中本有的道德光明；"心生滅門"就是"人慾"——被私慾遮蔽的狀態。修行的目的，就是讓"良知"重新顯現。

同時，彭際清又把《起信論》和淨土宗聯繫起來："信"就是信阿彌陀佛，"願"就是願生西方，"行"就是念佛。這樣，一部教理論書就變成了淨土修行的指南。

### 現代啓示

1. **跨文化理解**：彭際清以儒解佛，啓示我們：不同文化傳統之間可以相互理解、相互借鑑。在全球化時代，這種跨文化的能力尤爲重要。

2. **居士佛學**：彭際清作爲居士，其佛學造詣不遜於出家人。這啓示我們：學佛不必出家，在家同樣可以深入研究。

3. **會通精神**："會閱"的精神——廣泛閱讀、融會貫通，對今天的終身學習有重要啓示。

### 閱讀建議

- 彭際清的疏解較爲自由，適合有一定儒學和佛學基礎的讀者
- 可配合《居士傳》《華嚴唸佛三昧論》一同閱讀，瞭解其整體思想
- 注意其"儒釋會通"的特點，理解清代居士佛學的獨特視角
