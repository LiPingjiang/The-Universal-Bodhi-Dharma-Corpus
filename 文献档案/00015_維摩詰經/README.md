## 維摩詰經源流骨架

《維摩詰經》（Vimalakīrti-nirdeśa-sūtra，"維摩詰所說經"，"淨名經"）是大乘佛教最重要的經典之一，也是最具文學魅力的佛教經典之一。"維摩詰"（Vimalakīrti，"淨名"，即"清淨的名聲"）是一位在家居士（非出家僧侶），卻擁有超越一切出家菩薩的智慧和辯才。

```
        维摩诘经思想体系
        ├── 不二法门（Advaya-dharma-paryāya）
        │       ├── 超越一切对立 → 生死与涅槃不二
        │       ├── 维摩诘的沉默 → 最高的不二表达
        │       └── 文殊的赞叹 → 沉默胜于言说
        ├── 在家菩萨道（Gṛhastha-bodhisattva-mārga）
        │       ├── 维摩诘示疾 → 以病为方便，教化众生
        │       ├── 在家居士可以成佛 → 不必出家
        │       └── 世间即涅槃 → 不离世间而证涅槃
        └── 不思议解脱（Acintya-vimokṣa）
                ├── 方丈室容纳三万二千师子座 → 超越空间限制
                ├── 天女散花 → 花不著声闻，著菩萨
                └── 维摩诘的神通 → 超越一切常规
```

《維摩詰經》的核心思想是「不二法門」（advaya-dharma-paryāya）：超越一切對立（生死與涅槃、染與淨、有與無），進入不二的境界。維摩詰以"沉默"來回答"什麼是不二法門"，被文殊菩薩讚歎爲"真入不二法門"——沉默勝於任何言說，因爲不二超越了語言的對立。

《維摩詰經》對中國佛教的影響極爲深遠，特別是對禪宗。禪宗的"不立文字"、"以心傳心"、"棒喝"等方法，都與《維摩詰經》的"沉默"和"不二"思想有密切關聯。鳩摩羅什（Kumārajīva，344-413）的漢譯本《維摩詰所說經》（T475）是最流行的譯本。
