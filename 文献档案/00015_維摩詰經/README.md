## 维摩诘经源流骨架

《维摩诘经》（Vimalakīrti-nirdeśa-sūtra，"维摩诘所说经"，"净名经"）是大乘佛教最重要的经典之一，也是最具文学魅力的佛教经典之一。"维摩诘"（Vimalakīrti，"净名"，即"清净的名声"）是一位在家居士（非出家僧侣），却拥有超越一切出家菩萨的智慧和辩才。

```
        维摩诘经思想体系
        ├── 不二法门（Advaya-dharma-paryāya）
        │       ├── 超越一切对立 → 生死与涅槃不二
        │       ├── 维摩诘的沉默 → 最高的不二表达
        │       └── 文殊的赞叹 → 沉默胜于言说
        ├── 在家菩萨道（Gṛhastha-bodhisattva-mārga）
        │       ├── 维摩诘示疾 → 以病为方便，教化众生
        │       ├── 在家居士可以成佛 → 不必出家
        │       └── 世间即涅槃 → 不离世间而证涅槃
        └── 不思议解脱（Acintya-vimokṣa）
                ├── 方丈室容纳三万二千师子座 → 超越空间限制
                ├── 天女散花 → 花不著声闻，著菩萨
                └── 维摩诘的神通 → 超越一切常规
```

《维摩诘经》的核心思想是「不二法门」（advaya-dharma-paryāya）：超越一切对立（生死与涅槃、染与净、有与无），进入不二的境界。维摩诘以"沉默"来回答"什么是不二法门"，被文殊菩萨赞叹为"真入不二法门"——沉默胜于任何言说，因为不二超越了语言的对立。

《维摩诘经》对中国佛教的影响极为深远，特别是对禅宗。禅宗的"不立文字"、"以心传心"、"棒喝"等方法，都与《维摩诘经》的"沉默"和"不二"思想有密切关联。鸠摩罗什（Kumārajīva，344-413）的汉译本《维摩诘所说经》（T475）是最流行的译本。
