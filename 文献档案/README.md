### 三件套命名

每個文獻目錄內統一使用繁體裸名三件套：

| 檔案 | 說明 |
|------|------|
| `原文.md` | 原典文本（巴利/梵/漢/藏對照或單語） |
| `解讀_學術.md` | 學術解讀：文獻學資訊、歷史地位、教義分析、研究現狀 |
| `解讀_白話.md` | 白話解讀：現代白話通俗講解、核心思想、現代啟示 |

部分文獻目錄另有 `索引卡.md`（文獻信息卡）和 `README.md`（導覽）。

重複保留的檔案使用後綴：`原文_備份.md`、`解讀_學術_備份.md`、`解讀_白話_備份.md`、`*_根目錄版.md`。

### 文獻來源

## 分期總覽

### 第一期　原始佛教·口傳與早期結集（約前6–前3世紀）

佛陀住世及滅後百年，法藏口耳相傳，諸大結集，奠定一切佛典之根基。

- 00001 佛陀口傳法藏（佛語 Buddhavacana）— 363部經
- 00002 第一次結集（王舍城，佛滅當年）
- 00003 第二次結集（毗舍離，佛滅百年）
- 00004 阿育王法敕（約前269–前231年）
- 00005 巴利律藏 Vinaya Piṭaka

### 第二期　部派佛教·各部派典籍（約前4–前1世紀）

統一僧團歷三百年「分家」，裂為上座、大眾兩系十八／二十部。

- 00006 部派分裂總覽
- 00007 上座部及其傳承
- 00008 大眾部
- 00009 論事 Kathāvatthu
- 00010 彌蘭陀王問經
- 00011 阿毗達磨論藏興起

### 第三期　早期大乘·般若等經典（約前1–3世紀）

般若空慧橫空出世，法華唱一乘，維摩示不二，華嚴現重重無盡。

- 00012 般若經系總覽
- 00013 八千頌般若
- 00014 法華經
- 00015 維摩詰經
- 00016 華嚴經
- 00017 無量壽經與淨土經典

### 第四期　大乘論書·中觀唯識（約3–7世紀）

龍樹立中觀、無著世親建唯識，佛教哲學登峰造極。

- 00018 龍樹與中觀學派
- 00019 中論
- 00020 無著世親與唯識學派
- 00021 唯識三十頌
- 00022 如來藏思想
- 00023 佛教因明（陳那·法稱）

### 第五期　密乘·金剛乘典籍（約7–12世紀）

真言陀羅尼匯為大流，兩部曼荼羅立正純密教。

- 00024 密乘金剛乘總覽
- 00025 大日經與金剛頂經
- 00026 時輪續與無上瑜伽

### 第六期　漢傳·藏傳本土化典籍（約5–20世紀）

佛法東流，漢地成八宗、禪淨雙修；高原立四派、顯密圓融。

- 00028 六祖壇經
- 00029 藏傳佛教總覽

西學東漸，佛典入學院；文獻學、比較哲學、認知科學多維交照。
