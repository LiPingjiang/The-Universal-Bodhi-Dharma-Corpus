# 佛教佛法文献深度档案

本目录在《佛教佛法文献古籍大全.xlsx》索引基础上，**逐部建立深度档案**。从最古老、最原始的文献开始，按年代顺序逐一梳理。

## 每份档案的固定结构

1. **文献信息卡** —— 编号、年代、原始语言、所属传统、类型、在大藏经中的编号、原文出处链接。
2. **名称还原解释** —— 把音译的、文言的名称拆开，还原成大白话，逐词解释「到底是什么意思」。
3. **原始内容（落盘）** —— 该文献的原文。短经全录，长经录核心段落与著名名句（含原始语言与对照译文）。
4. **大白话内容总结** —— 用最通俗的中文，讲清楚这部文献到底讲了什么、为什么重要。

## 编号与命名规则

文件名格式：`编号_文献名称.md`，编号与 Excel 主表「序号」一一对应，便于交叉检索。

## 文献来源

以权威开放库为主：SuttaCentral（巴利及多语对照）、CBETA（汉文大正藏）、84000（藏译英）、GRETIL / DSBC（梵文）、维基文库等。每份档案在「原始内容」处标注具体出处。
