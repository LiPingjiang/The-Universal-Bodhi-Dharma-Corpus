## 學術解讀

### 一、文獻學信息

《大乘起信論義疏》是唐代新羅僧人元曉（617-686）所撰。元曉，俗姓薛，海東（今朝鮮半島）人，爲新羅佛教史上最具影響力的學者之一。此疏是《大乘起信論》現存最早的註疏之一，對後世法藏、宗密等人的註疏有直接影響。

元曉一生著述宏富，有《華嚴經疏》《金剛三昧經論》《十門和諍論》等傳世。其學風通達，主張"和諍"——融合佛教內部各宗派的爭論。此疏充分體現了這一特點，以圓融的態度解釋《起信論》的義理。

### 二、歷史背景

《大乘起信論》於隋代傳入中國，唐代初期即受到重視。元曉於唐高宗時期入唐求法，與玄奘、窺基等同時代。他在長安期間深入研究《起信論》，回國後撰成此疏。當時新羅佛教正處於蓬勃發展期，元曉此疏成爲海東佛教界理解《起信論》的主要依據，對後來韓國佛教的如來藏思想傳統有深遠影響。

### 三、思想史定位

元曉此疏的最大貢獻在於對"一心二門"結構的系統解釋。他提出"一心二門"不是兩個心，而是"一心"的兩種"門"——即兩種觀察角度。"心真如門"是從本體角度觀察，"心生滅門"是從現象角度觀察。這一解釋後來被法藏《義記》所吸收，成爲《起信論》解釋的標準範式。元曉還提出"本覺"與"始覺"的關係問題，認爲"始覺"即是"本覺"的顯現，這一思想對後來禪宗的"頓悟"說有影響。

## 白話解讀

### 核心內容

《大乘起信論義疏》是新羅（今韓國）高僧元曉對《大乘起信論》的解釋。元曉大師的特點是善於融合——他不偏向任何宗派，而是把各家的長處都吸收進來。

元曉對"一心二門"的解釋特別清楚：不是有兩個心，而是一個心的兩個"門"。就像一座房子有兩個門，一個前門一個後門，但進去都是同一個房子。"心真如門"是讓我們看到心的本質（清淨、光明），"心生滅門"是讓我們看到心的現象（念頭、情緒）。

元曉還提出"本覺"和"始覺"的概念：我們本來就有覺悟的能力（本覺），但需要通過修行讓它顯現出來（始覺）。就像太陽本來就在，只是被雲遮住了。

### 現代啓示

1. **多元視角**：元曉"和諍"的思想啓示我們：面對爭論，不要急於站隊，而要理解各方的合理之處，尋求融合。

2. **本具潛能**："本覺"思想告訴我們：每個人內在都有智慧的光芒，教育和工作不是"安裝"新東西，而是"喚醒"已有的東西。

3. **觀察角度**："一心二門"啓示我們：同一個事物，從不同角度看會有不同的理解。學會切換視角，是智慧的表現。

### 閱讀建議

- 元曉的疏解較爲平實，適合初學者
- 可配合法藏《義記》對比閱讀，理解同一文本的不同解釋角度
- 注意元曉"和諍"的立場，理解其融合而非對立的思維方式
