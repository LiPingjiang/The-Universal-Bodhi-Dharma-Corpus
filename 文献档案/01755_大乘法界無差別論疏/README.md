# 大乘法界無差別論疏

## 基本信息

- **編號**：01755
- **類別**：中觀般若疏釋
- **狀態**：已完成

## 學術解讀

### 一、文獻學信息

《大乘法界無差別論疏》是唐代法藏（643-712）所撰，爲《大乘法界無差別論》的註疏。該論原題爲堅慧菩薩造，收於《大正藏》第三十一冊，全論一卷，以"一乘"思想爲核心，闡述法界與衆生無差別的義理。

法藏，華嚴宗實際創始人，世稱"賢首大師"，曾參與玄奘譯場，後獨立弘揚華嚴教義。其註疏以圓融無礙、事事無礙爲特點，善於以十門、六相等方式組織義理。

### 二、歷史背景

《大乘法界無差別論》與《大乘起信論》在思想上有密切關聯，均屬於如來藏系論書。法藏作此疏時，正值華嚴宗鼎盛期，其疏解不僅闡發原論義理，更融入華嚴宗"法界緣起"的思想體系。該疏在唐代即受到重視，後傳入日本，對東瀛華嚴宗影響深遠。

### 三、思想史定位

本疏的核心思想是將"法界無差別"與華嚴宗的"性起"思想相融通。法藏認爲，一切衆生本具如來藏，與法界無二無別，此"無差別"非泯滅一切差別的虛無，而是在差別中見平等、在平等中顯差別的圓融境界。這一思想對後世中國佛教的"即煩惱即菩提""生死即涅槃"等命題有重要啓發。

## 白話解讀

### 核心內容

《大乘法界無差別論疏》是法藏大師對《大乘法界無差別論》的詳細解釋。核心要旨是：法界（宇宙萬物的真實本體）與一切衆生在本質上是完全平等的，沒有任何差別。

法藏大師用華嚴宗的圓融思想來解釋這一點：我們平常看到的種種差別——貧富、貴賤、智愚、善惡——都是表面的現象。在深層本質上，一切衆生都具備成佛的可能性（如來藏），都與宇宙的真實本體（法界）無二無別。

### 現代啓示

1. **平等觀**：在現代社會中，種族、性別、階層的歧視依然存在。"法界無差別"提醒我們，每個人在人格尊嚴和內在潛能上是完全平等的。

2. **自我認知**：很多人因爲外在條件而自卑或自傲。認識到自己本具如來藏，有助於建立健康的自我認知——既不妄自菲薄，也不傲慢自大。

3. **包容精神**：理解"無差別"不是否定差異，而是在尊重差異的基礎上看到共同的本質。這對多元文化社會的和諧共處有重要啓示。

### 閱讀建議

- 建議先通讀《大乘法界無差別論》原文，再讀法藏疏
- 可配合法藏《華嚴金師子章》一同學習，理解華嚴宗的思維方式
- 注意區分"無差別"與"無分別"：前者是境界，後者是功夫
