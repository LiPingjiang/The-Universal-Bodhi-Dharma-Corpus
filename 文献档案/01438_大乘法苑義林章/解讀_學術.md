# 大乘法苑义林章 · 學術解讀

## 一、文獻定位

《大乘法苑义林章》为唐代唯识学大师窥基（632–682）所撰，是唐代唯识宗（法相宗）最重要的综合性义学著作之一。全书凡七卷，以"义林章"体例系统阐述大乘佛教法相唯识学的核心教义，涵盖唯识理论、因明逻辑、戒律轨范、观行实践等唯识学全部关键领域。该书是窥基在玄奘译场参与翻译、整理唯识经典后，对其师玄奘所传唯识学的系统总结与发挥，是研究唐代唯识宗思想体系、玄奘译传及中国佛教义学巅峰时期的关键文本。

## 二、版本與傳承

- **作者**：窥基（俗姓尉迟，京兆长安人，玄奘高足）
- **成书年代**：约唐高宗时期（650–683）
- **原始語言**：漢文撰述
- **現存版本**：

  - 敦煌写本残卷

## 三、思想史分析

### 3.1 歷史背景

窥基是玄奘（602–664）最杰出的弟子之一，参与玄奘译场翻译工作长达二十余年。玄奘从印度带回并翻译了大量唯识学经典，包括《瑜伽师地论》《成唯识论》《唯识三十颂》等。然而，这些经典义理深奥、体系庞大，需要系统的整理和阐释。窥基在玄奘去世后，承担起整理和发扬唯识学的重任，《大乘法苑义林章》正是这一工作的结晶。

唐代是中国佛教宗派佛教的鼎盛时期，唯识宗与华严宗、天台宗、禅宗等并列为大宗。窥基的著作代表了唯识宗义学的最高水平，其精密的概念分析和严密的逻辑论证，在中国佛教思想史上独树一帜。

### 3.2 核心論題

《大乘法苑义林章》全书分为多个"义林"（义类之林，即教义分类），每林下列细目，系统论述。主要论题包括：

1. **唯识义林**：阐述"万法唯识"的基本原理，分析识的体性、功能与转变，是全书的核心。
2. **因明义林**：系统介绍印度因明（佛教逻辑学）的基本概念、论证规则和推理方法，是研究唐代因明学的核心资料。
3. **法相义林**：分类阐述佛教法相（一切法的现象与本质），包括五蕴、十二处、十八界、百法等。
4. **戒律义林**：论述大乘戒律的体性、种类与持守方法，体现唯识宗对戒律的理解。
5. **观行义林**：阐述唯识宗的修行观行，包括五重唯识观、四寻思观等具体修行方法。
6. **佛身义林**：分析法、报、化三身的体性、相状与关系，与《大乘义章》的三身论相互参照。
7. **判教义林**：对佛教经典进行判教分类，建立唯识宗的教相体系。

### 3.3 學術評價

- **吕澂**在《中国佛学源流略讲》中评价窥基为"玄奘门下第一人"，认为《大乘法苑义林章》是"唐代唯识学义理的集大成之作"。
- **日本学者**如宇井伯寿、山口益等，将窥基的著作视为研究印度唯识学中国化的基本文献，尤其重视其因明学和法相学的精密分析。
- **现代学者**关注该书与玄奘译传的关系，认为窥基在忠实传承印度唯识学的同时，也融入了中国佛教的思维方式，体现了"印度佛学中国化"的典型路径。
- 当代研究还关注该书与《成唯识论述记》的互补关系，前者侧重义理分类，后者侧重经典注释。

## 四、術語對照

| 漢譯術語 | 梵文/巴利原文 | 現代語義 |
|---------|-------------|---------|
| 唯識 | vijñāna-mātra / vijñapti-mātra | 一切现象皆为识所变现，外境无实，唯识实有 |
| 因明 | hetu-vidyā | 印度佛教逻辑学与认识论，包括现量、比量、圣教量等 |
| 法相 | dharma-lakṣaṇa | 一切法的现象特征与本质属性，唯识宗以此为核心研究对象 |
| 五蘊 | pañca-skandha | 色、受、想、行、识五种构成生命的基本要素 |
| 百法 | 无对应梵文 | 唯识宗将一切法分为百种，包括心法、心所法、色法、不相应行法、无为法 |
| 五重唯識觀 | 无对应梵文 | 唯识宗修行的五个层次：遣虚存实、舍滥留纯、摄末归本、隐劣显胜、遣相证性 |
| 三自性 | tri-svabhāva | 遍计所执性（虚妄分别）、依他起性（缘起现象）、圆成实性（真实本性） |
| 轉識成智 | āśraya-parivṛtti | 将八识转变为四智，是唯识宗修行的最终目标 |
