# 三律仪论 · 白話解讀

## 这部论典是什么？

《三律仪论》是萨迦班智达写的一部关于戒律的重要著作。"三律仪"指的是佛教的三种戒律体系：别解脱戒（出家人和在家人的基本戒律）、菩萨戒（发菩提心者的戒律）、密乘戒（修习密宗者的戒律）。这部论典的核心问题是：这三种戒律是什么关系？可以互相取代吗？

## 核心内容

萨迦班智达在这部论典中明确指出：

**三种戒律各有其独立性**：别解脱戒是基础，菩萨戒是在此基础上的升华，密乘戒是更高层次的修行规范。三者不能互相取代，也不能混淆。

**批判"密乘高于一切"的错误观点**：当时有些人认为，修了密宗就不需要遵守基本戒律了。萨迦班智达明确批判了这种观点，强调戒律的基础性地位。

**三种戒律的关系**：三种戒律是层层递进的关系，修行者应该在持守基本戒律的基础上，逐步受持菩萨戒和密乘戒。

## 现代启示

这部论典的核心思想——戒律是修行的基础，不能因为追求高深法门而忽视基本戒律——在今天依然非常重要。很多人急于修习高深法门，却忽视了基本的道德规范，这正是萨迦班智达所批判的。

## 阅读建议

这部论典涉及专业的戒律学知识，建议先了解佛教基本戒律（五戒、八戒等），再来学习这部论典。

## 关键概念表

| 概念 | 解释 |
|------|------|
| 别解脱戒 | 佛教基本戒律，包括出家戒和在家戒 |
| 菩萨戒 | 发菩提心者受持的戒律 |
| 密乘戒 | 修习密宗者受持的三昧耶戒 |
| 三律仪 | 三种戒律体系的总称 |
