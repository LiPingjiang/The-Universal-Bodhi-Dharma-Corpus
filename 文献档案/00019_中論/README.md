## 源流与背景

公元2世纪中叶，南印度出身的论师龙树（Nāgārjuna，约150–250 CE）在般若经系「空性」思想的基础上，撰写了《根本中观颂》（Mūlamadhyamakakārikā，简称《中论》或 MMK）。全书27品约450颂，以极其精炼的偈颂体，系统性地运用归谬法破斥当时各派哲学体系——说一切有部的「三世实有」、犊子部的「补特伽罗实有」、数论的「自性」、胜论的「实句义」等——确立「一切法无自性、缘起即空」的根本立场。龙树由此开创中观学派，与后来无著、世亲的瑜伽行派（唯识）并列为大乘两大哲学支柱。

《中论》的论证策略极为独特：它不直接建立自己的主张，而是接受对手的前提，然后通过严格的逻辑推导揭示其中的矛盾，迫使对手承认其立场不能自洽。最著名的实例是第一品开篇的「四门不生」——不自生、不他生、不共生、不无因生，穷尽「生」的一切可能性后予以逐一破斥。第二十四品则回应了最尖锐的批评：「如果一切法空，那四圣谛、三宝、因果报应岂不都不存在了？」龙树的回答是：正因为空、正因为无自性，因果与修行才可能成立；若有固定自性，则一切法僵死不动，反而无从建立任何因果关系。

此论现存完整梵本（de Jong 1977 校订本，叶少勇 2011 改进校订本）、鸠摩罗什汉译（T1564，409年译出，含青目释，是东亚佛教最通行版本）及藏译（收入丹珠尔）。注释传统以月称《明句论》（Prasannapadā，梵文完整保存）最为重要，佛护《中论释》与清辨《般若灯论》则分别代表了应成派与自续派的诠释分野——这一分歧后来深刻影响了藏传佛教的宗义架构。龙树所确立的「缘起性空」思想，经提婆、佛护、月称、清辨等人发扬，又经鸠摩罗什传入汉地催生三论宗，最终成为整个东亚大乘佛教的核心哲学基石。

## 关键概念

- **缘起**（pratītyasamutpāda）：龙树哲学的核心。第二十四品第18颂给出全论最精炼的表述：「凡是缘起的，就是空性；它即是假名施设，这本身就是中道。」缘起、空性、假名、中道四者等价——这是中观学派的根本命题。
- **自性**（svabhāva）：龙树一切论证的攻击靶子。指事物「不依赖因缘条件的独立固有本质」。龙树论证：一切经验事物都依赖条件，因此无自性；无自性故空。学术界对 svabhāva 的精确含义有强实在论（说一切有部法体）、弱本质论（内在属性）、独立存在论三种解读，当代趋势倾向于综合后两者。
- **空性**（śūnyatā）：非「什么都没有」的虚无，而是「无自性」的同义词。龙树强调空性本身也是空的（śūnyatā-śūnyatā）——若把空性执为新的实体，则落入另一种执着。空性的功能是「治疗性」的：如药治病，病愈药弃。
- **二谛**（satya-dvaya）：世俗谛（saṃvṛti-satya）与胜义谛（paramārtha-satya）。龙树在第二十四品确立：世俗谛中因果、修行、四圣谛皆有效成立；胜义谛中一切法空、无自性。二谛不矛盾——空性正是因果得以成立的前提。不了解二谛分别，则不知龙树真实义。
- **中道**（madhyamā pratipad）：远离「有」与「无」两边。执有则落入常见（认为事物有固定自性），执无则落入断见（认为一切不存在）。龙树的中道不是折中，而是以缘起超越二元对立——事物既非实有（无自性），也非全无（缘起显现），如幻而有。
