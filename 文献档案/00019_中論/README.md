## 源流與背景

公元2世紀中葉，南印度出身的論師龍樹（Nāgārjuna，約150–250 CE）在般若經系「空性」思想的基礎上，撰寫了《根本中觀頌》（Mūlamadhyamakakārikā，簡稱《中論》或 MMK）。全書27品約450頌，以極其精煉的偈頌體，系統性地運用歸謬法破斥當時各派哲學體系——說一切有部的「三世實有」、犢子部的「補特伽羅實有」、數論的「自性」、勝論的「實句義」等——確立「一切法無自性、緣起即空」的根本立場。龍樹由此開創中觀學派，與後來無著、世親的瑜伽行派（唯識）並列爲大乘兩大哲學支柱。

《中論》的論證策略極爲獨特：它不直接建立自己的主張，而是接受對手的前提，然後通過嚴格的邏輯推導揭示其中的矛盾，迫使對手承認其立場不能自洽。最著名的實例是第一品開篇的「四門不生」——不自生、不他生、不共生、不無因生，窮盡「生」的一切可能性後予以逐一破斥。第二十四品則回應了最尖銳的批評：「如果一切法空，那四聖諦、三寶、因果報應豈不都不存在了？」龍樹的回答是：正因爲空、正因爲無自性，因果與修行纔可能成立；若有固定自性，則一切法僵死不動，反而無從建立任何因果關係。

此論現存完整梵本（de Jong 1977 校訂本，葉少勇 2011 改進校訂本）、鳩摩羅什漢譯（T1564，409年譯出，含青目釋，是東亞佛教最通行版本）及藏譯（收入丹珠爾）。註釋傳統以月稱《明句論》（Prasannapadā，梵文完整保存）最爲重要，佛護《中論釋》與清辨《般若燈論》則分別代表了應成派與自續派的詮釋分野——這一分歧後來深刻影響了藏傳佛教的宗義架構。龍樹所確立的「緣起性空」思想，經提婆、佛護、月稱、清辨等人發揚，又經鳩摩羅什傳入漢地催生三論宗，最終成爲整個東亞大乘佛教的核心哲學基石。

## 關鍵概念

- **緣起**（pratītyasamutpāda）：龍樹哲學的核心。第二十四品第18頌給出全論最精煉的表述：「凡是緣起的，就是空性；它即是假名施設，這本身就是中道。」緣起、空性、假名、中道四者等價——這是中觀學派的根本命題。
- **自性**（svabhāva）：龍樹一切論證的攻擊靶子。指事物「不依賴因緣條件的獨立固有本質」。龍樹論證：一切經驗事物都依賴條件，因此無自性；無自性故空。學術界對 svabhāva 的精確含義有強實在論（說一切有部法體）、弱本質論（內在屬性）、獨立存在論三種解讀，當代趨勢傾向於綜合後兩者。
- **空性**（śūnyatā）：非「什麼都沒有」的虛無，而是「無自性」的同義詞。龍樹強調空性本身也是空的（śūnyatā-śūnyatā）——若把空性執爲新的實體，則落入另一種執着。空性的功能是「治療性」的：如藥治病，病癒藥棄。
- **二諦**（satya-dvaya）：世俗諦（saṃvṛti-satya）與勝義諦（paramārtha-satya）。龍樹在第二十四品確立：世俗諦中因果、修行、四聖諦皆有效成立；勝義諦中一切法空、無自性。二諦不矛盾——空性正是因果得以成立的前提。不瞭解二諦分別，則不知龍樹真實義。
- **中道**（madhyamā pratipad）：遠離「有」與「無」兩邊。執有則落入常見（認爲事物有固定自性），執無則落入斷見（認爲一切不存在）。龍樹的中道不是折中，而是以緣起超越二元對立——事物既非實有（無自性），也非全無（緣起顯現），如幻而有。
