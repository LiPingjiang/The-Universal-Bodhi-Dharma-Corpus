# 大乘起信論疏義

## 學術解讀

### 一、文獻學信息

《大乘起信論疏義》是唐代子璇（約8世紀）所撰。子璇，俗姓不詳，爲唐代天台宗高僧，活躍於唐玄宗時期，以弘揚《大乘起信論》和《楞嚴經》著稱。此疏以"疏義"爲名，取"疏通義理，闡幽發微"之意，是天台宗系統解釋《起信論》的重要著作。

子璇是天台宗"中興"時期的重要人物，當時天台宗經過隋代智顗的創立和唐代前期的傳承，需要重新整理和弘揚。子璇致力於將天台宗的"性具""一念三千"等核心概念與《起信論》的"一心二門"相融合，形成天台宗獨特的《起信論》解釋體系。

### 二、歷史背景

唐玄宗時期，佛教各宗派競相發展，天台宗與華嚴宗、禪宗之間的思想競爭尤爲激烈。天台宗以"性具"思想爲特色，強調善惡諸法皆本具於法性，與華嚴宗的"性起"思想形成對照。子璇此疏正是在這一背景下產生，旨在以天台宗的立場重新解釋《起信論》，確立天台宗在如來藏思想傳統中的話語權。

### 三、思想史定位

子璇此疏的最大特色在於以天台宗的"一念三千"和"性具"思想來解釋《起信論》。他認爲，《起信論》的"一心"即是天台宗所說的"一念心"，這一念心中具足三千世間（十法界×十如是×三種世間）。"心真如門"即是"法性"的清淨面，"心生滅門"即是"法性"的隨緣面。子璇進一步將"無明"解釋爲"性惡"的顯現，將"修行"解釋爲"修惡即善"的過程。這一解釋對後來天台宗與《起信論》的關係有決定性影響。

## 白話解讀

### 核心內容

《大乘起信論疏義》是唐代天台宗高僧子璇對《大乘起信論》的解釋。子璇大師用天台宗獨特的"一念三千"思想來理解《起信論》。

天台宗認爲：我們當下的"一念心"中，具足了一切——佛、菩薩、天人、甚至地獄，都在這一念心中。這不是說一念心創造了世界，而是說一念心與整個宇宙是"互具"的。

子璇大師說：《起信論》的"一心"就是這個"一念心"，"心真如門"是這一念心的清淨面，"心生滅門"是這一念心的變化面。而且，因爲"性具"，善惡都在這一念心中本具，所以修行不是消滅惡，而是"修惡即善"——在惡中修行，轉化惡爲善。

### 現代啓示

1. **一念的力量**：現代人常常忽視"當下"的重要性。"一念三千"告訴我們：每一個念頭都有無限的可能性，當下的選擇影響深遠。

2. **接納與轉化**："性具"思想啓示我們：接納自己的全部（包括缺點），然後加以轉化。不是自我否定，而是自我超越。

3. **整體觀**："一念三千"的整體觀啓示我們：個人與宇宙是相互關聯的。個人的改變會影響整體，整體的改變也會影響個人。

### 閱讀建議

- 子璇的疏解需要天台宗的基礎知識，建議先讀《摩訶止觀》或《法華玄義》
- 可對比智旭以天台宗解《起信論》的方式，理解天台宗解釋傳統的延續
- 注意"一念三千"與"一心二門"的融合，理解天台宗的獨特視角
