## 學術解讀

### 一、文獻學信息

《大乘起信論發隱》是明代李贄（1527-1602）所撰。李贄，號卓吾，泉州晉江（今福建晉江）人，爲明代著名思想家、文學家，以"異端"思想著稱。此疏以"發隱"爲名，取"發抉隱微，揭示奧義"之意，是《大乘起信論》註疏中較爲獨特的一部，因爲它出自一位非僧人的"居士"之手，且帶有強烈的個人色彩。

李贄一生思想激進，批判傳統儒學，推崇佛教禪宗，主張"童心說"，認爲人應保持純真的本心。其著作《焚書》《續焚書》《藏書》等在明代思想界引起巨大反響。此疏作於其晚年，是他以禪宗立場解釋《起信論》的代表作。

### 二、歷史背景

明代中後期，王陽明心學盛行，"心即理""致良知"等思想深入人心。李贄受心學影響，進而歸向禪宗，認爲佛教禪宗纔是"心學"的究竟歸宿。當時《大乘起信論》作爲"一心二門"思想的經典，自然受到李贄的重視。此疏正是在"心學"與"禪學"交融的背景下產生，體現了明代思想界儒釋道三教合流的趨勢。

### 三、思想史定位

李贄此疏的最大特色在於以"童心"概念來解釋《起信論》的"心真如門"。他認爲，"心真如"即是"童心"——未被世俗污染的純真本心。"心生滅門"則是"習心"——被後天習氣所遮蔽的心。李贄進一步將"始覺"解釋爲"恢復童心"的過程，將"修行"解釋爲"去除習心、迴歸童心"的實踐。這一解釋雖然偏離了《起信論》的原意，但反映了明代"心學"對佛教思想的吸收和改造，對後來近代思想啓蒙有間接影響。

## 白話解讀

### 核心內容

《大乘起信論發隱》是明代思想家李贄對《大乘起信論》的解釋。李贄不是僧人，是一位思想激進的文人。

李贄用他著名的"童心"概念來解釋《起信論》："心真如門"就是"童心"——小孩子那種純真、沒有虛僞的心；"心生滅門"就是"習心"——被大人的世界污染了的心。修行的目的，就是去掉"習心"，恢復"童心"。

李贄還說：很多人學佛學成了"假道學"——表面上仁義道德，內心卻充滿算計。真正的學佛，就是保持"童心"，做一個真實的人。

### 現代啓示

1. **保持純真**：現代社會充滿虛僞和算計。李贄的"童心"思想提醒我們：保持內心的純真和真實，是一種難得的勇氣。

2. **反對假道學**：李贄批判"假道學"，啓示我們：任何信仰如果變成了虛僞的包裝，就失去了意義。真正的修行是內心的真實。

3. **獨立思考**：李贄以"異端"自居，不盲從權威。這種獨立思考的精神，對今天的學習和工作都有啓示。

### 閱讀建議

- 李贄的疏解帶有強烈的個人色彩，適合對明代思想感興趣的讀者
- 可配合《焚書》中的《童心說》一同閱讀，理解其整體思想
- 注意李贄以"童心"解"心真如"的特點，理解其思想的時代背景
