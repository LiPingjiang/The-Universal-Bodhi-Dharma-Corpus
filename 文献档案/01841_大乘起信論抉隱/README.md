# 大乘起信論抉隱

## 學術解讀

### 一、文獻學信息

《大乘起信論抉隱》是明代真可（1543-1604）的弟子或再傳弟子所撰，一說爲憨山德清（1546-1623）所撰。此疏以"抉隱"爲名，取"抉發隱微，揭示幽奧"之意，是明代《大乘起信論》註疏中較爲深入的一部。"抉隱"與紫柏真可的《釋要》、憨山德清的《裂網疏》風格相近，均以禪宗立場解釋《起信論》，但更注重對"隱微"義理的挖掘。

明代中後期，禪宗與教理的融合成爲趨勢。此疏的作者（無論是紫柏門人還是憨山本人）都繼承了這一傳統，以禪宗的"明心見性"爲核心，深入挖掘《起信論》中蘊含的禪宗義理。

### 二、歷史背景

明代萬曆年間，禪宗復興運動興起，紫柏真可、憨山德清等人主張"以禪爲體，以教爲用"。他們認爲，禪宗的"頓悟"必須有教理的支撐，否則容易流於空疏；教理的研習也必須有禪宗的"明心"爲歸趣，否則容易淪爲知解。此疏正是在這一背景下產生，旨在從《起信論》中"抉發"出禪宗的"隱微"義理，爲禪宗學人提供教理依據。

### 三、思想史定位

此疏的特色在於以"一心二門"與禪宗"三關"（初關、重關、牢關）相對應。作者認爲，"心真如門"即是"破初關"——見本性；"心生滅門"即是"透重關"——在日用中保任；"二門不二"即是"砸牢關"——理事圓融、事事無礙。這一解釋將《起信論》完全納入禪宗的修行體系，對後來禪宗與教理的深度融合有重要推動作用。

## 白話解讀

### 核心內容

《大乘起信論抉隱》是明代禪宗學人對《大乘起信論》的深入解釋。"抉隱"就是挖掘隱藏的道理。

此疏用禪宗"三關"的概念來解釋《起信論》：
- **心真如門** = **破初關**：突然明白，原來我的心本來就是清淨的！這就是"見性"。
- **心生滅門** = **透重關**：雖然明白了本性，但日常生活中還有煩惱、還有習氣。要在煩惱中保持覺悟，這就是"保任"。
- **二門不二** = **砸牢關**：最後明白，清淨的心和煩惱的心不是兩個，而是同一個。到這個時候，處處是道場，事事是修行。

### 現代啓示

1. **覺悟的層次**：禪宗"三關"啓示我們：成長有不同的層次。不是"一朝頓悟，萬事大吉"，而是需要持續的修行和保任。

2. **在日常生活中修行**："透重關"就是在日常生活中保持覺悟。現代人不需要去深山閉關，在工作、家庭、社交中都可以修行。

3. **超越二元對立**："砸牢關"就是超越"清淨"與"煩惱"的二元對立。不是追求一個"清淨"的境界，而是認識到清淨與煩惱本質相同。

### 閱讀建議

- 此疏較爲深入，適合有一定禪宗基礎的讀者
- 可配合紫柏《釋要》、憨山《裂網疏》一同閱讀，理解明代禪宗對《起信論》的解釋傳統
- 注意"三關"與"一心二門"的對應，理解禪宗的修行次第
