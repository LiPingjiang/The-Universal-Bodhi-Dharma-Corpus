## 甘珠尔与丹珠尔——藏文大藏经

### 甘珠尔（bKa' 'gyur）

**甘珠尔** = bKa'（佛语/教言）+ 'gyur（翻译）= "佛语的翻译"——对应汉传"经"和"律"。

| 部 | 内容 | 典籍数 |
|---|------|--------|
| 律部 | 根本说一切有部律 | 13部 |
| 般若部 | 般若经系列 | 23部 |
| 华严部 | 华严经等 | 6部 |
| 宝积部 | 大宝积经诸会 | 49部 |
| 经部 | 其他大乘经 | 270余部 |
| 续部 | **密续**（Tantra） | 450余部 |

甘珠尔最大的特色：**密续部**——占了将近一半的篇幅——这是藏传佛教与汉传佛教最大的典籍差异。汉文大藏经中密教文献相对较少，而甘珠尔中密续极为丰富。

主要版本：德格版（sDe dge）、北京版、奈塘版（sNar thang）、卓尼版。

### 丹珠尔（bsTan 'gyur）

**丹珠尔** = bsTan（论/论述）+ 'gyur（翻译）= "论述的翻译"——对应汉传"论"。

约3500余部论典——涵盖：
- 印度论师的中观/唯识/因明/阿毗达磨论著
- 密续注释
- 诗学/语法/医学/天文/工艺等"五明"文献

## 宗派核心典籍

### 宁玛派（rNying ma）

| 典籍 | 内容 |
|------|------|
| **宁玛十万续**（rNying ma rgyud 'bum） | 宁玛派特有的密续集成——不包含在甘珠尔中 |
| **龙钦七宝藏** | 龙钦饶降巴（Longchenpa）的系统论著——大圆满（Dzogchen）的权威阐述 |
| **伏藏**（gTer ma） | 莲花生大士"埋藏"的法教——由后世"掘藏师"（gTer ston）发掘 |

**大圆满**（Dzogchen / rDzogs chen）：宁玛派的最高法门——"心的本性就是佛——不需要修——只需要认出"。核心文本包括《心部》《界部》《口诀部》三大分类。

### 噶举派（bKa' brgyud）

| 典籍 | 内容 |
|------|------|
| **那洛六法** | 那洛巴传玛尔巴——六种修身/修气脉明点的密法 |
| **大手印** | Mahamudra——噶举派的最高见地——与大圆满类似但方法不同 |
| **密勒日巴道歌集** | 密勒日巴的证悟之歌——藏文文学的瑰宝 |
| **解脱庄严宝论** | 冈波巴——菩萨道与大手印的综合 |

### 萨迦派（Sa skya）

| 典籍 | 内容 |
|------|------|
| **道果法** | Lam 'bras——萨迦派的核心修行体系——"轮回与涅槃无别" |
| **萨迦五祖文集** | 五位萨迦祖师的著作集成 |
| **量理宝藏** | 萨迦班智达——因明学的藏传权威论著 |

### 格鲁派（dGe lugs）

| 典籍 | 内容 |
|------|------|
| **菩提道次第广论** | 宗喀巴——显教修行的完整道路（已有档案030） |
| **密宗道次第广论** | 宗喀巴——密教修行的完整道路 |
| **中观广释** | 宗喀巴——对入中论的详细注释 |
| **辨了不了义善说藏** | 宗喀巴——判教论——区分佛经的了义与不了义 |

### 五大论（格鲁派僧院课程）

1. **因明**：释量论（法称）
2. **般若**：现观庄严论（弥勒/无著）
3. **中观**：入中论（月称）
4. **俱舍**：俱舍论（世亲）
5. **律**：律经（功德光）

## 大白话总结

藏传佛教的典籍体系与汉传有一个根本差异：**密续**。汉传大藏经以经/律/论为主——密教文献只占很小一部分。但在藏传的甘珠尔中——密续占了将近一半——这反映了藏传佛教"显密双修"的特色。

另一个独特之处是**伏藏**传统——特别是宁玛派——认为莲花生大士在8世纪入藏时"埋藏"了大量法教——等到未来时机成熟时再由"掘藏师"发掘出来。这个传统使得藏传佛教的"经典"不是封闭的——而是持续"被发现"的——直到20世纪仍有新的伏藏被发掘。

对于格鲁派来说——"五大论"构成了僧人教育的骨架——一个格鲁派僧人从入寺到毕业可能需要15-25年——其中大部分时间都在学习和辩论这五部论典。这种系统的学术训练使格鲁派的僧院教育成为人类历史上持续时间最长的"研究生项目"之一。
