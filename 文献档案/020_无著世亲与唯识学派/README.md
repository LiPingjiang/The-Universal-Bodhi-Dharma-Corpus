## 概述

唯识学派（Vijñānavāda，"唯识论"；Yogācāra，"瑜伽行派"）是大乘佛教的两大哲学传统之一（另一为中观学派）。无著（Asaṅga，约4世纪）和世亲（Vasubandhu，约4-5世纪）兄弟是唯识学派的奠基人。唯识学派的核心主张是：一切现象都是识（vijñāna）的变现，没有独立于识之外的外境（bāhya-artha）。

唯识学派对佛教心理学、认识论和修行论有深远影响，是中国法相宗（玄奘、窥基）和藏传佛教的重要思想基础。

## 核心概念

- **无著**（Asaṅga）：唯识学派奠基人，约4世纪，著有《摄大乘论》、《瑜伽师地论》
- **世亲**（Vasubandhu）：无著之弟，著有《唯识二十论》、《唯识三十颂》
- **唯识**（vijñaptimātratā）：一切现象都只是识的变现
- **阿赖耶识**（ālaya-vijñāna）："藏识"，储存一切业种子的根本识
- **三性**（tri-svabhāva）：遍计所执性、依他起性、圆成实性
- **种子**（bīja）：储存在阿赖耶识中的业力潜能
- **转依**（āśraya-parāvṛtti）：修行的目标，转化阿赖耶识
