## 学术解读

### 一、文献学信息

《大乘起信论发隐》是明代李贽（1527-1602）所撰。李贽，号卓吾，泉州晋江（今福建晋江）人，为明代著名思想家、文学家，以"异端"思想著称。此疏以"发隐"为名，取"发抉隐微，揭示奥义"之意，是《大乘起信论》注疏中较为独特的一部，因为它出自一位非僧人的"居士"之手，且带有强烈的个人色彩。

李贽一生思想激进，批判传统儒学，推崇佛教禅宗，主张"童心说"，认为人应保持纯真的本心。其著作《焚书》《续焚书》《藏书》等在明代思想界引起巨大反响。此疏作于其晚年，是他以禅宗立场解释《起信论》的代表作。

### 二、历史背景

明代中后期，王阳明心学盛行，"心即理""致良知"等思想深入人心。李贽受心学影响，进而归向禅宗，认为佛教禅宗才是"心学"的究竟归宿。当时《大乘起信论》作为"一心二门"思想的经典，自然受到李贽的重视。此疏正是在"心学"与"禅学"交融的背景下产生，体现了明代思想界儒释道三教合流的趋势。

### 三、思想史定位

李贽此疏的最大特色在于以"童心"概念来解释《起信论》的"心真如门"。他认为，"心真如"即是"童心"——未被世俗污染的纯真本心。"心生灭门"则是"习心"——被后天习气所遮蔽的心。李贽进一步将"始觉"解释为"恢复童心"的过程，将"修行"解释为"去除习心、回归童心"的实践。这一解释虽然偏离了《起信论》的原意，但反映了明代"心学"对佛教思想的吸收和改造，对后来近代思想启蒙有间接影响。

## 白话解读

### 核心内容

《大乘起信论发隐》是明代思想家李贽对《大乘起信论》的解释。李贽不是僧人，是一位思想激进的文人。

李贽用他著名的"童心"概念来解释《起信论》："心真如门"就是"童心"——小孩子那种纯真、没有虚伪的心；"心生灭门"就是"习心"——被大人的世界污染了的心。修行的目的，就是去掉"习心"，恢复"童心"。

李贽还说：很多人学佛学成了"假道学"——表面上仁义道德，内心却充满算计。真正的学佛，就是保持"童心"，做一个真实的人。

### 现代启示

1. **保持纯真**：现代社会充满虚伪和算计。李贽的"童心"思想提醒我们：保持内心的纯真和真实，是一种难得的勇气。

2. **反对假道学**：李贽批判"假道学"，启示我们：任何信仰如果变成了虚伪的包装，就失去了意义。真正的修行是内心的真实。

3. **独立思考**：李贽以"异端"自居，不盲从权威。这种独立思考的精神，对今天的学习和工作都有启示。

### 阅读建议

- 李贽的疏解带有强烈的个人色彩，适合对明代思想感兴趣的读者
- 可配合《焚书》中的《童心说》一同阅读，理解其整体思想
- 注意李贽以"童心"解"心真如"的特点，理解其思想的时代背景
