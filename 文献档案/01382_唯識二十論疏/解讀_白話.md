# 唯识二十论疏 · 白話解讀

## 這是什麼書？

《唯識二十論疏》是唐代唯識宗大師窺基為世親的《唯識二十論》寫的注釋。世親是印度唯識宗的集大成者，被稱為「千部論主」。《唯識二十論》只有二十頌（類似二十首短詩），但內容極其精煉，用嚴密的邏輯論證了一個核心命題：「萬法唯識」——一切現象都是心識變現的。

窺基的《疏》就像一位嚴謹的邏輯學教授，把世親的每一個論證都展開來講，用因明（佛教邏輯學）的三支作法，一步一步地證明為什麼「萬法唯識」是對的。

簡單說，這本書回答的是：「為什麼說一切現象都是心識變現的？如果沒有客觀世界，為什麼大家看到的東西都差不多？」

## 核心內容

### 1. 主要說了什麼？

**萬法唯識：一切現象都是心識的變現**

唯識宗的核心命題是「萬法唯識」。這不是說「我想什麼就有什麼」（那不是魔法嗎？），而是說：我們所經驗的世界，是透過心識的結構和內容來呈現的。沒有離開心識而獨立存在的「客觀世界」。

世親用幾個精妙的論證來支持這個觀點：

**夢喻論證**

世親說：你做夢的時候，夢裡有山河大地、有人有事，你覺得很真實。但醒來後發現，一切都是心識變現的。我們的「現實世界」也是一樣——它看起來很真實，但本質上也是心識的變現。

窺基解釋說：夢中的世界和醒時的世界，在本質上沒有區別。兩者都是心識的產物。唯一的區別是，醒時的世界有更多的「共業」支撐，所以看起來更穩定、更持久。

**地獄論證**

世親說：地獄眾生看到烈火、刀山、鬼差，感到極度痛苦。但這些烈火刀山不是客觀存在的——如果是客觀存在的，那麼鬼差走在火裡也應該感到痛苦，但鬼差不會。所以地獄的烈火是業力所現，不是客觀實有。

窺基解釋說：同理，我們人間看到的世界，也是業力與心識所現。只是因為我們的「共業」相似，所以看到的東西也差不多。

**同緣論證**

這是世親最精妙的論證之一：

如果外境是客觀實有的，那麼每個人應該看到不同的東西（因為每個人的心識不同）。但實際上，很多人可以同時看到同一個東西（比如一群人同時看到月亮）。這說明大家看到的不是「客觀的外境」，而是「相似的心識顯現」。

就像很多人同時做相似的夢，夢裡都有月亮。這不是因為有一個客觀的月亮在夢外面，而是因為大家的夢境有相似的結構。

**時分論證**

世親說：如果外境是客觀實有的，它應該是恆常不變的。但我們看到的一切現象都在剎那剎那地變化——花開花謝、雲卷雲舒。這說明現象不是客觀實體，而是心識的流變。

窺基解釋說：心識的特點就是剎那生滅，所以心識變現的現象也是剎那生滅的。如果外境是客觀實有的，它應該是穩定的、不變的。

**種子與現行：宇宙是怎麼運作的？**

唯識宗用「種子」和「現行」來解釋宇宙的運作：

- **種子**：儲存在阿賴耶識（第八識）裡的潛在傾向。你過去所有的經驗，都會留下「種子」。
- **現行**：種子遇到適當的條件，就會「現行」——產生現實的經驗和現象。
- **熏習**：現實的經驗又會產生新的種子，儲存回阿賴耶識。

這樣就形成了一個無盡的循環：種子生現行，現行熏種子，種子又生現行……這就是我們的生死輪迴。

**三性：認識的三種狀態**

唯識宗用「三性」來描述我們認識世界的三種狀態：

- **遍計所執性**：我們錯誤地認為世界有獨立自存的實體，並且對這些虛構的實體產生執著。這就是凡夫的狀態——活在幻覺中。
- **依他起性**：一切現象都是依賴各種條件而生起的，沒有獨立自存的實體。這是現象的真實狀態——緣起性空。
- **圓成實性**：認識到依他起性的真相，不再執著於遍計所執的虛構，證得圓滿真實。這是覺悟的狀態。

修行的過程，就是從遍計所執，到認識依他起，最終證得圓成實。

### 2. 對現代人有什麼啟發？

**關於「現實是建構的」**：唯識宗說「萬法唯識」，現代認知科學也說我們的現實是透過大腦建構的。兩者都指出：我們不是被動地接收現實，而是主動地建構現實。改變心識，就改變了世界。這不是唯心主義，而是對認知過程的深刻洞察。

**關於「共業」的社會意義**：唯識宗說，大家看到相似的世界，是因為有「共業」。對現代人來說，這意味著：我們的社會現實是共同建構的。如果大家都有貪婪的種子，就會共同建構一個貪婪的社會；如果大家都有慈悲的種子，就會共同建構一個慈悲的社會。改變社會，從改變每個人的心識開始。

**關於「夢喻」的哲學意義**：莊子也說「夢蝶」——不知道是自己夢見變成蝴蝶，還是蝴蝶夢見變成自己。唯識宗的夢喻與此相通：我們的「現實」和「夢境」在本質上沒有區別，都是心識的顯現。這讓我們對「什麼是真實」有了更深刻的思考。

**關於邏輯思維的訓練**：窺基用因明三支作法來論證唯識，這種嚴密的邏輯訓練對現代人非常有益。在信息爆炸的時代，我們需要學會辨別論證的有效性，不被似是而非的觀點迷惑。

### 3. 怎麼讀這部書？

- **適合誰讀**：對邏輯和哲學感興趣的人；想了解佛教認識論的人；對「現實是怎麼建構的」這個問題有好奇心的人。
- **從哪裡開始**：《唯識二十論》只有二十頌，非常精煉。建議先讀這份白話解讀，了解核心論證；再讀原文，配合窺基《述記》。
- **需要注意什麼**：
  - 「萬法唯識」不是主觀唯心論，不是說「我想什麼世界就是什麼」。而是說我們經驗的世界是透過心識結構呈現的。
  - 唯識宗與中觀宗（空宗）不是對立的。中觀破「遍計所執」之實有，唯識立「依他起」之緣起。兩者相輔相成。
  - 因明的術語（宗、因、喻）需要一些時間適應，但掌握後會發現它是非常嚴密的邏輯工具。
- **搭配閱讀》：玄奘《唯識二十論》、窺基《唯識二十論述記》、陳兵《佛教心理學》、楊維中《中國唯識宗通史》。

| 萬法唯識 | 一切諸法皆唯識所變，為唯識宗的核心命題 |
| 八識 | 心識的八個層次：前五識（感官）、意識（思考）、末那識（我執）、阿賴耶識（倉庫） |
| 阿賴耶識 | 根本識，像一個巨大倉庫，儲存了我們過去所有經驗的種子 |
| 種子 | 潛在的功能傾向，儲存在阿賴耶識中，遇緣就會發芽 |
| 現行 | 種子遇到條件後產生的現實活動和經驗 |
| 遍計所執性 | 錯誤地認為世界有獨立實體，並對這些虛構產生執著 |

| 圓成實性 | 認識到依他起性的真相，證得圓滿真實 |

## 讀後思考

1. 世親用「夢喻」說現實和夢境在本質上沒有區別。你認同這個觀點嗎？為什麼？

2. 「同緣論證」說大家看到相似的世界是因為「共業」。你認為這個觀點如何解釋現代社會的集體現象（如流行文化、經濟危機）？

3. 如果「萬法唯識」，那麼科學研究的對象是什麼？科學發現的規律是客觀的還是心識的建構？

4. 唯識宗說「遍計所執」是錯覺，「依他起」是真相。你生活中有哪些「遍計所執」——你以為真實存在、其實只是虛構的東西？

5. 窺基用因明的嚴密邏輯來論證唯識。你認為這種邏輯訓練對現代人有什麼價值？

6. 唯識宗與中觀宗（空宗）的區別，你認為哪一個更適合現代人的理解方式？為什麼？
