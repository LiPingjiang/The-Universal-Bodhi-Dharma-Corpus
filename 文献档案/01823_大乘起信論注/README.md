## 學術解讀

### 一、文獻學信息

《大乘起信論注》是唐代道宣（596-667）所撰。道宣，俗姓錢，丹徒（今江蘇鎮江）人，爲唐代律宗創始人，世稱"南山律師"，被尊爲中國律宗南山宗初祖。此疏以"注"爲名，取"註解文句，疏通義理"之意，是《大乘起信論》註疏中較爲早期的一部。

道宣一生致力於律學的整理和弘揚，著有《四分律行事鈔》《四分律羯磨疏》《四分律含注戒本疏》等，被合稱爲"南山三大部"。同時，道宣也精通經論，著有《廣弘明集》《續高僧傳》等。此疏作於其中年，體現了其不僅精通律學，也深達教理的學術素養。

### 二、歷史背景

道宣活躍於唐初，當時佛教正處於從隋代復興向唐代鼎盛過渡的時期。唐初皇室重視佛教，但更傾向於秩序和規範。道宣作爲律宗創始人，強調戒律是佛教的基礎，同時也不廢教理。此疏正是在這一背景下產生，旨在說明《起信論》的"信"與戒律的"持"是一致的：信是內心的基礎，戒是行爲的規範，二者相輔相成。

### 三、思想史定位

道宣此疏的特色在於以律宗的立場來解釋《起信論》。他認爲，《起信論》的"信心"是持戒的基礎，沒有信心，戒律只是外在的形式；有了信心，戒律纔是內心的自然流露。道宣進一步將"一心二門"與戒律實踐相聯繫："心真如門"即是"戒體"（本有的清淨佛性），"心生滅門"即是"戒相"（具體的戒律規範）。這一解釋雖然非主流，但體現了律宗對《起信論》的獨特理解，豐富了《起信論》的解釋傳統。

## 白話解讀

### 核心內容

《大乘起信論注》是唐代律宗創始人道宣大師對《大乘起信論》的註解。道宣大師是戒律專家，但他也深達教理。

道宣大師的獨特視角是：把《起信論》和戒律聯繫起來。他說：《起信論》講"信"，戒律講"持"，信和戒是一體的。沒有內心的信，守戒只是外在的形式；有了內心的信，守戒就是自然的行爲。

道宣大師還用戒律的語言來解釋"一心二門"："心真如門"就是"戒體"——我們受戒時得到的那個清淨的佛性；"心生滅門"就是"戒相"——具體的戒律條文。戒體是內在的、清淨的，戒相是外在的、變化的，但二者本是一體。

### 現代啓示

1. **內外一致**：現代人常常"說一套做一套"。道宣大師告訴我們：真正的道德不是外在的規則，而是內心的自然流露。信是戒的內在基礎，戒是信的外在表現。

2. **自律與自由**：很多人認爲戒律是束縛。但道宣大師說：戒律是"戒體"的顯現，而戒體是清淨的佛性。所以守戒不是被束縛，而是迴歸本性。

3. **信心與行動**："信"和"戒"的關係，啓示我們：任何行動都需要內心的信念支撐。沒有信念的行動是空洞的，沒有行動的信念是虛弱的。

### 閱讀建議

- 道宣的註疏較爲平實，適合初學者
- 可配合《四分律行事鈔》一同閱讀，瞭解律宗的基本思想
- 注意其"以戒解教"的特點，理解律宗對教理的獨特視角
