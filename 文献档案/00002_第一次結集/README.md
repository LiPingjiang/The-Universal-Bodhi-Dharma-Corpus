## 關於本目錄

本目錄收錄與**佛教第一次結集**（梵語：Saṅgīti，巴利語：Saṅgāyana）直接相關的原始文獻與解讀資料。第一次結集是佛陀入滅後，由大迦葉（Mahākassapa）主持、在王舍城（Rājagaha）附近的七葉窟（Sattapaṇṇi-guhā）舉行的經典結集，是佛教史上最早、最重要的一次經典整理活動。此次結集確立了經藏（Sutta-piṭaka）與律藏（Vinaya-piṭaka）的基本框架，奠定了後世佛教三藏傳承的基礎。

本目錄中每部文獻均提供**三件套**：
- **原文** — 巴利語或漢文原始文本節錄，含甲譯（白話意譯）與乙譯（嚴謹對照）
- **學術解讀** — 文獻學、歷史學、思想史角度的專業分析
- **白話解讀** — 面向普通讀者的通俗講解，佛教術語均作白話化翻譯並附腳註

## 第一次結集的歷史意義

第一次結集發生在佛陀涅槃（約公元前5世紀）後的第一個雨安居期間，地點在王舍城附近的七葉窟。參加結集的是五百位阿羅漢（已徹底斷除煩惱的聖者），由大迦葉主持。結集的核心成果是：

1. **經藏的誦出**：阿難誦出佛陀生前所說的教法（Dhamma），形成經藏（Sutta-piṭaka）的雛形。
2. **律藏的誦出**：優波離誦出佛陀制定的戒律（Vinaya），形成律藏（Vinaya-piṭaka）的雛形。
3. **經典的確認**：通過五百阿羅漢的共同認證，確立了"如是我聞"（Evaṃ me sutaṃ）的經典開頭格式，保證了傳承的可靠性。

第一次結集不僅是佛教經典傳承的起點，也確立了佛教僧團內部以"結集"（Saṅgīti）方式整理與確認經典的制度，爲後世第二次結集（毘舍離結集）、第三次結集（華氏城結集）乃至第四次結集（阿育王時代、迦膩色迦王時代）奠定了基礎。
