# 六足论疏 · 白話解讀

## 這是什麼書？

《六足論疏》是對佛教「六足論」的注解。「六足論」是說一切有部（部派佛教的重要部派）的六部基礎論典，就像一個人的六條腿（比喻），支撐著《發智論》（身論）這個主體。這六部論從不同角度系統地分析了佛教哲學。

## 核心內容

### 1. 主要說了什麼？

**六部論典各有側重**：六足論包括：《集異門足論》（分析各種佛教術語的不同名稱）、《法蘊足論》（系統論述各種修行方法）、《施設足論》（論述世界的形成和眾生的種類）、《識身足論》（論述心識的種類）、《品類足論》（對一切法進行分類）、《界身足論》（論述十八界的性質）。

**三十七道品**：《法蘊足論》對三十七道品（四念處、四正勤、四神足、五根、五力、七覺支、八正道）有系統論述。三十七道品是佛陀所說的三十七種修行方法，是修行的完整體系。

**十八界**：《界身足論》論述十八界——六根（眼耳鼻舌身意）、六境（色聲香味觸法）、六識（眼識耳識鼻識舌識身識意識）。這是佛教對認識活動的系統分析。

### 2. 對現代人有什麼啟發？

**修行的系統性**：三十七道品提供了一套完整的修行體系，從四念處（正念）到八正道，層層深入，系統全面。

**認識的分析**：十八界的分析，幫助我們理解認識活動的結構：感官（根）接觸對象（境），產生認識（識）。這與現代認知科學的研究有相通之處。

**術語的精確性**：《集異門足論》對各種術語的精確分析，體現了佛教哲學對語言和概念的嚴格態度。

### 3. 怎麼讀這部書？

六足論是比較艱深的論典，建議先讀一些部派佛教的入門書籍，再選擇感興趣的論典閱讀。《法蘊足論》中關於三十七道品的部分，對修行者最有實際意義。

| 六足論 | 說一切有部的六部基礎論典 |
| 三十七道品 | 三十七種修行方法的總稱 |
| 四念處 | 觀身、觀受、觀心、觀法四種正念修行 |

| 十八界 | 六根、六境、六識，認識活動的三個要素 |
