## 源流与背景

般若波罗蜜多（Prajñāpāramitā，意为「智慧的圆满」）经系是大乘佛教最早兴起的经典群之一。其中《八千颂般若》（Aṣṭasāhasrikā Prajñāpāramitā，意为「八千行/颂的般若波罗蜜多」）是现存最古老的般若经文本，被学术界普遍认为是般若经群的原始形态——后世更为庞大的《二万五千颂》《十万颂》等大品般若，均是在八千颂基础上的扩展与增广。龙树在《大智度论》中所注释的「大品般若」与八千颂有密切关联，中观学派的核心概念「空」（śūnyatā）直接承自此经。

经文以佛陀在灵鹫山（Gṛdhrakūṭa）与须菩提（Subhūti）的对话为主线，共32品。核心内容反复阐明：一切法（包括菩萨、般若波罗蜜多本身）都不可得、无自性、如幻如化——菩萨正是在这种「无所得」中行六度、度众生。经中遍布的「三段否定公式」（佛说X即非X是名X）后来在《金刚经》中被提炼为最精练的形式，而八千颂中已有的「不住」「不可得」「如幻」等概念，构成了整个大乘空宗思想的原典基础。

关于成书环境，学术界有多个假说。Conze（1960）和 Lamotte（1962）主张西北印度犍陀罗地区，依据是最早写本残片出土于此；平川彰（1963）和印顺（1981）则关联南印度大众部系；Schopen（1975）和 Harrison（2003）提出非单一地点产生的修正版。当前学术风向倾向于承认多中心性，但犍陀罗-北印度区域可能是最初的主要编纂地。支娄迦谶于东汉光和二年（179 CE）译出的《道行般若经》（T224）是此经最早有确切纪年的汉译本，将梵文本的成形推至至少公元1世纪中叶。

## 关键概念

- **般若波罗蜜多**（Prajñāpāramitā）：prajñā（智慧）+ pāramitā（到彼岸/圆满），意为「到达彼岸的智慧」。但经文开宗明义宣称「不存在一个叫般若波罗蜜多的东西」——连自身最推崇的概念都加以否定，以此彻底堵住对佛法的执着。
- **空性**（śūnyatā）：在八千颂中尚未如后世中观哲学那样成为精确定义的哲学概念，而是更接近一种修辞-冥想策略——通过对每个概念反复施加「不可得」，训练修行者不在任何对象上停驻心念。空性涵盖对象空、修行空、语言空三个层次。
- **不可得**（anupalabdha / na vidyate）：vid = 「找到」「存在」；na vidyate = 「不被找到」「不存在为可指认的实体」。这是般若经系最核心的手法：对一切法反复宣说「不可得」，包括般若本身。
- **不住**（na pratitiṣṭhet）：prati（向着）+ sthā（站立）= 「以……为落脚点」。「不住」即做布施时不把任何东西当作可依靠的实在基础。后来《金刚经》「应无所住而生其心」正是此思想的极度精简版。
- **三段否定公式**（佛说X / 即非X / 是名X）：逻辑结构为——我们使用「X」这个名称；但此名称不指向独立实在的本体；正因无固定本体，「X」才能作为灵活的约定名称使用。这区分了约定层面的有效使用与实体层面的真实存在，是般若经系认识论的核心工具。
