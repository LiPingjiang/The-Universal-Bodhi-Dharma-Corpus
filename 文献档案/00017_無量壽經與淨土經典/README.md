## 概述

淨土經典（Sukhāvatī-vyūha-sūtra，"極樂世界莊嚴經"）是大乘佛教淨土宗（Pure Land Buddhism）的核心文獻，以阿彌陀佛（Amitābha，"無量光"）的極樂世界（Sukhāvatī，"極樂"）爲核心，闡述往生淨土的修行方法。主要包括《無量壽經》（大經）、《觀無量壽經》（觀經）、《阿彌陀經》（小經），合稱"淨土三經"。

淨土思想在東亞佛教中影響極爲深遠，是中國、日本、朝鮮、越南等地最廣泛流傳的佛教修行傳統之一。

## 核心概念

- **阿彌陀佛**（Amitābha/Amitāyus）：無量光/無量壽，西方極樂世界的佛
- **極樂世界**（Sukhāvatī）：阿彌陀佛的淨土，修行的理想環境
- **四十八願**（catvāriṃśad-aṣṭa-praṇidhāna）：法藏比丘成佛前發的四十八個願望
- **第十八願**（aṣṭādaśa-praṇidhāna）：唸佛往生願，淨土宗的核心
- **唸佛**（buddha-smṛti/nian-fo）：稱念阿彌陀佛名號的修行方法
- **往生**（upapatti）：死後往生極樂世界
- **一心不亂**（ekāgra-citta）：專注唸佛，心不散亂
