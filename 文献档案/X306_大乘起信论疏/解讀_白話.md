# 大乘起信论疏 · 白話解讀

## 這是什麼書？

《大乘起信論疏》是唐朝華嚴宗大師法藏對《大乘起信論》的詳細解釋。《大乘起信論》是一部非常重要的佛教論典，它用簡潔的語言解釋了「為什麼眾生本來是佛，卻又迷失在煩惱中」這個根本問題。法藏的疏釋讓這部論典更容易理解。

## 核心內容

### 1. 主要說了什麼？

**一心的秘密**：這部論的核心是「一心」。我們每個人的心，同時具有兩個面向：一個是「真如門」，就是心的本來面目，清淨、永恆、不生不滅；另一個是「生滅門」，就是我們日常體驗到的心，有喜怒哀樂，隨著外境起伏變化。這兩個面向不是兩個心，而是同一個心的兩面。

**為什麼會迷失？** 論中說，因為「無明」（根本的無知），清淨的心起了一個念頭，就像平靜的海面起了波浪。這個念頭帶動了更多的念頭，最終形成了我們複雜的心理世界和外在的物質世界。

**怎麼回歸本心？** 通過信心（信真如、信佛、信法、信僧）和修行（布施、持戒、忍辱、精進、禪定、智慧），逐漸消除無明，讓心回歸本來的清淨狀態。

### 2. 對現代人有什麼啟發？

**心的兩面性**：我們的心既有清淨的一面（善良、智慧、慈悲），也有染污的一面（貪婪、憤怒、愚癡）。修行就是讓清淨的一面越來越強，染污的一面越來越弱。這與現代心理學中「積極心理學」的理念有相通之處。

**習氣的力量**：論中說到「熏習」，就是習慣的力量。好的習慣會薰染我們的心，讓我們越來越好；壞的習慣也會薰染我們，讓我們越來越差。這提醒我們要注意日常生活中的習慣養成。

**信心的重要性**：論中把「信心」放在修行的首位。對自己本具佛性的信心，是修行的根本動力。

### 3. 怎麼讀這部書？

建議先讀《大乘起信論》原文（很短，只有一萬多字），再讀法藏的疏釋。可以配合印順法師的《大乘起信論講記》一起閱讀，更容易理解。重點理解「一心二門」的架構，這是整部論的骨幹。

## 關鍵概念（白話版）

| 佛學術語 | 白話解釋 |
|---------|---------|
| 一心 | 眾生本具的清淨心，也是宇宙萬法的根本 |
| 真如門 | 心的本來面目，清淨不變 |
| 生滅門 | 心的現象面，隨緣起伏 |
| 如來藏 | 眾生心中本具的佛性種子 |
| 無明 | 根本的無知，迷失本心的原因 |
| 熏習 | 習氣的薰染，好壞習慣都會影響我們的心 |
| 三大 | 心的體（本質）、相（功德）、用（作用）三個面向 |

## 讀後思考

1. 「一心二門」的說法，讓你對自己的心有什麼新的認識？
2. 「無明」是迷失的根源，你認為在日常生活中，什麼是我們最常見的「無明」？
3. 如果你相信自己本具佛性，這種信念會如何改變你對自己和他人的態度？
