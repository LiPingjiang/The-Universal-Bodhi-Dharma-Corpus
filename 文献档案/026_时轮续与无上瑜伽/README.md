## 时轮续源流骨架

《时轮续》（Kālacakra-tantra，"时轮怛特罗"，字面义"时间之轮的密续"）是藏传佛教无上瑜伽续（Anuttara-yoga-tantra）中最晚出、最复杂的密续之一，约成立于公元10–11世纪印度东部：

```
        时轮续体系
        ├── 《时轮根本续》（Mūla-Kālacakra-tantra）— 原典，12000颂，已佚
        ├── 《时轮略续》（Laghu-Kālacakra-tantra）— 现存根本续，1047颂
        │       ├── 第一品：外时轮（宇宙论）
        │       ├── 第二品：内时轮（身体论）
        │       ├── 第三品：别时轮（灌顶论）
        │       ├── 第四品：修行（生起次第）
        │       └── 第五品：智慧（圆满次第）
        ├── 《无垢光大疏》（Vimalaprabhā）— 最重要注疏，普贤慧著
        └── 藏传时轮传承
                ├── 觉囊派（Jonang）— 时轮续的主要传承者
                ├── 格鲁派（Gelug）— 宗喀巴弟子传承
                └── 萨迦派（Sakya）— 亦有传承
```

时轮续的独特之处在于：它将宇宙论（外时轮）、身体论（内时轮）、修行论（别时轮）三者紧密结合，认为宇宙的结构与人体的结构相互对应，通过修炼身体内部的气脉明点，可以与宇宙的节律（时轮）同步，最终证得时轮金刚（Kālacakra）的成就。

时轮续还包含了独特的天文历法体系（时轮历）和预言体系（香巴拉王国的预言），是藏传佛教中最具综合性的密续传统。
