# 大乘起信論闡微

## 學術解讀

### 一、文獻學信息

《大乘起信論闡微》是清代楊文會（1837-1911）所撰。楊文會，字仁山，安徽石埭（今安徽石臺）人，爲近代中國佛教復興運動的先驅，世稱"楊仁山"，被尊爲"近代佛教復興之父"。此疏以"闡微"爲名，取"闡發幽微，顯微闡幽"之意，是近代《大乘起信論》註疏中最具學術性和現代性的一部。

楊仁山一生致力於佛教經典的整理和刊刻，創辦"金陵刻經處"，刻印佛經三千餘卷，對近代佛教的復興貢獻巨大。他還創辦"祗洹精舍"，培養佛學人才，太虛、歐陽竟無等近代佛學大師皆受其影響。此疏作於其晚年，融會傳統義學與現代學術方法，開創了近代佛學研究的新範式。

### 二、歷史背景

清末民初，中國面臨"數千年未有之大變局"，佛教也面臨衰落與復興的雙重挑戰。楊仁山在日本留學期間，發現日本佛教保存了大量中國已佚的佛教典籍，遂立志復興中國佛教。他主張"佛教非宗教非哲學"，強調佛教的理性精神和學術價值。此疏正是在這一背景下產生，旨在以現代學術方法重新解釋《起信論》，爲近代佛學研究奠定基礎。

### 三、思想史定位

楊仁山此疏的最大特色在於以"比較宗教學"和"文獻學"的方法來研究《起信論》。他首次系統考證了《起信論》的作者問題（馬鳴菩薩是否真爲作者）、翻譯問題（真諦譯本與玄奘譯本的差異）、以及思想來源問題（印度如來藏思想與中國本土思想的融合）。楊仁山認爲，《起信論》雖然可能不是馬鳴菩薩的真作，但其思想價值不可否認，是中國佛教"心性論"的重要經典。這一"批判性繼承"的態度，對後來近代佛學研究有決定性影響。

## 白話解讀

### 核心內容

《大乘起信論闡微》是近代佛教復興先驅楊仁山對《大乘起信論》的研究。楊仁山用現代學術方法來研究這部古論。

楊仁山的獨特貢獻在於：他既尊重傳統，又敢於質疑。他考證了《起信論》的作者——傳統說是馬鳴菩薩寫的，但楊仁山發現有些疑點。不過，他說：即使作者有疑問，這部論的思想價值是不可否認的。

楊仁山還從"比較宗教學"的角度，把《起信論》與印度的《楞伽經》《密嚴經》等對比，發現它們都講"如來藏"思想。這說明《起信論》不是孤立的，而是印度佛教思想發展的一部分。

### 現代啓示

1. **批判性繼承**：楊仁山既尊重傳統，又敢於質疑。這種"批判性繼承"的態度，對今天的學習和研究都有啓示：不盲從權威，也不全盤否定。

2. **學術方法**：楊仁山引入現代學術方法（文獻學、比較宗教學）來研究佛學，啓示我們：傳統學問也可以用現代方法來研究，二者並不矛盾。

3. **文化復興**：楊仁山在日本發現中國已佚的佛教典籍，立志復興中國佛教。這啓示我們：文化的傳承需要主動的努力，不能被動等待。

### 閱讀建議

- 楊仁山的疏解具有現代學術性，適合有一定佛學基礎且對近代學術感興趣的讀者
- 可配合《楊仁山文集》中的其他文章，如《佛教初學課本》《十宗略說》等一同閱讀
- 注意其"批判性繼承"的學術態度，理解近代佛學研究的轉型
