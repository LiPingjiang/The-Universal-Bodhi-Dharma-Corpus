## 学术解读

### 一、文献学信息

《大乘起信论阐微》是清代杨文会（1837-1911）所撰。杨文会，字仁山，安徽石埭（今安徽石台）人，为近代中国佛教复兴运动的先驱，世称"杨仁山"，被尊为"近代佛教复兴之父"。此疏以"阐微"为名，取"阐发幽微，显微阐幽"之意，是近代《大乘起信论》注疏中最具学术性和现代性的一部。

杨仁山一生致力于佛教经典的整理和刊刻，创办"金陵刻经处"，刻印佛经三千余卷，对近代佛教的复兴贡献巨大。他还创办"祗洹精舍"，培养佛学人才，太虚、欧阳竟无等近代佛学大师皆受其影响。此疏作于其晚年，融会传统义学与现代学术方法，开创了近代佛学研究的新范式。

### 二、历史背景

清末民初，中国面临"数千年未有之大变局"，佛教也面临衰落与复兴的双重挑战。杨仁山在日本留学期间，发现日本佛教保存了大量中国已佚的佛教典籍，遂立志复兴中国佛教。他主张"佛教非宗教非哲学"，强调佛教的理性精神和学术价值。此疏正是在这一背景下产生，旨在以现代学术方法重新解释《起信论》，为近代佛学研究奠定基础。

### 三、思想史定位

杨仁山此疏的最大特色在于以"比较宗教学"和"文献学"的方法来研究《起信论》。他首次系统考证了《起信论》的作者问题（马鸣菩萨是否真为作者）、翻译问题（真谛译本与玄奘译本的差异）、以及思想来源问题（印度如来藏思想与中国本土思想的融合）。杨仁山认为，《起信论》虽然可能不是马鸣菩萨的真作，但其思想价值不可否认，是中国佛教"心性论"的重要经典。这一"批判性继承"的态度，对后来近代佛学研究有决定性影响。

## 白话解读

### 核心内容

《大乘起信论阐微》是近代佛教复兴先驱杨仁山对《大乘起信论》的研究。杨仁山用现代学术方法来研究这部古论。

杨仁山的独特贡献在于：他既尊重传统，又敢于质疑。他考证了《起信论》的作者——传统说是马鸣菩萨写的，但杨仁山发现有些疑点。不过，他说：即使作者有疑问，这部论的思想价值是不可否认的。

杨仁山还从"比较宗教学"的角度，把《起信论》与印度的《楞伽经》《密严经》等对比，发现它们都讲"如来藏"思想。这说明《起信论》不是孤立的，而是印度佛教思想发展的一部分。

### 现代启示

1. **批判性继承**：杨仁山既尊重传统，又敢于质疑。这种"批判性继承"的态度，对今天的学习和研究都有启示：不盲从权威，也不全盘否定。

2. **学术方法**：杨仁山引入现代学术方法（文献学、比较宗教学）来研究佛学，启示我们：传统学问也可以用现代方法来研究，二者并不矛盾。

3. **文化复兴**：杨仁山在日本发现中国已佚的佛教典籍，立志复兴中国佛教。这启示我们：文化的传承需要主动的努力，不能被动等待。

### 阅读建议

- 杨仁山的疏解具有现代学术性，适合有一定佛学基础且对近代学术感兴趣的读者
- 可配合《杨仁山文集》中的其他文章，如《佛教初学课本》《十宗略说》等一同阅读
- 注意其"批判性继承"的学术态度，理解近代佛学研究的转型
