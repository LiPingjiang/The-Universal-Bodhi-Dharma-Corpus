# 六度集經學術解讀

## 一、文獻學信息

《六度集經》（梵名擬為 *Pāramitā-saṃgrāha-sūtra* 或 *Pāramitā-saṃgīti-sūtra*），凡八卷，為三國吳黃武年間（222-229年）至建興年間（252-253年）由康居國沙門康僧會於交趾（今越南）至建業（今南京）間譯出。康僧會為中國佛教史上早期最重要之譯經僧之一，其先祖康居人，世居天竺，故兼通梵漢。除《六度集經》外，尚譯有《吳品般若經》（《道行般若經》異譯）、《舊雜譬喻經》等，並於建業建立建初寺，為江南佛教之開拓者。

本經收錄九十一則本生故事（Jātaka），按「六波羅蜜」（ṣaṭ-pāramitā）分類編排：布施（dāna）、持戒（śīla）、忍辱（kṣānti）、精進（vīrya）、禪定（dhyāna）、明度（即般若，prajñā）。此種以「六度」為綱組織本生故事的編纂方式，在漢譯佛典中為首創，標誌著「菩薩道」敘事在中國的系統化呈現。收於《大正藏》第三冊第152號。

在版本源流方面，本經無完整梵本存世，但部分故事可在巴利文《本生經》（Jātaka，共547則）、梵文《本生鬘》（Jātakamālā，聖勇著，34則）、《菩薩本生鬘論》（Bodhisattvāvadānakalpalatā，月稱著，108則）及《天譬喻》（Divyāvadāna）中找到平行文本。藏譯本中亦有部分對應譯本。近代學者如呂澂、印順等，對此經之漢譯語言及故事源流有過專題研究。

## 二、在佛教文學與本緣故事傳統中的地位

《六度集經》在中國佛教文學史上佔有開創性地位。它是漢譯佛典中第一部以「菩薩道」為核心框架、系統化編排本生故事的專集。在此之前，漢譯本生故事多散見於律藏、阿含經及單行的譬喻集（如《舊雜譬喻經》），缺乏明確的修行階梯結構。《六度集經》以「六度」為綱，將紛繁的本生故事納入一個清晰的「菩薩修行次第」體系，這既是對印度大乘佛教「菩薩行」理論的敘事化表達，也是對中國讀者理解「成佛之道」的系統化引導。

在本緣故事傳統中，本經屬於「本生」（Jātaka）體裁，即佛陀述說其前生為菩薩時的種種修行經歷。與巴利文《本生經》相比，《六度集經》的故事更強調「菩薩的積極利他」——布施、持戒、忍辱等行為，不是為了個人解脫，而是為了積累功德、成就佛道、普度眾生。這種「大乘化」的敘事傾向，與康僧會本人之大乘信仰背景直接相關。

本經對後世文學的影響深遠。經中「尸毗王代鴿」、「薩埵太子捨身飼虎」、「睒子被射」等故事，成為中國佛教藝術（敦煌壁畫、雲岡石窟、龍門石窟）中最常見的「本生圖」題材；其敘事結構也影響了後來的《賢愚經》、《菩薩本生鬘論》等作品。

## 三、與梵文/巴利文原本的對應關係

《六度集經》的九十一則故事，絕大多數可在巴利文及梵文傳統中找到淵源：

- **須大拏太子**：對應巴利文《本生經》第547號（Vessantara-jātaka），為《本生經》最長的一則故事，也是整個南傳佛教傳統中最著名的本生譚。梵文《天譬喻》及藏譯本中亦有對應版本。康僧會的漢譯在情節上與巴利文版本大體一致，但對話與細節有較大出入，顯示譯者所據可能為說一切有部或大乘傳承之梵本。
- **尸毗王代鴿**：對應巴利文無直接等同本生，但可在《大智度論》卷四及《菩薩本生鬘論》中找到平行文本。這則故事在敦煌壁畫（如第254窟、第275窟）中被反覆描繪，是北朝佛教藝術的標誌性母題。
- **薩埵太子捨身飼虎**：對應巴利文《本生經》第540號（Suvaṇṇasāma-jātaka，但情節有差異），梵文《菩薩投身飼虎起塔因緣》（收於《三寶感應要略錄》）及《金光明經》中均有類似敘事。敦煌莫高窟第254窟、第428窟之「薩埵太子本生圖」，即據此經繪製。
- **睒道士（睒子）**：對應巴利文《本生經》第540號（Suvaṇṇasāma-jātaka，部分學者認為即睒子故事），在漢譯《大方便佛報恩經》、《雜寶藏經》中均有平行文本。北朝石塔（如北齊響堂山石窟）及敦煌壁畫中常見「睒子本生」圖像。

在語言學上，康僧會的譯文屬於三國時期佛經翻譯的「古譯」階段，其語言較之後來的鳩摩羅什「舊譯」更為古奧，部分詞彙保留了早期漢譯佛經的「格義」特徵（如以「道」譯「菩提」，以「無為」譯「涅槃」），對研究中古漢語早期佛教詞彙史具有重要價值。

## 四、主要內容與思想

《六度集經》的核心思想，是「菩薩道」的實踐論。經中每一則故事，都是對某一「波羅蜜」的極致詮釋：

1. **布施波羅蜜**：以「須大拏太子」為極致——不僅布施國土、財寶、車馬，甚至布施妻子、兒女，乃至自身血肉。這種「無所不捨」的布施，不是「殘忍」，而是對「我執」的最徹底打破。經中反覆強調：「菩薩視一切眾生，如父如母，如兄如弟」，故布施不是「施與」，而是「還債」——還報眾生累世之恩。
2. **持戒波羅蜜**：以「長壽王」等故事為例，展現即使面臨死亡威脅，也不殺生、不妄語的堅定持戒。持戒的深層意義，在於「心戒」——不僅行為不犯，心中不起惡念。
3. **忍辱波羅蜜**：以「睒子」被射為典範。睒子被毒箭誤射，瀕死之際，不生嗔恨，唯念父母無人供養。這種「外境不動於心」的忍辱，是「心忍」而非「力忍」。
4. **精進波羅蜜**：展現菩薩在無數劫中，於地獄、餓鬼、畜生道中，不捨菩提心，不廢菩薩行。
5. **禪定波羅蜜**：以動物本生（如鹿王、螺王）的故事，說明「定」不在山林，而在心不動亂。
6. **般若波羅蜜**：以「明度」為總綱，說明前五度若無智慧引導，如盲人行路；唯有般若，能令諸度趨向無上菩提。

值得注意的是，本經雖然強調「六度」，但其終極指向是「慈悲」與「利他」。每一度都不是孤立的「修行項目」，而是「利益眾生」的具體方式。這種思想，與後來《大智度論》中「五度如盲，般若為導」的說法遙相呼應。

## 五、學術研究價值

《六度集經》的學術價值體現在以下幾方面：

在文獻學上，它是三國時期佛經翻譯的標準文本之一，其語言、詞彙、句法，是研究中古漢語「佛經漢語」（Buddhist Chinese）形成期的關鍵語料。經中大量出現的「昔者菩薩」句式，成為後來本生故事翻譯的標準開場白。

在文學史上，它開創了「六度—本生」的敘事範式，直接影響了《賢愚經》、《經律異相》、《法苑珠林》等類書的編纂方式。經中「捨身飼虎」、「割肉代鴿」等極端布施敘事，也成為中國佛教文學中最具震撼力的「崇高美學」典範。

在藝術史上，本經的諸多故事是敦煌、雲岡、龍門、麥積山等石窟中本生圖的主要題材來源。對這些圖像的準確解讀，離不開對《六度集經》文本的精細研究。

在宗教學上，它是研究三國時期中國菩薩信仰、大乘佛教早期傳播、以及「菩薩道」實踐論中國化的重要文本。康僧會以「六度」為框架，將印度佛教的修行體系呈現給中國讀者，這本身就是佛教「中國化」過程的重要一環。
