# 大乘起信論解

## 學術解讀

### 一、文獻學信息

《大乘起信論解》是唐代良賁（約8世紀）所撰。良賁，俗姓不詳，爲唐代唯識宗高僧，參與玄奘、義淨譯場，精通《瑜伽師地論》《成唯識論》等。此疏以"解"爲名，取"解釋文義，疏通疑難"之意，是唐代《大乘起信論》註疏中較爲獨特的一部，因爲它嘗試以唯識宗的立場來解釋如來藏系的《起信論》。

良賁是唯識宗的重要傳人，曾參與《成唯識論》的編纂和講解。唯識宗以"萬法唯識"爲核心，強調阿賴耶識的緣起作用，與如來藏系的"心性本淨"思想有所不同。良賁此疏正是在這一學術背景下產生，試圖調和唯識與如來藏兩種思想體系。

### 二、歷史背景

唐代是佛教宗派並立的鼎盛期，唯識宗與華嚴宗、天台宗之間時有思想交鋒。玄奘從印度帶回的唯識學強調"阿賴耶識"的染淨二分，而《起信論》則強調"心性本淨"的如來藏思想。良賁此疏試圖以"會通"的方式解決這一矛盾：阿賴耶識的"清淨分"即是如來藏，"染污分"即是生滅門。這一解釋雖然受到後來華嚴宗學者的批評，但體現了唐代佛教內部思想交流的活躍。

### 三、思想史定位

良賁此疏的特色在於以唯識宗的"八識"理論來解釋《起信論》的"一心"。他認爲，《起信論》的"一心"不是超越八識的另一種心，而是"阿賴耶識"的"清淨分"。"心真如門"即是阿賴耶識的"真如性"，"心生滅門"即是阿賴耶識的"生滅相"。良賁進一步將"無明"解釋爲"第七識"（末那識）的"我執"，將"修行"解釋爲"轉識成智"的過程。這一解釋對後來唯識宗與如來藏思想的對話有重要推動作用。

## 白話解讀

### 核心內容

《大乘起信論解》是唐代唯識宗高僧良賁對《大乘起信論》的解釋。唯識宗認爲"萬法唯識"——一切現象都是心識的變現。良賁大師嘗試用唯識的理論來解釋《起信論》。

良賁大師說：《起信論》的"一心"其實就是唯識宗說的"阿賴耶識"——我們最深層的意識，像一個大倉庫，儲存着所有的"種子"（潛在能力）。"心真如門"就是這個倉庫的"清淨面"，"心生滅門"就是倉庫的"活動面"。

良賁大師還解釋了"無明"：無明就是第七識（末那識）的"我執"——總是把"我"當成一個固定不變的東西。修行的過程，就是"轉識成智"——把普通的意識轉化爲智慧。

### 現代啓示

1. **深層心理**：唯識宗的"八識"理論可以看作是對人類心理的深層分析。阿賴耶識就像"潛意識"，儲存着我們所有的經驗和潛能。

2. **自我認知**："末那識"的"我執"啓示我們：很多痛苦源於對"自我"的執着。認識到"自我"是流動的、變化的，有助於減輕焦慮。

3. **轉化成長**："轉識成智"的思想告訴我們：成長不是消滅舊我、創造新我，而是轉化現有的心理模式。

### 閱讀建議

- 良賁的疏解需要一定的唯識學基礎，建議先讀《成唯識論》或《八識規矩頌》
- 可對比法藏以華嚴宗解《起信論》的方式，理解不同宗派的解釋差異
- 注意唯識與如來藏兩種思想體系的差異和融合
