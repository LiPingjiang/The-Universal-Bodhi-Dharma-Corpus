# 五蕴论广释 · 學術解讀

## 一、文獻定位

《五蕴论广释》为唐代唯识学大师窥基（632–682）所撰，是世亲《大乘五蕴论》的注释书。《大乘五蕴论》是唯识宗阐述五蕴学说的核心论书，将一切有为法归纳为五蕴（色、受、想、行、识），是理解佛教"无我"教义的基础文本。窥基的《广释》对《五蕴论》进行了逐句详细的解释，并补充了大量的义理发挥，是研究唐代唯识宗五蕴思想、佛教心理学及"人无我"理论的关键文本。

## 二、版本與傳承

- **作者**：窥基（注释者）；世亲（原论作者，约320–400）

## 三、思想史分析

### 3.1 歷史背景

五蕴（pañca-skandha）是佛教最基本的教义之一，将一切有为法（有生灭变化的现象）归纳为五种要素：色（物质）、受（感受）、想（概念）、行（意志）、识（意识）。这一分类体系最早见于早期佛教经典（如《阿含经》），后来在大乘佛教中得到进一步的发展。

世亲的《大乘五蕴论》是在大乘唯识学的框架下重新阐述五蕴学说，将五蕴与唯识学的八识、百法等概念联系起来。玄奘在印度游学期间学习了世亲的五蕴论，回国后将其翻译。窥基作为玄奘的嫡传弟子，撰写了《五蕴论广释》，对五蕴的每一个方面进行了详细的解释。

### 3.2 核心論題

《五蕴论广释》围绕五蕴展开，窥基在注释中进行了大量的义理发挥：

1. **色蕴**：详细解释色蕴的范围，包括五根（眼耳鼻舌身）、五尘（色声香味触）以及法处所摄色。窥基特别分析了色蕴与心识的关系，指出色蕴虽然是物质性的，但其存在和认知离不开心识。
2. **受蕴**：解释受蕴的三种类型：苦受（痛苦的感受）、乐受（快乐的感受）、不苦不乐受（中性的感受）。窥基进一步分析了受蕴与贪爱的关系，指出受是贪爱生起的直接原因。
3. **想蕴**：解释想蕴的功能，即对事物进行概念化、标签化的认知活动。窥基分析了想蕴与语言、记忆的关系，指出想蕴是形成"自我"概念的重要环节。
4. **行蕴**：解释行蕴的范围，包括除受、想之外的一切心所法（心理活动），如思、触、作意、贪、嗔、信、精进等。窥基详细分析了行蕴与业力的关系，指出行蕴是造业的主体。
5. **识蕴**：解释识蕴的八种识（眼耳鼻舌身意末那阿赖耶），详细分析每一识的体性、功能、所缘和相互关系。窥基将五蕴与唯识学的八识理论贯通，指出识蕴是五蕴中最根本的，因为一切色、受、想、行都离不开识。

### 3.3 學術評價

- **吕澂**评价《五蕴论广释》为"理解佛教无我教义的最佳注释书"，认为五蕴分析是"佛教心理学的基石"。
- **现代心理学研究者**关注五蕴与现代心理学的对话，认为五蕴的分类与西方心理学的感知、情绪、认知、动机、意识等理论有可比性。
- **当代佛教学者**重视五蕴在修行实践中的价值，认为五蕴观（观察五蕴的无常、苦、无我）是佛教最基本的修行方法之一。
- 日本学者如宇井伯寿等，将《五蕴论广释》视为研究世亲五蕴思想中国化的重要资料。

## 四、術語對照

| 漢譯術語 | 梵文/巴利原文 | 現代語義 |
|---------|-------------|---------|
| 五蘊 | pañca-skandha | 构成生命和世界的五个要素：色、受、想、行、识 |
| 色蘊 | rūpa-skandha | 物质现象，包括身体、感官器官和感官对象 |
| 受蘊 | vedanā-skandha | 感受，包括苦、乐、不苦不乐三种 |
| 想蘊 | saṃjñā-skandha | 概念化认知，对事物进行标签和分类 |
| 行蘊 | saṃskāra-skandha | 意志和心理活动，包括除受想外的一切心所法 |
| 識蘊 | vijñāna-skandha | 意识，包括八识 |
| 有為法 | saṃskṛta-dharma | 因缘和合而生、有生灭变化的现象 |
| 人無我 | pudgala-nairātmya | 人没有独立自存的"自我"实体，只是五蕴的暂时聚合 |
