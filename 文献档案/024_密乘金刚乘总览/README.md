## 密乘源流骨架

密乘（Vajrayāna，"金刚乘"，又称 Mantrayāna "真言乘"、Tantrayāna "怛特罗乘"）是印度佛教发展的第三阶段，在大乘佛教（Mahāyāna）的基础上，融合了印度密教（Tantra）的修行方法，形成了以灌顶（abhiṣeka）、本尊（iṣṭadevatā）、咒语（mantra）、手印（mudrā）、坛城（maṇḍala）为核心的修行体系：

```
        密乘（Vajrayāna）发展脉络
        ├── 事续（Kriyā-tantra）— 以外在仪式为主，如《大日经》
        │       ├── 《苏悉地经》（Susiddhikara-sūtra）
        │       └── 《金刚顶经》（Vajraśekhara-sūtra）部分
        ├── 行续（Caryā-tantra）— 外在仪式与内在修行并重
        │       └── 《大日经》（Mahāvairocana-sūtra）
        ├── 瑜伽续（Yoga-tantra）— 以内在修行为主
        │       └── 《金刚顶经》（Vajraśekhara-sūtra）
        └── 无上瑜伽续（Anuttara-yoga-tantra）— 最高密续
                ├── 父续（Pitṛ-tantra）— 以方便（upāya）为主
                │       ├── 《密集金刚续》（Guhyasamāja-tantra）
                │       └── 《喜金刚续》（Hevajra-tantra）
                ├── 母续（Mātṛ-tantra）— 以智慧（prajñā）为主
                │       ├── 《胜乐轮续》（Cakrasaṃvara-tantra）
                │       └── 《时轮续》（Kālacakra-tantra）
                └── 不二续（Advaya-tantra）
                        └── 《时轮续》（部分分类）
```

密乘的核心特色在于：以"果位道"（phala-yāna）为修行方法，即将成佛的果位（buddhahood）作为修行的起点和方法，通过观想本尊（iṣṭadevatā，"所选择的神明"）、持诵咒语（mantra）、结手印（mudrā）、观想坛城（maṇḍala）等方法，在此生此世快速成就佛果。这与显教（Sūtrayāna）的"因位道"（hetu-yāna）——通过积累福德和智慧，经历三大阿僧祇劫才能成佛——形成鲜明对比。

密乘在印度约公元7–12世纪达到鼎盛，随后随着伊斯兰教的入侵而在印度本土衰落，但通过西藏（藏传佛教）、中国（汉传密教）、日本（东密、台密）等地的传播，至今仍是重要的佛教传统。

## 关键概念速查

| 概念 | 梵语 | 简释 |
|------|------|------|
| 金刚乘 | Vajrayāna | 密乘，以金刚（坚不可摧）为喻的修行乘 |
| 灌顶 | abhiṣeka | 密乘入门仪式，传授修法权限 |
| 本尊 | iṣṭadevatā | 修行者所选择观想的佛菩萨 |
| 咒语 | mantra | 具有神圣力量的音节或语句 |
| 手印 | mudrā | 具有象征意义的手势 |
| 坛城 | maṇḍala | 本尊及其眷属的宫殿图，修行的象征宇宙 |
| 续 | tantra | 密乘经典，字面义"连续""脉络" |
| 无上瑜伽 | Anuttara-yoga | 最高级别的密续，包含生起次第与圆满次第 |
| 生起次第 | utpatti-krama | 观想本尊的修行阶段 |
| 圆满次第 | sampanna-krama | 修炼气脉明点的修行阶段 |
