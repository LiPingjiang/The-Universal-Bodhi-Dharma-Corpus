# 《如何经》原文

> 收录编号：MN 103（中部·第103经 Kinti Sutta）
> 原始语言：巴利语 · 篇幅类型：中经
> 出处：SuttaCentral（suttacentral.net/mn103）
> 主题：比丘间如法处理诤事

---

## 一、巴利原文（拉丁转写，节录核心段落）

Evaṃ me sutaṃ — ekaṃ samayaṃ bhagavā kusinārāyaṃ viharati baliharaṇe vanasaṇḍe.

Tatra kho bhagavā bhikkhū āmantesi: "Bhikkhavo" ti. "Bhadante" ti te bhikkhū bhagavato paccassosuṃ.

Bhagavā etadavoca:

"Kinti vo, bhikkhave, mayi paṭibhāti — kimatthiyaṃ samaṇo gotamo dhammaṃ deseti, kiṃ pāripūriyaṃ samaṇassa gotamassa dhammadesanā" ti?

"Bhagavaṃmūlakā no, bhante, dhammā, bhagavaṃnettikā bhagavaṃpaṭisaraṇā. Sādhu vata, bhante, bhagavantaṃyeva paṭibhātu etassa bhāsitassa attho. Bhagavato sutvā bhikkhū dhāressantī" ti.

"Tena hi, bhikkhave, suṇātha, sādhukaṃ manasi karotha, bhāsissāmī" ti. "Evaṃ, bhante" ti kho te bhikkhū bhagavato paccassosuṃ.

Bhagavā etadavoca:

"Na kho me, bhikkhave, cīvarahetu dhammo desito, na piṇḍapātahetu, na senāsanahetu, na itibhavābhavahetu dhammo desito.

Api ca kho, bhikkhave, dukkhanirodhagāminī paṭipadā me desitā. Tasmātiha, bhikkhave, idha bhikkhu dhammaṃ deseti. Tassa ce, bhikkhave, dhammadesanāya bhikkhu bhikkhussa adhikaraṇaṃ kareyya, so bhikkhu evaṃ paccavekkheyya:

'Yena kho ahaṃ saha sabrahmacārī dhammaṃ deseti, tena kho me saha sabrahmacārinā adhikaraṇaṃ vūpasantaṃ. Api ca me eso sappaṭigho dhammo, yo ca me sappaṭigho dhammo taṃ pajahissāmī' ti.

Yo ca kho, bhikkhave, bhikkhu bhikkhussa adhikaraṇaṃ karoti so ca evaṃ paccavekkheyya:

'Ayaṃ kho āyasmā bahussuto āgatāgamo dhammadharaṃ vinayadharaṃ mātikādharaṃ; nayidaṃ āyasmā adhammaṃ byākareyya, nayidaṃ āyasmā avinayaṃ byākareyya. Yathā kho ayaṃ āyasmā byākaroti, tathāhaṃ na khamamī. Api ca khvāhaṃ ayaṃ āyasmantaṃ sahadhammikaṃ vuccamāno, khantiyā ce upasampajja vihareyyaṃ' ti.

Kathaṃ ca, bhikkhave, adhikaraṇaṃ vūpasammati?

Idha, bhikkhave, bhikkhu bhikkhuṃ vadati: 'Idaṃ kho, āvuso, atthaṃ sandhāya bhagavatā bhāsitaṃ; ayaṃ kho āvuso attho, imassāyaṃ attho' ti.

Itaro bhikkhu evaṃ vadeyya: 'Ahampi kho, āvuso, idameva atthaṃ sandhāya bhagavatā bhāsitaṃ vadāmi; ayaṃ kho, āvuso, attho' ti.

Tehi, bhikkhave, bhikkhūhi samannesamānehi evaṃ sallakkhitabbaṃ: 'Kiñcāpi kho idaṃ āyasmantehi nānāvohārehi byākataṃ, api catthato ekameva hoti, byañjanato nānan' ti — yadi evaṃ sallakkhenti:

'Atthatova ekameva hoti, byañjanato nānā' ti, 'byañjanato sāmaññaṃ kātuṃ na sakkā' ti — iti viditvā atthatoeva sāmaññaṃ katvā muñcitabbaṃ.

Api ca, bhikkhave, idha bhikkhu bhikkhuṃ vadati: 'Idaṃ kho, āvuso, atthaṃ sandhāya bhagavatā bhāsitaṃ' — itaro bhikkhu evaṃ vadeyya: 'Nayidaṃ, āvuso, atthaṃ sandhāya bhagavatā bhāsitaṃ, aññaṃ kho idaṃ atthaṃ sandhāya bhagavatā bhāsitan' ti.

Tehi, bhikkhave, bhikkhūhi samannesamānehi evaṃ sallakkhitabbaṃ: 'Kiñcāpi kho idaṃ āyasmantehi nānāvohārehi byākataṃ, atthatoeva ca nānaṃ byañjanatoeva ca nānaṃ — api ca bahu-bhikkhūnaṃ santike etaṃ adhikaraṇaṃ vūpasamessatī' ti.

'Api ca, bhikkhave, so adhammavādī bhikkhu bahukānaṃ bhikkhūnaṃ santike adhikaraṇaṃ netabbaṃ; yebhuyyasikāya vā nivesitabbaṃ' ti."

## 二、汉译对照

### 甲译（白话意译）

我是这样听说的[^1]——有一次，世尊[^2]住在拘尸那罗的波利哈罗那林中。

世尊对出家修行人们[^3]说："修行人们。""是的，老师。"他们恭敬回应。

世尊说：

"修行人们，你们是如何看待我的？——沙门乔达摩[^4]说法的目的是什么？沙门乔达摩教导的圆满指向什么？"

"老师，我们的教法以世尊为根本、以世尊为引导、以世尊为依归。老师，请世尊为我们解说这段话的含义，修行人们听后会牢记。"

"那么，修行人们，请你们仔细听好。

修行人们，我说法不是为了衣物、不是为了饭食、不是为了住处，也不是为了这样那样的生存。我所教导的是通往苦的止息的修行道路[^5]。

因此，修行人们，如果有一位修行人在说法时，另一位修行人对他提起了争端，那位修行人应当这样反思：

'这位同修在为我说法，我与这位同修之间的争端已经平息了。况且我这个对抗的心态，我应当放下它。'

而那位提起争端的修行人也应当这样反思：

'这位尊者是多闻者、通达经典者、持法者、持律者、持论母者。这位尊者不会作不如法的解说，不会作不如律的解说。既然这位尊者这样解说——虽然我还不能接受——但我应当对这位尊者所说的如法之语，以忍耐来面对。'

修行人们，争端如何平息呢？

在此，修行人们，一位修行人对另一位说：'朋友，世尊是针对这个义理而说的；这个义理就是这个意思。'

另一位修行人说：'朋友，我也认为世尊是针对同样的义理而说的；这个意思就是那个意思。'

修行人们，这两位修行人在探讨时应当这样观察：'虽然两位尊者的措辞不同，但在义理上其实是一样的，只是文句不同。'——如果他们这样认定：

'义理相同，文句不同，文句上无法统一'——那么了知这一点后，就在义理上取得一致，然后放下争论。

但是，修行人们，如果一位修行人对另一位说：'朋友，世尊是针对这个义理而说的'——另一位说：'朋友，世尊不是针对这个义理而说的，而是针对另一个义理而说的'——

修行人们，这两位修行人在探讨时应当这样观察：'虽然两位尊者的措辞不同，义理也不同——但是在更多修行人面前这个争端可以得到平息。'

'修行人们，那位说法不正确的修行人，应当将争端带到更多修行人面前；或者按多数决来裁定[^6]。'"

[^1]: "我是这样听说的"：巴利语 evaṃ me sutaṃ，经文开篇定式。
[^2]: 世尊：巴利语 bhagavā，对佛陀的尊称。
[^3]: 出家修行人：巴利语 bhikkhu（比丘）。
[^4]: 沙门乔达摩：巴利语 samaṇa Gotama，外人对佛陀的称呼，乔达摩是佛陀俗家姓氏。
[^5]: 通往苦的止息的修行道路：巴利语 dukkhanirodhagāminī paṭipadā，即八正道。
[^6]: 多数决：巴利语 yebhuyyasikā，僧团处理争端的七灭诤法之一，按多数人的意见裁定。

### 乙译（严谨译）

如是我闻——一时，世尊住拘尸那罗波利哈罗那林中。

尔时世尊告诸比丘："诸比丘。""大德。"诸比丘应诺。

世尊说言：

"诸比丘，汝等于我云何想？沙门瞿昙说法何所为？沙门瞿昙法之教授何所圆满？"

"世尊，我等之法以世尊为根本、以世尊为导、以世尊为所归。善哉，世尊，唯愿为说此义。诸比丘闻世尊已当受持。"

"诸比丘，谛听，善思念之，当说。""唯然，世尊。"

世尊说言：

"诸比丘，我非为衣故说法，非为食故说法，非为住处故说法，非为有无之生存故说法。

然诸比丘，我所教者乃苦灭道迹。是故诸比丘，若有比丘说法时，另一比丘于彼起诤事，其比丘应如是思惟：

'彼同梵行者为我说法，我与彼同梵行者之诤事已息。况我此有对法，我当舍彼有对法。'

彼起诤之比丘亦应如是思惟：

'此尊者多闻、通达阿含、持法、持律、持论母。此尊者非作非法之说，非作非律之说。彼尊者如是说——虽我未能认可——然我于此尊者如法之语，应以忍辱而住。'

诸比丘，云何诤事得息？

此中诸比丘，一比丘语一比丘言：'友，世尊为此义故说此；此义即如是。'

彼比丘言：'友，我亦谓世尊为此同义故说；此义即如是。'

诸比丘，彼二比丘共审察时应如是观：'虽诸尊者以异语说，然义实一，唯文句异。'若如是知——

'义同文异'，'文不可强同'——了知此已，于义取一致，放下争论。

诸比丘，若一比丘言：'友，世尊为此义说'——彼比丘言：'友，世尊非为此义说，乃为别义说'——

诸比丘，彼二比丘共审察时应如是观：'虽诸尊者异语异义——然于多数比丘前此诤可息。'

'诸比丘，说非法之比丘应将诤事提于多数比丘前；或以多数决裁之。'"

---

*说明：本经（MN 103 Kinti Sutta）是佛陀关于如何处理僧团内部教义争端的重要指导。佛陀首先强调说法的目的是苦灭道迹，然后教导争论双方应各自反思，最终以义理一致或多数决来解决分歧。原文据巴利圣典协会（PTS）校订本与 SuttaCentral，汉译参酌庄春江、元亨寺等通行译本整理。*
