# 《舍利弗相应偈》原文

> 收录编号：SN 28.1（相应部·舍利弗相应·第一经 Paṭhamajhāna-sutta）
> 原始语言：巴利语 · 体裁：散文与偈颂混合（geyya）
> 主题：舍利弗自述入初禅的体验——不以禅定为我所有
> 出处：SuttaCentral · 巴利圣典协会（PTS）校订本

---

## 一、巴利原文（拉丁转写）

Ekaṃ samayaṃ āyasmā Sāriputto Sāvatthiyaṃ viharati Jetavane Anāthapiṇḍikassa ārāme.

Atha kho āyasmā Sāriputto pubbaṇhasamayaṃ nivāsetvā pattacīvaramādāya Sāvatthiṃ piṇḍāya pāvisi. Atha kho āyasmā Sāriputto: "Atippago kho tāva Sāvatthiyaṃ piṇḍāya carituṃ. Yannūnāhaṃ yena aññatitthiyānaṃ paribbājakānaṃ ārāmo tenupasaṅkameyyan"ti.

Atha kho āyasmā Sāriputto yena aññatitthiyānaṃ paribbājakānaṃ ārāmo tenupasaṅkami.

Tatra kho āyasmā Sāriputto tehi aññatitthiyehi paribbājakehi saddhiṃ sammodi. Sammodanīyaṃ kathaṃ sāraṇīyaṃ vītisāretvā ekamantaṃ nisīdi.

Tena kho pana samayena tesaṃ aññatitthiyānaṃ paribbājakānaṃ sannisinnānaṃ sannipatitānaṃ ayamantarākathā udapādi: "Yo kho, āvuso, paṭhamaṃ jhānaṃ samāpajjati, tassa evaṃ hoti: 'Paṭhamaṃ jhānaṃ samāpanno'smī'"ti.

Atha kho āyasmā Sāriputto te aññatitthiye paribbājake etadavoca:

"Na kho, āvuso, paṭhamaṃ jhānaṃ samāpannassa evaṃ hoti: 'Paṭhamaṃ jhānaṃ samāpanno'smī'ti. Api ca, āvuso, paṭhamaṃ jhānaṃ samāpannassa paṭhamajhānasahagataṃ ñāṇaṃ hoti.

Seyyathāpi, āvuso, gāvī khīraṃ denti — na tassā evaṃ hoti: 'Khīraṃ dentī'smī'ti — api ca khīradāyikāya gāviyā khīradānasahagataṃ sukhaṃ hoti.

Evameva kho, āvuso, paṭhamaṃ jhānaṃ samāpannassa na evaṃ hoti: 'Paṭhamaṃ jhānaṃ samāpanno'smī'ti. Api ca paṭhamaṃ jhānaṃ samāpannassa paṭhamajhānasahagataṃ ñāṇaṃ hotī"ti.

## 二、汉译对照

### 甲译（白话）

有一次，舍利弗尊者[^1]住在舍卫城祇树给孤独园。

那时舍利弗尊者在上午穿好衣服、持钵进入舍卫城托钵[^2]。然后舍利弗尊者想："现在去舍卫城托钵还太早。不如我先去外道游行者[^3]的园林。"

于是舍利弗尊者前往外道游行者的园林。

在那里舍利弗尊者与那些外道游行者互相问候。寒暄之后坐在一旁。

那时那些外道游行者坐在一起集会时，生起了这样的讨论："朋友们，进入第一禅定[^4]的人会这样想：'我进入了第一禅定。'"

于是舍利弗尊者对那些外道游行者说：

"朋友们，进入第一禅定的人不会这样想：'我进入了第一禅定。'然而，朋友们，进入第一禅定的人会有与第一禅定相伴的智慧[^5]。

朋友们，就像一头母牛在产奶——她不会这样想：'我在产奶'——然而产奶的母牛会有与产奶相伴的快乐。

同样地，朋友们，进入第一禅定的人不会这样想：'我进入了第一禅定。'然而进入第一禅定的人会有与第一禅定相伴的智慧。"

---

### 乙译（严谨）

一时，尊者舍利弗住舍卫城祇树给孤独园。

尔时尊者舍利弗于晨朝着衣持钵入舍卫城乞食。时尊者舍利弗念："于舍卫城乞食尚早。我宁往外道游行者之园。"

尊者舍利弗即往外道游行者之园。

于彼处尊者舍利弗与彼等外道游行者共相问讯。问讯已坐于一面。

尔时彼等外道游行者集坐时生此论议："诸友，入初禅者如是念：'我入初禅。'"

时尊者舍利弗语彼等外道游行者言：

"诸友，入初禅者非如是念：'我入初禅。'然，诸友，入初禅者有与初禅俱之智。

诸友，譬如牝牛出乳——彼非如是念：'我出乳'——然出乳之牝牛有与出乳俱之乐。

如是，诸友，入初禅者非如是念：'我入初禅。'然入初禅者有与初禅俱之智。"

## 脚注

[^1]: 舍利弗尊者（Āyasmā Sāriputto）：佛陀智慧第一弟子。
[^2]: 托钵（piṇḍāya）：出家人持钵乞食。
[^3]: 外道游行者（aññatitthiya paribbājaka）：非佛教的其他宗教修行者。
[^4]: 第一禅定（paṭhamaṃ jhānaṃ）：四色界禅之第一，具有寻、伺、喜、乐、一境性五禅支。
[^5]: 与第一禅定相伴的智慧（paṭhamajhānasahagataṃ ñāṇaṃ）：在禅定中自然生起的了知，而非自我意识的反思。

---

*说明：本经展示了舍利弗对禅定体验的深刻理解——真正的禅定中没有"我在禅定中"的自我意识，正如母牛产奶时不会想"我在产奶"。这体现了佛教禅修中"无我"的核心原则。原文据巴利圣典协会（PTS）校订本与 SuttaCentral（suttacentral.net/sn28.1），汉译参酌菩提比丘英译及庄春江中译整理。*
