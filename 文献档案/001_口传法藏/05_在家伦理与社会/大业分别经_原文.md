# 《大业分别经》原文

> 收录编号：MN 136（中部）
> 原始语言：巴利语 · 篇幅类型：长经·节录
> 出处：SuttaCentral（suttacentral.net/mn136）
> 主题：业果复杂性——行善受苦、行恶享乐的四种情况

---

## 一、巴利原文（拉丁转写，节录核心段落）

Ekaṃ samayaṃ bhagavā rājagahe viharati veḷuvane kalandakanivāpe. Tena kho pana samayena āyasmā samiddhi bhagavato avidūre nisinno hoti.

Atha kho potaliputto paribbājako yenāyasmā samiddhi tenupasaṅkami... "Idha me, āvuso samiddhi, purisapuggalo pāṇātipātī hoti... so kāyassa bhedā paraṃ maraṇā sugatiṃ saggaṃ lokaṃ upapajjati. Idha panekacco purisapuggalo pāṇātipātā paṭivirato hoti... so kāyassa bhedā paraṃ maraṇā apāyaṃ duggatiṃ vinipātaṃ nirayaṃ upapajjatī"ti.

Bhagavā etadavoca: "Cattārome, bhikkhave, puggalā santo saṃvijjamānā lokasmiṃ. Katame cattāro?

Idha, bhikkhave, ekacco puggalo pāṇātipātī hoti, adinnādāyī hoti, kāmesumicchācārī hoti, musāvādī hoti... so kāyassa bhedā paraṃ maraṇā apāyaṃ duggatiṃ vinipātaṃ nirayaṃ upapajjati.

Idha pana, bhikkhave, ekacco puggalo pāṇātipātī hoti... so kāyassa bhedā paraṃ maraṇā sugatiṃ saggaṃ lokaṃ upapajjati.

Idha pana, bhikkhave, ekacco puggalo pāṇātipātā paṭivirato hoti... so kāyassa bhedā paraṃ maraṇā sugatiṃ saggaṃ lokaṃ upapajjati.

Idha pana, bhikkhave, ekacco puggalo pāṇātipātā paṭivirato hoti... so kāyassa bhedā paraṃ maraṇā apāyaṃ duggatiṃ vinipātaṃ nirayaṃ upapajjati."

"Tatra, bhikkhave, yvāyaṃ puggalo pāṇātipātī hoti... so kāyassa bhedā paraṃ maraṇā sugatiṃ saggaṃ lokaṃ upapajjati — so diṭṭheva dhamme vā aññasmiṃ vā attabhāve vā sucaritassa vipākaṃ paṭisaṃvedeti sukhavedanīyaṃ."

## 二、汉译对照

> **甲译**：白话意译，通顺自然，不拘泥于原文句式，适合直接阅读。
> **乙译**：严谨译，逐句对应巴利原文，术语精确，供对照研究。

### 甲译（白话）

有一次，世尊住在王舍城竹林迦兰陀迦园。那时，尊者三弥提坐在世尊不远处。

游方者¹哺多利子来到尊者三弥提面前……说道："朋友三弥提，这里有人杀生……他身坏命终后²却投生到善趣³、天界。又有人远离杀生……他身坏命终后却投生到恶趣⁴、地狱。"

世尊说道："比丘们，世间有四种人存在。是哪四种呢？

比丘们，这里有人杀生、偷盗、邪淫⁵、妄语……他身坏命终后投生到恶趣、地狱。

比丘们，又有人杀生……他身坏命终后却投生到善趣、天界。

比丘们，又有人远离杀生……他身坏命终后投生到善趣、天界。

比丘们，又有人远离杀生……他身坏命终后却投生到恶趣、地狱。"

"比丘们，其中那个杀生的人……身坏命终后却投生到善趣天界——他是在今生或其他某一世⁶感受到过去善行的乐果⁷。"

### 乙译（严谨）

一时，世尊住王舍城竹林迦兰陀迦园。尔时，尊者三弥提坐于世尊不远处。

时，游方者哺多利子诣尊者三弥提所……言："友三弥提，此有补特伽罗杀生者……彼身坏命终后生于善趣天界。复有补特伽罗离杀生者……彼身坏命终后生于恶趣、堕处、地狱。"

世尊如是说："诸比丘，世间有四种补特伽罗。何等为四？

诸比丘，此有一补特伽罗，杀生者、不与取者、欲邪行者、妄语者……彼身坏命终后生于恶趣、堕处、地狱。

诸比丘，复有一补特伽罗，杀生者……彼身坏命终后生于善趣天界。

诸比丘，复有一补特伽罗，离杀生者……彼身坏命终后生于善趣天界。

诸比丘，复有一补特伽罗，离杀生者……彼身坏命终后生于恶趣、堕处、地狱。"

"诸比丘，其中彼杀生之补特伽罗……身坏命终后生于善趣天界者——彼于现法或于余生感受善行之异熟乐受。"

## 三、脚注

1. **游方者**（paribbājaka）：非佛教的游行沙门，四处游历辩论的修行者。
2. **身坏命终后**（kāyassa bhedā paraṃ maraṇā）：死后，身体崩坏之后。
3. **善趣**（sugati）：善的去处，包括人间和天界。
4. **恶趣**（apāya/duggati）：恶的去处，包括地狱、饿鬼、畜生。
5. **邪淫**（kāmesumicchācāra）：不正当的性行为，在家人指与他人配偶等发生关系。
6. **其他某一世**（aññasmiṃ attabhāve）：另一个生命形态中，指过去世或中间世。
7. **乐果**（sucaritassa vipāka）：善行的异熟果报，即过去善业成熟所带来的乐受。

---

*说明：本经解释了业果的复杂性——为何有人行恶却暂时享乐（因过去善业成熟），有人行善却暂时受苦（因过去恶业成熟）。佛陀指出不能以片面观察否定因果法则，业的成熟有其时间差与复杂条件。原文据巴利圣典协会（PTS）校订本与 SuttaCentral，汉译参酌庄春江、元亨寺等通行译本整理。*
