# 《戏论经》原文

> 收录编号：SN 42.2（相应部·村长相应）
> 原始语言：巴利语 · 篇幅类型：短经·全录
> 出处：SuttaCentral（suttacentral.net/sn42.2）

---

## 一、巴利原文（拉丁转写，节录核心段落）

Atha kho tālapuṭo naṭagāmaṇi yena bhagavā tenupasaṅkami... ekamantaṃ nisinno kho tālapuṭo naṭagāmaṇi bhagavantaṃ etadavoca:

"Sutaṃ metaṃ, bhante, pubbakānaṃ ācariyapācariyānaṃ naṭānaṃ bhāsamānānaṃ: 'Yo so naṭo raṅgamajjhe samajjamajjhe janassa hāsaṃ upaḍḍhāpeti musāvādena, so kāyassa bhedā paraṃ maraṇā pahāsānaṃ devānaṃ sahabyataṃ upapajjatī'ti. Idha bhagavā kiṃ āhā"ti?

"Alaṃ, gāmaṇi, tiṭṭhatetaṃ. Mā maṃ etaṃ pucchī"ti.

Dutiyampi kho... tatiyampi kho tālapuṭo naṭagāmaṇi bhagavantaṃ etadavoca...

"Na kho tyāhaṃ, gāmaṇi, etaṃ anuññāsiṃ... api ca tyāhaṃ byākarissāmi. Yo so, gāmaṇi, naṭo raṅgamajjhe samajjamajjhe janassa rāgajanīyesu dhammesu rāgaṃ upaḍḍhāpeti, dosajanīyesu dhammesu dosaṃ upaḍḍhāpeti, mohajanīyesu dhammesu mohaṃ upaḍḍhāpeti — so attanā pamatto paraṃ pamajjāpeti. So kāyassa bhedā paraṃ maraṇā 'pahāso' nāma nirayo, tattha upapajjati.

Sace kho panassa evaṃ diṭṭhi hoti: 'Yo so naṭo raṅgamajjhe samajjamajjhe janassa hāsaṃ upaḍḍhāpeti musāvādena, so kāyassa bhedā paraṃ maraṇā pahāsānaṃ devānaṃ sahabyataṃ upapajjatī'ti — sāssa hoti micchādiṭṭhi. Micchādiṭṭhikassa kho panāhaṃ, gāmaṇi, dvinnaṃ gatīnaṃ aññataraṃ gatiṃ vadāmi — nirayaṃ vā tiracchānayoniṃ vā"ti.

Evaṃ vutte, tālapuṭo naṭagāmaṇi parodi...

## 二、汉译对照

> **甲译**：白话意译，通顺自然，不拘泥于原文句式，适合直接阅读。
> **乙译**：严谨译，逐句对应巴利原文，术语精确，供对照研究。

### 甲译（白话）

那时，戏子村长多罗弗来到世尊面前……坐好后对世尊说：

"世尊，我听过去的老师们这样说：'那个戏子在舞台上、在集会中，以虚假表演¹让人们欢笑，他身坏命终后会投生到欢笑天²与诸天同住。'世尊对此怎么说？"

"够了，村长，别问这个。不要问我这个。"

第二次……第三次，戏子村长又问了同样的问题……

"村长，我并没有允许你问……但我还是会回答你。村长，那个戏子在舞台上、在集会中，在能引发贪欲³的事物上激起人们的贪欲，在能引发嗔恨⁴的事物上激起人们的嗔恨，在能引发愚痴⁵的事物上激起人们的愚痴——他自己放逸⁶，又使他人放逸。他身坏命终后，会投生到名为'欢笑'的地狱⁷。

如果他还持有这样的见解：'戏子在舞台上以虚假表演让人欢笑，死后会投生到欢笑天'——那就是邪见⁸。村长，持有邪见的人，我说他只有两种去处之一——地狱或畜生道⁹。"

听到这话，戏子村长多罗弗哭了起来……

### 乙译（严谨）

时，戏子村长多罗弗诣世尊所……一面坐已，白世尊言：

"大德，我闻过去诸师之师——诸戏子如是言：'彼戏子于舞台中、集会中，以妄语令众人欢笑，彼身坏命终后生于欢笑天众中。'世尊于此云何说？"

"止，村长，置此事。勿问我此。"

第二次……第三次，戏子村长复白世尊……

"村长，我未许汝此……然我当为汝解说。村长，彼戏子于舞台中、集会中，于引生贪之法令众生贪，于引生嗔之法令众生嗔，于引生痴之法令众生痴——彼自放逸，复令他放逸。彼身坏命终后，生于名'欢笑'之地狱。

若彼复有如是见：'戏子于舞台中、集会中以妄语令众人欢笑，身坏命终后生于欢笑天众中'——此为其邪见。村长，邪见者，我说有二趣之一——地狱或畜生。"

如是说已，戏子村长多罗弗涕泣……

## 三、脚注

1. **虚假表演**（musāvāda）：此处指以不实的表演、夸张的演出来娱乐观众。
2. **欢笑天**（pahāsānaṃ devā）：字面意为"欢笑的天人"，戏子传统中认为能让人笑的人死后生此天。
3. **贪欲**（rāga）：对感官享受的贪求与执着。
4. **嗔恨**（dosa）：愤怒、厌恶、恶意。
5. **愚痴**（moha）：无明，对事物真相的无知与迷惑。
6. **放逸**（pamāda）：不谨慎、懈怠、沉溺于感官享乐而忘失正念。
7. **名为"欢笑"的地狱**（pahāso nāma nirayo）：佛陀以讽刺的方式指出——不是"欢笑天"，而是"欢笑地狱"。
8. **邪见**（micchādiṭṭhi）：错误的见解，此处指相信以不善行为能获得善果。
9. **畜生道**（tiracchānayoni）：旁生，三恶道之一。

---

*说明：佛陀破斥"戏子死后生欢笑天"的民间信仰——以表演激发众人贪嗔痴的戏子，实际上是在造恶业，死后不生天而堕恶趣。原文据巴利圣典协会（PTS）校订本与 SuttaCentral，汉译参酌庄春江、元亨寺等通行译本整理。*
