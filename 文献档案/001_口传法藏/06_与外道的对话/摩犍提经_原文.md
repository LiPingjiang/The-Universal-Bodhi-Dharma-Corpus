# 《摩犍提经》原文

> 收录编号：MN 75（中部第七十五经）
> 原始语言：巴利语 · 篇幅类型：中经
> 巴利名：Māgaṇḍiya Sutta
> 主题：以麻风病人喻说明欲乐实苦

---

## 一、巴利原文（拉丁转写，节录核心段落）

**§1 游方者摩犍提指责佛陀是「毁灭者」**

> Evaṃ me sutaṃ — ekaṃ samayaṃ bhagavā Kammāsadhamme viharati Bhāradvājagotte brāhmaṇassa agyāgāre. Atha kho Māgaṇḍiyo paribbājako jaṅghāvihāraṃ anucaṅkamamāno ... yena bhagavā tenupasaṅkami ...

> "Bhūnahu samaṇo Gotamo"ti.
> "Kena me, Māgaṇḍiya, evaṃ vadesi: 'Bhūnahu samaṇo Gotamo'ti?"
> "Samaṇo hi me, bho Gotama, Gotamo bhūnahūti dhammiyā kathāya ovadatī"ti.

**§2 佛陀以比喻说明自己已离欲**

> "Seyyathāpi, Māgaṇḍiya, puriso kuṭṭhī assa kaṇḍu-paccanaṅgo ... aṅgārakāsuyā kāyaṃ paritāpeyya ... so evaṃ vadeyya — 'Arogā imesu pañcasu kāmaguṇesu viharanti sukhan'ti."

> "Taṃ kiṃ maññasi, Māgaṇḍiya, api nu so puriso sukhaṃ vedeti? ... No h'idaṃ, bho Gotama. Yathā taṃ rogo — rogassa vikaṭaṃ paṭicca."

**§3 麻风病人的比喻——欲乐如灼伤之痒**

> "Seyyathāpi, Māgaṇḍiya, puriso kuṭṭhī kaṇḍu-paccanaṅgo vaṇappaṭikujjito aṅgārakāsuyā kāyaṃ paritāpeyya. Yathā yathā kho, Māgaṇḍiya, so puriso kuṭṭhī kaṇḍu-paccanaṅgo vaṇappaṭikujjito aṅgārakāsuyā kāyaṃ paritāpeyya, tathā tathāssa tāni vaṇamukhāni asucitarāni ceva bhavissanti duggandhatarāni ca. Hoti tv eva kocideva assādo — seyyathidaṃ — kaṇḍūnaṃ paṭikamanā.

> Evameva kho, Māgaṇḍiya, kāmā rogā ti vuccanti ... kāmā gaṇḍā ... kāmā sallā ... kāmā aggi ... Seyyathāpi, Māgaṇḍiya, kuṭṭhī puriso ... so tamhā ābādhā parimucceyya, arogapaccayo hoti sukho sukhavihārī. Tamenaṃ puriso dve kuṭṭhī purisā ubhohatthehi gahetvā aṅgārakāsuyā upakaḍḍheyyuṃ ... so tāya aṅgārakāsuya hatthapāde vā saṃhareyya aññataraṃ vā aññataraṃ vā aṅgaṃ saṃhareyya. Taṃ kissa hetu? Viditā hissa, Māgaṇḍiya, tassa aṅgārakāsuya ādīpitattā ca mahāpariḷāhattā ca."

## 二、汉译对照

### 甲译（白话意译）

**§1** 我是这样听闻的——有一次，世尊住在甘马萨达摩村¹婆罗陀若婆罗门的火祭堂中。游方修行者摩犍提散步时来到世尊面前……

摩犍提说："沙门乔达摩是一个毁灭者²！"
"摩犍提，你为什么这样说我？"
"因为乔达摩先生用教法劝导人远离欲乐³——在我看来，这就是毁灭。"

**§2** "摩犍提，假设有一个麻风病人⁴，浑身发痒、长满疮口，他用指甲抓痒，用火坑⁵来烤热自己的身体……另一个健康的人看到这一切，会说那些在五种感官享受⁶中生活的人是快乐的。

"你觉得呢？那个麻风病人把身体在火坑上烤的时候，他感受到的是快乐吗？""不是的，乔达摩先生，那只不过是疾病——因为疾病才觉得烤火舒服。"

**§3** "摩犍提，就像那个麻风病人把身体在火坑上烤——他越烤，伤口就越脏、越臭。他得到的那种所谓的舒适感，不过是止止痒⁷罢了。

"同样道理，欲乐被称为疾病⁸……欲乐被称为脓疮……欲乐被称为箭刺……欲乐被称为烈火。摩犍提，就像那个麻风病人后来痊愈了⁹，恢复了健康，快快乐乐地生活。这时如果有两个麻风病人抓住他的双手，硬把他拖向火坑……他一定会缩回手脚、全力挣脱。为什么？因为他已经知道了那火坑是灼热的¹⁰、是令人剧痛的。

"同样，摩犍提，觉悟者已经远离了感官欲望，就像痊愈的人远离了火坑一样——即使有人把全世界的欲乐放在他面前，他也不会动心。"

### 乙译（严谨对照译）

**§1** 如是我闻——一时，世尊住甘马萨达摩婆罗陀若婆罗门之火堂。游方者摩犍提……诣世尊所……

"沙门瞿昙是毁灭者。"
"摩犍提，汝何故如是说我为毁灭者？"
"瞿昙，沙门以法教语呵责欲乐，故我谓之毁灭者。"

**§2** "摩犍提，譬如有人为癞病者，身痒疮烂，以爪搔痒，以身近火坑而炙……他人——无病者——见彼于五欲功德中享受快乐。

汝意云何？彼癞者以身炙于火坑，受乐耶？""否也，瞿昙。彼唯是病——以病之变故而觉舒适。"

**§3** "摩犍提，譬如癞者以身炙于火坑——愈炙则疮口愈秽、愈臭。然有些许安慰——所谓止痒耳。

如是，摩犍提，诸欲说为病……说为痈……说为箭……说为火。譬如癞者后愈，无病而安乐住。若二癞人以两手执彼，牵拽至火坑……彼必缩手足、退缩肢体。何以故？彼已知火坑之炽然与大热恼故。

如是，摩犍提，觉者已离诸欲，如无病人之离火坑——纵有世间一切欲乐现前，彼亦不动心。"

## 脚注

1. **甘马萨达摩**（Kammāsadhamma）：拘留国的一个市镇。
2. **毁灭者**（bhūnahu）：字面义为"杀胎者/毁灭者"——摩犍提认为佛陀劝人远离欲乐是在毁灭人生的幸福。
3. **欲乐**（kāma）：五种感官享受——色、声、香、味、触带来的快感。
4. **麻风病人**（kuṭṭhī）：这是佛教最著名的比喻之一——用麻风病人的止痒快感来说明凡人对欲乐的追求其实是病态的。
5. **火坑**（aṅgārakāsu）：燃烧着炭火的坑，麻风病人用它来止痒。
6. **五种感官享受**（pañca kāmaguṇā）：可爱的色、声、香、味、触——凡人追求的五欲功德。
7. **止止痒**（kaṇḍūnaṃ paṭikamanā）：暂时缓解瘙痒，但并非真正的快乐——比喻欲乐只是暂时满足渴求。
8. **欲乐被称为疾病**（kāmā rogā）：此处连用疾病、脓疮、箭刺、烈火四喻，说明欲乐的本质是苦而非乐。
9. **麻风病人后来痊愈了**：比喻觉悟者——一旦烦恼之病痊愈，就不会再被欲乐所吸引。
10. **灼热的**（ādīpitattā）：觉悟者以智慧之眼看到欲乐的真实面目——如同痊愈者看到火坑的灼热一样清楚。

*说明：本经巴利原文据巴利圣典协会（PTS）校订本与 SuttaCentral（suttacentral.net/mn75）录入。"麻风病人"比喻是巴利经藏中最有冲击力的譬喻之一——深刻揭示了欲乐的本质不是真正的快乐，而是对"渴求之痒"的暂时止息。汉译参酌庄春江、菩提比丘英译本整理。*
