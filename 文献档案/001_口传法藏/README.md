# 001 · 佛陀口傳法藏

> ← [返回總目錄](../../README.md)

**Buddhavacana（佛語）**——佛陀親口宣說之教法，為一切佛典之根源。

佛陀住世約四十五年（約前5世紀），隨機應化，說法無數。彼時無文字記錄，全憑弟子口耳相傳。佛滅當年，五百阿羅漢於王舍城結集，誦出經、律二藏，此為法藏成文之始。本節所收，即此口傳源頭之代表性經典。

## 體例說明

每部經典備有三種讀法，讀者可按需取用：

**原文** — 巴利/梵文原典落盤，附漢譯對照，存真第一。

**解讀·學術** — 術語精確，含文獻考證、版本異同、思想史定位，適合有一定基礎的讀者深入研究。

**解讀·白話** — 通篇以現代中文日常熟詞講解，術語悉數沉入腳注，流暢如讀一篇現代文章，適合初入門者及一般讀者。
