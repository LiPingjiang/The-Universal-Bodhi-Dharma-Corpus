# 001 · 佛陀口傳法藏

> ← [返回總目錄](../README.md)

**Buddhavacana（佛語）**——佛陀親口宣說之教法，為一切佛典之根源。

佛陀住世約四十五年（約前5世紀），隨機應化，說法無數。彼時無文字記錄，全憑弟子口耳相傳。佛滅當年，五百阿羅漢於王舍城結集，誦出經、律二藏，此為法藏成文之始。

本節按**教義主題**分類收錄佛陀口傳經文，每部經文均附原文（巴利+漢譯對照）、學術解讀、白話解讀三件套。去掉增支部/相應部中的極短數字變體重複後，有獨立內容的經文約500–700篇。

## 體例說明

每部經典備有三種讀法，讀者可按需取用：

**原文** — 巴利/梵文原典落盤，附漢譯對照，存真第一。

**解讀·學術** — 術語精確，含文獻考證、版本異同、思想史定位，適合深入研究。

**解讀·白話** — 通篇以現代中文日常熟詞講解，術語悉數沉入腳注，適合初入門者。
