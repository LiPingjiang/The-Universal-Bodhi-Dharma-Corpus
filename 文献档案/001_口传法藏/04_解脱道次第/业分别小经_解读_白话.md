# 《业分别小经》整理与解释

> ← [返回目錄](../README.md)
> 节点ID：001.4.6 · 对应原文档案：业分别小经_原文.md
> 收录编号：MN 135（中部第一三五经）· 估计年代：佛陀住世（约公元前5世纪）

---

## 一、名称还原解释

**「业分别小经」**，巴利原名 **Cūḷakammavibhaṅga Sutta**：

- **cūḷa（小）**＝"简明版"。还有一部"大业分别经"（MN 136）讨论更复杂的情况。
- **kamma（业）**＝"行为及其后果"。你做的事——好事坏事——都是你的"业"，它们会产生相应的结果。
- **vibhaṅga（分别）**＝"分类说明"。把一个大话题拆开来，一条一条讲清楚。
- **Sutta（经）**＝贯串教义的文本。

所以 **Cūḷakammavibhaṅga Sutta** 就是**「对行为因果的简明分类说明」**——一部用最直白的方式告诉你"做什么得什么"的经典。

## 二、这部经讲了什么（大白话）

**一个年轻人问了一个人人都想知道的问题：**

婆罗门青年须婆来找佛陀，问了一个极其朴素但极其根本的问题：

"**为什么人与人之间有这么大的差别？**有人长寿有人短命，有人健康有人多病，有人漂亮有人丑，有人有权有人没权，有人富有人穷，有人聪明有人笨——这到底是为什么？"

**佛陀的回答只有一句核心：众生各有自己的业。**

"业是你自己的财产，业是你继承的遗产，业是你出生的源头，业是你最亲的亲属，业是你唯一的靠山。是业将众生分为高下不同。"

然后佛陀一对一对地详细解说：

**杀生 → 短命；不杀生 → 长寿。** 经常杀害生命的人，来世投生为人时寿命短暂。远离杀生、爱护生命的人，来世寿命长久。

**伤害他人 → 多病；不伤害 → 健康。** 用拳头、石头、棍子折磨别人的人，来世多病。不去伤害他人的人，来世少病。

**脾气暴躁 → 丑陋；温和不忿 → 端正。** 动不动就发火、怨恨、记仇的人，来世容貌丑陋。心性温和、不轻易动怒的人，来世相貌端正。

**嫉妒别人 → 没权势；随喜别人 → 有权势。** 看见别人好就嫉妒的人，来世权势微薄。能随喜他人成就的人，来世权势隆盛。

**悭吝不施 → 贫穷；慷慨布施 → 富有。** 有钱但不肯给予的人，来世贫穷。乐善好施的人，来世富裕。

**傲慢自大 → 出身卑微；谦恭有礼 → 出身高贵。** 对修行人、长辈傲慢无礼的人，来世出身低微。恭敬谦逊的人，来世出身尊贵。

**不学习不思考 → 愚痴；好学亲近智者 → 有智慧。** 从不去请教"什么是善什么是恶"的人，来世愚笨。主动亲近智者、请问善恶的人，来世聪慧。

## 三、为什么它重要

第一，**它是佛教"因果报应"最直接、最系统的教科书**。如果有人问"佛教的因果到底怎么讲的？"——把这部经拿给他看就行了。七对因果，清清楚楚，一目了然。

第二，**它把"命运"从神的手里拿回到了自己手里**。在婆罗门教看来，人的高低贵贱是梵天创造的、是与生俱来不可改变的。佛陀说不——你的命运取决于你自己的行为。这意味着：无论你现在是什么出身、什么处境，通过改变行为，你的未来是可以改变的。这是一种深刻的赋权。

第三，**它对日常生活有极强的指导性**。不需要出家、不需要打坐、不需要高深的禅修——你只要在日常生活中不杀生、不伤人、不发怒、不嫉妒、肯布施、懂谦逊、愿学习，就在为自己的未来积累善因。这是最接地气的佛法。

第四，**要注意它的局限性**。这是"简明版"——真实的因果运作比这复杂得多。佛陀在另一部"大业分别经"（MN 136）里讨论了为什么有时候好人遭殃、坏人享福的"反例"——因为业报的成熟受到很多因素影响，不是简单的"一因一果"。两部经合起来看，才是佛教业报思想的完整图景。

简单说：这部经像一张最基本的"行为→结果对照表"——做什么得什么，写得明明白白。它是佛教道德教育的"第一课"，适合所有人——不管你是不是佛教徒，这七对因果里蕴含的生活智慧都值得思考。
