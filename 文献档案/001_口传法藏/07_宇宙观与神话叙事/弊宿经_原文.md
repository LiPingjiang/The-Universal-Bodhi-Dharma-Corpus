# 《弊宿经》原文

> 节点ID：001.7.4 · 收录编号：DN 23（长部第23经）；对应汉译《长阿含经》卷7《弊宿经》（T1）及《中阿含经》卷16《蜱肆经》（T26）
> 原始语言：巴利语 · 篇幅类型：长经·节录核心段落
> 出处：SuttaCentral（suttacentral.net/dn23）、CBETA 汉文电子佛典

## 一、巴利原文（拉丁转写，节录核心段落）

**弊宿王的断灭论立场：**

> Pāyāsi ... rājañño evaṁdiṭṭhiko ahosi: "Natthi paro loko, natthi sattā opapātikā, natthi sukatadukkaṭānaṁ kammānaṁ phalaṁ vipāko"ti.

**鸠摩罗迦叶的到来：**

> Tena kho pana samayena āyasmā Kumārakassapo ... Setavyāyaṁ viharati Uttarena Setavyaṁ Siṁsapāvane.

**"罐中灵魂"的比喻——弊宿的论证：**

> "Idha me, bho Kassapa, ñātī vā honti mittā vā ... te ahaṁ evaṁ vadāmi: 'tāta, tumhe ... pāpakammaṁ karonti ... kumbhiyā pakkhipitvā ... mukhaṁ pidahitvā ... vivarantā na passāma sattaṁ nikkhamantaṁ' — tena me evaṁ hoti: 'natthi paro loko'"ti.

**鸠摩罗迦叶的反驳——"睡眠者"的比喻：**

> "Seyyathāpi, rājañña, puriso niddāyamāno supinena passeyya ārāmavanarāmaṇīyakaṁ ... so paccuṭṭhahitvā na passeyya ... api nu tvaṁ, rājañña, evaṁ vadeyyāsi: 'natthi supino'"ti?

**"铜釜"的比喻：**

> "Seyyathāpi, rājañña, puriso tattaṁ ayoguḷaṁ ādittaṁ sampajjalitaṁ sajotibhūtaṁ gaṇheyya ayosandāsena ... api nu kho tattha aggi nikkhamanto disseyya"ti?

**"萤火虫"的比喻：**

> "Seyyathāpi, rājañña, khattādhipurisena vā ... soṇḍikena vā ... saṅkhaṁ dhamantassa saṅkhamhā saddo nikkhamati ... api nu kho taṁ saddaṁ gaṇhituṁ sakkhissasi?"

## 二、汉译对照

> **甲译**：白话意译，通顺自然，不拘泥于原文句式，适合直接阅读。
> **乙译**：严谨译，逐句对应巴利原文，术语精确，供对照研究。

---

**弊宿王[^payasi]的观点**

**甲译：** **弊宿王是一位地方统治者，他坚定地持有"断灭见"[^ucchedavada]——不相信有来世、不相信有自然化生的众生、不相信善恶行为有果报。他到处宣称："死了就什么都没有了。"**

**乙译：** 弊宿王持如是邪见："无后世，无化生有情，无善恶业之异熟果报。"

---

**弊宿的"实验"论证**

**甲译：** **弊宿王说："我做过实验！我让人把一个该下地狱的杀人犯装进一口大罐子里，封死盖子，等他死了再打开——我没看到任何灵魂从里面出来。所以灵魂不存在，来世也不存在。"**

他又说："我还让人把一个有德行的人称过体重，等他死了再称一次——重量一样！说明没有什么'灵魂'离开了身体。"

**乙译：** "我令人取罪人封于瓮中，密封其口。彼死后开瓮观之，不见有情出。故知无后世。"

---

**鸠摩罗迦叶[^kumarakassapa]的比喻论证**

**甲译：**

鸠摩罗迦叶（佛陀的大弟子之一）用一系列精彩的反比喻回应：

**比喻一（梦境）：** "大王，一个人做梦时看到美丽的花园——醒来后花园不见了。你会因此说'梦不存在'吗？看不到不等于不存在。"

**比喻二（烧红的铁球）：** "大王，你把铁球烧得通红发光——那'火'在铁球里面还是外面？你能用钳子把'火'从铁球里夹出来吗？夹不出来，但你不会说铁球里没有火。"

**比喻三（吹螺号）：** "大王，号手吹海螺时发出声音——你能把'声音'从螺壳里抓出来吗？抓不出来，但你不会说声音不存在。"

**比喻四（火堆中的火）：** "你看着一堆火燃烧——火灭了之后，你能指出'火跑到哪里去了'吗？指不出来，但你不会说火从来没存在过。"

每一个比喻都指向同一个论点：**"看不到"不等于"不存在"。你用物质的方法去验证非物质的东西，方法论本身就是错的。**

**乙译：** "大王，如人梦见园林殊胜……醒已不见。汝岂言'无梦'耶？" "如人以铁钳持烧铁丸……汝能见火出否？" "如人吹螺……汝能执取声否？"

---

**弊宿王被说服**

**甲译：** **经过一连串的辩论，弊宿王最终被说服了。但有趣的是——他虽然承认"来世可能存在"，却仍然对布施和修行持消极态度。鸠摩罗迦叶又用"商人囤积"和"赌博"的比喻劝他：即使你不确定来世存在，押注"行善"也比押注"不行善"合算——这是一种类似"帕斯卡赌注"的论证。**

**乙译：** 弊宿王言："善哉！我今信有后世、有化生、有业报[^kamma]。"

---

[^payasi]: **弊宿**（Pāyāsi）：经文中的一位地方首领（rājañña），统治斯多婆提（Setavyā）地区。他代表了当时印度"断灭论"思潮——认为死后一切断灭，没有来世、没有灵魂、没有善恶果报。

[^ucchedavada]: **断灭见**（ucchedavāda）：认为人死后一切断灭、什么都不存在的观点。佛教将其视为两种极端错误观点之一（另一种是"常见"——认为有永恒不变的灵魂）。

[^kumarakassapa]: **鸠摩罗迦叶**（Kumārakassapa）：佛陀的大弟子之一，以善于辩论和说法著称。经文中是他（而非佛陀本人）与弊宿王辩论——这表明佛教传法不限于佛陀一人，弟子们也有足够的智慧应对挑战。

[^kamma]: **业报**（kamma-vipāka）：kamma（业）指有意的行为，vipāka（果报）指行为产生的后果。佛教认为善行带来好的果报，恶行带来坏的果报——这不是神的奖惩，而是一种自然的因果规律。

---

*说明：本经为长部第二十三经，记录佛弟子鸠摩罗迦叶与断灭论者弊宿王的精彩辩论。鸠摩罗迦叶以一系列巧妙的比喻论证了"看不到≠不存在"，动摇了弊宿王的唯物主义立场。本经是巴利三藏中论证"业报轮回"最为生动的文本之一。原文据 PTS 校订本与 SuttaCentral，汉译参酌《长阿含经》卷七及现代译本整理。*
