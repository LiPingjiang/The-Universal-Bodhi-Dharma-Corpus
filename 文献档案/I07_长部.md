## 名称还原与解释

**长部**（Dīgha Nikāya）：
- Dīgha = 长的（指每篇经文篇幅较长）
- Nikāya = 集/部/类聚

= "长篇经文的汇集"。与中部（中等长度）、相应部（按主题分类）、增支部（按数字递增）并列为经藏四大部（加小部为五部）。

## 原始内容落盘

### 三品结构与重要经典

**戒蕴品（Sīlakkhandha-vagga）**·13经——主题：佛教vs.外道，戒律的殊胜：

| # | 经名 | 要点 |
|---|------|------|
| DN1 | 梵网经（Brahmajāla Sutta） | 列举62种外道见解并一一破斥——佛教"正见"的反面清单 |
| DN2 | 沙门果经（Sāmaññaphala Sutta） | 阿阇世王问六师外道与佛法之区别——修行的具体利益（戒→定→慧→神通→解脱） |
| DN4 | 种德经（Soṇadaṇḍa Sutta） | 与婆罗门辩论"什么是真正的婆罗门"——德行胜于出身 |
| DN9 | 布吒婆楼经（Poṭṭhapāda Sutta） | 讨论"想"的本质——认知心理学的早期探索 |

**大品（Mahā-vagga）**·10经——主题：佛陀生平重大事件与核心教法：

| # | 经名 | 要点 |
|---|------|------|
| DN14 | 大本经（Mahāpadāna Sutta） | 过去七佛的故事——佛陀不是唯一的 |
| DN15 | 大因缘经（Mahānidāna Sutta） | 十二缘起的最详细展开之一 |
| DN16 | 大般涅槃经（Mahāparinibbāna Sutta） | **长部最重要的经**——佛陀最后三个月的行程、遗教、入灭全过程（6品） |
| DN17 | 大善见王经（Mahāsudassana Sutta） | 佛陀前世为转轮圣王的故事 |
| DN22 | 大念处经（Mahāsatipaṭṭhāna Sutta） | **四念处修法的完整版**——正念禅修的根本依据（001已收录） |

**波梨品（Pāṭika-vagga/Pāthika-vagga）**·11经——主题：杂录，含宇宙论、伦理、社会：

| # | 经名 | 要点 |
|---|------|------|
| DN26 | 转轮圣王师子吼经（Cakkavatti-sīhanāda Sutta） | 社会退化与弥勒佛将来出世的预言 |
| DN27 | 起世因本经（Aggañña Sutta） | 佛教版"创世神话"——世界与社会阶层的起源 |
| DN31 | 教授尸迦罗越经（Sigālovāda Sutta） | "在家人伦理手册"——六方礼（父母/师长/妻子/朋友/仆从/沙门） |
| DN33 | 等诵经（Saṅgīti Sutta） | 教法分类汇总（一法→十法）——类似"考试大纲" |
| DN34 | 十上经（Dasuttara Sutta） | 十组教法主题的系统归纳 |

### 长部与长阿含的对照

| 巴利长部 | 汉译长阿含 |
|----------|------------|
| 34经，三品 | 30经，四分 |
| 上座部传本 | 法藏部传本 |
| 有梵网经 | 有梵动经（对应） |
| 有大般涅槃经 | 有游行经（对应但内容有异） |
| 无世记经 | 有世记经（长阿含独有） |

两版本核心经典高度重合，但各有独特篇章——证明两者源自同一古本但经过各自部派数百年的传承而各有增损。

## 大白话总结

长部是巴利经藏中"块头最大"的经文集——每一篇都是长篇大论。它收集了佛教最"正式"的场景：佛陀与国王辩论、佛陀临终遗言、佛陀解释宇宙起源、佛陀讲授完整的修行路线图。

如果经藏是一个大学的话，长部就是"公开讲座系列"——规模宏大、面向广泛听众、系统而完整。中部更像"课堂教学"（每篇针对特定问题），相应部像"主题笔记"，增支部像"要点手册"。

长部中最不可替代的三篇：《大般涅槃经》（佛陀的临终记录——任何想了解"佛教创始人是怎么面对死亡的"都必须读）、《大念处经》（禅修的根本操作手册）、《梵网经》（佛教与所有其他思想流派的"关系地图"）。
