# 雜阿含經 學術解讀

## 一、文獻學基本信息

**譯者**：求那跋陀羅（Guṇabhadra，394–468 CE），中天竺（印度中部）人，精通大小乘，信仰以大乘為主，但翻譯範圍廣泛。於南朝劉宋時代來華，在廣州、建康（今南京）、荊州等地譯經，共譯出約七十八部、一百三十七卷。  
**譯時**：劉宋元嘉十二年至二十二年（435–445 CE）。  
**底本語言**：梵文，底本屬說一切有部（Sarvāstivāda）傳誦，此為現代學界基本共識。  
**大正藏**：T02, No. 99，第二冊，頁1–373。  
**卷帙**：五十卷，但實際傳承有問題（見下文）。  

### 版本傳承的混亂問題

《雜阿含經》的傳本存在嚴重的混亂：現行大正藏本中有大量「插入」現象——部分卷次明顯是後來從其他文本插入的，如有學者發現其中若干卷次的文字實際上屬於《別譯雜阿含經》（T02, No. 100）或其他文本。

印順法師在其巨著《雜阿含經論會編》（1983）中，花費數十年工夫，依據巴利《相應部》的對應結構、《瑜伽師地論·攝事分》（Yogācārabhūmi，玄奘譯，引用了大量《雜阿含》原文）及《阿毗達磨大毗婆沙論》的引文，重新整理出《雜阿含》的原始架構，恢復了其「相應」（saṃyutta）的組織方式。這是20世紀漢傳佛教文獻學最重要的成果之一。

## 二、在佛教史上的地位與影響

《雜阿含經》在四部阿含中具有最高的哲學密度，是早期佛教教義的最核心文獻庫：

**1. 玄奘的引用依據**：唐代玄奘在翻譯《瑜伽師地論》時，在〈攝事分〉部分引用了大量《雜阿含》的原文（以「契經」形式），並加以論疏。這表明說一切有部的阿毗達磨論師視《雜阿含》為其教義分析的直接文獻依據，地位遠高於其他三部阿含。

**2. 印度阿毗達磨的母典**：根據說一切有部的傳統（及學術研究的佐證），《雜阿含》是說一切有部論書（如《品類足論》《法蘊足論》）的原始依據。這些論書的分類框架（如五蘊、十二處、十八界的三科分析）直接來自《雜阿含》的組織原則。

**3. 緣起論的最完整文本**：十二因緣在《雜阿含》中有最系統、最反覆的陳述，多達數十篇相關經典，從各個角度論述緣起的結構、意義和修行意涵，是任何研究佛教緣起論必須首先參考的文獻。

**4. 三法印的標準陳述**：無常、苦、無我（三法印，或加涅槃為四法印）的最精練、最哲學化的陳述集中在《雜阿含》的五蘊相應部分。

## 三、與南傳巴利尼柯耶的對應關係

《雜阿含經》與巴利《相應部》（Saṃyutta Nikāya，SN）的對應是四部阿含中最為密切的。兩者採用相同的「相應」（按主題分類）組織原則，很多個別經典的文字幾乎可以逐句對照：

| 《雜阿含》相應 | 巴利《相應部》 |
|-------------|-------------|
| 五陰相應 | SN 22: Khandhasaṃyutta |
| 六入處相應 | SN 35: Saḷāyatanasaṃyutta |
| 因緣相應 | SN 12: Nidānasaṃyutta |
| 諦相應 | SN 56: Saccasaṃyutta |
| 道品相應 | SN 45–51 |
| 比丘尼相應 | SN 5: Bhikkhunīsaṃyutta |
| 諸天相應 | SN 1–2: Devatāsaṃyutta, Devaputtasaṃyutta |
| 有偈（魔相應） | SN 4: Mārasaṃyutta |

學界認為兩傳的共同祖本可以追溯至佛陀滅後第一次結集（或其後不久），是四阿含中歷史最為古老的文本類型。

**主要差異**：
- 巴利《相應部》結構更完整，包含六十個相應；漢譯《雜阿含》因傳承混亂，目前可辨識的相應較少。
- 某些哲學細節上兩傳有差異，反映說一切有部（漢譯底本）與銅鍱部（巴利底本）的教義分歧。

## 四、主要教義內容概述

### 1. 三科：五蘊、十二處、十八界

《雜阿含》以「三科」作為分析人之存在的基本框架：

- **五蘊**（skandha）：色（物質）、受（感受）、想（認知）、行（意志）、識（意識）。這是從「人的構成」角度分析存在。
- **十二處**（āyatana）：六根（眼、耳、鼻、舌、身、意）與六境（色、聲、香、味、觸、法）。這是從「認識活動的條件」角度分析存在。
- **十八界**（dhātu）：六根+六境+六識（眼識、耳識、鼻識、舌識、身識、意識）。這是最完整的認識論分析。

三科分析的哲學目的是：在哲學上徹底解析「我」是什麼，最終論證任何組合中都找不到一個固定不變的「自我」，從而動搖「我執」的哲學基礎。

### 2. 十二因緣

《雜阿含》中的因緣相應是十二緣起（nidānasaṃyutta）研究的第一手材料。這五十餘篇經典從不同角度闡述緣起：
- 正向緣起（流轉）：無明→……→老死苦
- 逆向緣起（還滅）：無明滅→……→老死苦滅
- 緣起的「此有故彼有」邏輯公式
- 緣起與「中道」的關係（不落常見、不落斷見）

### 3. 四聖諦的詳細闡述

《雜阿含》中的諦相應是四聖諦（苦、集、滅、道）最詳細的哲學展開，涉及苦諦的定義、集諦的渴愛分析、滅諦（涅槃）的性質，以及道諦（八正道）的每一支的詳細界定。

### 4. 修行次第：戒、定、慧三學

《雜阿含》的道品相應呈現了三學（戒、定、慧）的修行次第：持戒是基礎，禪定（samādhi）在戒的基礎上培育心的穩定和集中，智慧（paññā）在禪定的基礎上洞察三法印，最終導向解脫。

## 五、學術研究現狀與爭議

### 印順法師的重編工作

印順法師的《雜阿含經論會編》（1983）是20世紀漢傳佛教文獻學最重要的成果之一。他的方法是：依據《瑜伽師地論·攝事分》中玄奘引用的《雜阿含》原文及注解，結合巴利《相應部》的對應結構，重新確定《雜阿含》各卷的原始位置，糾正了大正藏本中的混亂。這一重編工作至今是研究《雜阿含》的標準參考。

### 底本歸屬的確認

《雜阿含》的底本屬說一切有部這一點，由無著比丘（Bhikkhu Analayo）等學者通過細緻的文本對比進一步確認。判斷依據包括：（1）某些術語和定義與說一切有部論書一致；（2）《瑜伽師地論·攝事分》（有部背景）引用《雜阿含》作為解說對象；（3）部分教義細節（如識的性質、觸的定義）與有部阿毗達磨吻合。

### 《雜阿含》與《別譯雜阿含》的關係

大正藏T02, No. 100《別譯雜阿含經》與《雜阿含》的關係至今有爭議。兩者之間存在大量重複的題材，但文字不同，說明底本不同（《別譯》底本可能屬飲光部或法藏部）。現行大正藏本《雜阿含》中確有部分混入了《別譯雜阿含》的內容，印順法師和後來的研究者在重編工作中對此有詳細辨析。

### 求那跋陀羅的翻譯風格

求那跋陀羅的翻譯總體上較為忠實，但他的語言習慣帶有濃厚的中天竺梵語特色，某些術語的翻譯選擇有時與同時代其他譯者不同，給後來的比較研究帶來了一些技術困難。
