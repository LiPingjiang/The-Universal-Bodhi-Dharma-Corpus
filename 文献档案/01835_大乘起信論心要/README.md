# 大乘起信論心要

## 基本信息

- **編號**：01835

## 學術解讀

### 一、文獻學信息

《大乘起信論心要》是唐代慧沼（650-714）所撰。慧沼，俗姓劉，淄川（今山東淄博）人，爲唐代唯識宗二祖，窺基大師高足，世稱"淄州大師"。此疏以"心要"爲名，取"心法之要，提綱挈領"之意，是唯識宗系統解釋《大乘起信論》的重要著作。

慧沼一生弘揚唯識教義，著有《成唯識論了義燈》《因明入正理論義纂要》等，對唯識宗的發展貢獻巨大。他嚴格繼承玄奘、窺基的唯識學統，同時也不排斥與其他宗派的對話。此疏作於武則天時期，是唯識宗對如來藏思想進行系統回應的代表作。

### 二、歷史背景

唐代初期，唯識宗與華嚴宗、天台宗之間的思想競爭十分激烈。唯識宗以"阿賴耶識緣起"爲核心，強調"轉識成智"的修行次第；而《起信論》則以"心性本淨"爲核心，強調"本覺"與"始覺"的不二。兩種思想體系在"心"的問題上存在根本分歧。慧沼此疏正是在這一背景下產生，試圖以唯識宗的嚴謹義理來重新詮釋《起信論》，確立唯識宗在如來藏思想傳統中的解釋權。

### 三、思想史定位

慧沼此疏的特色在於以唯識宗的"五重唯識觀"來解釋《起信論》的"一心二門"。他認爲，《起信論》的"一心"可以納入唯識宗的"第八識"框架，但需要做細緻的區分："心真如門"對應第八識的"清淨分"（無漏種子），"心生滅門"對應第八識的"染污分"（有漏種子）。慧沼進一步將"本覺"解釋爲"本有無漏種子"，"始覺"解釋爲"有漏種子漸斷、無漏種子漸顯"的過程。這一解釋雖然被華嚴宗學者批評爲"將《起信論》唯識化"，但體現了唐代佛教義學的深度和複雜性。

## 白話解讀

### 核心內容

《大乘起信論心要》是唐代唯識宗高僧慧沼對《大乘起信論》的解釋。慧沼大師是玄奘大師再傳弟子，唯識學的權威。

慧沼大師用唯識宗的"種子"理論來解釋《起信論》：我們的心（第八識）就像一個倉庫，裏面儲存着兩種"種子"——"無漏種子"（清淨的、能成佛的潛能）和"有漏種子"（染污的、導致輪迴的潛能）。"心真如門"就是倉庫裏的"無漏種子"，"心生滅門"就是倉庫裏的"有漏種子"在發芽。

修行的過程，就是逐漸減少"有漏種子"的影響力，讓"無漏種子"顯現出來。這就是從"始覺"到"本覺"的過程。

### 現代啓示

1. **潛能開發**："種子"理論啓示我們：每個人都有無限的潛能，只是有些潛能被遮蔽了。教育、修行、自我提升，都是"讓好種子發芽"的過程。

2. **習慣養成**："有漏種子"就像壞習慣，"無漏種子"就像好習慣。習慣的形成和改變，都可以從"種子"的角度來理解。

3. **循序漸進**：唯識宗強調"漸修"，啓示我們：成長是一個漸進的過程，不要期望一蹴而就。

### 閱讀建議

- 慧沼的疏解需要唯識學基礎，建議先讀《八識規矩頌》或《成唯識論》入門
- 可對比良賁以唯識解《起信論》的方式，理解唯識宗解釋傳統的延續
- 注意唯識宗與如來藏宗在"心"問題上的根本分歧
