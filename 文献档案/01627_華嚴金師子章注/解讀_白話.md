# 华严金师子章注 · 白話解讀

## 這是什麼書？

《華嚴金師子章》是唐朝法藏大師的一篇短文，他用「金獅子」這個具體的例子，來解釋華嚴宗最深奧的哲學道理。後人對這篇文章的注解，就是「金師子章注」。這篇文章雖然很短，但包含了華嚴宗最精華的思想。

## 核心內容

### 1. 主要說了什麼？

**金獅子的比喻**：法藏大師指著殿前的金獅子說：「金」代表宇宙的本體（理），「獅子的形狀」代表現象（事）。金和獅子形狀是不可分離的——沒有金就沒有獅子，沒有獅子形狀金也無法顯現。這說明本體和現象是相即不離的。

**六相圓融**：一隻獅子有整體（總相）和各個部分（別相），各部分有共同點（同相）也有差異（異相），各部分組合在一起（成相）也可以分開（壞相）。這六種關係同時存在，圓融無礙。

**因陀羅網**：想像一個無限大的網，每個網結上都有一顆寶珠，每顆寶珠都映照出所有其他寶珠的影像，而那些影像中又有更多的映照……這就是華嚴宗說的「重重無盡」，宇宙萬物相互包含、相互映照。

### 2. 對現代人有什麼啟發？

**整體與部分的關係**：現代系統論也強調整體不等於部分之和，整體中的每個部分都影響著整體。華嚴的「六相圓融」提供了一種看待複雜系統的智慧。

**相互依存的世界觀**：「因陀羅網」的比喻讓我們看到，世界上沒有任何事物是孤立存在的，一切都相互依存、相互影響。這與現代生態學、系統論的觀點高度吻合。

**包容差異的智慧**：華嚴說「同相」和「異相」同時存在，提醒我們在看到差異的同時，也要看到共同點；在強調共同點的同時，也要尊重差異。

### 3. 怎麼讀這部書？

《金師子章》原文很短，建議先讀原文，再讀注疏。可以配合法藏的《華嚴一乘教義分齊章》一起閱讀，更能理解華嚴宗的整體思想體系。重點理解「理事無礙」和「事事無礙」的區別。

## 關鍵概念（白話版）

| 佛學術語 | 白話解釋 |
|---------|---------|
| 理 | 宇宙的本體、本質，如金的本質 |
| 事 | 現象、具體事物，如獅子的形狀 |
| 六相 | 整體、部分、共性、差異、組合、分解六種關係 |
| 十玄 | 十種玄妙的相互關係，說明萬物圓融無礙 |
| 因陀羅網 | 重重無盡的相互映照，比喻萬物相互包含 |
| 事事無礙 | 現象與現象之間也是圓融無礙的 |

## 讀後思考

1. 「因陀羅網」的比喻讓你想到現代社會中什麼現象？（比如互聯網、社交網絡）
2. 「六相圓融」的思想如何幫助我們理解一個組織或團隊的運作？
3. 如果萬物都是相互依存的，這對我們的環保意識和社會責任感有什麼啟示？
