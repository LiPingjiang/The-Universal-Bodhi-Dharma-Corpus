# 瑜伽师地论略纂 · 學術解讀

## 一、文獻定位

《瑜伽师地论略纂》为唐代唯识学大师窥基（632–682）所撰，是玄奘译《瑜伽师地论》的精要纂辑和注释书。全书凡十六卷，对《瑜伽师地论》一百卷的庞大内容进行了系统的提炼和阐释，是研究唐代唯识宗思想、瑜伽行派学说及印度佛教论书中国化的关键文本。该书与窥基的《成唯识论述记》《因明入正理论疏》共同构成其三大代表作，全面展现了窥基对玄奘所传瑜伽行派学说的系统理解和发挥。

## 二、版本與傳承

- **作者**：窥基（俗姓尉迟，京兆长安人，玄奘高足）
- **成书年代**：约唐高宗时期（650–683）
- **所释经典**：玄奘译《瑜伽师地论》（《大正藏》第30冊 No.1579）
- **現存版本**：
  - 《大正藏》第43冊 No.1829（十六卷本）
  - 日本京都帝国大学文学部藏旧钞本
- **相關文獻**：窥基《成唯识论述记》《大乘法苑义林章》《因明入正理论疏》等

## 三、思想史分析

### 3.1 歷史背景

《瑜伽师地论》是印度瑜伽行派（唯识学派）的根本论书，相传为弥勒菩萨所造，由无著传述。全书一百卷，规模宏大，内容涵盖佛教教相、观行、果位等全部领域，是印度佛教论藏中篇幅最长、体系最完整的著作之一。玄奘在印度游学期间，对《瑜伽师地论》进行了深入研究，回国后于贞观二十年至二十二年（646–648）将其翻译为中文。

然而，《瑜伽师地论》一百卷的篇幅对一般学者来说过于庞大，且义理深奥，需要系统的导读。窥基作为玄奘的嫡传弟子，在玄奘的指导下对《瑜伽师地论》进行了深入研究，并撰写《瑜伽师地论略纂》，对全书内容进行提炼和阐释，为学习者提供了一条进入这部巨著的捷径。

### 3.2 核心論題

《瑜伽师地论》全书分为五分（五个部分），窥基的《略纂》对此五分进行了系统的提炼：

1. **本地分**：这是全书的核心，将瑜伽师的修行地分为十七地，从五识身相应地到无余依涅槃地，系统阐述从凡夫到佛的全部修行阶位。窥基在《略纂》中详细解释了十七地的含义、内容和相互关系。
2. **摄决择分**：对本地分中重要论题的深入抉择和论证。窥基提炼了其中关于五蕴、十二处、十八界、四谛、缘起等核心教义的分析。
3. **摄释分**：对佛教经典体裁和解释方法的阐述。窥基解释了佛教经典的十二分教（十二种体裁）及其解释原则。
4. **摄异门分**：对佛教术语不同名称的解释。窥基系统整理了佛教中同一概念的不同表述方式。
5. **摄事分**：对佛教具体事项（如戒律、禅定、智慧等）的详细阐述。窥基提炼了其中关于戒律、禅定、智慧的修行方法。

### 3.3 學術評價

- **吕澂**评价《瑜伽师地论略纂》为"进入《瑜伽师地论》的门户"，认为没有此书，学者很难把握《瑜伽师地论》的庞大体系。
- **日本学者**如宇井伯寿、结城令闻等，将《略纂》视为研究玄奘译传和瑜伽行派中国化的重要资料，尤其重视窥基对十七地的解释。
- **现代学者**关注该书与《瑜伽师地论》原文的关系，认为窥基在忠实于原文的基础上，也融入了自己的理解和发挥，体现了唐代唯识学的特色。
- 当代研究还关注该书的修行论价值，因为十七地的修行体系是佛教修行论最完整的系统之一，对现代佛教修行实践仍有参考价值。

| 瑜伽行派 | Yoga-cāra | 印度佛教大乘学派，以瑜伽修行为核心，主张万法唯识 |
| 十七地 | sapta-daśa-bhūmi | 瑜伽师修行的十七个阶位，从凡夫到佛 |
| 本地分 | maula-bhūmi | 《瑜伽师地论》第一分，阐述十七地的本地修行 |
| 五識身相應地 | 无对应梵文 | 十七地中的第一地，与五识（眼耳鼻舌身）相应的修行阶位 |
| 無餘依涅槃地 | 无对应梵文 | 十七地中的最高地，断除一切烦恼和习气，证得无余涅槃 |
| 攝決擇分 | 无对应梵文 | 《瑜伽师地论》第二分，对本地分重要论题的深入抉择 |
