# 大乘起信論了義

## 基本信息

- **編號**：01847
- **類別**：大乘起信論疏釋
- **狀態**：已完成

## 學術解讀

### 一、文獻學信息

《大乘起信論了義》是唐代賢首法藏（643-712）的再傳弟子或門人所撰，一說爲宋代淨源（1011-1088）所撰。淨源，俗姓楊，字伯長，泉州晉水（今福建晉江）人，爲宋代華嚴宗復興的關鍵人物，世稱"晉水淨源"，被尊爲宋代華嚴宗的中興祖師。此疏以"了義"爲名，取"了達義理，究竟圓滿"之意，是華嚴宗解釋《大乘起信論》的重要著作。

淨源一生致力於華嚴宗的復興，當時華嚴宗經唐末五代戰亂後典籍散佚，淨源從日本、高麗求回大量華嚴宗典籍，重新整理弘揚。著有《華嚴經疏注》《華嚴原人論發微錄》等。此疏作於其中年，是其以華嚴宗"法界緣起"思想統攝《起信論》的代表作。

### 二、歷史背景

宋代是華嚴宗復興的重要時期。淨源從日本、高麗求回法藏、澄觀等人的著作，重新建立華嚴宗的學統。當時禪宗盛行，教理之學衰落，淨源主張"教禪並重"，以華嚴宗的圓融思想來統攝各宗。《起信論》作爲"一心二門"思想的經典，自然成爲淨源復興華嚴宗的重要教理資源。此疏正是在這一背景下產生。

### 三、思想史定位

淨源此疏的特色在於以華嚴宗的"法界三觀"（真空觀、理事無礙觀、周遍含容觀）來解釋《起信論》的"一心二門"。他認爲，"心真如門"即是"真空觀"——觀心的本體爲空、爲清淨；"心生滅門"即是"理事無礙觀"——觀心的現象與本體不相障礙；"一心二門不二"即是"周遍含容觀"——觀一心含容萬法、萬法歸於一心的圓融境界。淨源進一步將"修行"解釋爲"稱性修行"——依本性的功德而起修行，不是從外面獲得什麼。這一解釋對後來華嚴宗與《起信論》的關係有重要影響。

## 白話解讀

### 核心內容

《大乘起信論了義》是宋代華嚴宗大師淨源對《大乘起信論》的解釋。淨源大師是華嚴宗復興的關鍵人物，他從日本、高麗找回了很多失傳的華嚴宗典籍。

淨源大師用華嚴宗的"法界三觀"來解釋《起信論》：
- **心真如門** = **真空觀**：觀察心的本體，它是空的、清淨的、不生不滅的。
- **心生滅門** = **理事無礙觀**：觀察心的現象（念頭、情緒），它們與本體不是對立的，而是相互無礙的。
- **二門不二** = **周遍含容觀**：最後明白，一心包含萬法，萬法歸於一心的圓融境界。

淨源大師還說：修行不是從外面得到什麼，而是"稱性修行"——依本性的功德而起修行。就像點燈：燈本來就亮，只是被燈罩遮住了。修行就是去掉燈罩，讓光明自然顯現。

### 現代啓示

1. **圓融思維**：華嚴宗的"周遍含容"啓示我們：不要把自己侷限在狹小的視角里。每一個個體都與整體相連，整體也包含在每一個個體中。

2. **稱性修行**："稱性修行"告訴我們：修行不是"做加法"（從外面獲得），而是"做減法"（去掉遮蔽）。這與現代心理學的"自我實現"理念相通。

3. **教禪並重**：淨源主張"教禪並重"，啓示我們：理論學習和實踐修行不可偏廢。只學理論容易空談，只重實踐容易盲修。
