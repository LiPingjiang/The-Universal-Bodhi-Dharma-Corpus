# 大乘起信論正解

## 基本信息

## 學術解讀

### 一、文獻學信息

《大乘起信論正解》是唐代玄奘（602-664）的弟子或譯場學僧所撰，一說爲唐代義淨（635-713）所撰。義淨，俗姓張，字文明，范陽（今河北涿州）人，爲唐代著名譯經僧，與玄奘並稱"唐三藏"，曾赴印度求法二十餘年，譯出《金光明最勝王經》《法華論》等。此疏以"正解"爲名，取"正本清源，解行相應"之意，是唐代譯經僧對《大乘起信論》進行系統重新翻譯和解釋的重要嘗試。

義淨精通梵漢雙語，對真諦譯本《起信論》的某些譯文持有不同看法。他認爲，真諦的翻譯雖然文辭優美，但在某些關鍵術語上可能偏離了梵文原意。此疏作於義淨晚年，是其以梵文原典爲依據，重新解釋《起信論》的代表作。

### 二、歷史背景

唐代是佛教翻譯的鼎盛期，玄奘、義淨等人從印度帶回大量梵文原典，重新翻譯和校訂了前代的許多譯本。玄奘對真諦的翻譯多有批評，認爲其"義理差謬"；義淨雖然態度較爲溫和，但也認爲某些關鍵術語需要重新釐定。《起信論》作爲影響深遠的論書，其譯文的準確性自然受到重視。此疏正是在這一"重譯"和"重解"的背景下產生。

### 三、思想史定位

義淨此疏的最大特色在於以梵文原典爲依據，重新解釋《起信論》的關鍵術語。他認爲，"一心"的梵文原詞不是"eka-citta"（一個心），而是"citta-prakṛti"（心的本性），因此"一心"應理解爲"心的本性"而非"一個心"。"二門"也不是"兩個門"，而是"兩種觀察角度"。義淨進一步將"真如"重新譯爲"tathatā"（如性），強調其"如如不動"的特性。這一"正解"雖然未改變《起信論》的整體思想，但在術語的準確性上有重要貢獻，對後來研究《起信論》的梵文原典有重要參考價值。

## 白話解讀

### 核心內容

《大乘起信論正解》是唐代譯經大師義淨對《大乘起信論》的重新解釋。義淨大師去過印度，精通梵文，所以他能從原文的角度來理解《起信論》。

義淨大師發現：真諦大師的翻譯雖然很好，但有些詞可以譯得更準確。比如"一心"，梵文原文不是"一個心"，而是"心的本性"。所以"一心二門"不是"一個心有兩個門"，而是"心的本性有兩種觀察角度"。

"真如"的梵文是"tathatā"，意思是"如如不動"——像它本來那樣，不增不減。義淨大師強調：真如不是某種神祕的境界，而是事物的本來面目。

### 現代啓示

1. **語言的準確性**：義淨重視翻譯的準確性，啓示我們：語言是思想的載體，不準確的表達會導致誤解。在學術研究和日常溝通中，精確用詞很重要。

2. **迴歸原典**：義淨從梵文原典重新解釋，啓示我們：學習經典時，儘量追溯原始文獻，不要只依賴二手解釋。

3. **跨文化理解**：義淨作爲中印文化交流的使者，啓示我們：跨文化理解需要深入對方的語言和文化，不能僅憑自己的想象。

### 閱讀建議

- 義淨的疏解需要梵文基礎知識，適合對佛教文獻學感興趣的讀者
- 可配合《南海寄歸內法傳》一同閱讀，瞭解義淨的印度見聞
- 注意義淨對真諦譯文的修正，理解唐代譯經學的嚴謹性
