# 法華經文句

## 一、經典概說

《法華經文句》是隋代天台宗實際創立者智者大師（538—597）撰寫的重要著作，是對《妙法蓮華經》經文的逐句解釋。此書與《法華經玄義》、《摩訶止觀》並稱為「天台三大部」，是天台宗理論體系的重要組成部分。智者大師以「四釋」——因緣釋、約教釋、本跡釋、觀心釋——為方法，對《法華經》二十八品的經文進行了詳細的解釋，是天台宗最重要的經文注釋之一。

智者大師俗姓陳，字德安，生於荊州華容（今湖北監利）。他十八歲出家，先後師從慧思、慧曠等名僧，深研《法華經》等大乘經典。後來他入天台山修行，證得「法華三昧」，創立了天台宗。他的著作極為豐富，除「天台三大部」外，還有《觀音玄義》、《觀音義疏》、《金光明經玄義》、《金光明經文句》、《觀經疏》等，對中國佛教的發展影響深遠。

《法華經》是印度大乘佛教的重要經典，由鳩摩羅什譯出，共七卷二十八品。此經以「三乘歸一」、「會三歸一」為核心思想，認為聲聞、緣覺、菩薩三乘最終都歸於佛乘，一切眾生都有佛性，都能成佛。這種「一佛乘」的思想，是大乘佛教的重要教義。

由於《法華經》經文優美、義理深奧，需要逐句的注釋來幫助理解。智者大師的《法華經文句》正是這樣一部著作，它與《法華經玄義》互相配合：《玄義》偏重理論闡發，《文句》偏重經文解釋。兩者合讀，才能全面理解《法華經》的義理。

## 二、核心內容

《法華經文句》的核心內容是對《法華經》二十八品經文的逐句解釋。主要包括以下幾個方面：

**四釋方法**：智者大師在解釋經文時，使用了「四釋」的方法：
- **因緣釋**：從歷史因緣的角度解釋經文，說明佛為什麼在這個時候、這個地點、對這些人說這些話。
- **約教釋**：從教理的角度解釋經文，說明這句經文屬於哪一種教法（藏、通、別、圓）。
- **本跡釋**：從「本」和「跡」的角度解釋經文。「本」是佛的本來面目，「跡」是佛示現的化身。這種解釋方法，是天台宗的特色。
- **觀心釋**：從觀心修行的角度解釋經文，說明這句經文對我們的修行有什麼啟發。

這「四釋」方法，使經文的解釋更加全面、更加深入，是天台宗的重要貢獻。

**二十八品解釋**：《法華經》分為二十八品，每一品都有不同的主題和內容。智者大師對每一品都進行了詳細的解釋：
- **序品**：解釋經文的序分，說明說經的時間、地點、聽眾等。
- **方便品**：解釋「會三歸一」的道理，說明佛為什麼要說三乘。
- **譬喻品**：解釋「三車喻」、「窮子喻」等著名比喻。
- **信解品**：解釋菩薩對「一佛乘」的信解。
- **藥草喻品**：解釋「藥草喻」，說明佛如何根據眾生的根機說法。
- **授記品**：解釋佛為聲聞弟子授記成佛。
- **化城喻品**：解釋「化城喻」，說明二乘是「化城」，不是究竟的目的地。
- **五百弟子授記品**：解釋佛為五百弟子授記。
- **授學無學人記品**：解釋佛為有學、無學弟子授記。
- **法師品**：解釋如何弘揚《法華經》。
- **見寶塔品**：解釋多寶佛塔的出現，象徵「法身常在」。
- **提婆達多品**：解釋提婆達多也能成佛，顯示「一佛乘」的普遍性。
- **持品**：解釋如何受持《法華經》。
- **安樂行品**：解釋修行《法華經》的四種安樂行。
- **從地湧出品**：解釋無量菩薩從地湧出，顯示《法華經》的殊勝。
- **如來壽量品**：解釋佛的壽量無量，顯示「法身常在」。
- **分別功德品**：解釋受持《法華經》的功德。
- **隨喜功德品**：解釋隨喜《法華經》的功德。
- **法師功德品**：解釋法師的功德。
- **常不輕菩薩品**：解釋常不輕菩薩的故事，顯示「一切眾生皆可成佛」。
- **如來神力品**：解釋佛顯示神力，護持《法華經》。
- **囑累品**：解釋佛囑咐菩薩弘揚《法華經》。
- **藥王菩薩本事品**：解釋藥王菩薩的故事，顯示為法捨身的精神。
- **妙音菩薩品**：解釋妙音菩薩的故事，顯示菩薩的方便善巧。
- **觀世音菩薩普門品**：解釋觀世音菩薩的普門示現，救度眾生。
- **陀羅尼品**：解釋陀羅尼的功德。
- **妙莊嚴王本事品**：解釋妙莊嚴王的故事，顯示「一佛乘」的普遍性。
- **普賢菩薩勸發品**：解釋普賢菩薩勸發眾生受持《法華經》。

## 三、經典結構

《法華經文句》的結構分為以下幾個部分：

**第一部分：總序**。說明撰寫此書的緣起、目的、方法等。智者大師強調，此書是為了幫助眾生理解《法華經》的經文，與《法華經玄義》互相配合，共同闡發《法華經》的義理。

**第二部分：逐品解釋**。按照《法華經》二十八品的順序，逐品進行解釋。每一品的解釋通常包括：說明此品的結構、解釋經文的字面意義、用四釋方法闡發深層義理、聯繫其他經論、總結此品的要義等。

**第三部分：結論**。總結全書的要點，強調《法華經》的殊勝性，勸導讀者勤奮學習。

## 四、主要思想特色

**（一）四釋方法**：《法華經文句》的最大特色是「四釋」方法。這種從四個層面來解釋經文的方法，使經文的解釋更加全面、更加深入。因緣釋偏重歷史，約教釋偏重教理，本跡釋偏重本體，觀心釋偏重實踐。四者合起來，構成了完整的詮釋體系。

**（二）本跡思想**：「本跡釋」是天台宗的特色。智者大師認為，佛有「本」和「跡」兩個層面：「本」是佛的真實法身，「跡」是佛示現的應化身。《法華經》中說的佛，有時是「本」，有時是「跡」，需要仔細分辨。這種「本跡」思想，是天台宗對《法華經》的重要詮釋。

**（三）觀心實踐**：「觀心釋」是天台宗的重要特色。智者大師強調，解釋經文不是為了知識，而是為了修行。每一句經文都可以與我們的心相應，都可以成為觀心的對象。這種「觀心釋」的方法，體現了天台宗「解行相應」的傳統。

**（四）圓教思想**：智者大師在《法華經文句》中處處體現「圓教」的思想。圓教是天台宗最高的教法，認為一切諸法都是圓融無礙的，一即一切，一切即一。這種圓融的思想，貫穿於《法華經文句》的每一品解釋中。

**（五）會通各家**：智者大師雖然是天台宗的學者，但他的視野非常開闊。他在《法華經文句》中不僅闡述天台宗的思想，還會通其他宗派的觀點，如三論宗、華嚴宗、唯識宗等。這種「會通各家」的態度，體現了智者大師的博學和開放。

## 五、歷史影響

《法華經文句》在中國佛教史上佔有極為重要的地位。天台宗是中國佛教最早成立的宗派之一，智者大師被尊為「東土小釋迦」。《法華經文句》是天台宗最重要的經文注釋之一，對天台宗的發展影響深遠。

此書對其他宗派也有重要影響。華嚴宗的法藏大師、禪宗的六祖惠能、淨土宗的善導大師等，都曾研讀《法華經文句》，並從中汲取思想資源。特別是淨土宗，雖然以淨土經典為依據，但也重視《法華經》的「一佛乘」思想。

此書對日本佛教的影響也很大。日本天台宗在平安時代由最澄大師從中國傳入，成為日本佛教最重要的宗派之一。日本天台宗的學者對《法華經文句》進行了深入的研究，形成了獨特的詮釋傳統。近代以來，隨著佛學研究的復興，此書重新受到學術界的重視。

此書不僅是宗教修行的指南，也是研究中國佛教思想史的重要文獻。智者大師對《法華經》的詮釋，反映了隋代佛教思想的特點，對於研究中國佛教的發展具有重要價值。書中對「四釋」、「本跡」、「觀心」、「圓教」等概念的闡發，也體現了中國佛教哲學的獨特性，對於中國哲學史和宗教學的研究都有重要意義。特別是「四釋」方法，是中國佛教詮釋學的重要貢獻，對後來的經典解釋產生了深遠的影響。
