## 大众部源流骨架

大众部（Mahāsāṅghika，"大众"，即人数众多的一派）是部派佛教中人数最多、影响最广的一派。根本分裂后，大众部主张灵活、开放，被现代学界普遍视为大乘思想的先驱与孵化器：

```
        大众部系（Mahāsāṅghika，大众一系）
        ├── 一说部（Ekavyāvahārika）→ 一切皆名言假立
        ├── 说出世部（Lokottaravāda）→ 佛陀超越世间，出世间法
        │       └── 《大事》（Mahāvastu）→ 菩萨道文学
        ├── 鸡胤部（Kukkuṭika / Gokulika）→ 世间法皆假，出世间法实有
        ├── 多闻部（Bahuśrutīya）→ 广学多闻，兼容并蓄
        ├── 说假部（Prajñaptivāda）→ 一切皆假名施设
        └── 制多山部（Caitika）→ 南印度大众部
                ├── 西山住部（Aparaśaila）
                └── 北山住部（Uttaraśaila）
```

大众部的核心特色在于：主张佛陀超越世间（出世间论）、阿罗汉有缺陷（非圆满论）、心性本净（心性清净论）、一念顿悟（顿悟论）、一切法空（空观先驱）。这五大倾向直接为后来大乘佛教的佛身论、菩萨道、般若空观奠定了思想基础。
