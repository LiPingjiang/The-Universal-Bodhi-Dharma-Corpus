## 大衆部源流骨架

大衆部（Mahāsāṅghika，"大衆"，即人數衆多的一派）是部派佛教中人數最多、影響最廣的一派。根本分裂後，大衆部主張靈活、開放，被現代學界普遍視爲大乘思想的先驅與孵化器：

```
        大众部系（Mahāsāṅghika，大众一系）
        ├── 一说部（Ekavyāvahārika）→ 一切皆名言假立
        ├── 说出世部（Lokottaravāda）→ 佛陀超越世间，出世间法
        │       └── 《大事》（Mahāvastu）→ 菩萨道文学
        ├── 鸡胤部（Kukkuṭika / Gokulika）→ 世间法皆假，出世间法实有
        ├── 多闻部（Bahuśrutīya）→ 广学多闻，兼容并蓄
        ├── 说假部（Prajñaptivāda）→ 一切皆假名施设
        └── 制多山部（Caitika）→ 南印度大众部
                ├── 西山住部（Aparaśaila）
                └── 北山住部（Uttaraśaila）
```

大衆部的核心特色在於：主張佛陀超越世間（出世間論）、阿羅漢有缺陷（非圓滿論）、心性本淨（心性清淨論）、一念頓悟（頓悟論）、一切法空（空觀先驅）。這五大傾向直接爲後來大乘佛教的佛身論、菩薩道、般若空觀奠定了思想基礎。
