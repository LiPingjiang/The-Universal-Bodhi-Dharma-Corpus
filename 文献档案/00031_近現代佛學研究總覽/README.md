## 目錄簡介

本目錄收錄近現代佛學研究的核心文獻，涵蓋19世紀以來西方學術界對佛教的科學研究、中國近現代佛學復興運動，以及當代佛教與科學、心理學、哲學的對話。

**時代範圍**：19世紀至今  
**核心主題**：西方佛教學、中國近現代佛學復興、佛教與科學對話、人間佛教

## 關鍵概念

- **東方學**（Orientalism）：19世紀西方對東方文化的學術研究
- **巴利文學會**（Pali Text Society）：1881年成立，系統整理巴利文佛典
- **人間佛教**：太虛大師提出，強調佛教在現實社會中的實踐
- **批判佛教**（Critical Buddhism）：日本學者提出，批評如來藏思想
- **心智科學**（Mind Science）：佛教禪修與認知科學的交叉研究
