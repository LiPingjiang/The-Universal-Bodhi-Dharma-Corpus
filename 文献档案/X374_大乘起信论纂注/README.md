## 学术解读

### 一、文献学信息

《大乘起信论纂注》是明代云栖祩宏（1535-1615）所撰。祩宏，俗姓沈，字佛慧，号莲池，世称"云栖大师"，为明代四大高僧之一，净土宗第八代祖师。此疏以"纂注"为名，取"纂集诸家注释，会归一心"之意，是《大乘起信论》注疏中较为晚出但影响深远的一部。

祩宏一生融通禅、净、教、戒，主张"禅净双修""儒释合一"。其《纂注》不仅疏解《起信论》文义，更结合净土思想，将"一心二门"与"念佛往生"相融通，形成独特的诠释体系。

### 二、历史背景

明代万历年间，佛教内部宗派林立，禅净之争、教观之争时有发生。祩宏居杭州云栖寺，广收徒众，倡导各宗融合。此疏作于其晚年，旨在为学人提供一条简明扼要的《起信论》入门路径。祩宏在疏中大量引用前代注疏（如法藏《义记》、元晓《疏》、宗密《注疏》等），加以裁择会通，故名"纂注"。

### 三、思想史定位

祩宏此疏的最大特色是将《起信论》的"一心二门"与净土宗的"信愿行"相结合。他认为，"信"不仅是《起信论》的开端，也是净土宗的核心——"信有西方净土，信有阿弥陀佛，信有往生之事"。祩宏将"心真如门"解释为"自性弥陀"，"心生灭门"解释为"唯心净土"，从而打通了教理与修行的关节。这一思想对后世"净土即禅""禅净不二"的思潮有重要影响。

## 白话解读

### 核心内容

《大乘起信论纂注》是莲池大师祩宏对《大乘起信论》的集注。莲池大师把历代高僧对《起信论》的解释收集起来，选择最精要的部分，加上自己的理解，编成这部注疏。

莲池大师的独特贡献在于：他把《起信论》的"信"与净土宗的"信"联系起来。《起信论》说"信"是修行的起点，莲池大师说：信什么？信自己本有佛性，信阿弥陀佛的救度，信西方净土的真实。这样，教理（《起信论》）和修行（净土法门）就打通了。

### 现代启示

1. **信仰的力量**：现代人常将"信"理解为盲目相信，但佛教中的"信"是"正信"——基于理解的相信。莲池大师告诉我们：信不是放弃思考，而是思考后的选择。

2. **整合智慧**：莲池大师融通禅、净、教、戒，启示我们在面对复杂问题时，不要局限于单一视角，而要整合多种智慧资源。

3. **简化修行**：莲池大师主张"持名念佛"，认为最简单的修行方式往往最有效。这对现代人忙碌的生活特别有启示。

### 阅读建议

- 莲池大师的注疏较为平实，适合初学者
- 可配合《云栖法汇》中的其他著作，如《弥陀疏钞》《竹窗随笔》等一同阅读
- 注意莲池大师融通各宗的特点，理解其"会归净土"的立场
