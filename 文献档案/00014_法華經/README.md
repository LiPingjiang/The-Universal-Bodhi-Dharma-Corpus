## 法華經源流骨架

《法華經》（Saddharma-puṇḍarīka-sūtra，"妙法蓮華經"，"正法白蓮花經"）是大乘佛教最重要的經典之一，也是中國佛教天台宗的根本經典。"妙法"指佛陀最深妙的教法，"蓮華"象徵出污泥而不染的清淨，"蓮華"也是佛陀教法的比喻——就像蓮花在污泥中生長卻保持清淨，佛陀的教法在世間流傳卻超越世間。

```
        法华经思想体系
        ├── 一乘思想（Ekayāna）
        │       ├── 开权显实 → 三乘（声闻、缘觉、菩萨）皆归一佛乘
        │       ├── 会三归一 → 一切修行道路最终归于成佛
        │       └── 佛乘唯一 → 只有一乘，无有二乘三乘
        ├── 久远实成（Ādi-buddha）
        │       ├── 佛陀久远劫前已成佛 → 非历史上才成佛
        │       ├── 寿量无限 → 佛陀常住不灭
        │       └── 方便示现 → 在世间示现成道、入灭，皆是方便
        └── 普度思想
                ├── 龙女成佛 → 女性可以成佛
                ├── 提婆达多授记 → 恶人可以成佛
                └── 观音普门 → 观音菩萨普度一切众生
```

《法華經》的核心思想是「一乘」（Ekayāna，"一個車乘"）：佛陀過去所說的三乘（聲聞乘、緣覺乘、菩薩乘）都是方便（upāya，"善巧方便"，即根據衆生根機而設的教法），最終都歸於一佛乘（Buddha-yāna）。這就是"開權顯實"——開顯方便（權），顯示真實（實）。

《法華經》對中國佛教的影響極爲深遠。天台宗以《法華經》爲根本經典，建立了"五時八教"的判教體系，將《法華經》判爲佛陀一生說法的最高峯。鳩摩羅什（Kumārajīva，344-413）的漢譯本《妙法蓮華經》（T262）是中國佛教史上最重要的譯本之一，文字優美，影響深遠。

## 關鍵概念速查

| 概念 | 梵/巴利語 | 簡釋 |
|------|-----------|------|
| 一乘 | ekayāna | 唯一的修行道路，即佛乘 |
| 開權顯實 | upāya-kauśalya | 開顯方便，顯示真實 |
| 方便 | upāya | 善巧方便，根據衆生根機而設的教法 |
| 久遠實成 | ādi-buddha | 佛陀久遠劫前已成佛，非歷史上才成佛 |
| 三乘 | triyāna | 聲聞乘、緣覺乘、菩薩乘 |
| 佛乘 | buddha-yāna | 最高的修行道路，成佛之道 |
