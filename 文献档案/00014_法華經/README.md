## 法华经源流骨架

《法华经》（Saddharma-puṇḍarīka-sūtra，"妙法莲华经"，"正法白莲花经"）是大乘佛教最重要的经典之一，也是中国佛教天台宗的根本经典。"妙法"指佛陀最深妙的教法，"莲华"象征出污泥而不染的清净，"莲华"也是佛陀教法的比喻——就像莲花在污泥中生长却保持清净，佛陀的教法在世间流传却超越世间。

```
        法华经思想体系
        ├── 一乘思想（Ekayāna）
        │       ├── 开权显实 → 三乘（声闻、缘觉、菩萨）皆归一佛乘
        │       ├── 会三归一 → 一切修行道路最终归于成佛
        │       └── 佛乘唯一 → 只有一乘，无有二乘三乘
        ├── 久远实成（Ādi-buddha）
        │       ├── 佛陀久远劫前已成佛 → 非历史上才成佛
        │       ├── 寿量无限 → 佛陀常住不灭
        │       └── 方便示现 → 在世间示现成道、入灭，皆是方便
        └── 普度思想
                ├── 龙女成佛 → 女性可以成佛
                ├── 提婆达多授记 → 恶人可以成佛
                └── 观音普门 → 观音菩萨普度一切众生
```

《法华经》的核心思想是「一乘」（Ekayāna，"一个车乘"）：佛陀过去所说的三乘（声闻乘、缘觉乘、菩萨乘）都是方便（upāya，"善巧方便"，即根据众生根机而设的教法），最终都归于一佛乘（Buddha-yāna）。这就是"开权显实"——开显方便（权），显示真实（实）。

《法华经》对中国佛教的影响极为深远。天台宗以《法华经》为根本经典，建立了"五时八教"的判教体系，将《法华经》判为佛陀一生说法的最高峰。鸠摩罗什（Kumārajīva，344-413）的汉译本《妙法莲华经》（T262）是中国佛教史上最重要的译本之一，文字优美，影响深远。

## 关键概念速查

| 概念 | 梵/巴利语 | 简释 |
|------|-----------|------|
| 一乘 | ekayāna | 唯一的修行道路，即佛乘 |
| 开权显实 | upāya-kauśalya | 开显方便，显示真实 |
| 方便 | upāya | 善巧方便，根据众生根机而设的教法 |
| 久远实成 | ādi-buddha | 佛陀久远劫前已成佛，非历史上才成佛 |
| 三乘 | triyāna | 声闻乘、缘觉乘、菩萨乘 |
| 佛乘 | buddha-yāna | 最高的修行道路，成佛之道 |
