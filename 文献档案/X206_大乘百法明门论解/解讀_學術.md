# 大乘百法明门论解 · 學術解讀

## 一、文獻定位

《大乘百法明门论解》为唐代唯识学大师窥基（632–682）所撰，是世亲《大乘百法明门论》的注释书。《大乘百法明门论》是唯识宗最重要的法相分类学著作，将大乘佛教所说的一切法（现象）归纳为五大类一百种，是进入唯识学的入门钥匙。窥基的《论解》对《百法明门论》进行了详细的逐句解释，是研究唐代唯识宗法相学、唯识学思想体系及佛教现象分类学的核心文本。

## 二、版本與傳承

- **作者**：窥基（注释者）；世亲（原论作者，约320–400）
- **成书年代**：约唐高宗时期（650–683）

## 三、思想史分析

### 3.1 歷史背景

《大乘百法明门论》是印度唯识学创始人世亲所撰的一篇短论，全文仅约一千字，但内容极其精炼，将大乘佛教所说的"一切法"（一切现象）系统地分为五大类一百种。这一分类体系是唯识学对世界最细致、最全面的分析，被称为"百法"。

玄奘在印度游学期间，对百法体系进行了深入学习，回国后将其翻译为中文。由于原文极其精炼，每个概念都只是点到为止，需要详细的解释。窥基作为玄奘的嫡传弟子，撰写了《大乘百法明门论解》，对百法中的每一个法进行了详细的解释，包括其定义、性质、功能和与其他法的关系。

### 3.2 核心論題

《大乘百法明门论》将一切法分为五大类，窥基的《论解》对此进行了详细阐释：

1. **心法（八法）**：即八识——眼识、耳识、鼻识、舌识、身识、意识、末那识、阿赖耶识。这是认识的主体，是一切心理活动的基础。
2. **心所法（五十一法）**：与心相应的心理活动，包括：
   - 遍行心所（五）：作意、触、受、想、思——一切心理活动都具备的基本要素
   - 别境心所（五）：欲、胜解、念、定、慧——特定情境下生起的心理活动
   - 善心所（十一）：信、精进、惭、愧、无贪、无嗔、无痴、轻安、不放逸、行舍、不害——善良的心理品质
   - 烦恼心所（六）：贪、嗔、慢、无明、疑、恶见——根本烦恼
   - 随烦恼心所（二十）：忿、恨、覆、恼、嫉、悭、诳、谄、憍、害、无惭、无愧、昏沉、掉举、不信、懈怠、放逸、失念、散乱、不正知——随根本烦恼而生起的次要烦恼
   - 不定心所（四）：悔、眠、寻、伺——性质不定，可为善可为恶的心理活动
3. **色法（十一法）**：物质现象，包括五根（眼耳鼻舌身）和五尘（色声香味触）以及法处所摄色（意识所缘的物质对象）。
4. **心不相应行法（二十四法）**：既不属于心也不属于色的心理和物理的抽象概念，如得、命根、众同分、异生性、无想定、灭尽定、无想报、名身、句身、文身、生、老、住、无常、流转、定异、相应、势速、次第、时、方、数、和合性、不和合性。
5. **无为法（六法）**：超越生灭变化的本体，包括虚空无为、择灭无为、非择灭无为、不动灭无为、想受灭无为、真如无为。

### 3.3 學術評價

- **吕澂**评价《大乘百法明门论解》为"进入唯识学的最佳入门书"，认为百法体系是"佛教现象学的最高成就"。
- **日本学者**如宇井伯寿、山口益等，将百法体系视为研究印度佛教心理学和认识论的核心资料，尤其重视其对心理活动的精细分类。
- **现代心理学研究者**关注百法与现代心理学的对话，认为心所法的分类与西方心理学的情绪理论、动机理论有可比性。
- 当代研究还关注百法体系的哲学意义，认为它将佛教"一切法无我"的教义具体化为对每一个法的分析，展示了佛教分析哲学的特色。

## 四、術語對照

| 漢譯術語 | 梵文/巴利原文 | 現代語義 |
|---------|-------------|---------|
| 百法 | 无对应梵文 | 唯识宗将一切法分为一百种，是佛教最系统的现象分类学 |
| 心法 | citta-dharma | 八识，认识的主体，一切心理活动的基础 |
| 心所法 | caitasika-dharma | 与心相应的心理活动，共五十一类 |
| 色法 | rūpa-dharma | 物质现象，包括五根、五尘和法处所摄色 |
| 不相應行法 | citta-viprayukta-saṃskāra | 既非心也非色的抽象概念，如时间、空间、数量等 |
| 無為法 | asaṃskṛta-dharma | 超越生灭变化的本体，如虚空、真如、涅槃 |
| 遍行心所 | sarvaga-caitasika | 一切心理活动都具备的基本要素，如作意、触、受、想、思 |
| 煩惱心所 | kleśa-caitasika | 根本烦恼，如贪、嗔、慢、无明、疑、恶见 |
