## 一、《金刚顶经》的文献学考察

《金刚顶经》（梵名 Vajraśekhara-sūtra，又名 Sarvatathāgata-tattvasaṃgraha）约成立于公元7世纪印度南部，是瑜伽续（Yoga-tantra）的代表经典。

汉译本：不空（Amoghavajra，705–774）译，共3卷（T865）。金刚智（Vajrabodhi，671–741）亦有部分译本。

梵文本：完整梵文本存于日本高野山，日本学者有校勘研究。

## 二、金刚界九会曼荼罗

金刚界曼荼罗由九个曼荼罗（九会）组成，共1461尊：

| 会 | 名称 | 特点 |
|----|------|------|
| 第一会 | 羯磨会（成身会） | 主会，37尊，立体形式 |
| 第二会 | 三昧耶会 | 以持物象征本尊 |
| 第三会 | 微细会 | 极微细的本尊形象 |
| 第四会 | 供养会 | 供养本尊的形象 |
| 第五会 | 四印会 | 四种手印 |
| 第六会 | 一印会 | 一种手印，大日如来 |
| 第七会 | 理趣会 | 般若理趣经的内容 |
| 第八会 | 降三世羯磨会 | 降三世明王 |
| 第九会 | 降三世三昧耶会 | 降三世明王的三昧耶形 |

## 三、五智与五蕴的转化

金刚界的核心思想是"转识成智"——将凡夫的五蕴（色受想行识）转化为五智：

| 五蕴 | 五智 | 五方佛 |
|------|------|--------|
| 识蕴 | 大圆镜智 | 阿閦佛 |
| 受蕴 | 平等性智 | 宝生佛 |
| 想蕴 | 妙观察智 | 阿弥陀佛 |
| 行蕴 | 成所作智 | 不空成就佛 |
| 色蕴 | 法界体性智 | 毗卢遮那佛 |
