## 两部大经源流骨架

《大日经》（Mahāvairocana-abhisaṃbodhi-vikurvitādhiṣṭhāna-vaipulya-sūtra，简称《大毗卢遮那成佛神变加持经》）与《金刚顶经》（Vajraśekhara-sūtra，全称《金刚顶一切如来真实摄大乘现证大教王经》）是正纯密教（汉传密教的核心）的两部根本大经，分别代表胎藏界（Garbhadhātu）和金刚界（Vajradhātu）两大法门：

```
        两部大经体系
        ├── 《大日经》（Mahāvairocana-sūtra）— 胎藏界
        │       ├── 成立：约7世纪，印度中部
        │       ├── 汉译：善无畏（Śubhākarasiṃha）与一行，开元十二年（724）
        │       ├── 藏译：存（Rnam par snang mdzad chen po mngon par rdzogs par byang chub pa）
        │       ├── 核心：大日如来（Mahāvairocana）的法身说法，胎藏界曼荼罗
        │       └── 思想：理（胎藏）与智（金刚）的统一，"六大"体大日
        └── 《金刚顶经》（Vajraśekhara-sūtra）— 金刚界
                ├── 成立：约7世纪，印度南部
                ├── 汉译：不空（Amoghavajra），天宝年间（742–756）
                ├── 藏译：存
                ├── 核心：金刚界曼荼罗，五智如来，三十七尊
                └── 思想：五智（五方佛）体系，金刚界九会曼荼罗
```

两部大经在日本密教（东密、台密）中被称为"两部大法"，是真言宗（东密）的根本经典。空海（弘法大师，774–835）将两部大法从中国带回日本，建立了东密传统。在汉传密教中，善无畏、金刚智、不空三藏（开元三大士）是传播两部大法的关键人物。
