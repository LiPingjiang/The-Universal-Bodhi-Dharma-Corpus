## 兩部大經源流骨架

《大日經》（Mahāvairocana-abhisaṃbodhi-vikurvitādhiṣṭhāna-vaipulya-sūtra，簡稱《大毗盧遮那成佛神變加持經》）與《金剛頂經》（Vajraśekhara-sūtra，全稱《金剛頂一切如來真實攝大乘現證大教王經》）是正純密教（漢傳密教的核心）的兩部根本大經，分別代表胎藏界（Garbhadhātu）和金剛界（Vajradhātu）兩大法門：

```
        两部大经体系
        ├── 《大日经》（Mahāvairocana-sūtra）— 胎藏界
        │       ├── 成立：约7世纪，印度中部
        │       ├── 汉译：善无畏（Śubhākarasiṃha）与一行，开元十二年（724）
        │       ├── 藏译：存（Rnam par snang mdzad chen po mngon par rdzogs par byang chub pa）
        │       ├── 核心：大日如来（Mahāvairocana）的法身说法，胎藏界曼荼罗
        │       └── 思想：理（胎藏）与智（金刚）的统一，"六大"体大日
        └── 《金刚顶经》（Vajraśekhara-sūtra）— 金刚界
                ├── 成立：约7世纪，印度南部
                ├── 汉译：不空（Amoghavajra），天宝年间（742–756）
                ├── 藏译：存
                ├── 核心：金刚界曼荼罗，五智如来，三十七尊
                └── 思想：五智（五方佛）体系，金刚界九会曼荼罗
```

兩部大經在日本密教（東密、臺密）中被稱爲"兩部大法"，是真言宗（東密）的根本經典。空海（弘法大師，774–835）將兩部大法從中國帶回日本，建立了東密傳統。在漢傳密教中，善無畏、金剛智、不空三藏（開元三大士）是傳播兩部大法的關鍵人物。
