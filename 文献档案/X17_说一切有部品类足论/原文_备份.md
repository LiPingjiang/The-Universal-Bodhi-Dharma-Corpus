# 說一切有部品類足論 · 原文

## 文獻基本信息

- **梵名**：Saṃgīti-paryāya-śāstra / Prakaraṇa-pāda-śāstra
- **巴利名**：Saṅgīti-paryāya（擬）
- **漢譯**：玄奘譯《阿毘達磨品類足論》
- **年代**：唐高宗顯慶年間（公元656-659年）
- **卷數**：十八卷
- **所屬部派**：說一切有部（Sarvāstivāda）
- **作者**：傳為世友菩薩（Vasumitra）所造
- **藏經編號**：大正藏第1542號（阿毘達磨部）

## 經典概要

《說一切有部品類足論》，簡稱《品類足論》，是說一切有部「六足論」之一，被視為《發智論》（身論）的重要輔助論書。此論由世友菩薩（Vasumitra）所造，玄奘法師於唐高宗顯慶年間譯出，共十八卷。品類足論之「品類」，意為對諸法進行分類整理；「足」，意為輔助身論（發智論）之足。

此論的特點在於以系統分類的方式，將佛教教理中的各種法相進行整理和歸類。全論分為八品，分別從不同角度對一切法進行分類，是研究說一切有部法相學的重要文獻。

## 核心內容

### 一、八品結構

《品類足論》分為八品，每品從一個特定的角度對諸法進行分類：

1. **辯五事品**（Pañca-vastu-vibhāṣā）：將一切法分為五類——色、心、心所、心不相應行、無為。這是說一切有部最基本的法相分類，後來被《俱舍論》繼承和發展。

2. **辯諸智品**（Jñāna-vibhāṣā）：對各種智慧進行分類和辨析。包括法智、類智、世俗智、苦智、集智、滅智、道智等，詳細說明每種智慧的性質、所緣和作用。

3. **辯諸處品**（Āyatana-vibhāṣā）：對十二處（六根和六塵）進行分類和辨析。分析每一處的性質、功能及其與認識的關係。

4. **辯七事品**（Sapta-vastu-vibhāṣā）：將一切法分為七類進行討論，包括五蘊、十二處、十八界等核心概念。

5. **辯隨眠品**（Anuśaya-vibhāṣā）：對各種隨眠（潛在的煩惱）進行分類和辨析。包括貪、瞋、癡、慢、疑、見等六種根本隨眠，以及各種衍生隨眠。

6. **辯攝等品**（Saṃgraha-vibhāṣā）：對各種攝（歸納）的方式進行討論，包括蘊攝、處攝、界攝等。

7. **辯聖諦品**（Satya-vibhāṣā）：對四聖諦（苦、集、滅、道）進行詳細的辨析和闡述。

8. **辯界品**（Dhātu-vibhāṣā）：對十八界（六根、六塵、六識）進行分類和辨析。

### 二、五事分類法

《品類足論》最重要的貢獻之一，是建立了「五事」分類法。這一分類法將一切法分為五類：

1. **色法**（rūpa）：物質現象，包括地、水、火、風四大種及由四大種所造的各種色法，如眼、耳、鼻、舌、身等根身，以及色、聲、香、味、觸等外境。

2. **心法**（citta）：心識活動，即六識（眼識、耳識、鼻識、舌識、身識、意識）。心法是能緣，即能認識的主體。

3. **心所法**（caitasika）：與心識相應的心理活動。說一切有部認為有四十六種心所法（據《俱舍論》），包括受（感受）、想（概念）、思（意志）、觸（接觸）、作意（注意）、欲（欲望）、勝解（決定）、念（記憶）、定（專注）、慧（智慧）等。

4. **心不相應行法**（citta-viprayukta-saṃskāra）：既非色法也非心法的現象，但與心識不相應（即不與心識同時生起）。包括命根（壽命）、眾同分（種類共性）、時（時間）、方（空間）、數（數量）、生、住、異、滅等。

5. **無為法**（asaṃskṛta）：非因緣生起的法，即不由因緣造作而存在的法。包括虛空無為、擇滅無為（通過智慧選擇而證得的涅槃）、非擇滅無為（不由智慧選擇而自然達成的寂滅）等。

這種五事分類法，為後來的《俱舍論》和唯識學派的「五位百法」奠定了基礎。

### 三、隨眠理論

《品類足論》的「辯隨眠品」對煩惱理論有詳細的闡述。隨眠（anuśaya）意為潛在的煩惱，即潛伏在內心深處、尚未現行的煩惱種子。說一切有部將隨眠分為六種根本隨眠：

1. **貪隨眠**（rāgānuśaya）：對欲界和色界的貪愛
2. **瞋隨眠**（pratighānuśaya）：對不順境的憤怒
3. **慢隨眠**（mānānuśaya）：自高自大的心理
4. **無明隨眠**（avidyānuśaya）：對真理的無知
5. **疑隨眠**（vicikitsānuśaya）：對佛法的懷疑
6. **見隨眠**（dṛṣṭyānuśaya）：各種錯誤的見解，包括身見、邊見、邪見、見取、戒禁取等

這六種隨眠又可以細分為九十八種（據《俱舍論》），分別與三界（欲界、色界、無色界）和九地相應。這種精細的分類，體現了說一切有部對煩惱心理學的深入研究。

### 四、四聖諦的辨析

《品類足論》的「辯聖諦品」對四聖諦進行了詳細的辨析。四聖諦是佛教教理的核心：

1. **苦諦**（Duḥkha-satya）：一切有漏法皆是苦。這裡的「苦」不僅指痛苦的感受，更指一切有為法的無常性和不圓滿性。

2. **集諦**（Samudaya-satya）：苦的原因在於煩惱和業。貪、瞋、癡等煩惱驅使眾生造作惡業，惡業又導致苦果。

3. **滅諦**（Nirodha-satya）：苦可以滅除，滅除苦的境界就是涅槃。

4. **道諦**（Mārga-satya）：通向滅苦的道路，即八正道。

《品類足論》從多個角度對四聖諦進行分析，包括四聖諦的性質、所緣、作用、證得方式等。

## 結構說明

《品類足論》十八卷的結構，按照八品組織如下：

### 第一至第三卷：辯五事品
- 將一切法分為色、心、心所、心不相應行、無為五類
- 對每一類進行詳細的定義和說明

### 第四至第六卷：辯諸智品
- 對各種智慧進行分類和辨析
- 詳細說明每種智慧的性質和作用

### 第七至第八卷：辯諸處品
- 對十二處進行分類和辨析
- 分析每一處的功能和特性

### 第九至第十卷：辯七事品
- 將一切法分為七類進行討論
- 包括五蘊、十二處、十八界等

### 第十一至第十三卷：辯隨眠品
- 對各種隨眠進行分類和辨析
- 詳細說明每種隨眠的性質和對治方法

### 第十四至第十五卷：辯攝等品
- 對各種攝的方式進行討論
- 分析不同分類法之間的關係

### 第十六至第十七卷：辯聖諦品
- 對四聖諦進行詳細的辨析
- 從多個角度闡述四聖諦的內涵

### 第十八卷：辯界品
- 對十八界進行分類和辨析
- 總結全論的內容

這種結構體現了說一切有部對佛教教理的系統化整理，從基礎概念到核心教義，層次分明。

## 思想價值

《品類足論》的思想價值主要體現在以下幾個方面：

1. **建立法相學體系**：通過五事分類法，建立了佛教法相學的基本框架，為後來的佛教哲學發展奠定了基礎。

2. **深化煩惱理論**：對隨眠的精細分類和分析，深化了佛教對煩惱心理的理解。

3. **系統化教理**：通過八品的結構，將佛教教理從多個角度進行系統化的整理，便於學習和傳承。
