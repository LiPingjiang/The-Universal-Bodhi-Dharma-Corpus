# 小石柱法敕（Minor Pillar Edicts）原文

**文献编号**：MPE（Minor Pillar Edicts）  
**刻写年代**：约公元前249—前243年（阿育王在位第二十至二十六年）  
**文字**：婆罗米文（Brāhmī script）  
**语言**：摩揭陀语（Māgadhī Prākrit）  
**载体**：磨光砂岩石柱（Polished sandstone pillars）  

> **甲译**：白话意译，通顺自然，不拘泥于原文句式，适合直接阅读。术语均附 inline 解释，方便零基础读者理解。
> **乙译**：严谨译，逐句对应原文，术语精确，供对照研究。

---

## 说明

小石柱法敕是刻于石柱上的较短诏令，与大石柱法敕（7条）相区别。主要包括以下几类：

1. **菩提迦耶诏令（Bodh Gayā Edict）**：纪念佛陀成道地点
2. **桑奇诏令（Sāñcī Edict）**：关于僧团纪律，禁止分裂僧团
3. **萨尔纳特诏令（Sārnāth Edict）**：同上，禁止分裂僧团
4. **科萨姆比诏令（Kauśāmbī Edict）**：同上
5. **鹿野苑诏令（Deer Park Edict）**：关于僧团纪律
6. **尼格利萨加尔诏令（Nigālī Sāgar Edict）**：纪念迦那迦牟尼佛（Koṇāgamana Buddha）
7. **蓝毗尼诏令（Lumbinī Edict）**：纪念佛陀诞生地，免除蓝毗尼的税收

---

## 一、蓝毗尼诏令（Lumbinī Edict）

### 婆罗米文拉丁转写

```
devānaṃpiyena piyadasinā lājinā
vīsati-vasābhisitena
atana āgāca mahīyite
hida budhe jāte sakyamunī ti
silā-vigaḍabhī ca kālāpita
silā-thabhe ca usapāpita
hida bhagavaṃ jāte ti
Lummini-gāme ubalike kate
aṭha-bhāgiye ca
```

### 甲译

天爱喜见王（Devānāṃpriya Priyadarśin，阿育王的自称）受戒（即登基加冕）已满二十年，亲自来到这里礼拜。

这里，是释迦牟尼佛（Śākyamuni，意为"释迦族的圣者"，即历史上的佛陀）诞生的地方。

王命令建造石围栏，并竖立石柱，以纪念这一圣地。

此处，世尊（bhagavat，对佛陀的尊称）诞生于此。

蓝毗尼村（Lumbinī-grāma，今尼泊尔南部，佛陀诞生地）免除赋税，只需缴纳八分之一的收成，而不是通常的四分之一。

### 乙译

天爱喜见王受戒已满二十年，亲自来此礼拜。

此处，释迦牟尼佛诞生于此。

（王）命令建造石围栏，并竖立石柱。

此处，世尊诞生于此。

蓝毗尼村（Lumbinī-grāma）免除赋税，仅缴纳八分之一的收成（而非通常的四分之一）。

## 二、尼格利萨加尔诏令（Nigālī Sāgar Edict）

### 婆罗米文拉丁转写

```
devānaṃpiyena piyadasinā lājinā
cudasa-vasābhisitena
Koṇākamanasa Budhasa thūbe
duviyaṃ vaḍhite
vīsati-vasābhisitena ca
atana āgāca mahīyite
```

### 甲译

天爱喜见王受戒已满十四年，将迦那迦牟尼佛（Koṇāgamana Buddha，过去七佛之一，释迦牟尼佛的前辈佛陀）的窣堵波（stūpa，佛塔，供奉佛陀舍利的圆形建筑）扩建为原来的两倍。

受戒已满二十年，王亲自来到这里礼拜。

### 乙译

天爱喜见王受戒已满十四年，将迦那迦牟尼佛（Koṇāgamana Buddha）的窣堵波（stūpa，佛塔）扩建为原来的两倍。

受戒已满二十年，（王）亲自来此礼拜。

## 三、桑奇诏令（Sāñcī Edict）——禁止分裂僧团

### 婆罗米文拉丁转写

```
devānaṃpiye piyadasi lājā hevaṃ āhā —
saṃghe anāpasāde kāte
no ca bhikhu vā bhikkhunī vā saṃghaṃ bhindeyya
yo pana bhikhu vā bhikkhunī vā saṃghaṃ bhindeyya
so odāpana-vattha-kārite
āvasathā ca parivasāpetavyo
```

### 甲译

天爱喜见王宣告：

僧团（saṅgha，出家修行者组成的团体）不得分裂。

无论是比丘（bhikkhu，男性出家修行者）还是比丘尼（bhikkhunī，女性出家修行者），都不得分裂僧团。

凡是分裂僧团的比丘或比丘尼，都应被强制换上白衣（即还俗，白衣是在家人的服装，与出家人的袈裟相对），并被驱逐出寺院，不得再住在僧团之中。

这道诏令应当宣读给比丘众和比丘尼众，也应当宣读给在家信众。

### 乙译

天爱喜见王如是宣告：

僧团不得分裂。

无论是比丘还是比丘尼，都不得分裂僧团。

凡是分裂僧团的比丘或比丘尼，都应被强制穿白衣（即还俗），并被驱逐出寺院，不得居住于僧团之中。

此诏令应被宣读给比丘众和比丘尼众，并应被宣读给在家信众。

## 四、萨尔纳特诏令（Sārnāth Edict）

### 婆罗米文拉丁转写

（与桑奇诏令基本相同，以下为附加段落）

```
iyaṃ ca likhāpite
mahāmātā ca savata
savata ca mahāmātā
iyaṃ likhāpite
iyaṃ ca ātapā iyaṃ ca usāhe
iyaṃ ca nisevā
```

### 甲译

（前段与桑奇诏令相同，参见上文）

这道诏令已被刻写下来。

法大官（mahāmātra，专门负责推广法、监督宗教事务的官员）应在各地宣读此诏令，各地的法大官也应宣读此诏令。

此乃我之精进，此乃我之努力，此乃我之修习。

### 乙译

（前段与桑奇诏令相同）

此诏令已被刻写。

法大官（mahāmātra）应在各地宣读此诏令，各地的法大官也应宣读此诏令。

此乃我之精进，此乃我之努力，此乃我之修习。

## 五、科萨姆比诏令（Kauśāmbī Edict）

### 甲译

天爱喜见王宣告：

这道诏令应当宣读给科萨姆比城（Kauśāmbī，古印度重要城市，今印度北方邦）的官员。

无论是比丘还是比丘尼，都不得分裂僧团。

凡是分裂僧团的比丘或比丘尼，都应被强制换上白衣，并被驱逐出寺院。

这道诏令应当宣读给比丘众和比丘尼众，也应当宣读给在家信众。

### 乙译

天爱喜见王如是宣告：

此诏令应被宣读给科萨姆比城的官员。

无论是比丘还是比丘尼，都不得分裂僧团。

凡是分裂僧团的比丘或比丘尼，都应被强制穿白衣，并被驱逐出寺院。

此诏令应被宣读给比丘众和比丘尼众，并应被宣读给在家信众。

## 六、菩提迦耶诏令（Bodh Gayā Edict）

### 甲译

天爱喜见王受戒已满十年，亲自前往菩提迦耶（Sambodhi，即佛陀成道之地，今印度比哈尔邦菩提迦耶）礼拜。

这里，是世尊在菩提树（Bodhi tree，佛陀在此树下证悟成佛）下成道的地方。

王命令在此建造石围栏，并种植菩提树的幼苗，以纪念这一圣地。

这里应受到一切人的礼敬，无论是婆罗门、沙门还是在家信众。

### 乙译

天爱喜见王受戒已满十年，亲自前往菩提迦耶（Sambodhi）礼拜。

此处，世尊在菩提树下成道。

（王）命令在此建造石围栏，并种植菩提树的幼苗，以纪念此圣地。

此处应受到一切人的礼敬，无论是婆罗门、沙门还是在家信众。

---

## 说明

1. 以上转写依据Hultzsch（1925）标准版本，并参照Falk（2006）的校勘意见。
2. 蓝毗尼诏令和尼格利萨加尔诏令是阿育王亲自前往佛教圣地礼拜的直接证据，具有极高的历史价值。
3. 桑奇、萨尔纳特、科萨姆比三处诏令内容基本相同，均关于禁止分裂僧团，是研究第三次结集（华氏城结集）的重要资料。
4. 蓝毗尼诏令中"八分之一"的税收减免，是研究孔雀王朝税收制度的重要资料。
