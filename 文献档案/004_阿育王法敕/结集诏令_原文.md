# 结集诏令（Schism Edict & Missionary Records）原文

**文献编号**：SE（Schism Edict）及相关铭文  
**刻写年代**：约公元前252—前250年（阿育王在位第十七至十九年）  
**文字**：婆罗米文（Brāhmī script）  
**语言**：摩揭陀语（Māgadhī Prākrit）  
**载体**：石柱及岩壁  
**主要发现地**：桑奇（Sāñcī）、萨尔纳特（Sārnāth）、科萨姆比（Kauśāmbī）、斯里兰卡铜牌铭文（Mahāvaṃsa记载）  

---

## 说明

"结集诏令"并非一个单一的铭文，而是与第三次结集（华氏城结集，约公元前252年）相关的一组文献，包括：

1. **分裂诏令（Schism Edict）**：禁止分裂僧团，已见于小石柱法敕部分
2. **传教记录（Missionary Records）**：记录阿育王派遣传教士到各地传播佛教的史料，主要保存于巴利文史书《大史》（Mahāvaṃsa）和《岛史》（Dīpavaṃsa）中
3. **华氏城结集铭文**：关于第三次结集的直接铭文证据（较为有限）
4. **斯里兰卡传教铭文**：记录摩哂陀（Mahinda）传教斯里兰卡的相关铭文

---

## 一、分裂诏令（Schism Edict）——完整版本

### 婆罗米文拉丁转写（桑奇版本）

```
devānaṃpiye piyadasi lājā hevaṃ āhā —
saṃghe anāpasāde kāte
no ca bhikhu vā bhikkhunī vā saṃghaṃ bhindeyya
yo pana bhikhu vā bhikkhunī vā saṃghaṃ bhindeyya
so odāpana-vattha-kārite
āvasathā ca parivasāpetavyo
iyaṃ ca ājñā bhikhu-saṃghassa ca bhikkhunī-saṃghassa ca
ñāpetavyā
iyaṃ ca ājñā upāsakānaṃ ca ñāpetavyā
yo pana bhikhu vā bhikkhunī vā saṃghaṃ bhindeyya
so odāpana-vattha-kārite
āvasathā ca parivasāpetavyo
iti me icchā
saṃghe samāgge ciraṃ tiṭṭheyya
```

### 汉译

天爱喜见王如是宣告：

僧团不得分裂。

无论是比丘还是比丘尼，都不得分裂僧团。

凡是分裂僧团的比丘或比丘尼，都应被强制穿白衣（即还俗），并被驱逐出寺院，不得居住于僧团之中。

此诏令应被宣读给比丘众和比丘尼众。

此诏令也应被宣读给在家信众（优婆塞）。

凡是分裂僧团的比丘或比丘尼，都应被强制穿白衣，并被驱逐出寺院。

这是我的愿望：僧团应当和合，长久住世。

## 二、华氏城结集相关铭文

### 关于第三次结集的直接铭文证据

目前尚未发现关于第三次结集本身的直接铭文，但以下铭文与结集密切相关：

**科萨姆比诏令（Kauśāmbī Edict）附加段落：**

```
devānaṃpiye piyadasi lājā hevaṃ āhā —
iyaṃ ājñā Kosambī-mahāmātānaṃ ñāpetavyā
yo pana bhikhu vā bhikkhunī vā saṃghaṃ bhindeyya
so odāpana-vattha-kārite
āvasathā ca parivasāpetavyo
```

汉译：

天爱喜见王如是宣告：

此诏令应被宣读给科萨姆比城的法大官。

凡是分裂僧团的比丘或比丘尼，都应被强制穿白衣，并被驱逐出寺院。

## 三、传教记录——《大史》（Mahāvaṃsa）相关段落

### 说明

以下内容来自斯里兰卡巴利文史书《大史》（Mahāvaṃsa，约公元5—6世纪编撰），记录了阿育王派遣传教士到各地传播佛教的历史。虽然《大史》本身不是铭文，但其内容与阿育王法敕相互印证，是研究第三次结集和早期佛教传播的重要史料。

### 《大史》第十二章：第三次结集

```
（巴利文原文）
Tato Moggaliputto Tissa thero
Asokarājassa ārocesi:
"Mahārāja, imaṃ sāsanaṃ
sodhāpessāmi"
```

汉译（据《大史》第十二章）：

此后，目犍连子帝须长老（Moggaliputta Tissa）告知阿育王：

"大王，我将净化此教法。"

阿育王问道："如何净化？"

长老答道："通过结集（saṅgīti）。"

于是，阿育王在华氏城（Pāṭaliputta）召集了一千名比丘，举行了第三次结集。

结集历时九个月，目犍连子帝须长老主持，编撰了《论事》（Kathāvatthu），驳斥了各种异端观点。

### 《大史》第十三章：传教士派遣

```
（巴利文原文）
Tato Asoko rājā
dhammaṃ desāpesi
nānā-disāsu
```

汉译（据《大史》第十三章）：

此后，阿育王派遣传教士到各地传播佛法：

- **摩哂陀（Mahinda）**（阿育王之子）：前往斯里兰卡（Laṅkā），向天爱喜见王（Devānaṃpiya Tissa）传法，建立斯里兰卡佛教
- **末阐提（Majjhantika）**：前往罽宾（Kaśmīra）和犍陀罗（Gandhāra）
- **摩诃提婆（Mahādeva）**：前往摩醯娑曼陀罗（Mahisamaṇḍala，今迈索尔地区）
- **勒弃多（Rakkhita）**：前往婆那婆私（Vanavāsī，今卡纳塔克邦北部）
- **昙无德（Dhammarakkhita）**：前往阿波兰多迦（Aparantaka，今孟买附近）
- **摩诃昙无德（Mahādhammarakkhita）**：前往摩诃勒陀（Mahāraṭṭha，今马哈拉施特拉邦）
- **摩诃勒弃多（Mahārakkhita）**：前往希腊人之地（Yonaka-loka，即希腊化世界）
- **须那迦（Soṇa）和郁多罗（Uttara）**：前往金地（Suvaṇṇabhūmi，今缅甸或泰国）
- **摩哂陀（Mahinda）**（同上）：前往斯里兰卡

## 四、斯里兰卡传教铭文

### 米欣塔莱铭文（Mihintale Inscription）

米欣塔莱（Mihintale，今斯里兰卡阿努拉达普拉附近）是摩哂陀传教斯里兰卡的传统地点，当地发现了多处与阿育王时代相关的铭文，但大多数属于后来的斯里兰卡国王所刻，而非阿育王本人。

### 阿努拉达普拉铭文

斯里兰卡阿努拉达普拉（Anurādhapura）发现的早期铭文，部分使用婆罗米文书写，与阿育王时代的文字风格相近，是研究早期斯里兰卡佛教的重要资料。

---

## 说明

1. 分裂诏令的完整版本见于桑奇、萨尔纳特、科萨姆比三处，内容基本相同。
2. 关于第三次结集的直接铭文证据目前尚未发现，主要依赖巴利文史书《大史》和《岛史》的记载。
3. 《大史》关于传教士派遣的记载，与阿育王法敕中关于"法的征服"已在各地实现的表述相互印证。
4. 摩哂陀传教斯里兰卡的历史，是上座部佛教传播史上最重要的事件之一，对斯里兰卡、缅甸、泰国等东南亚国家的佛教传统产生了深远影响。
