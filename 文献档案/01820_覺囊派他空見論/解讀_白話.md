# 觉囊派他空见论 · 白話解讀

## 这部文献讲什么？

他空见是觉囊派最核心的哲学思想，也是整个藏传佛教思想史上最有争议的话题之一。简单来说，他空见主张：真正需要"空"掉的是外加的染污（他），而心性本身——也就是如来藏——是真实不空的。这与格鲁派主张的"一切法自性空"形成了根本性的对立。

## 核心内容

**他空与自空的根本区别** 自空派（以格鲁派为代表）认为一切法包括如来藏在内，都是自性空的，没有任何法在胜义谛中真实存在。而他空派（觉囊派）则认为，如来藏——众生本具的清净觉性——在胜义谛中是真实存在的，需要被空掉的只是外加的客尘染污。用比喻来说，自空派认为镜子和影像都是空的，而他空派认为镜子本身是真实的，只有上面的灰尘和影像才是空的。

**三性理论的重新诠释** 觉囊派运用唯识宗的三性理论来阐述他空见：遍计所执性和依他起性是空的（他空），而圆成实性——即如来藏——是不空的（自性有）。这种诠释方式使觉囊派的他空见与唯识宗和如来藏系经论建立了紧密的理论联系。

**二谛的界定** 觉囊派在二谛理论上有独特立场：胜义谛是指如来藏本身的清净状态，世俗谛是指被客尘遮蔽的状态。这与自空派将胜义谛定义为"空性本身"有本质区别。

## 现代启示

他空见的争论提醒我们：对"空"的理解不应简单化。空性不是虚无主义，也不是否定一切存在的价值。觉囊派的立场启发我们思考：在否定染污的同时，是否也应肯定心性本具的清净觉性？这一思考对现代人的心灵修养也有启发意义。

## 关键概念表

| 概念 | 解释 |
|------|------|
| 他空 | 客体染污是空的，心性本体真实不空 |
| 自空 | 一切法自性空，包括如来藏 |
| 如来藏 | 众生本具的清净佛性 |
| 客尘 | 遮蔽如来藏的外加染污 |
| 胜义谛 | 觉囊派指如来藏清净本体 |
