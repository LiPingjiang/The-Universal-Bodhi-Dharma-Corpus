## 概述

净土经典（Sukhāvatī-vyūha-sūtra，"极乐世界庄严经"）是大乘佛教净土宗（Pure Land Buddhism）的核心文献，以阿弥陀佛（Amitābha，"无量光"）的极乐世界（Sukhāvatī，"极乐"）为核心，阐述往生净土的修行方法。主要包括《无量寿经》（大经）、《观无量寿经》（观经）、《阿弥陀经》（小经），合称"净土三经"。

净土思想在东亚佛教中影响极为深远，是中国、日本、朝鲜、越南等地最广泛流传的佛教修行传统之一。

## 核心概念

- **阿弥陀佛**（Amitābha/Amitāyus）：无量光/无量寿，西方极乐世界的佛
- **极乐世界**（Sukhāvatī）：阿弥陀佛的净土，修行的理想环境
- **四十八愿**（catvāriṃśad-aṣṭa-praṇidhāna）：法藏比丘成佛前发的四十八个愿望
- **第十八愿**（aṣṭādaśa-praṇidhāna）：念佛往生愿，净土宗的核心
- **念佛**（buddha-smṛti/nian-fo）：称念阿弥陀佛名号的修行方法
- **往生**（upapatti）：死后往生极乐世界
- **一心不乱**（ekāgra-citta）：专注念佛，心不散乱
