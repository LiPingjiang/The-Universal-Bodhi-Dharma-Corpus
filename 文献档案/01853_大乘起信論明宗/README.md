## 學術解讀

### 一、文獻學信息

《大乘起信論明宗》是唐代智者大師（538-597）的再傳弟子或門人所撰，一說爲唐代灌頂（561-632）所撰。灌頂，俗姓吳，字法雲，臨海章安（今浙江臨海）人，爲智者大師高足，世稱"章安尊者"，被尊爲天台宗二祖。此疏以"明宗"爲名，取"顯明宗要，會歸一心"之意，是天台宗早期解釋《大乘起信論》的重要著作。

灌頂一生隨侍智者大師，記錄整理其講說，編成《法華玄義》《摩訶止觀》等天台宗根本典籍。他本人也精通各宗教義，著有《涅槃經疏》《觀心論疏》等。此疏作於隋代，是天台宗建立初期對《起信論》進行系統詮釋的嘗試。

### 二、歷史背景

隋代是天台宗創立和發展的重要時期。智者大師創立天台宗，以"一念三千""性具"等思想爲特色。當時《起信論》已在中國流傳，其"一心二門"思想與天台宗的"一念心"概念有相通之處。灌頂作爲智者大師的繼承者，需要將天台宗的思想與當時流行的《起信論》進行對話，確立天台宗在佛教思想界的地位。此疏正是在這一背景下產生。

### 三、思想史定位

灌頂此疏的特色在於以天台宗早期的"觀心"思想來解釋《起信論》的"一心二門"。他認爲，《起信論》的"一心"即是天台宗"觀心法門"所觀的"一念心"，"心真如門"即是"觀心"所見的"清淨性"，"心生滅門"即是"觀心"所見的"煩惱相"。灌頂進一步將"修行"解釋爲"觀心"——通過觀察當下的一念心，在生滅中見真如，在煩惱中見佛性。這一解釋奠定了天台宗解釋《起信論》的基本範式，對後來子璇、湛然、知禮等人的解釋有深遠影響。

## 白話解讀

### 核心內容

《大乘起信論明宗》是隋代天台宗大師灌頂對《大乘起信論》的解釋。灌頂大師是智者大師的弟子，天台宗的二祖。

灌頂大師用天台宗的"觀心"方法來解釋《起信論》：不需要去找什麼玄妙的"真心"，只要觀察當下的一念心。這個念頭是善是惡、是淨是染，都不要緊。重要的是：在觀察這個念頭的過程中，你會發現——念頭來來去去，但能"知道念頭來去"的那個覺知，始終不動。這個"不動的覺知"就是"心真如門"，那些"來來去去的念頭"就是"心生滅門"。

灌頂大師說：修行就是"觀心"——每天觀察自己的念頭，在生滅中見真如，在煩惱中見佛性。

### 現代啓示

1. **當下修行**：灌頂大師強調"觀當下一念心"，啓示我們：修行不需要特殊的條件，就在此刻、就在當下。

2. **自我覺察**："觀心"就是現代的"自我覺察"（self-awareness）——不評判地觀察自己的念頭和情緒。這是心理健康的重要能力。

3. **在煩惱中見佛性**：灌頂說"在煩惱中見佛性"，啓示我們：不要逃避負面情緒，而是在面對它們的過程中成長。

### 閱讀建議

- 灌頂的疏解較爲古樸，適合對天台宗早期思想感興趣的讀者
- 可配合《觀心論疏》一同閱讀，瞭解天台宗"觀心"法門
- 注意灌頂"以觀心解《起信論》"的特點，理解天台宗解釋傳統的源頭
