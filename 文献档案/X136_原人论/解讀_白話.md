# 原人论 · 白話解讀

## 這是什麼書？

《原人論》是唐代一位名叫宗密的大和尚寫的一本小書，全名叫做《華嚴原人論》。「原人」就是「追本溯源，弄清楚人到底是怎麼回事」的意思。

這本書特別有意思的地方在於：它不單單講佛教的道理，而是把儒家、道教和佛教三家放在一起比較。宗密大師就像一位耐心的老師，一層一層地分析：儒道兩家說人從哪裡來，為什麼不夠完整；佛教內部各個宗派又是怎麼回答這個問題的；最後，他用華嚴宗最圓融的智慧，給出一個最究竟的答案。

簡單說，這是一本「人從哪裡來、要到哪裡去」的哲學指南，而且是用唐代中國人最能理解的方式寫的。

## 核心內容

### 1. 主要說了什麼？

宗密大師把古往今來關於「人」的說法，分成五個層次，像爬樓梯一樣，一級比一級高：

**第一層：儒道兩家的說法**

儒家和道教都說，人是從「元氣」或「大道」生出來的。萬物都是由這種原始能量變化而來，人只是其中比較靈秀的一種。宗密大師說，這個說法不算錯，但太膚淺了——只看到了身體和物質的層面，沒有解釋為什麼人有苦有樂、有善有惡，更沒說清楚人死了以後怎麼辦。

**第二層：佛教小乘的說法**

小乘佛教說，人是由「業力」造成的。因為我們過去做了各種各樣的事（業），所以感召了現在這個身體和生命。這個解釋比儒道深入，因為它講到了因果，但小乘只講到「人沒有固定不變的自我」，卻認為構成世界的元素是實在的，還不夠徹底。

**第三層：大乘唯識宗的說法**

唯識宗說，一切都是「心」變現出來的。我們看到的世界，其實是我們內心深處一個叫做「阿賴耶識」的倉庫裡，儲存的種子不斷生長、變現的結果。這個說法很深刻，但宗密認為，唯識宗把「識」看得太實在了，好像真的有一個倉庫在運作，這還是「執著」。

**第四層：大乘空宗的說法**

空宗（如中觀派）說，一切都空，連「識」也是空的。什麼都破掉，什麼都不執著。這個很厲害，但宗密說，它「只破不立」——把房子拆光了，卻沒告訴你真正的家在哪裡。容易讓人誤解為什麼都沒有，變成虛無主義。

**第五層：華嚴圓教的說法（最究竟）**

最後，宗密亮出華嚴宗的答案：《華嚴經》說，「一切眾生皆有如來智慧德相」。每個人內心深處，都有一個本來清淨、圓滿無缺的「真心」。這個心從來沒有污染過，只是被無明（不明真相的癡迷）暫時遮住了。一旦覺悟，這個心就顯現出它本有的光明和妙用。

這個「真心」就是人的真正本源。儒道說的「元氣」、小乘說的「業力」、唯識說的「識」、空宗說的「空」，都是這個真心在不同層面上的隨緣顯現。就像大海起波浪，波浪有種種形狀，但本質都是水。

### 2. 對現代人有什麼啟發？

**關於自我認同的焦慮**：現代人常常問「我是誰」、「我為什麼活著」。《原人論》告訴我們，你的真正本質不是你的名字、職位、社會角色，甚至不是你的思想和情緒——這些都是「波浪」。你的真正本源是那片「大海」，是本來清淨、具足一切可能的真心。這不是玄學，而是一種深刻的自我肯定：你本來就圓滿，不需要向外追逐來證明自己的價值。

**關於多元文化的理解**：宗密沒有簡單否定儒道，而是說它們都是「真心」的某種顯現。這給我們一個啟發：不同的文化、宗教、哲學，可能都是同一個終極真理在不同層面上的表達。真正的智慧不是排斥異己，而是能夠在差異中看到統一。

**關於修行與生活**：宗密說，覺悟不是變成另一個人，而是認出自己本來的面目。這對現代人的啟示是：修行不必離群索居，在日常工作、人際關係中，保持對「真心」的覺察，就是修行。

### 3. 怎麼讀這部書？

- **適合誰讀**：對中國哲學、佛教思想感興趣的人；想了解「三教合一」是怎麼回事的人；對「我是誰」這個問題有好奇心的人。
- **從哪裡開始**：可以先讀白話解讀，了解整體框架；再讀原文，感受宗密的論證邏輯。原文不長，只有一卷，約五千字。
- **需要注意什麼**：宗密的五教判釋，是從華嚴宗的立場出發的。其他宗派（如天台宗）也有自己的判教體系，不必執著於「誰對誰錯」，而要欣賞他論證的精妙和心胸的開闊。
- **搭配閱讀**：可以配合《華嚴經》的〈如來出現品〉、宗密的《禪源諸詮集都序》一起讀，會有更完整的理解。

## 關鍵概念（白話版）

| 佛學術語 | 白話解釋 |
|---------|---------|
| 原人 | 追本溯源，弄清楚人的真正本源是什麼 |
| 真心 | 每個人內心深處本來清淨、圓滿、光明的本性，從未被污染 |
| 無明 | 不明白真相的狀態，就像烏雲遮住太陽，但太陽本身從未消失 |
| 業力 | 我們身口意所做的一切，會像種子一樣儲存起來，影響未來 |
| 阿賴耶識 | 唯識宗說的「深層意識倉庫」，儲存了我們過去所有的經驗種子 |
| 五蘊 | 色（身體）、受（感受）、想（概念）、行（意志）、識（認知），組成「我」的五個要素 |
| 一乘顯性教 | 華嚴宗認為最圓滿的教法，直接顯示眾生本有的佛性 |
| 元氣 | 中國哲學中的原始能量，儒道認為是萬物生成的根本 |
| 法界 | 一切現象的總體，也是最高的真理境界 |
| 隨緣不變 | 真心能隨各種條件顯現種種現象（隨緣），但本身從不改變（不變） |

## 讀後思考

1. 宗密說儒道只講到「形氣」的層面，你認為現代科學（如進化論、基因學）對「人從哪裡來」的回答，相當於他五層樓梯中的哪一層？

2. 如果每個人內心都有一個「本來清淨的真心」，你覺得這對處理現代人的自卑、焦慮、自我懷疑有什麼幫助？

3. 宗密對三教的態度是「先破後立」——先指出局限，再歸於圓融。你認為這種態度在當今多元文化對話中，是優點還是缺點？

4. 「隨緣不變，不變隨緣」這個說法，如何應用到你的日常生活中？比如在工作中遇到變化時，如何保持內心的穩定？

5. 唯識宗說「萬法唯識」，空宗說「一切皆空」，華嚴宗說「真心顯現」。你個人比較傾向哪一種理解世界的方式？為什麼？
