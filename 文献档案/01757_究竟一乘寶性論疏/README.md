# 究竟一乘寶性論疏

- **編號**：01757
- **類別**：中觀般若疏釋

## 學術解讀

### 一、文獻學信息

《究竟一乘寶性論疏》是唐代法藏所撰，爲堅慧菩薩《究竟一乘寶性論》的註疏。該論收於《大正藏》第三十一冊，全論五章，系統闡述如來藏思想，是印度如來藏學說的集大成之作。法藏此疏以華嚴宗義理爲框架，對原論進行深度闡發。

《究竟一乘寶性論》簡稱《寶性論》，與《大乘起信論》並列爲如來藏思想最重要的兩部論書。原論以七金剛句（佛、法、僧、界、菩提、功德、業）組織全篇，結構嚴謹，義理深邃。

### 二、歷史背景

《寶性論》的漢譯本由勒那摩提於北魏正始五年（508）譯出，是較早傳入中國的如來藏系論書。法藏作疏時，正值武則天崇佛時期，華嚴宗受到皇室大力支持。法藏借疏解《寶性論》之機，將印度如來藏思想與華嚴宗的"性起""法界"等概念相融合，形成獨具特色的中國化如來藏學說。

### 三、思想史定位

本疏在中國佛教思想史上具有橋樑作用：一方面承繼印度中觀與如來藏的傳統，另一方面開啓華嚴宗"性起"思想的先河。法藏強調"一切衆生皆有佛性"不僅是一種可能性，更是一種現實性——佛性不是將來要獲得的，而是本來具足的。這一思想對後來禪宗"頓悟"說、天台宗"性具"說都有深遠影響。

## 白話解讀

### 核心內容

《究竟一乘寶性論》的核心觀點是：一切衆生都像礦石中含有金子一樣，本來具足佛性（寶性）。這個佛性不是從外面得來的，而是本來就有的。法藏大師的疏解進一步闡明：這個佛性不是潛在的可能性，而是當下就具足的現實。

全論用七個要點（七金剛句）來組織：佛、法、僧是三寶；界是衆生本具的佛性；菩提是覺悟；功德是佛性的種種作用；業是修行實踐。這七個方面共同構成了完整的成佛之道。

### 現代啓示

1. **內在價值**：現代人常常向外追求成功和認可，"寶性"思想提醒我們：每個人內在都有無價之寶，不需要通過外在成就來證明自己。

2. **教育平等**：如果每個孩子都本具佛性，那麼教育的目的不是"灌輸"知識，而是"啓發"內在潛能。這與中國古代"因材施教"的教育理念相通。

3. **心理健康**：很多心理問題的根源在於自我否定。認識到自己本具"寶性"，有助於建立穩定的自我價值感。

### 閱讀建議

- 建議先讀《寶性論》原文的"佛寶品"和"法寶品"，再讀其他章節
- 可配合《大乘起信論》對比閱讀，理解如來藏思想的共同脈絡
- 注意"佛性"與"佛果"的區別：有佛性不等於已經是佛，還需要修行
