## 源流與背景

般若波羅蜜多（Prajñāpāramitā，意爲「智慧的圓滿」）經系是大乘佛教最早興起的經典羣之一。其中《八千頌般若》（Aṣṭasāhasrikā Prajñāpāramitā，意爲「八千行/頌的般若波羅蜜多」）是現存最古老的般若經文本，被學術界普遍認爲是般若經羣的原始形態——後世更爲龐大的《二萬五千頌》《十萬頌》等大品般若，均是在八千頌基礎上的擴展與增廣。龍樹在《大智度論》中所註釋的「大品般若」與八千頌有密切關聯，中觀學派的核心概念「空」（śūnyatā）直接承自此經。

經文以佛陀在靈鷲山（Gṛdhrakūṭa）與須菩提（Subhūti）的對話爲主線，共32品。核心內容反覆闡明：一切法（包括菩薩、般若波羅蜜多本身）都不可得、無自性、如幻如化——菩薩正是在這種「無所得」中行六度、度衆生。經中遍佈的「三段否定公式」（佛說X即非X是名X）後來在《金剛經》中被提煉爲最精練的形式，而八千頌中已有的「不住」「不可得」「如幻」等概念，構成了整個大乘空宗思想的原典基礎。

關於成書環境，學術界有多個假說。Conze（1960）和 Lamotte（1962）主張西北印度犍陀羅地區，依據是最早寫本殘片出土於此；平川彰（1963）和印順（1981）則關聯南印度大衆部系；Schopen（1975）和 Harrison（2003）提出非單一地點產生的修正版。當前學術風向傾向於承認多中心性，但犍陀羅-北印度區域可能是最初的主要編纂地。支婁迦讖於東漢光和二年（179 CE）譯出的《道行般若經》（T224）是此經最早有確切紀年的漢譯本，將梵文本的成形推至至少公元1世紀中葉。

## 關鍵概念

- **般若波羅蜜多**（Prajñāpāramitā）：prajñā（智慧）+ pāramitā（到彼岸/圓滿），意爲「到達彼岸的智慧」。但經文開宗明義宣稱「不存在一個叫般若波羅蜜多的東西」——連自身最推崇的概念都加以否定，以此徹底堵住對佛法的執着。
- **空性**（śūnyatā）：在八千頌中尚未如後世中觀哲學那樣成爲精確定義的哲學概念，而是更接近一種修辭-冥想策略——通過對每個概念反覆施加「不可得」，訓練修行者不在任何對象上停駐心念。空性涵蓋對象空、修行空、語言空三個層次。
- **不可得**（anupalabdha / na vidyate）：vid = 「找到」「存在」；na vidyate = 「不被找到」「不存在爲可指認的實體」。這是般若經系最核心的手法：對一切法反覆宣說「不可得」，包括般若本身。
- **不住**（na pratitiṣṭhet）：prati（向着）+ sthā（站立）= 「以……爲落腳點」。「不住」即做佈施時不把任何東西當作可依靠的實在基礎。後來《金剛經》「應無所住而生其心」正是此思想的極度精簡版。
- **三段否定公式**（佛說X / 即非X / 是名X）：邏輯結構爲——我們使用「X」這個名稱；但此名稱不指向獨立實在的本體；正因無固定本體，「X」才能作爲靈活的約定名稱使用。這區分了約定層面的有效使用與實體層面的真實存在，是般若經系認識論的核心工具。
