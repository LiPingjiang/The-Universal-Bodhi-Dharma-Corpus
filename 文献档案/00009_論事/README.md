## 源流与背景

佛灭后约两百年，阿育王（Aśoka，约前268–232年在位）一统印度，护持佛教。据南传记载，当时僧团中异见纷起，各部派学说混杂，阿育王遂于华氏城（Pāṭaliputta）召集第三次结集，由**目犍连子帝须**（Moggaliputta Tissa）长老主持。结集的核心成果之一便是《论事》（Kathāvatthu）——一部以辩论问答体裁系统破斥异部学说的论书。

《论事》是巴利论藏（Abhidhamma Piṭaka）七部论中唯一被传统明确归于特定作者的作品。全书分23品，含约217–219个论题，每一论题皆以正反双方问答展开：一方提出他宗命题，另一方（上座部分别说部正统立场）通过逐步追问逼出矛盾，最终引经为证、判定该命题不成立。辩论对象涵盖犊子部的补特伽罗实有论、大众部系的阿罗汉退转论、说一切有部的三世实有论等，几乎囊括了部派佛教时期所有重大教义争端。

现代学术界对成书年代有更精细的讨论：核心层（前数品）确实对应公元前3世纪的部派格局，但后续品目中涉及的某些教义立场（如成熟的说一切有部体系）暗示文本至少经历了前2至前1世纪的增补。学界主流认为《论事》有一个前3世纪的核心，之后经历两三个世纪的逐步增补，最终在兰卡第四次结集前后定型（Bareau 1955, Norman 1983）。此论的辩论格式（pañcavāra，五段论式）被认为是佛教逻辑学的雏形，也是印度因明发展史上的重要一环。

## 关键概念

- **补特伽罗**（puggala/pudgala）：犊子部主张的「不可说的自我载体」，介于五蕴相同与相异之间，作为轮回主体。《论事》第一论题即破斥此说，认为它等于偷偷引入佛陀否定的「我」（attā）。
- **五段论式**（pañcavāra）：《论事》每个论题遵循的标准辩论程序——正问、逆问、破斥确认、倒转重演、经证归结，是归谬法在印度思想史上最早的系统化应用之一。
- **阿罗汉退转**（arahato parihāni）：部分部派主张已证阿罗汉果者仍可能退转，此争端直接呼应根本分裂中大天五事的核心争议。
- **胜义**（paramattha）：即「最终极意义上的真实」。《论事》反复追问对方的主张是否在「胜义」层面成立，以此区分约定层面的有效使用与本体层面的真实存在。
- **他宗/自宗**（parappavāda/sakavāda）：《论事》辩论中正方代表他宗（被破斥的异部命题），反方代表自宗（上座部分别说部正统立场），通过问答暴露他宗的逻辑矛盾。
