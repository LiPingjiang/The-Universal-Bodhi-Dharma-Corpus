## 源流與背景

佛滅後約兩百年，阿育王（Aśoka，約前268–232年在位）一統印度，護持佛教。據南傳記載，當時僧團中異見紛起，各部派學說混雜，阿育王遂於華氏城（Pāṭaliputta）召集第三次結集，由**目犍連子帝須**（Moggaliputta Tissa）長老主持。結集的核心成果之一便是《論事》（Kathāvatthu）——一部以辯論問答體裁系統破斥異部學說的論書。

《論事》是巴利論藏（Abhidhamma Piṭaka）七部論中唯一被傳統明確歸於特定作者的作品。全書分23品，含約217–219個論題，每一論題皆以正反雙方問答展開：一方提出他宗命題，另一方（上座部分別說部正統立場）通過逐步追問逼出矛盾，最終引經爲證、判定該命題不成立。辯論對象涵蓋犢子部的補特伽羅實有論、大衆部系的阿羅漢退轉論、說一切有部的三世實有論等，幾乎囊括了部派佛教時期所有重大教義爭端。

現代學術界對成書年代有更精細的討論：核心層（前數品）確實對應公元前3世紀的部派格局，但後續品目中涉及的某些教義立場（如成熟的說一切有部體系）暗示文本至少經歷了前2至前1世紀的增補。學界主流認爲《論事》有一個前3世紀的核心，之後經歷兩三個世紀的逐步增補，最終在蘭卡第四次結集前後定型（Bareau 1955, Norman 1983）。此論的辯論格式（pañcavāra，五段論式）被認爲是佛教邏輯學的雛形，也是印度因明發展史上的重要一環。

## 關鍵概念

- **補特伽羅**（puggala/pudgala）：犢子部主張的「不可說的自我載體」，介於五蘊相同與相異之間，作爲輪迴主體。《論事》第一論題即破斥此說，認爲它等於偷偷引入佛陀否定的「我」（attā）。
- **五段論式**（pañcavāra）：《論事》每個論題遵循的標準辯論程序——正問、逆問、破斥確認、倒轉重演、經證歸結，是歸謬法在印度思想史上最早的系統化應用之一。
- **阿羅漢退轉**（arahato parihāni）：部分部派主張已證阿羅漢果者仍可能退轉，此爭端直接呼應根本分裂中大天五事的核心爭議。
- **勝義**（paramattha）：即「最終極意義上的真實」。《論事》反覆追問對方的主張是否在「勝義」層面成立，以此區分約定層面的有效使用與本體層面的真實存在。
- **他宗/自宗**（parappavāda/sakavāda）：《論事》辯論中正方代表他宗（被破斥的異部命題），反方代表自宗（上座部分別說部正統立場），通過問答暴露他宗的邏輯矛盾。
