## 概述

《唯識三十頌》（Triṃśikā-vijñaptimātratāsiddhi）是世親（Vasubandhu，約4-5世紀）晚年的重要著作，以三十首偈頌（kārikā）系統闡述了唯識學派的核心思想。玄奘（602-664）將其譯爲漢文，並綜合十大論師的註釋，撰寫了《成唯識論》（T1585），成爲中國法相宗（Faxiang）的根本論典。

《唯識三十頌》是唯識學派最精煉、最系統的文本，涵蓋了阿賴耶識、三性、轉依等核心概念。

## 核心概念

- **世親**（Vasubandhu）：唯識學派奠基人，著有《唯識三十頌》、《唯識二十論》
- **八識**（aṣṭa-vijñāna）：眼識、耳識、鼻識、舌識、身識、意識、末那識、阿賴耶識
- **末那識**（manas-vijñāna）：第七識，執著阿賴耶識爲"我"的識
- **識轉變**（vijñāna-pariṇāma）：識的變現，產生見分和相分
- **見分**（darśana-bhāga）：識的能見部分
- **相分**（nimitta-bhāga）：識的所見部分（被認爲是外境的部分）
- **五位百法**（pañca-dharma-skandha）：唯識學派對一切現象的分類體系
