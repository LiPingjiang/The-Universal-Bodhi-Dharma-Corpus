## 概述

《唯识三十颂》（Triṃśikā-vijñaptimātratāsiddhi）是世亲（Vasubandhu，约4-5世纪）晚年的重要著作，以三十首偈颂（kārikā）系统阐述了唯识学派的核心思想。玄奘（602-664）将其译为汉文，并综合十大论师的注释，撰写了《成唯识论》（T1585），成为中国法相宗（Faxiang）的根本论典。

《唯识三十颂》是唯识学派最精炼、最系统的文本，涵盖了阿赖耶识、三性、转依等核心概念。

## 核心概念

- **世亲**（Vasubandhu）：唯识学派奠基人，著有《唯识三十颂》、《唯识二十论》
- **八识**（aṣṭa-vijñāna）：眼识、耳识、鼻识、舌识、身识、意识、末那识、阿赖耶识
- **末那识**（manas-vijñāna）：第七识，执著阿赖耶识为"我"的识
- **识转变**（vijñāna-pariṇāma）：识的变现，产生见分和相分
- **见分**（darśana-bhāga）：识的能见部分
- **相分**（nimitta-bhāga）：识的所见部分（被认为是外境的部分）
- **五位百法**（pañca-dharma-skandha）：唯识学派对一切现象的分类体系
