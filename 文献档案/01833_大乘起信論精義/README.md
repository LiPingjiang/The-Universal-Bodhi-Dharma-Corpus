## 学术解读

### 一、文献学信息

《大乘起信论精义》是近代太虚（1890-1947）所撰。太虚，俗姓吕，字太虚，浙江崇德（今浙江桐乡）人，为近代中国佛教改革运动的领袖，世称"太虚大师"，被尊为"人间佛教"的倡导者。此疏以"精义"为名，取"精择要义，会归一心"之意，是近代《大乘起信论》注疏中最具系统性和影响力的一部。

太虚一生致力于佛教改革，主张"教理革命""僧制革命""寺产革命"，提出"人生佛教"（后称"人间佛教"）的理念，强调佛教应关注现实人生、参与社会建设。他精通各宗派教义，著有《太虚大师全书》六十四册。此疏作于其中年，是其"以大乘思想统摄各宗"学术体系的重要组成部分。

### 二、历史背景

近代中国内忧外患，佛教面临衰落危机。太虚认为，佛教衰落的原因之一是"重死轻生"——过于关注死后的解脱，忽视现实的人生。他主张佛教应回归"大乘"精神，关注众生疾苦，参与社会改造。《起信论》的"大乘"精神和"一心二门"思想，恰好为太虚的"人生佛教"提供了教理基础。此疏正是在这一背景下产生，旨在以《起信论》为基础，建立适应现代社会的佛教思想体系。

### 三、思想史定位

太虚此疏的最大特色在于以"人生佛教"的立场来解释《起信论》。他认为，《起信论》的"心真如门"不是脱离现实的"玄谈"，而是每个人当下可以体验的"清净心"。"心生灭门"也不是抽象的哲学概念，而是每个人日常生活中的"烦恼心"。太虚进一步将"修行"解释为"在生活中修行"：工作、学习、家庭、社会，都是修行的道场。这一解释对后来"人间佛教"的发展有决定性影响，至今仍是台湾、香港等地佛教的主流思想。

## 白话解读

### 核心内容

《大乘起信论精义》是近代佛教改革领袖太虚大师对《大乘起信论》的解释。太虚大师主张"人生佛教"——佛教不应该只关注死后的世界，而应该关注现实的人生。

太虚大师用"人生佛教"的语言来解释《起信论》："心真如门"不是遥不可及的玄妙境界，而是每个人当下就能体验的清净心。当你帮助别人、做善事的时候，那个"愿意做好事"的心，就是"心真如门"的显现。

"心生灭门"也不是抽象的哲学概念，而是我们每天的烦恼——工作中的压力、家庭中的矛盾、社会上的竞争。太虚大师说：这些烦恼本身就是修行的材料。不要逃避生活，要在生活中修行。

### 现代启示

1. **生活即修行**：现代人常常觉得"修行"就是去寺庙打坐。太虚大师告诉我们：工作、学习、家庭、社会，都是修行的道场。真正的修行不在远方，就在当下。

2. **关注现实**：佛教不应该"重死轻生"。太虚大师的"人生佛教"启示我们：宗教应该关注现实问题，帮助人们解决生活中的困难。

3. **社会责任**：太虚大师主张佛教参与社会建设。这启示我们：个人修行与社会责任不是对立的，而是相辅相成的。

### 阅读建议

- 太虚大师的疏解具有现代性和实践性，适合当代读者
- 可配合《太虚大师全书》中的《人生佛教》《佛法僧义》等一同阅读
- 注意其"人间佛教"的立场，理解近代佛教改革的脉络
