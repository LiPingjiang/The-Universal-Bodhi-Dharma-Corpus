# 大乘起信論精義

## 學術解讀

### 一、文獻學信息

《大乘起信論精義》是近代太虛（1890-1947）所撰。太虛，俗姓呂，字太虛，浙江崇德（今浙江桐鄉）人，爲近代中國佛教改革運動的領袖，世稱"太虛大師"，被尊爲"人間佛教"的倡導者。此疏以"精義"爲名，取"精擇要義，會歸一心"之意，是近代《大乘起信論》註疏中最具系統性和影響力的一部。

太虛一生致力於佛教改革，主張"教理革命""僧製革命""寺產革命"，提出"人生佛教"（後稱"人間佛教"）的理念，強調佛教應關注現實人生、參與社會建設。他精通各宗派教義，著有《太虛大師全書》六十四冊。此疏作於其中年，是其"以大乘思想統攝各宗"學術體系的重要組成部分。

### 二、歷史背景

近代中國內憂外患，佛教面臨衰落危機。太虛認爲，佛教衰落的原因之一是"重死輕生"——過於關注死後的解脫，忽視現實的人生。他主張佛教應迴歸"大乘"精神，關注衆生疾苦，參與社會改造。《起信論》的"大乘"精神和"一心二門"思想，恰好爲太虛的"人生佛教"提供了教理基礎。此疏正是在這一背景下產生，旨在以《起信論》爲基礎，建立適應現代社會的佛教思想體系。

### 三、思想史定位

太虛此疏的最大特色在於以"人生佛教"的立場來解釋《起信論》。他認爲，《起信論》的"心真如門"不是脫離現實的"玄談"，而是每個人當下可以體驗的"清淨心"。"心生滅門"也不是抽象的哲學概念，而是每個人日常生活中的"煩惱心"。太虛進一步將"修行"解釋爲"在生活中修行"：工作、學習、家庭、社會，都是修行的道場。這一解釋對後來"人間佛教"的發展有決定性影響，至今仍是臺灣、香港等地佛教的主流思想。

## 白話解讀

### 核心內容

《大乘起信論精義》是近代佛教改革領袖太虛大師對《大乘起信論》的解釋。太虛大師主張"人生佛教"——佛教不應該只關注死後的世界，而應該關注現實的人生。

太虛大師用"人生佛教"的語言來解釋《起信論》："心真如門"不是遙不可及的玄妙境界，而是每個人當下就能體驗的清淨心。當你幫助別人、做善事的時候，那個"願意做好事"的心，就是"心真如門"的顯現。

"心生滅門"也不是抽象的哲學概念，而是我們每天的煩惱——工作中的壓力、家庭中的矛盾、社會上的競爭。太虛大師說：這些煩惱本身就是修行的材料。不要逃避生活，要在生活中修行。

### 現代啓示

1. **生活即修行**：現代人常常覺得"修行"就是去寺廟打坐。太虛大師告訴我們：工作、學習、家庭、社會，都是修行的道場。真正的修行不在遠方，就在當下。

2. **關注現實**：佛教不應該"重死輕生"。太虛大師的"人生佛教"啓示我們：宗教應該關注現實問題，幫助人們解決生活中的困難。

3. **社會責任**：太虛大師主張佛教參與社會建設。這啓示我們：個人修行與社會責任不是對立的，而是相輔相成的。

### 閱讀建議

- 太虛大師的疏解具有現代性和實踐性，適合當代讀者
- 可配合《太虛大師全書》中的《人生佛教》《佛法僧義》等一同閱讀
- 注意其"人間佛教"的立場，理解近代佛教改革的脈絡
