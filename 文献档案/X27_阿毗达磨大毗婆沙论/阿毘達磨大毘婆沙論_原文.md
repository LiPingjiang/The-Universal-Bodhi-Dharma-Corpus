# 阿毘達磨大毘婆沙論 · 原文

## 文獻基本信息

- **梵名**：Abhidharma-mahāvibhāṣā-śāstra
- **巴利名**：Abhidhamma-mahāvibhaṅga（擬）
- **漢譯**：玄奘譯《阿毘達磨大毘婆沙論》
- **年代**：唐顯慶年間（公元656-660年）
- **卷數**：二百卷
- **所屬部派**：說一切有部（Sarvāstivāda）
- **編纂者**：迦濕彌羅（罽賓）論師眾
- **藏經編號**：大正藏第1545號（阿毘達磨部）

## 經典概要

《阿毘達磨大毘婆沙論》，簡稱《大毘婆沙論》，是說一切有部最重要、最龐大的論書，也是整個佛教文獻中最龐大的單一論書之一。此論由迦濕彌羅（罽賓）的說一切有部論師集體編纂，玄奘法師於唐朝顯慶年間譯出，共二百卷。

「毘婆沙」（Vibhāṣā）意為「廣解」、「分別解」。「大毘婆沙」意為「廣大的分別解」。此論是對《發智論》的詳細註釋和廣解，將《發智論》中簡要提及的每個問題，都進行了極其詳細的分析和討論。

此論的規模極其龐大，二百卷的篇幅，涵蓋了說一切有部教理的幾乎所有方面。它不僅是對《發智論》的註釋，還是一部獨立的、系統的教理百科全書。

## 核心內容

### 一、對《發智論》的廣解

《大毘婆沙論》的核心內容，是對《發智論》的廣解。具體來說：

1. **逐句解釋**：對《發智論》中的每個句子，進行詳細的解釋。
2. **問題的展開**：對《發智論》中提到的每個問題，展開詳細的討論。
3. **多種觀點的呈現**：對每個問題，呈現多種不同的觀點和論證。
4. **結論的確定**：在比較多種觀點後，確定最正確的結論。

這種廣解的方式，使《大毘婆沙論》成為一部內容極其豐富的論書。

### 二、法的詳細分類

《大毘婆沙論》對法進行了極其詳細的分類，主要包括：

1. **色法**：物質現象，包括四大種及所造色。此論對色法的分類極其詳細，包括各種微細的物質現象。
2. **心法**：心識活動，包括六識。此論對心識的運作機制進行了詳細的分析。
3. **心所法**：與心識相應的心理活動，包括受、想、思等。此論對心所法的分類極其詳細，列出了數十種心所法。
4. **心不相應行法**：既非色法也非心法的現象。此論對這類法的分類也非常詳細。
5. **無為法**：非因緣生起的法。此論對無為法的種類和性質進行了詳細的討論。

這種詳細的分類，體現了說一切有部對法的分析達到了極高的精細程度。

### 三、煩惱的詳細分析

《大毘婆沙論》對煩惱進行了極其詳細的分析：

1. **根本煩惱**：貪、瞋、癡、慢、疑、見。此論對每種根本煩惱的性質、作用、對治方法都進行了詳細的討論。
2. **隨煩惱**：忿、恨、覆、惱、嫉、慳等。此論對每種隨煩惱的性質和作用也進行了詳細的分析。
3. **煩惱的生起機制**：此論詳細分析了煩惱是如何生起的，包括各種不同的因緣條件。
4. **對治的詳細方法**：此論對每種煩惱的對治方法，都進行了詳細的說明。

### 四、業與果報

《大毘婆沙論》對業與果報的問題進行了詳細的討論：

1. **業的種類**：善業、惡業、無記業。
2. **業的運作機制**：業是如何造作、如何保存、如何產生果報的。
3. **果報的種類**：現報、生報、後報。
4. **業與因果**：業與因果法則的關係。

此論對業的討論，是佛教業力思想最詳細的闡述之一。

### 五、修行與證果

《大毘婆沙論》對修行和證果的問題進行了詳細的說明：

1. **修行的次第**：從初學到證果的完整次第。
2. **各種禪定**：四禪八定的詳細說明。
3. **各種果位**：預流果、一來果、不還果、阿羅漢果的詳細說明。
4. **證果的條件**：證得各種果位所需要的條件。

## 結構說明

《大毘婆沙論》二百卷的結構，是按照《發智論》的結構來組織的。具體來說：

### 雜蘊（第一卷至第二十卷）
- 對《發智論》「雜蘊」的廣解
- 討論各種雜類的問題

### 結蘊（第二十一卷至第四十卷）
- 對《發智論》「結蘊」的廣解
- 討論煩惱（結）的問題

### 智蘊（第四十一卷至第六十卷）
- 對《發智論》「智蘊」的廣解
- 討論智慧的問題

### 業蘊（第六十一卷至第八十卷）
- 對《發智論》「業蘊」的廣解
- 討論業的問題

### 大種蘊（第八十一卷至第一百卷）
- 對《發智論》「大種蘊」的廣解
- 討論四大種的問題

### 根蘊（第一百零一卷至第一百二十卷）
- 對《發智論》「根蘊」的廣解
- 討論根（感官和能力）的問題

### 定蘊（第一百二十一卷至第一百四十卷）
- 對《發智論》「定蘊」的廣解
- 討論禪定的問題

### 見蘊（第一百四十一卷至第一百六十卷）
- 對《發智論》「見蘊」的廣解
- 討論各種見解的問題

### 後續部分（第一百六十一卷至第二百卷）
- 補充討論
- 總結和結論

這種結構體現了《大毘婆沙論》作為《發智論》廣解的特點。

## 思想價值

《阿毘達磨大毘婆沙論》的思想價值主要體現在以下幾個方面：

1. **教理的百科全書**：此論涵蓋了說一切有部教理的幾乎所有方面，是佛教教理的百科全書。

2. **思想的寶庫**：此論保存了大量的佛教思想資料，包括許多現已失傳的論師的觀點。

3. **論證的典範**：此論的論證方法極其嚴密，是佛教因明學的重要資料。

4. **歷史的記錄**：此論記錄了說一切有部發展到高峰時期的思想狀況，是研究佛教思想史的重要文獻。
