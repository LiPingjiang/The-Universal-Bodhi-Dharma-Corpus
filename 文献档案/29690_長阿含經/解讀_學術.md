# 長阿含經 學術解讀

## 一、文獻學基本信息

**譯者**：佛陀耶舍（Buddhayaśas），罽賓（即迦溼彌羅，Kashmir）人，精通三藏，尤諳律部；竺佛念，涼州人，漢地著名義學僧，長於翻譯。  
**譯時**：後秦弘始十五年（413 CE），於長安譯出。  
**底本語言**：梵文（Sanskrit），底本已佚，學界推測原本屬法藏部（Dharmaguptaka）或說一切有部（Sarvāstivāda）傳誦。  
**大正藏**：T01, No. 1，第一冊，頁1–149。  
**卷帙**：二十二卷，共三十經。  

《長阿含經》的底本問題至今爭議未决。翻譯底本說法有二：（1）法藏部本，以日本學者赤沼智善（Akanuma Chizen）為代表；（2）說一切有部本，以德國學者溫特尼茲（Moriz Winternitz）等為代表。近年大多數學者（包括Bhikkhu Analayo）傾向認為底本與巴利長部同源，但各自獨立流傳，並非同一部派文本的直接翻譯。《世記經》（卷二十二）在巴利《長部》中無對應文本，可能是漢譯底本獨有之增補。

## 二、在佛教史上的地位與影響

《長阿含經》在漢傳佛教文獻史上具有奠基性地位。它是四部漢譯阿含（長、中、增壹、雜）之首，依大正藏排序置於卷首，象徵阿含類文獻在印度原始佛教中的核心位置。

就歷史影響而言，至少有以下幾個面向：

1. **宇宙論資源**：《世記經》詳述三界宇宙觀（三千大千世界、地獄、餓鬼、天界）及劫數計算，成為後世中國佛教宇宙論敘述的基礎來源，影響《立世阿毗曇論》及天台、華嚴的宇宙論建構。

2. **涅槃敘事**：《遊行經》（對應巴利 Mahāparinibbāna-sutta）是現存最完整的佛陀最後之旅記錄，影響中土佛教對涅槃的理解，並對後世《大般涅槃經》（大乘）的撰述提供了原型敘述框架。

3. **梵志批判**：《種德經》《究羅檀頭經》等諸經記錄佛陀與婆羅門的辯論，確立了早期佛教「人不因出身而高貴，因行為而高貴」的平等主義倫理立場，此思路深刻影響漢傳佛教的倫理觀念。

4. **六十二見**：《梵動經》對六十二種邪見的系統批判，是佛教知識批判史的重要文獻，後世論師（如清辨、法稱）在論難外道時屢屢援引此框架。

## 三、與南傳巴利尼柯耶的對應關係

《長阿含經》（漢譯）與巴利《長部》（Dīgha Nikāya，DN）最相近，但對應並非一一相同。主要對應如下：

| 漢譯《長阿含》 | 巴利《長部》 |
|--------------|-------------|
| 大本經（卷1） | DN 14: Mahāpadāna-sutta |
| 遊行經（卷2–4） | DN 16: Mahāparinibbāna-sutta |
| 典尊經（卷5） | DN 19: Mahāgovinda-sutta |
| 小緣經（卷7–8） | DN 27: Aggañña-sutta |
| 大緣方便經（卷13） | DN 15: Mahānidāna-sutta |
| 梵動經（卷14） | DN 1: Brahmajāla-sutta |
| 究羅檀頭經（卷15） | DN 5: Kūṭadanta-sutta |
| 種德經（卷17） | DN 4: Soṇadaṇḍa-sutta |
| 堅固經（卷18） | DN 11: Kevaṭṭa-sutta |
| 世記經（卷20–22） | **無對應**（巴利《長部》無此內容） |

值得注意的是，《世記經》在巴利《長部》中完全沒有對應，顯示漢譯底本所屬部派在宇宙論方面有獨立的文獻積累。同時，部分巴利《長部》的經典（如 DN 2: Sāmaññaphala-sutta 沙門果經，DN 21: Sakkapañha-sutta 帝釋所問經）在《長阿含》中有對應但編排不同。

## 四、主要教義內容概述

### 1. 緣起論
《大緣方便經》詳述十二支緣起，並追溯意識在名色生起中的角色。佛陀告阿難：「識緣名色，名色亦緣識」——此雙向緣起說（識與名色互為緣起）是佛教心理哲學的深刻洞見，後世學者（如Bhikkhu Bodhi、Bhikkhu Sujato）對此解讀頗多討論。

### 2. 宇宙論與輪王思想
《世記經》《轉輪聖王修行經》構建了佛教特有的宇宙觀與政治理想：轉輪聖王（cakravartin）以正法統治，七寶出現，人民安樂；但當社會道德衰敗時，壽命縮短，苦難增加。此乃佛教倫理政治哲學的重要來源，影響古代亞洲政教關係的理論建構。

### 3. 六十二見批判
《梵動經》將當時印度哲學界的見解歸納為六十二種，核心分為常見（eternalism）和斷見（annihilationism）兩大類，佛陀以緣起論超越這一二元困境，確立「中道」的知識論立場。

### 4. 平等主義倫理
《小緣經》《種德經》通過對種姓制度的批判，建立了佛教的平等倫理：四姓（婆羅門、刹帝利、吠舍、首陀羅）本無高下，行為（業）才是人格尊嚴的根本。

## 五、學術研究現狀與爭議

### 主要研究成果
- 赤沼智善《漢巴四部四尼柯耶對照録》（1929）：系統比對漢巴兩傳，至今仍是基礎工具書。
- Bhikkhu Analayo《A Comparative Study of the Majjhima-nikāya》（2011）及多篇針對長阿含個別經典的比較研究論文。
- 無著比丘（Bhikkhu Analayo）的研究尤其強調，漢巴兩傳的差異常常反映部派傳誦中的獨立演化，而非相互翻譯。

### 主要爭議
1. **底本歸屬**：底本是法藏部本還是說一切有部本？由於底本梵文早已失傳，此問題至今無定論。近年犍陀羅（Gandhāra）梵文寫本的陸續出版，提供了新的比較材料，但尚不能確定《長阿含》底本的直接歸屬。

2. **《世記經》的性質**：《世記經》是原始文獻一部分，還是後來附加的宇宙論擴充？部分學者認為其阿毗達磨色彩濃厚，可能是部派論師的整理增補，而非源自佛陀時代的直接傳誦。

3. **遊行經的歷史性**：《遊行經》作為佛陀最後旅程的記錄，其歷史可靠性問題（哪些是真實史料，哪些是後世添加的神異敘述）在學術界有持續討論，E. J. Thomas、Hermann Oldenberg 等早期學者均曾就此發表意見。

4. **竺佛念的翻譯風格**：竺佛念的漢譯偏向意譯，有時遠離原文。與後來玄奘的直譯風格相比，其譯文的文獻準確性較難評估，給依漢譯本進行語義重建帶來困難。
