## 目錄簡介

本目錄收錄漢傳佛教本土化進程的核心文獻，涵蓋佛教傳入中國後與儒道文化融合、形成獨特宗派體系的歷史與思想。

**梵文名稱**：Cīna-bauddha-dharma（中國佛法）  
**時代範圍**：東漢至宋代（約公元1世紀—13世紀）  
**核心主題**：格義佛教、判教體系、宗派分化、儒佛道三教融合

## 關鍵概念

- **格義**（gé yì）：用儒道概念比附佛教概念的翻譯方法
- **判教**（pàn jiào）：對佛陀一生教法進行分類排序的體系
- **五時八教**：天台宗智顗的判教體系
- **五教十宗**：華嚴宗法藏的判教體系
- **禪淨雙修**：禪宗與淨土宗的融合修行
