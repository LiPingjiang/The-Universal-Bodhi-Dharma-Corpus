## 學術解讀

### 一、文獻學信息

《大乘起信論釋要》是明代高僧紫柏真可（1543-1604）所撰。真可，字達觀，號紫柏，世稱"紫柏大師"，爲明代四大高僧之一，與憨山德清、雲棲祩宏、藕益智旭齊名。此疏以"釋要"爲名，取"解釋要義，提綱挈領"之意，是《大乘起信論》註疏中較爲精要的一部。

紫柏真可一生致力於佛教復興，主張禪、教、淨融通，尤其重視經論的刊刻與流通。他曾發起刊刻《大藏經》（即《嘉興藏》），對佛教文獻的保存貢獻巨大。此疏作於其弘法中期，以禪宗立場解釋《起信論》，文字簡潔而義理深刻。

### 二、歷史背景

明代中後期，禪宗衰落，教典湮沒。紫柏真可與憨山德清等人發起佛教復興運動，主張"以禪爲體，以教爲用"。紫柏認爲，禪宗的"明心見性"必須有教理的支撐，否則容易流於空疏。《起信論》作爲"教理"與"修行"之間的橋樑，自然受到重視。此疏正是在這一背景下產生，旨在爲禪宗學人提供教理基礎。

### 三、思想史定位

紫柏此疏的特色在於以禪宗"頓悟"思想來解釋《起信論》的"始覺"概念。他認爲，"始覺"不是漸進的覺悟過程，而是當下即是的頓悟——一念回光，即見本心。紫柏進一步將"一心二門"與禪宗"平常心是道"相聯繫："心真如門"即是"平常心"，"心生滅門"即是"煩惱心"，但二門本是一心，煩惱即菩提、平常心即是道。這一解釋對後來禪宗與教理的融合有重要推動作用。

## 白話解讀

### 核心內容

《大乘起信論釋要》是紫柏真可大師對《大乘起信論》的精要解釋。"釋要"就是抓住要點，不囉嗦。

紫柏大師是禪宗大德，所以他用禪宗的語言來解釋《起信論》：不要慢慢修、一點點覺悟，而是當下就覺悟！"始覺"不是從明天開始覺悟，而是此刻就迴光返照，看到自己的本心。

紫柏大師說："心真如門"就是禪宗說的"平常心"——不造作、不刻意、本來如此的心；"心生滅門"就是煩惱。但這兩個不是兩個心，而是一個心的兩個表現。就像你生氣的時候，那個能"知道自己在生氣"的覺知，就是"平常心"。

### 現代啓示

1. **當下覺醒**：現代人總在等"條件成熟"再行動。紫柏大師的"頓悟"思想告訴我們：覺醒不需要等待，就在此刻。

2. **平常心**：在競爭激烈的社會中，"平常心"是一種稀缺的能力。認識到"平常心是道"，有助於在壓力中保持內心的平衡。

3. **煩惱轉化**：紫柏"煩惱即菩提"的思想不是讓人放縱煩惱，而是看到煩惱背後的覺知。當你焦慮時，覺察那個焦慮——覺察本身就是解脫的開始。

### 閱讀建議

- 紫柏大師的文風簡潔有力，適合有一定禪宗基礎的讀者
- 可配合《紫柏老人集》中的其他文章，如《法語》《開示》等一同閱讀
- 注意其"以禪解教"的特點，理解禪宗對教理的獨特詮釋
