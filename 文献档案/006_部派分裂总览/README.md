## 部派源流骨架

佛灭后约百年，统一僧团经历长达三百年的「分家」，裂为上座、大众两系，再各分出十余部派，为后世大乘与上座部的分立埋下远因。

```
        根本僧团（佛灭后约前5世纪）
        ├── 上座部系（Sthavira，长老一系）
        │     ├── 说一切有部（Sarvāstivāda）→ 六足一身论、大毗婆沙论
        │     ├── 犊子部（Vātsīputrīya）→ 补特伽罗论
        │     ├── 化地部（Mahīśāsaka）→ 五分律
        │     ├── 法藏部（Dharmaguptaka）→ 四分律
        │     ├── 经量部（Sautrāntika）→ 经部师
        │     └── 雪山部（Haimavata）等
        └── 大众部系（Mahāsāṅghika，大众一系）
              ├── 一说部（Ekavyāvahārika）
              ├── 说出世部（Lokottaravāda）→ 佛陀超世间说
              ├── 鸡胤部（Kukkuṭika）
              ├── 多闻部（Bahuśrutīya）
              ├── 说假部（Prajñaptivāda）
              └── 制多山部（Caitya）等
```
