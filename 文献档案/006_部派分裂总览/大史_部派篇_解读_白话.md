## 一、《大史》是什么？

### 1.1 斯里兰卡的「史记」+「史诗」

《大史》（Mahāvaṃsa）是斯里兰卡最著名的佛教历史书，约写于公元5世纪，相当于中国的南北朝时期。书名「大史」就是「伟大的历史」或「大历史」。

**打个比方**：
- 《大史》就像斯里兰卡的「史记」+「史诗」
- 既有历史记录，又有文学故事
- 是研究早期佛教史最重要的文献之一

### 1.2 与《岛史》的关系

《大史》有一个「前身」叫《岛史》（Dīpavaṃsa），约写于公元4世纪：

- 《岛史》 = 原始版，更古老，更简洁
- 《大史》 = 升级版，更详细，更文学化
- 两者内容相似，但《大史》加了更多故事和细节
- 《大史》更受重视，影响更大

**现代类比**：
- 就像《三国演义》与《三国志》的关系
- 《岛史》是「三国志」，更原始、更历史
- 《大史》是「三国演义」，更详细、更文学

### 1.3 作者是谁？

《大史》的作者叫摩诃那摩（Mahānāma），是摩揭陀国（印度中部）的比丘，但长期在斯里兰卡活动。

**关于摩诃那摩**：
- 生活在公元5世纪
- 是斯里兰卡大寺（Mahāvihāra）的僧人
- 写《大史》的目的是确立斯里兰卡佛教的正统地位

## 二、佛教为什么分裂？——《大史》的说法

### 2.1 第二次结集：耶舍长老的故事

《大史》详细记载了第二次结集的故事，增加了耶舍长老的生平。

#### 耶舍长老是谁？

耶舍（Yasa Kākaṇḍakaputta）是迦乾陀子，佛陀的弟子之一。他在毘舍离游行时，听到跋耆子比丘提出十项戒律放宽的主张。

**耶舍的故事**：

1. **发现问题**：耶舍在毘舍离听到比丘们提出「十事非法」
2. **试图说服**：耶舍先试图说服跋耆子放弃，但未果
3. **召集长老**：耶舍前往各地邀请上座长老来毘舍离
4. **举行结集**：最终召集了七百比丘，举行第二次结集
5. **宣布非法**：结集宣布十事非法，维持戒律严格

**现代类比**：
- 耶舍就像「纪检干部」，发现违规，向上级报告
- 或者像「 whistleblower（吹哨人）」，发现问题，揭露真相
- 或者像「传统派领袖」，召集会议，维护传统

#### 七百比丘的结集

第二次结集有七百比丘参加，所以也叫「七百结集」或「毘舍离结集」。

**结集的过程**：
- **诵经**：阿难系弟子诵经
- **诵律**：优波离系弟子诵律
- **讨论**：讨论十事是否非法
- **决议**：宣布十事非法，维持戒律严格

**现代看法**：
- 七百比丘的数字可能是象征性的，表示「众多」
- 结集的历史真实性较高，但细节可能有文学加工
- 耶舍的故事可能是后来加上的，目的是增加结集的合法性

### 2.2 十事非法：戒律之争

《大史》对十事非法的记载与《岛史》基本一致，但增加了更多细节。

#### 十件事的通俗解释

| 序号 | 名称 | 通俗解释 | 现代类比 |
|------|------|----------|----------|
| 1 | 角盐净 | 允许把盐装在牛角里备用 | 允许提前准备调料 |
| 2 | 二指净 | 太阳偏西两指宽还可以吃饭 | 允许「稍微」迟到吃饭 |
| 3 | 村落间净 | 在村落之间可以吃东西 | 允许在通勤路上吃饭 |
| 4 | 住处净 | 在同一住处可以分开吃饭 | 允许在同一地方多次吃饭 |
| 5 | 许可净 | 吃完后可以请求再吃点 | 允许「再要一份」 |
| 6 | 习惯净 | 按当地习惯行事 | 允许「入乡随俗」 |
| 7 | 不搅净 | 牛奶不搅拌就可以喝 | 允许喝未加工的乳制品 |
| 8 | 饮阇楼净 | 可以喝未发酵的棕榈汁 | 允许喝「天然饮料」 |
| 9 | 无沿座具净 | 可以用没有边沿的坐垫 | 允许用「简易坐垫」 |
| 10 | 金银净 | 可以接受金银钱财 | 允许接受金钱供养 |

#### 为什么这十件事引起分裂？

**表面原因**：戒律的「严格」与「放宽」之争

- **上座派**：认为戒律要严格，不能放宽
- **跋耆子派**：认为戒律可以适应环境，适当放宽

**深层原因**：佛教的发展方向之争

- **保守派**：坚持原始传统，不改变
- **改革派**：适应时代变化，适当调整

**现代类比**：
- 就像公司制度：要严格按制度办事，还是灵活处理？
- 或者像宗教改革：要坚持传统教义，还是适应现代社会？
- 或者像法律：要严格执法，还是酌情处理？

### 2.3 根本分裂：大众部 vs 上座部

《大史》记载，第二次结集后，不接受决议的比丘形成大众部（Mahāsāṅghika），接受决议的比丘形成上座部（Theravāda）。

#### 《大史》的独特记载

《大史》有一句话是《岛史》没有的：

> 「大众部人多，上座部人少」

这句话暗示：
- 大众部人数众多，可能代表「多数派」
- 上座部人数少，但坚持传统，可能代表「精英派」
- 分裂不仅是教义之争，也是人数与质量之争

**现代类比**：
- 就像民主选举：人多的一方获胜，但不一定正确
- 或者像学术争论：主流观点人多，但少数派可能更正确
- 或者像公司决策：多数人的意见 vs 专家的意见

#### 分裂的结果

- **大众部**：改革派，人数多，主张适应时代
- **上座部**：保守派，人数少，主张坚持传统

**历史影响**：
- 大众部后来发展出大乘佛教
- 上座部传入斯里兰卡，形成南传佛教
- 两种传统至今并存

### 2.4 阿育王与第三次结集

《大史》详细记载了阿育王时代的第三次结集，这是《岛史》没有的。

#### 阿育王是谁？

阿育王（Aśoka，约公元前304-前232年）是印度孔雀王朝的第三代皇帝，统一了印度大部分地区。他最初是暴君，后来皈依佛教，成为佛教最著名的护法。

**阿育王的转变**：

1. **早期**：征服羯陵伽国，屠杀十万人，后悔
2. **中期**：皈依佛教，拜目犍连子帝须为师
3. **后期**：大力推广佛教，建立石柱法敕，派遣传教士

**现代类比**：
- 就像「暴君变圣人」：从秦始皇到唐太宗
- 或者像「战争罪犯变和平使者」：从战犯到诺贝尔和平奖得主
- 或者像「黑帮老大变慈善家」：从恶人到善人

#### 第三次结集

阿育王统一印度后，发现僧团内部因分裂而混乱，不同部派互相攻击。于是他邀请目犍连子帝须（Moggaliputta Tissa）主持第三次结集。

**第三次结集的过程**：

1. **召集僧团**：阿育王召集各派僧人
2. **讨论争议**：讨论各部派的教义分歧
3. **统一经律**：确立上座部为正统
4. **派遣传教**：派遣传教士到各地传播佛教

**第三次结集的结果**：

- **统一经律**：确立了上座部的正统地位
- **派遣传教**：传教士被派往各地，包括斯里兰卡、缅甸、中亚等
- **确立正统**：上座部成为「分别说部」（Vibhajjavāda）

**现代看法**：
- 第三次结集的历史真实性有争议
- 部分学者认为是上座部为了确立正统而「创造」的历史
- 但阿育王的支持对佛教传播确实有重要作用

#### 佛教传入斯里兰卡

第三次结集后，阿育王派遣其子摩哂陀（Mahinda）到斯里兰卡传教。

**摩哂陀的故事**：

1. **出家**：摩哂陀是阿育王的儿子，出家为比丘
2. **传教**：受阿育王派遣，到斯里兰卡传播佛教
3. **成功**：斯里兰卡国王天爱帝须（Devānāmpiya Tissa）皈依佛教
4. **建立大寺**：在斯里兰卡建立大寺（Mahāvihāra），成为南传佛教中心

**现代类比**：
- 就像「太子出家传教」：从王子到传教士
- 或者像「文化输出」：从印度到斯里兰卡
- 或者像「宗教扩张」：从发源地到海外

## 三、部派分裂的「家谱」

### 3.1 《大史》的部派谱系

《大史》对部派谱系的记载比《岛史》更详细、更系统：

**大众部系**：
```
大众部（Mahāsāṅghika）
    ├── 一说部（Ekabyohārika）
    ├── 鸡胤部（Gokulika）
    │       └── 说出世部（Lokottaravāda）
    ├── 多闻部（Bahuśrutīya）
    │       └── 说假部（Prajñaptivāda）
    └── 后来发展出大乘佛教
```

**上座部系**：
```
上座部（Theravāda）
    ├── 化地部（Mahiṃsāsaka）
    ├── 犊子部（Vajjiputtaka）
    │       ├── 法上部（Dhammuttarika）
    │       ├── 贤胄部（Bhaddayānika）
    │       └── 正量部（Saṃmitīya）
    └── 说一切有部（Sabbatthivāda）
            ├── 法藏部（Dhammaguttika）
            ├── 饮光部（Kassapika）
            ├── 说转部（Saṅkantika）
            └── 经量部（Sutavāda）
```

### 3.2 部派分裂的「家谱图」

```
原始佛教
    │
    ├── 第一次结集（佛灭后不久）
    │
    ├── 第二次结集（佛灭后100年）
    │       │
    │       ├── 上座部（Theravāda）── 保守派，人少
    │       │       │
    │       │       ├── 第一波分裂：化地部、犊子部、法上部、贤胄部、正量部
    │       │       │
    │       │       └── 第二波分裂：说一切有部、法藏部、饮光部、说转部、经量部
    │       │
    │       └── 大众部（Mahāsāṅghika）── 改革派，人多
    │               │
    │               ├── 一说部、鸡胤部、说出世部、多闻部、说假部
    │               │
    │               └── 后来发展出大乘佛教
    │
    └── 第三次结集（佛灭后200多年，阿育王时代）
            │
            ├── 上座部传入斯里兰卡，形成南传佛教
            │
            └── 大众部在印度继续发展，形成大乘佛教
```

## 四、《大史》的史学价值与局限

### 4.1 史学价值

《大史》作为历史文献具有以下价值：

- **权威性**：是南传佛教最权威的史传，影响深远
- **详细性**：提供了大量历史细节，便于研究
- **系统性**：构建了完整的佛教史框架
- **文学性**：采用诗歌形式，可读性强
- **对比性**：与北传文献对比，可重构更完整的历史

### 4.2 史学局限

《大史》作为历史文献也有以下局限：

- **宗教偏见**：作者有明显的上座部立场，对大众部有偏见
- **传说成分**：部分记载可能是传说而非历史
- **文学加工**：为了文学效果，可能改变历史事实
- **政治目的**：为王权服务，可能歪曲历史
- **时代错位**：部分记载可能反映作者时代而非历史时代

### 4.3 批判性阅读

阅读《大史》需要注意：

- **区分历史与传说**：哪些是可信的历史，哪些是文学加工
- **识别偏见**：作者的上座部立场如何影响记载
- **对比其他文献**：与北传文献、考古证据对比
- **分析动机**：作者为什么这样写，有什么目的

## 五、现代启示

### 5.1 对修行者的启示

《大史》记载的部派分裂告诉我们：

1. **佛教是多元的**：佛教内部有不同观点，这是正常的、健康的
2. **戒律重要**：十事非法的核心是戒律，说明戒律是佛教的基础
3. **传统与创新**：既要尊重传统，也要适应时代
4. **团结与分裂**：分裂虽然遗憾，但也促进了佛教的多样化发展
5. **信仰与历史**：了解历史可以加深信仰，但信仰不依赖于历史

### 5.2 对佛教研究的启示

《大史》的研究告诉我们：

1. **文献要对比**：南传和北传文献要对比研究，才能接近历史真相
2. **批判性阅读**：宗教文献有偏见，需要批判性分析
3. **历史是复杂的**：佛教分裂不是简单的「好人 vs 坏人」，而是复杂的历史过程
4. ** oral 传统的局限**：口传传统容易失真，需要文献考证
5. **文学与历史**：文学作品可以提供历史信息，但需要批判性分析

### 5.3 对现代社会的启示

《大史》记载的部派分裂与现代社会的一些现象相通：

1. **传统 vs 改革**：任何组织都面临传统与改革的张力
2. **规则 vs 灵活**：规则要严格，但也要适应环境
3. **统一 vs 多元**：统一有力量，多元有活力
4. **历史记忆**：历史记忆会影响现实认同
5. **领袖作用**：阿育王的例子说明领袖对宗教发展有重要影响

---

## 六、总结

《大史》是研究佛教部派分裂最重要的南传文献，其详细记载和文学价值使其成为佛教史研究的必读文献。通过《大史》，我们可以了解南传佛教对部派分裂的记载，以及与北传记载的差异。

佛教分裂的历史告诉我们：

> **佛教不是一成不变的，而是不断发展的。分裂虽然带来痛苦，但也促进了佛教的多样化和大乘佛教的兴起。了解历史，可以让我们更深刻地理解佛教的多元性和包容性。**

今天的修行者，既可以从上座部学习「尊重传统」的精神，也可以从大众部学习「勇于改革」的态度。两者结合，才是完整的佛教修行。
