## 關鍵概念

- **前弘期**（7—9世紀）：松贊干布至朗達瑪滅佛時期
- **後弘期**（10世紀至今）：佛教在西藏的復興
- **寧瑪派**（Nyingma）：舊譯派，最古老的宗派
- **薩迦派**（Sakya）：道果法傳承
- **噶舉派**（Kagyu）：大手印傳承
- **格魯派**（Gelug）：宗喀巴改革派，達賴喇嘛所屬
