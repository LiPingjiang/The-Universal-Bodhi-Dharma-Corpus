## 关键概念

- **前弘期**（7—9世纪）：松赞干布至朗达玛灭佛时期
- **后弘期**（10世纪至今）：佛教在西藏的复兴
- **宁玛派**（Nyingma）：旧译派，最古老的宗派
- **萨迦派**（Sakya）：道果法传承
- **噶举派**（Kagyu）：大手印传承
- **格鲁派**（Gelug）：宗喀巴改革派，达赖喇嘛所属
