## 目錄簡介

本目錄收錄宗喀巴大師《菩提道次第廣論》（Lam rim chen mo）的核心文獻，這是藏傳佛教格魯派最重要的修行指南，系統整理了從初學者到成佛的完整修行次第。

**梵文名稱**：Bodhi-mārga-krama-mahā-śāstra（菩提道次第廣論）  
**藏文名稱**：Byang chub lam rim chen mo  
**作者**：宗喀巴（Tsongkhapa，1357—1419年）  
**成書時代**：約公元1402年  
**核心主題**：三士道、止觀雙運、顯密圓融

## 關鍵概念

- **三士道**：下士道（求人天善趣）、中士道（求個人解脫）、上士道（求一切衆生解脫）
- **奢摩他**（śamatha）：止，心的專注與平靜
- **毗婆舍那**（vipaśyanā）：觀，對空性的直接洞察
- **中觀應成派**（Prāsaṅgika-Mādhyamaka）：宗喀巴所依據的最高哲學立場
- **自續派**（Svātantrika）：中觀的另一分支，宗喀巴認爲低於應成派
