# 大乘起信論宗要

## 學術解讀

### 一、文獻學信息

《大乘起信論宗要》是唐代湛然（711-782）所撰。湛然，俗姓戚，常州荊溪（今江蘇宜興）人，爲唐代天台宗第九祖，世稱"荊溪大師"或"妙樂大師"，被尊爲天台宗"中興"的關鍵人物。此疏以"宗要"爲名，取"宗派之要，會歸一心"之意，是天台宗解釋《大乘起信論》的代表性著作。

湛然一生致力於復興天台宗，面對當時華嚴宗、禪宗的興盛，他以"性具"思想爲武器，系統闡發天台宗的獨特教義。著有《摩訶止觀輔行傳弘決》《法華玄義釋籤》《金剛錍》等。此疏作於其中年，是其以天台宗義理統攝《起信論》的重要嘗試。

### 二、歷史背景

唐代中期，天台宗經歷了一段衰落期。湛然立志中興天台宗，提出"性具"思想與華嚴宗的"性起"思想相抗衡。他認爲，華嚴宗只說"性起"（從清淨法性起現萬法），忽略了"性具"（善惡諸法皆本具於法性）。湛然將《起信論》納入天台宗的框架，旨在證明天台宗的"性具"思想纔是對《起信論》最忠實的解釋。此疏正是在這一宗派競爭背景下產生。

### 三、思想史定位

湛然此疏的核心貢獻在於以"性具"思想重新解釋《起信論》的"一心二門"。他認爲，《起信論》的"一心"不是單純的"清淨心"，而是"具足一切法"的心——善也具、惡也具、淨也具、染也具。"心真如門"是"性具"的"理體"，"心生滅門"是"性具"的"事用"。湛然進一步將"無明"解釋爲"性惡"的顯現，將"修行"解釋爲"修惡即善"——不是消滅惡，而是在惡中見善。這一解釋對後來天台宗與《起信論》的關係有決定性影響。

## 白話解讀

### 核心內容

《大乘起信論宗要》是唐代天台宗中興祖師湛然對《大乘起信論》的解釋。湛然大師用天台宗獨特的"性具"思想來理解《起信論》。

湛然大師說：我們的心不是單純的"清淨"或"染污"，而是"什麼都具足"——善也具足、惡也具足、佛也具足、衆生也具足。這就是"性具"。"心真如門"是這種"具足"的"理體"（本質），"心生滅門"是這種"具足"的"事用"（表現）。

湛然大師特別強調：惡不是從外面來的，而是本性具足的。所以修行不是消滅惡，而是"修惡即善"——在惡中修行，轉化惡爲善。就像鍊金：礦石中有金子（善），也有雜質（惡），修行的過程就是提煉金子，而不是把礦石扔掉。

### 現代啓示

1. **接納完整自我**：現代人常常自我否定，想要"消滅"自己的缺點。湛然的"性具"思想告訴我們：接納完整的自己，包括缺點，然後加以轉化。

2. **轉化而非消滅**：面對負面情緒和不良習慣，與其對抗消滅，不如理解其根源，加以轉化。這就像治水：堵不如疏。

3. **一念三千的豐富性**："一念三千"告訴我們：每一個念頭都蘊含無限可能。當下的選擇影響深遠，所以要謹慎對待每一個念頭。

### 閱讀建議

- 湛然的疏解需要天台宗基礎，建議先讀《摩訶止觀》或《法華玄義》
- 可對比子璇以天台宗解《起信論》的方式，理解天台宗解釋傳統的延續
- 注意"性具"與"性起"的差異，理解天台宗與華嚴宗的根本分歧
