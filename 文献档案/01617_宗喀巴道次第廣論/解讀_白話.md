# 菩提道次第广论 · 白話解讀

## 这部论典是什么？

《菩提道次第广论》简称"广论"，是宗喀巴大师最重要的著作，写于1402年。这本书的名字可以这样理解："菩提"是觉悟，"道次第"是修行的阶梯，"广论"是详细的论述。简单说，这是一本告诉你如何一步一步走向觉悟的完整指南。

## 核心内容

广论把修行分成三个层次，就像爬楼梯一样：

**第一层（下士道）**：认识到生命的无常和苦，发心求得来世的善趣（好的投生）。这一层的修行包括：思维死亡无常、思维恶趣之苦、皈依三宝、修习业果。

**第二层（中士道）**：认识到整个轮回都是苦，发心求得解脱。这一层深入学习四圣谛，理解苦的根源在于烦恼和业。

**第三层（上士道）**：发菩提心，不仅为自己，更为所有众生求得圆满觉悟。这是最高的发心，包括修习六度（布施、持戒、忍辱、精进、禅定、智慧）和止观双运。

## 现代启示

广论的智慧在今天依然实用：它告诉我们，真正的成长需要循序渐进，不能跳过基础。就像学数学要先学加减法再学微积分，修行也要先打好基础，再追求高深境界。

## 阅读建议

广论篇幅较长，建议先读《菩提道次第略论》（广论的简化版）。读广论时，最好配合善知识的讲解，因为其中涉及很多专业术语和深奥义理。法尊法师的汉译本是目前最流通的版本。

## 关键概念表

| 概念 | 简单解释 |
|------|---------|
| 三士道 | 修行的三个层次：求善趣、求解脱、求觉悟 |
| 菩提心 | 为利益一切众生而求觉悟的心 |
| 止（奢摩他） | 让心安住、专注的禅定 |
| 观（毗婆舍那） | 用智慧观察实相的禅修 |
| 空性 | 万物没有固有的独立存在 |
| 业果 | 行为（业）会产生相应的结果（果） |

## 读后思考

1. 为什么宗喀巴强调修行要从"下士道"开始，而不是直接修高深法门？
2. "止观双运"的修行方式，与现代正念冥想有什么异同？
3. 菩提心的修习如何改变我们对待他人的方式？
