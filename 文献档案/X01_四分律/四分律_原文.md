## 二、文獻概要

《四分律》為佛教根本律藏之一，屬法藏部（曇無德部）所傳之廣律，是漢傳佛教流傳最廣、影響最為深遠的律典。全律因內容分為四部分（四分）而得名，為中國律宗所依根本經典，亦為漢傳佛教僧尼受戒持戒之主要依據。

本律於姚秦弘始年間，由罽賓沙門佛陀耶舍於長安譯出。佛陀耶舍誦出梵本，竺佛念筆受譯為漢文。譯出後，經道安、慧遠等大德弘弘揚，漸成為漢地最通行之律典。唐代道宣律師據此律撰《四分律刪繁補闕行事鈔》等「南山三大部」，正式創立律宗，使《四分律》成為漢傳佛教唯一通行之廣律。

## 三、核心內容

### 初分：比丘戒（比丘犍度）

初分共二十一卷，詳述比丘二百五十條戒律，分為八段（波羅夷、僧殘、不定、捨、捨墮、單墮、眾學法、滅諍法、滅法），是比丘戒法的完整記錄。

- **波羅夷法**（四**（四條）：殺、盜、婬、大妄語，犯者永棄佛法之外，如斷頭不可復生。
- **僧殘法**（十三條）：犯者須於僧眾中行摩那埵懺悔，如人殘命尚存。
- **不定法**（二條）：依可信舉發人之言而待查證之戒。
- **捨墮法**（三十條）：涉及衣缽等財物之戒，犯者須捨其物於僧中懺悔。
- **單墮法**（九十條）：輕罪，但須對一人懺悔。
- **眾學法**（一百條）：威儀細行，屬輕戒，但數量多，須時時學習。
- **滅諍法**（七條）：處理僧團諍論之程序規定。
- **滅法**：有關滅諍之方法。

### 二分：比丘尼戒（比丘尼犍度）

二分共十五卷，記載比丘尼三百四十八條戒律。比丘尼戒較比丘戒為多，係因佛陀為保護尼眾而增設若干防護條款。比丘尼戒亦分八段，其中波羅夷八條（較比丘多四條），僧殘十七條，捨墮三十條，單墮一百七十八條等。

此部分體現了早期佛教對女性出家者的制度性安排，反映了當時印度社會背景與佛教僧團管理的實際需要。

### 三分：法犍度（受戒、說戒等二十犍度）

三分共十五卷，收錄二十種「犍度」（Skandhaka，即篇章），涵蓋僧團運作的各種制度與儀軌：

1. **受戒犍度**：規定出家受具足戒之條件與程序
2. **說戒犍度**：布薩誦戒之制度
3. **安居犍度**：雨季安居之規定
4. **自恣犍度**：安居結束自恣懺悔之儀式
5. **皮革犍度**：關於革履等皮革用品之規定
6. **衣犍度**：袈裟等衣著之規定
7. **藥犍度**：醫藥飲食之規定
8. **迦絺那衣犍度**：功德衣之制度
9. **拘睒彌犍度**：處理僧團分裂之方法
10. **瞻波犍度**：如法羯磨之條件
11. **呵責犍度**：處罰惡性比丘之程序
12. **人犍度**：關於懺悔出罪之規定
13. **覆藏犍度**：覆藏罪之處理
14. **遮犍度**：遮說戒之規定
15. **破僧犍度**：破和合僧之果報
16. **滅諍犍度**：七滅諍法之詳細說明
17. **比丘尼犍度**：比丘尼特有之制度
18. **法犍度**：各種雜法
19. **房舍犍度**：僧房建設之規定
20. **雜犍度**：其他雜項規定

### 四分：房舍犍度與雜犍度

四分共九卷，主要包含房舍犍度與雜犍度之詳細內容，以及五百集法與七百集法之記載——即佛陀涅槃後第一次與第二次結集之情形，具有極高的歷史文獻價值。

此外，四分之末附有「調部」與「毗尼增一」，前者為戒條之分類整理，後者為律文之增補說明。

## 五、術語註釋

| 術語 | 解釋 |
|------|------|
| 犍度（Skandhaka） | 律藏中篇章之名，意為「蘊」、「聚」，即同類事項之彙集 |
| 波羅夷（Pārājika） | 極重罪，犯者如斷頭，永棄僧團之外 |
| 僧殘（Saṅghāvaśeṣa） | 重罪之一，犯者須於僧中行摩那埵懺悔 |
| 羯磨（Karma） | 僧團中之表決儀式，有白羯磨、白二羯磨、白四羯磨等 |
| 布薩（Uposatha） | 僧團半月一次之誦戒集會 |
| 安居（Vārṣika） | 雨季三個月間僧眾定居一處修行之制度 |
| 自恣（Pravāraṇā） | 安居結束時，僧眾互相檢舉過失之儀式 |
| 摩那埵（Mānatta） | 僧殘罪之懺悔法，須行六夜 |
