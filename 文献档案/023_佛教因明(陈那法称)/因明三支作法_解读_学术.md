## 一、陈那因明的历史定位

### 1.1 从五支到三支：因明的革命

古因明（旧因明）以五支作法为核心：宗（命题）、因（理由）、喻（例证）、合（应用）、结（结论）。这一体系源自印度正理派（Nyāya）的传统，弥勒、无著等早期佛教论师沿用此体系。

陈那（Dignāga，约480–540 CE）的革命性贡献在于：将五支简化为三支（宗、因、喻），并通过"因三相"（tri-rūpa）建立了严格的有效性标准。这一简化不仅是形式上的，更是逻辑结构上的深化：

- **五支作法**：宗→因→喻→合→结，是一种演示性的论证程序
- **三支作法**：宗→因→喻，是一种逻辑上完备的推理结构，"合"与"结"被视为冗余

陈那的核心洞见是：有效的推理必须满足"因三相"，即因与宗的主词、同品、异品之间的三重关系。这一标准使因明从修辞性论辩转变为逻辑性推理。

### 1.2 陈那的主要著作

陈那的因明著作现存情况：

| 著作 | 梵名 | 现存情况 |
|------|------|----------|
| 《集量论》 | Pramāṇasamuccaya | 梵文残存，藏译完整（Tshad ma kun las btus pa） |
| 《因明正理门论》 | Nyāyamukha | 梵文佚失，汉译（玄奘译，T1628）、藏译存 |
| 《观所缘缘论》 | Ālambanaparīkṣā | 梵文存，汉译（玄奘译，T1624）、藏译存 |
| 《掌中论》 | Hastavālaprakaraṇa | 梵文佚失，汉译（义净译，T1621）存 |

《集量论》是陈那因明体系的集大成之作，共六章：现量品、为自比量品、为他比量品、喻似喻品、遮诠品、观他量品。

### 1.3 法称对陈那的继承与发展

法称（Dharmakīrti，约600–660 CE）是陈那之后最重要的因明论师，其"七论"（Saptapadārtha）是藏传因明的核心教材：

| 著作 | 梵名 | 特点 |
|------|------|------|
| 《量评释》 | Pramāṇavārttika | 对陈那《集量论》的系统注释与发展，四章 |
| 《量决择》 | Pramāṇaviniścaya | 简明纲要，三章 |
| 《正理一滴》 | Nyāyabindu | 入门要典，三章 |
| 《因滴》 | Hetubindu | 专论因的有效性 |
| 《观相属》 | Sambandhaparīkṣā | 论关系的本质 |
| 《诤理》 | Vādanyāya | 论辩规则 |
| 《成他相续》 | Santānāntarasiddhi | 论他心的存在 |

法称的主要贡献是将因明与佛教认识论（特别是唯识学）紧密结合，并对"遍充关系"（vyāpti，因与宗之间的必然联系）进行了深入分析。

## 二、因三相的逻辑分析

### 2.1 因三相的形式结构

因三相（tri-rūpa）是陈那因明的核心，其形式结构如下：

设论式为：宗（S is P），因（S is M），喻（M is P）

- **第一相**（遍是宗法性，pakṣa-dharmatā）：M ∈ S（因M是宗的主词S的性质）
- **第二相**（同品定有性，sapakṣe sattva）：∃x（x ∈ P ∧ x ∈ M）（在同品P中，因M存在）
- **第三相**（异品遍无性，vipakṣe asattva）：∀x（x ∉ P → x ∉ M）（在异品非P中，因M不存在）

以"声无常"论式为例：
- 宗：声（S）是无常（P）
- 因：所作性（M）
- 第一相：声（S）具有所作性（M）✓
- 第二相：无常的事物（P，如瓶）中，所作性（M）存在 ✓
- 第三相：常住的事物（非P，如虚空）中，所作性（M）不存在 ✓

### 2.2 九句因

陈那在《集量论》中建立了"九句因"（nava-padārtha），通过同品有无与异品有无的组合，系统分析因的有效性：

| 句 | 同品 | 异品 | 判断 |
|----|------|------|------|
| 第一句 | 有 | 有 | 不定因（两俱不定） |
| 第二句 | 有 | 无 | 正因（同品有，异品无） |
| 第三句 | 有 | 俱 | 不定因（异品一分转） |
| 第四句 | 无 | 有 | 相违因（同品无，异品有） |
| 第五句 | 无 | 无 | 不定因（两俱不定） |
| 第六句 | 无 | 俱 | 相违因（异品一分转） |
| 第七句 | 俱 | 有 | 不定因（同品一分转） |
| 第八句 | 俱 | 无 | 不定因（同品一分转） |
| 第九句 | 俱 | 俱 | 不定因（两俱不定） |

其中，第二句（同品有，异品无）为正因；第四句（同品无，异品有）为相违因（导致相反结论）；其余七句均为不定因（无法得出确定结论）。

### 2.3 遍充关系（vyāpti）

法称对陈那因明的最重要发展是对"遍充关系"（vyāpti，"普遍伴随"）的分析。遍充关系是因与宗之间的必然联系，是推理有效性的基础。

法称认为，遍充关系有两种基础：
- **自性关系**（svabhāva-pratibandha）：因与宗之间存在本质上的同一性，如"所作性"与"无常性"之间的关系
- **因果关系**（kārya-kāraṇa-bhāva）：因与宗之间存在因果关系，如"烟"与"火"之间的关系

## 三、汉传因明的发展

### 3.1 玄奘的传译

玄奘（602–664 CE）在印度那烂陀寺（Nālandā）学习因明，归国后翻译了《因明正理门论》（陈那著）和《因明入正理论》（商羯罗主著），将新因明引入中国。

玄奘在曲女城（Kanauj）大会上，以"真唯识量"论式挑战全印度论师，无人能破，被誉为"大乘天"（Mahāyānadeva）。"真唯识量"是：

> 宗：极成色，非离眼识。
> 因：自许初三摄，眼所不摄故。
> 喻：如眼识。

### 3.2 窥基的注疏

窥基（632–682 CE）是玄奘的首席弟子，著《因明入正理论疏》（《因明大疏》，T1840），是汉传因明最重要的注疏。窥基对三支作法、因三相、似宗、似因、似喻等进行了系统阐释，奠定了汉传因明的基本框架。

### 3.3 汉传因明的特点

汉传因明与印度因明（特别是法称体系）的主要差异：

| 方面 | 汉传因明 | 藏传因明（法称体系） |
|------|----------|---------------------|
| 核心文献 | 《因明入正理论》（商羯罗主） | 法称七论 |
| 重点 | 论式形式、似因似喻 | 认识论、遍充关系 |
| 发展 | 唐代后逐渐衰落 | 至今仍是寺院必修 |
| 与哲学的关系 | 相对独立 | 与中观、唯识紧密结合 |
