## 佛教因明源流骨架

佛教因明（Hetuvidyā，"因的学问"，即关于推理与论证的学问）是印度佛教哲学中最具系统性的逻辑认识论传统。其发展脉络可分为古因明与新因明两大阶段：

```
        佛教因明发展脉络
        ├── 古因明（旧因明）
        │       ├── 足目（Akṣapāda Gautama）《正理经》— 印度逻辑总源
        │       ├── 弥勒（Maitreya）《瑜伽师地论》— 佛教因明初建
        │       └── 无著（Asaṅga）《显扬圣教论》— 五支作法
        ├── 新因明（陈那体系）
        │       ├── 陈那（Dignāga，约480–540）
        │       │       ├── 《集量论》（Pramāṇasamuccaya）— 新因明奠基
        │       │       ├── 《因明正理门论》（Nyāyamukha）— 三支作法
        │       │       └── 《观所缘缘论》（Ālambanaparīkṣā）— 认识论
        │       └── 法称（Dharmakīrti，约600–660）
        │               ├── 《量评释》（Pramāṇavārttika）— 因明集大成
        │               ├── 《量决择》（Pramāṇaviniścaya）— 简明纲要
        │               └── 《正理一滴》（Nyāyabindu）— 入门要典
        └── 汉传因明
                ├── 玄奘（602–664）— 传入中国，翻译《因明入正理论》
                └── 窥基（632–682）— 《因明大疏》，汉传因明集大成
```

陈那（Dignāga）是佛教因明史上的革命性人物。他将古因明的五支作法（宗、因、喻、合、结）简化为三支作法（宗、因、喻），并建立了以现量（pratyakṣa，直接知觉）和比量（anumāna，推理）为核心的认识论体系。法称（Dharmakīrti）在陈那基础上进一步深化，将因明与佛教哲学（特别是唯识学）紧密结合，成为印度佛教哲学史上最重要的逻辑学家之一。

汉传因明以玄奘从印度带回的《因明入正理论》（商羯罗主著）和《因明正理门论》（陈那著）为核心，经窥基系统注疏，形成了独具特色的汉传因明传统。藏传因明则以法称七论为核心，至今仍是藏传佛教寺院教育的必修科目。

## 关键概念速查

| 概念 | 梵语 | 简释 |
|------|------|------|
| 因明 | Hetuvidyā | 关于推理与论证的学问，佛教逻辑学 |
| 现量 | pratyakṣa | 直接知觉，不经推理的直接认识 |
| 比量 | anumāna | 推理，通过因（理由）推知宗（结论） |
| 三支作法 | tri-rūpa-liṅga | 宗（命题）、因（理由）、喻（例证）三支论式 |
| 宗 | pakṣa | 论式的命题，待证的结论 |
| 因 | hetu | 论式的理由，支持宗的根据 |
| 喻 | dṛṣṭānta | 论式的例证，同喻与异喻 |
| 九句因 | nava-padārtha | 陈那建立的因的九种形式，判断因的有效性 |
| 量 | pramāṇa | 有效的认识手段，现量与比量 |
| 自比量 | svārthānumāna | 为自己推理，内心的推理过程 |
| 他比量 | parārthānumāna | 为他人论证，公开的论式表达 |
