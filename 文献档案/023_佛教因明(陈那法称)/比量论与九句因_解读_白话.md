## 一、比量：用推理认识世界

比量（anumāna）就是推理——通过已知的事实，推断出未知的结论。

最经典的例子：你看到山上有烟，推断山上有火。你没有直接看到火（那是现量），而是通过烟推断出来的——这就是比量。

比量在日常生活中无处不在：看到地面湿了，推断刚才下过雨；看到朋友皱眉，推断他心情不好；看到股价下跌，推断公司业绩不好……这些都是比量。

## 二、九句因：好理由的九种情况

陈那发现，一个理由（因）与结论的关系只有九种可能。他把这九种情况列出来，分析哪些是有效的理由，哪些是无效的。

### 2.1 最简单的理解

想象你是一个侦探，要判断一个证据是否有效：

- **有效证据**：这个证据在"有罪的人"中出现，在"无罪的人"中不出现——这样才能区分有罪和无罪
- **无效证据**：这个证据在有罪和无罪的人中都出现，或者都不出现——无法区分
- **相违证据**：这个证据在无罪的人中出现，在有罪的人中不出现——反而证明了无罪

九句因就是把这种分析系统化，列出所有可能的情况。

### 2.2 九句因简表

| 情况 | 同品（与结论相同的事物） | 异品（与结论相反的事物） | 结果 |
|------|------|------|------|
| 第一句 | 有 | 有 | 无效（两边都有，无法区分） |
| **第二句** | **有** | **无** | **有效！正因** |
| 第三句 | 有 | 有时有 | 无效（不确定） |
| 第四句 | 无 | 有 | 相违（得出相反结论） |
| 第五句 | 无 | 无 | 无效（两边都没有） |
| 第六句 | 无 | 有时有 | 相违（不确定） |
| 第七句 | 有时有 | 有 | 无效（不确定） |
| 第八句 | 有时有 | 无 | 无效（不确定） |
| 第九句 | 有时有 | 有时有 | 无效（不确定） |

只有第二句（同品有，异品无）是有效的正因。

## 三、遍充关系：为什么理由能证明结论？

### 3.1 什么是遍充？

遍充（vyāpti）就是因（理由）和宗（结论）之间的必然联系。

"声无常，所作性故"这个论式之所以有效，是因为"所作性"（被制作出来）和"无常性"（会消失）之间有必然联系——凡是被制作出来的东西，必然是无常的。这种必然联系就是遍充。

### 3.2 两种必然联系

法称认为，遍充关系只有两种真正的基础：

**同一性**：因和宗在本质上是同一的。比如"所作性"和"无常性"——被制作出来的东西必然是无常的，这是本质上的联系，不是偶然的。

**因果性**：因是宗的结果。比如"烟"和"火"——烟是由火产生的，所以看到烟就能推知有火。

偶然的伴随关系不能作为遍充的基础。比如"每次我出门，天就下雨"——这只是偶然的伴随，不是必然联系，不能用来推理。

## 四、总结

比量论和九句因告诉我们：推理要有效，理由必须与结论有必然联系（遍充关系），而且这种联系必须是本质上的（同一性或因果性），而不是偶然的。

> **比量的启示：好的推理需要好的理由，好的理由需要必然的联系。**

这不仅是逻辑学的原则，也是我们日常思考和判断的基础。
