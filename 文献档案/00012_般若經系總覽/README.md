## 般若经系源流骨架

般若经系（Prajñāpāramitā，"般若波罗蜜多"，意为"到彼岸的智慧"）是大乘佛教最重要的经典群，也是整个大乘思想的根基。般若经系的核心主题是「空性」（śūnyatā）——一切法无自性，无固定不变的本质。般若经系的形成是一个漫长的历史过程，从公元前1世纪到公元5世纪，经历了从短本到长本的不断扩充：

```
        般若经系（Prajñāpāramitā）
        ├── 短本系列
        │       ├── 《八千颂般若》（Aṣṭasāhasrikā-prajñāpāramitā）→ 最古老，约公元前1世纪
        │       ├── 《金刚经》（Vajracchedikā-prajñāpāramitā）→ 约公元1-2世纪
        │       └── 《心经》（Prajñāpāramitā-hṛdaya）→ 约公元1-3世纪
        ├── 中本系列
        │       ├── 《二万五千颂般若》（Pañcaviṃśatisāhasrikā）→ 大品般若
        │       └── 《一万八千颂般若》（Aṣṭādaśasāhasrikā）
        └── 长本系列
                └── 《十万颂般若》（Śatasāhasrikā-prajñāpāramitā）→ 最长，约公元3-5世纪
```

般若经系的核心思想可以概括为三个层次：第一，「空性」（śūnyatā）——一切法无自性，无固定不变的本质；第二，「不可得」（anupalabdha）——一切法不可执取，不可把握；第三，「三段否定」（三轮体空）——施者、受者、所施之物，三者皆空。这三个层次构成了般若思想的核心逻辑，也是理解整个大乘佛教的关键。

般若经系对中国佛教的影响极为深远。鸠摩罗什（Kumārajīva，344-413）翻译了《大品般若》《小品般若》《金刚经》等，奠定了中国大乘佛教的思想基础。玄奘（602-664）则翻译了《大般若经》六百卷，是中国佛教史上规模最大的翻译工程之一。

## 关键概念速查

| 概念 | 梵/巴利语 | 简释 |
|------|-----------|------|
| 般若 | prajñā | 智慧，特指洞见空性的智慧 |
| 空性 | śūnyatā | 一切法无自性，无固定不变的本质 |
| 不可得 | anupalabdha | 一切法不可执取，不可把握 |
| 三轮体空 | tri-maṇḍala-śūnyatā | 施者、受者、所施之物三者皆空 |
| 般若波罗蜜多 | prajñāpāramitā | 到彼岸的智慧，般若的完成 |
| 色即是空 | rūpaṃ śūnyatā | 物质现象即是空性 |
| 应无所住 | apratiṣṭhita | 不执著于任何事物 |
