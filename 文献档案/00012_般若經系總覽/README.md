## 般若經系源流骨架

般若經系（Prajñāpāramitā，"般若波羅蜜多"，意爲"到彼岸的智慧"）是大乘佛教最重要的經典羣，也是整個大乘思想的根基。般若經系的核心主題是「空性」（śūnyatā）——一切法無自性，無固定不變的本質。般若經系的形成是一個漫長的歷史過程，從公元前1世紀到公元5世紀，經歷了從短本到長本的不斷擴充：

```
        般若经系（Prajñāpāramitā）
        ├── 短本系列
        │       ├── 《八千颂般若》（Aṣṭasāhasrikā-prajñāpāramitā）→ 最古老，约公元前1世纪
        │       ├── 《金刚经》（Vajracchedikā-prajñāpāramitā）→ 约公元1-2世纪
        │       └── 《心经》（Prajñāpāramitā-hṛdaya）→ 约公元1-3世纪
        ├── 中本系列
        │       ├── 《二万五千颂般若》（Pañcaviṃśatisāhasrikā）→ 大品般若
        │       └── 《一万八千颂般若》（Aṣṭādaśasāhasrikā）
        └── 长本系列
                └── 《十万颂般若》（Śatasāhasrikā-prajñāpāramitā）→ 最长，约公元3-5世纪
```

般若經系的核心思想可以概括爲三個層次：第一，「空性」（śūnyatā）——一切法無自性，無固定不變的本質；第二，「不可得」（anupalabdha）——一切法不可執取，不可把握；第三，「三段否定」（三輪體空）——施者、受者、所施之物，三者皆空。這三個層次構成了般若思想的核心邏輯，也是理解整個大乘佛教的關鍵。

般若經系對中國佛教的影響極爲深遠。鳩摩羅什（Kumārajīva，344-413）翻譯了《大品般若》《小品般若》《金剛經》等，奠定了中國大乘佛教的思想基礎。玄奘（602-664）則翻譯了《大般若經》六百卷，是中國佛教史上規模最大的翻譯工程之一。
