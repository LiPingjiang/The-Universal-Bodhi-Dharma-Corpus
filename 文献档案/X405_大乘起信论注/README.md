## 学术解读

### 一、文献学信息

《大乘起信论注》是唐代道宣（596-667）所撰。道宣，俗姓钱，丹徒（今江苏镇江）人，为唐代律宗创始人，世称"南山律师"，被尊为中国律宗南山宗初祖。此疏以"注"为名，取"注解文句，疏通义理"之意，是《大乘起信论》注疏中较为早期的一部。

道宣一生致力于律学的整理和弘扬，著有《四分律行事钞》《四分律羯磨疏》《四分律含注戒本疏》等，被合称为"南山三大部"。同时，道宣也精通经论，著有《广弘明集》《续高僧传》等。此疏作于其中年，体现了其不仅精通律学，也深达教理的学术素养。

### 二、历史背景

道宣活跃于唐初，当时佛教正处于从隋代复兴向唐代鼎盛过渡的时期。唐初皇室重视佛教，但更倾向于秩序和规范。道宣作为律宗创始人，强调戒律是佛教的基础，同时也不废教理。此疏正是在这一背景下产生，旨在说明《起信论》的"信"与戒律的"持"是一致的：信是内心的基础，戒是行为的规范，二者相辅相成。

### 三、思想史定位

道宣此疏的特色在于以律宗的立场来解释《起信论》。他认为，《起信论》的"信心"是持戒的基础，没有信心，戒律只是外在的形式；有了信心，戒律才是内心的自然流露。道宣进一步将"一心二门"与戒律实践相联系："心真如门"即是"戒体"（本有的清净佛性），"心生灭门"即是"戒相"（具体的戒律规范）。这一解释虽然非主流，但体现了律宗对《起信论》的独特理解，丰富了《起信论》的解释传统。

## 白话解读

### 核心内容

《大乘起信论注》是唐代律宗创始人道宣大师对《大乘起信论》的注解。道宣大师是戒律专家，但他也深达教理。

道宣大师的独特视角是：把《起信论》和戒律联系起来。他说：《起信论》讲"信"，戒律讲"持"，信和戒是一体的。没有内心的信，守戒只是外在的形式；有了内心的信，守戒就是自然的行为。

道宣大师还用戒律的语言来解释"一心二门"："心真如门"就是"戒体"——我们受戒时得到的那个清净的佛性；"心生灭门"就是"戒相"——具体的戒律条文。戒体是内在的、清净的，戒相是外在的、变化的，但二者本是一体。

### 现代启示

1. **内外一致**：现代人常常"说一套做一套"。道宣大师告诉我们：真正的道德不是外在的规则，而是内心的自然流露。信是戒的内在基础，戒是信的外在表现。

2. **自律与自由**：很多人认为戒律是束缚。但道宣大师说：戒律是"戒体"的显现，而戒体是清净的佛性。所以守戒不是被束缚，而是回归本性。

3. **信心与行动**："信"和"戒"的关系，启示我们：任何行动都需要内心的信念支撑。没有信念的行动是空洞的，没有行动的信念是虚弱的。

### 阅读建议

- 道宣的注疏较为平实，适合初学者
- 可配合《四分律行事钞》一同阅读，了解律宗的基本思想
- 注意其"以戒解教"的特点，理解律宗对教理的独特视角
