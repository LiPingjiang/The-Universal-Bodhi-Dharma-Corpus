# 华严金狮子章 · 白話解讀

## 這是什麼書？

《華嚴金獅子章》是法藏法師用一個金獅子做比喻，給武則天講解華嚴宗深奧道理的記錄。這是華嚴宗最通俗、最好懂的入門書——用一個生動的比喻，把最難懂的哲學講得明明白白。

## 核心內容

### 1. 金獅子的比喻

法藏法師看到殿前有一座金獅子，就用它來做比喻：

金 = 理（本質、真如）
獅子 = 事（現象、形象）

金和獅子的關係，就是本質和現象的關係——金做成獅子，獅子就是金。兩者不即不離、圓融無礙。

### 2. 十玄門的金獅子版

法藏用金獅子逐一解釋十玄門。比如：
- 同時具足——金和獅子同時存在，不可分離
- 一多相容——獅子的每一根毛都包含整個獅子
- 隱密顯了——從金的角度看，獅子相是隱的；從獅子的角度看，金性是隱的

### 3. 六相圓融的金獅子版

總相——整個獅子
別相——眼、耳、鼻、舌、身各部分
同相——都屬於一個獅子
異相——各部分形狀不同
成相——合力構成獅子
壞相——各部分各守自性

六相同時成立，圓融無礙。

### 4. 五教的金獅子版

小教——只看到獅子，不見金
始教——知道金和獅子有區別
終教——明白金就是獅子
頓教——悟到金和獅子雙亡
圓教——看到金獅重重無盡

## 現代啟示

1. **比喻的力量**：法藏法師用金獅子講哲學，啟發我們用比喻來理解複雜概念。

2. **本質與現象**：金和獅子的關係，就是本質和現象的關係——看事物不只看表面（獅子），還要看本質（金）。

3. **整體與部分**：六相圓融用獅子做比喻，生動說明整體和部分不對立而圓融。

4. **認識的層次**：五教的金獅子版說明，同樣看一個東西，認識層次不同，看到的東西也不同。
