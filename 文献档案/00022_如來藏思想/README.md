## 概述

如來藏思想（Tathāgatagarbha，"如來的胎藏"）是大乘佛教的重要思想傳統，主張一切衆生都具有如來藏（tathāgatagarbha），即成佛的潛能或佛性（buddha-dhātu）。如來藏思想與唯識學派和中觀學派有複雜的互動關係，是東亞佛教（特別是禪宗、華嚴宗、天台宗）和藏傳佛教的重要思想基礎。

主要文獻包括《如來藏經》（Tathāgatagarbha-sūtra）、《勝鬘經》（Śrīmālādevī-siṃhanāda-sūtra）、《大乘起信論》（Mahāyāna-śraddhotpāda-śāstra）等。

## 核心概念

- **如來藏**（tathāgatagarbha）："如來的胎藏"，一切衆生成佛的潛能
- **佛性**（buddha-dhātu/buddha-gotra）：成佛的種性或本性
- **一心**（eka-citta）：《大乘起信論》的核心概念，真如與生滅的統一
- **真如**（tathatā）：如實的本性，不生不滅
- **生滅門**（utpāda-nirodha-dvāra）：現象生滅的層面
- **真如門**（tathatā-dvāra）：真如不變的層面
- **本覺**（ādi-bodhi）：本來具有的覺性
- **始覺**（ādi-bodhi-utpāda）：修行中覺悟的開始
