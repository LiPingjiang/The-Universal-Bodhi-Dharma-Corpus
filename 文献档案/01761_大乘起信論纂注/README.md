## 學術解讀

### 一、文獻學信息

《大乘起信論纂注》是明代雲棲祩宏（1535-1615）所撰。祩宏，俗姓沈，字佛慧，號蓮池，世稱"雲棲大師"，爲明代四大高僧之一，淨土宗第八代祖師。此疏以"纂注"爲名，取"纂集諸家註釋，會歸一心"之意，是《大乘起信論》註疏中較爲晚出但影響深遠的一部。

祩宏一生融通禪、淨、教、戒，主張"禪淨雙修""儒釋合一"。其《纂注》不僅疏解《起信論》文義，更結合淨土思想，將"一心二門"與"唸佛往生"相融通，形成獨特的詮釋體系。

### 二、歷史背景

明代萬曆年間，佛教內部宗派林立，禪淨之爭、教觀之爭時有發生。祩宏居杭州雲棲寺，廣收徒衆，倡導各宗融合。此疏作於其晚年，旨在爲學人提供一條簡明扼要的《起信論》入門路徑。祩宏在疏中大量引用前代註疏（如法藏《義記》、元曉《疏》、宗密《註疏》等），加以裁擇會通，故名"纂注"。

### 三、思想史定位

祩宏此疏的最大特色是將《起信論》的"一心二門"與淨土宗的"信願行"相結合。他認爲，"信"不僅是《起信論》的開端，也是淨土宗的核心——"信有西方淨土，信有阿彌陀佛，信有往生之事"。祩宏將"心真如門"解釋爲"自性彌陀"，"心生滅門"解釋爲"唯心淨土"，從而打通了教理與修行的關節。這一思想對後世"淨土即禪""禪淨不二"的思潮有重要影響。

## 白話解讀

### 核心內容

《大乘起信論纂注》是蓮池大師祩宏對《大乘起信論》的集註。蓮池大師把歷代高僧對《起信論》的解釋收集起來，選擇最精要的部分，加上自己的理解，編成這部註疏。

蓮池大師的獨特貢獻在於：他把《起信論》的"信"與淨土宗的"信"聯繫起來。《起信論》說"信"是修行的起點，蓮池大師說：信什麼？信自己本有佛性，信阿彌陀佛的救度，信西方淨土的真實。這樣，教理（《起信論》）和修行（淨土法門）就打通了。

### 現代啓示

1. **信仰的力量**：現代人常將"信"理解爲盲目相信，但佛教中的"信"是"正信"——基於理解的相信。蓮池大師告訴我們：信不是放棄思考，而是思考後的選擇。

2. **整合智慧**：蓮池大師融通禪、淨、教、戒，啓示我們在面對複雜問題時，不要侷限於單一視角，而要整合多種智慧資源。

3. **簡化修行**：蓮池大師主張"持名唸佛"，認爲最簡單的修行方式往往最有效。這對現代人忙碌的生活特別有啓示。

### 閱讀建議

- 蓮池大師的註疏較爲平實，適合初學者
- 可配合《雲棲法匯》中的其他著作，如《彌陀疏鈔》《竹窗隨筆》等一同閱讀
- 注意蓮池大師融通各宗的特點，理解其"會歸淨土"的立場
