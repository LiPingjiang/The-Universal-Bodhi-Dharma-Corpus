# 大乘起信論指要

## 學術解讀

### 一、文獻學信息

《大乘起信論指要》是宋代知禮（960-1028）所撰。知禮，俗姓金，字約言，四明（今浙江寧波）人，爲宋代天台宗山家派創始人，世稱"四明尊者"或"法智大師"。此疏以"指要"爲名，取"指示要義，直指人心"之意，是宋代天台宗解釋《大乘起信論》的代表性著作。

知禮一生致力於天台宗的復興和整理，面對當時禪宗的興盛和天台宗內部的"山家山外"之爭，他以嚴謹的義學捍衛天台宗的正統。著有《觀音玄義》《十不二門指要鈔》《觀無量壽佛經疏妙宗鈔》等。此疏作於其晚年，是其以天台宗"山家"義理統攝《起信論》的集大成之作。

### 二、歷史背景

宋代是天台宗復興的重要時期。知禮與慶昭、智圓等人發生"山家山外"之爭，核心問題是"真心"與"妄心"的關係。山外派（如慶昭）主張"真心觀"，認爲觀心應以"真心"爲對象；山家派（如知禮）主張"妄心觀"，認爲觀心應以"妄心"爲對象。知禮此疏正是在這一背景下產生，以《起信論》的"一心二門"爲教理依據，論證"妄心觀"的合理性。

### 三、思想史定位

知禮此疏的核心貢獻在於以"山家"的"妄心觀"來解釋《起信論》。他認爲，《起信論》的"心生滅門"即是"妄心"，"心真如門"即是"真心"，但二者不是兩個獨立的心，而是"一念心"的"真妄不二"。知禮強調：修行不是直接觀"真心"（因爲真心不可觀），而是觀"妄心"——在妄心中見真心，即"妄心即真心"。這一解釋對後來天台宗的觀心法門有決定性影響。

## 白話解讀

### 核心內容

《大乘起信論指要》是宋代天台宗大師知禮對《大乘起信論》的解釋。知禮大師是"山家派"的創始人，他用"山家"的"妄心觀"來理解《起信論》。

知禮大師說：《起信論》的"心生滅門"就是"妄心"——我們平常亂七八糟的念頭；"心真如門"就是"真心"——清淨的本心。但這兩個不是兩個心，而是"一念心"的兩個面。

知禮大師特別強調：修行不要去找"真心"（因爲真心本來就存在，不需要找），而是要觀"妄心"——在妄心中看到真心。就像看雲：雲（妄心）飄來飄去，但天空（真心）始終在那裏。觀雲的時候，自然就看到天空了。

### 現代啓示

1. **不逃避煩惱**：現代人常常想逃避煩惱。知禮大師告訴我們：不要逃避，要在煩惱中修行。煩惱本身就是修行的材料。

2. **當下即真**："妄心即真心"啓示我們：不要等待"清淨"的時刻纔開始修行。當下的每一個念頭，無論善惡，都是修行的入口。

3. **觀心方法**：知禮的"妄心觀"可以轉化爲現代的"正念"練習：不評判地觀察自己的念頭，在觀察中自然發現內心的平靜。

### 閱讀建議

- 知禮的疏解需要天台宗"山家山外"之爭的背景知識
- 可配合《十不二門指要鈔》一同閱讀，理解"山家"義理
- 注意"妄心觀"與"真心觀"的差異，理解知禮的學術立場
