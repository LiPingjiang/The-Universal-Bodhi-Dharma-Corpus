## 時輪續源流骨架

《時輪續》（Kālacakra-tantra，"時輪怛特羅"，字面義"時間之輪的密續"）是藏傳佛教無上瑜伽續（Anuttara-yoga-tantra）中最晚出、最複雜的密續之一，約成立於公元10–11世紀印度東部：

```
        时轮续体系
        ├── 《时轮根本续》（Mūla-Kālacakra-tantra）— 原典，12000颂，已佚
        ├── 《时轮略续》（Laghu-Kālacakra-tantra）— 现存根本续，1047颂
        │       ├── 第一品：外时轮（宇宙论）
        │       ├── 第二品：内时轮（身体论）
        │       ├── 第三品：别时轮（灌顶论）
        │       ├── 第四品：修行（生起次第）
        │       └── 第五品：智慧（圆满次第）
        ├── 《无垢光大疏》（Vimalaprabhā）— 最重要注疏，普贤慧著
        └── 藏传时轮传承
                ├── 觉囊派（Jonang）— 时轮续的主要传承者
                ├── 格鲁派（Gelug）— 宗喀巴弟子传承
                └── 萨迦派（Sakya）— 亦有传承
```

時輪續的獨特之處在於：它將宇宙論（外時輪）、身體論（內時輪）、修行論（別時輪）三者緊密結合，認爲宇宙的結構與人體的結構相互對應，通過修煉身體內部的氣脈明點，可以與宇宙的節律（時輪）同步，最終證得時輪金剛（Kālacakra）的成就。

時輪續還包含了獨特的天文曆法體系（時輪歷）和預言體系（香巴拉王國的預言），是藏傳佛教中最具綜合性的密續傳統。
