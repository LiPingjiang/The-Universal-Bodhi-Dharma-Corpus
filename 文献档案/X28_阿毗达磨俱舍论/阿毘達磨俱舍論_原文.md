# 阿毘達磨俱舍論 · 原文

## 文獻基本信息

- **梵名**：Abhidharma-kośa-śāstra
- **巴利名**：Abhidhamma-kosa（擬）
- **漢譯**：玄奘譯《阿毘達磨俱舍論》
- **年代**：唐永徽年間（公元650-655年）
- **卷數**：三十卷
- **所屬部派**：說一切有部（Sarvāstivāda），但對有部觀點有所批判
- **作者**：世親菩薩（Vasubandhu）
- **藏經編號**：大正藏第1558號（阿毘達磨部）

## 經典概要

《阿毘達磨俱舍論》，簡稱《俱舍論》，是佛教史上最重要、影響最深遠的論書之一。此論由世親菩薩（Vasubandhu）所造，玄奘法師於唐朝永徽年間譯出，共三十卷。

「俱舍」（Kośa）意為「藏」、「庫藏」。「俱舍論」意為「藏論」，即「包含一切阿毘達磨教理的寶藏」。此論雖然以說一切有部的教理為基礎，但對有部的一些觀點進行了批判和修正，因此被稱為「聰明論」或「智慧論」。

此論的特點在於：
1. **系統性**：將龐雜的阿毘達磨教理，進行了系統的整理和組織。
2. **簡潔性**：雖然內容豐富，但表達簡潔，便於學習。
3. **批判性**：對有部的一些觀點進行了批判，提出了更合理的解釋。
4. **影響力**：此論對後來的佛教思想產生了深遠的影響，被譽為「聰明論」。

## 核心內容

### 一、法的分類（蘊處界）

《俱舍論》對法進行了系統的分類，主要採用「蘊處界」三種分類法：

1. **五蘊**：色蘊、受蘊、想蘊、行蘊、識蘊。這是從心理學角度對法的分類。
2. **十二處**：眼處、耳處、鼻處、舌處、身處、意處，色處、聲處、香處、味處、觸處、法處。這是從認識論角度對法的分類。
3. **十八界**：眼界、耳界、鼻界、舌界、身界、意界，色界、聲界、香界、味界、觸界、法界，眼識界、耳識界、鼻識界、舌識界、身識界、意識界。這是從更詳細的認識論角度對法的分類。

這種分類法，是佛教教理中最基本、最常用的分類法。

### 二、四諦與因果

《俱舍論》對四聖諦和因果法則進行了詳細的闡述：

1. **苦諦**：一切有漏法皆是苦。此論對苦的種類和性質進行了詳細的分析。
2. **集諦**：苦的原因是煩惱和業。此論對煩惱和業的運作機制進行了詳細的說明。
3. **滅諦**：苦可以滅除，滅除苦的境界就是涅槃。此論對涅槃的性質進行了詳細的討論。
4. **道諦**：通向滅苦的道路就是八正道。此論對八正道的每個方面進行了詳細的說明。

### 三、煩惱與對治

《俱舍論》對煩惱和對治方法進行了系統的闡述：

1. **煩惱的分類**：根本煩惱（貪、瞋、癡、慢、疑、見）和隨煩惱（忿、恨、覆、惱、嫉、慳等）。
2. **煩惱的生起機制**：煩惱是如何生起的，包括各種不同的因緣條件。
3. **對治方法**：通過持戒、修定、開慧來對治煩惱。
4. **斷煩惱的次第**：從初學到斷盡煩惱的完整次第。

### 四、業與果報

《俱舍論》對業與果報的問題進行了系統的討論：

1. **業的種類**：善業、惡業、無記業。
2. **業的運作機制**：業是如何造作、如何保存、如何產生果報的。
3. **果報的種類**：現報、生報、後報。
4. **業與因果**：業與因果法則的關係。

### 五、修行與證果

《俱舍論》對修行和證果的問題進行了系統的說明：

1. **修行的次第**：從初學到證果的完整次第。
2. **各種禪定**：四禪八定的詳細說明。
3. **各種果位**：預流果、一來果、不還果、阿羅漢果的詳細說明。
4. **證果的條件**：證得各種果位所需要的條件。

## 結構說明

《俱舍論》三十卷的結構，分為八品（章）：

### 分別界品（第一卷至第三卷）
- 討論蘊處界的分類
- 說明法的各種分類方法

### 分別根品（第四卷至第六卷）
- 討論根（感官和能力）的問題
- 說明各種根的分類和作用

### 分別世間品（第七卷至第十二卷）
- 討論世界的構成和運作
- 說明三界（欲界、色界、無色界）的結構

### 分別業品（第十三卷至第十六卷）
- 討論業的問題
- 說明業的種類、運作機制和果報

### 分別隨眠品（第十七卷至第二十卷）
- 討論煩惱（隨眠）的問題
- 說明各種煩惱的性質和對治方法

### 分別聖道品（第二十一卷至第二十四卷）
- 討論聖道的問題
- 說明修行的次第和證果的方法

### 分別智品（第二十五卷至第二十七卷）
- 討論智慧的問題
- 說明各種智慧的性質和作用

### 分別定品（第二十八卷至第三十卷）
- 討論禪定的問題
- 說明各種禪定的性質和修法

這種結構體現了《俱舍論》系統整理阿毘達磨教理的特點。

## 思想價值

《阿毘達磨俱舍論》的思想價值主要體現在以下幾個方面：

1. **系統整理**：將龐雜的阿毘達磨教理，進行了系統的整理和組織，便於學習和理解。

2. **批判發展**：對說一切有部的一些觀點進行了批判，提出了更合理的解釋，推動了教理的發展。

3. **影響深遠**：此論對後來的佛教思想產生了深遠的影響，被譽為「聰明論」。

4. **跨部派價值**：此論雖然以有部為基礎，但超越了部派的局限，對各部派都有參考價值。
