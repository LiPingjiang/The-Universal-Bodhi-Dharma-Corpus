## 关键概念

- **三士道**：下士道（求人天善趣）、中士道（求个人解脱）、上士道（求一切众生解脱）
- **奢摩他**（śamatha）：止，心的专注与平静
- **毗婆舍那**（vipaśyanā）：观，对空性的直接洞察
- **中观应成派**（Prāsaṅgika-Mādhyamaka）：宗喀巴所依据的最高哲学立场
- **自续派**（Svātantrika）：中观的另一分支，宗喀巴认为低于应成派
