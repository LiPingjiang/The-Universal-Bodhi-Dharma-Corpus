## 华严经源流骨架

《华严经》（Avataṃsaka-sūtra，"大方广佛华严经"，"花环经"）是大乘佛教最宏大的经典，也是华严宗的根本经典。"华严"（Avataṃsaka，"花环"）象征佛陀的功德庄严，就像用无数花朵编成的花环，每一朵花都映照着其他所有花朵。

```
        华严经思想体系
        ├── 法界缘起（Dharmadhātu-pratītyasamutpāda）
        │       ├── 一即一切，一切即一 → 每一事物包含一切事物
        │       ├── 重重无尽 → 无限的相互映照
        │       └── 因陀罗网 → 宇宙如宝珠网，互相映照
        ├── 十玄门（Daśa-gūḍha-dvāra）
        │       ├── 同时具足相应门 → 一切同时存在
        │       ├── 广狭自在无碍门 → 大小相互包含
        │       └── 诸法相即自在门 → 一切法相互即入
        └── 普贤行愿（Samantabhadra-praṇidhāna）
                ├── 十大愿王 → 礼敬、称赞、供养、忏悔、随喜、请转法轮、请佛住世、常随佛学、恒顺众生、普皆回向
                ├── 善财童子五十三参 → 向五十三位善知识学习
                └── 入法界品 → 菩萨道的完整修行历程
```

《华严经》的核心思想是「法界缘起」（dharmadhātu-pratītyasamutpāda）：整个宇宙（法界）是一个相互依存、相互映照的整体，每一事物都包含一切事物，一切事物都包含每一事物。这就是"一即一切，一切即一"的华严境界。

《华严经》对中国佛教的影响极为深远。华严宗以《华严经》为根本经典，建立了"法界缘起"的哲学体系。法藏（643-712）的《华严经探玄记》《华严五教章》是华严宗的核心著作。《华严经》的"普贤行愿品"是中国佛教日常课诵的重要内容。
