## 華嚴經源流骨架

《華嚴經》（Avataṃsaka-sūtra，"大方廣佛華嚴經"，"花環經"）是大乘佛教最宏大的經典，也是華嚴宗的根本經典。"華嚴"（Avataṃsaka，"花環"）象徵佛陀的功德莊嚴，就像用無數花朵編成的花環，每一朵花都映照着其他所有花朵。

```
        华严经思想体系
        ├── 法界缘起（Dharmadhātu-pratītyasamutpāda）
        │       ├── 一即一切，一切即一 → 每一事物包含一切事物
        │       ├── 重重无尽 → 无限的相互映照
        │       └── 因陀罗网 → 宇宙如宝珠网，互相映照
        ├── 十玄门（Daśa-gūḍha-dvāra）
        │       ├── 同时具足相应门 → 一切同时存在
        │       ├── 广狭自在无碍门 → 大小相互包含
        │       └── 诸法相即自在门 → 一切法相互即入
        └── 普贤行愿（Samantabhadra-praṇidhāna）
                ├── 十大愿王 → 礼敬、称赞、供养、忏悔、随喜、请转法轮、请佛住世、常随佛学、恒顺众生、普皆回向
                ├── 善财童子五十三参 → 向五十三位善知识学习
                └── 入法界品 → 菩萨道的完整修行历程
```

《華嚴經》的核心思想是「法界緣起」（dharmadhātu-pratītyasamutpāda）：整個宇宙（法界）是一個相互依存、相互映照的整體，每一事物都包含一切事物，一切事物都包含每一事物。這就是"一即一切，一切即一"的華嚴境界。

《華嚴經》對中國佛教的影響極爲深遠。華嚴宗以《華嚴經》爲根本經典，建立了"法界緣起"的哲學體系。法藏（643-712）的《華嚴經探玄記》《華嚴五教章》是華嚴宗的核心著作。《華嚴經》的"普賢行願品"是中國佛教日常課誦的重要內容。
