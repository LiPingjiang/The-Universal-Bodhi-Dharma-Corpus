## 关键概念

- **东方学**（Orientalism）：19世纪西方对东方文化的学术研究
- **巴利文学会**（Pali Text Society）：1881年成立，系统整理巴利文佛典
- **人间佛教**：太虚大师提出，强调佛教在现实社会中的实践
- **批判佛教**（Critical Buddhism）：日本学者提出，批评如来藏思想
- **心智科学**（Mind Science）：佛教禅修与认知科学的交叉研究
