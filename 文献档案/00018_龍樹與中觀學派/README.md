# 00018 龍樹與中觀學派（Nāgārjuna-Mādhyamaka）

## 概述

龍樹（Nāgārjuna，約2世紀）是大乘佛教最重要的哲學家，中觀學派（Mādhyamaka，"中道學派"）的創始人。他的核心思想是空性（śūnyatā）：一切現象都沒有固有的自性（svabhāva），都是依緣起（pratītyasamutpāda）而存在的。中觀學派對後來的大乘佛教哲學影響極爲深遠，是藏傳佛教和東亞佛教的重要思想基礎。

## 核心概念

- **龍樹**（Nāgārjuna）：中觀學派創始人，約2世紀，南印度人
- **中觀**（Mādhyamaka）：中道之學，不落兩邊（有/無、常/斷）
- **空性**（śūnyatā）：一切現象無固有自性
- **自性**（svabhāva）：固有的、獨立的存在性
- **緣起**（pratītyasamutpāda）：一切現象依條件而生起
- **二諦**（dvi-satya）：世俗諦（saṃvṛti-satya）與勝義諦（paramārtha-satya）
- **戲論**（prapañca）：概念的增殖與執著
- **涅槃**（nirvāṇa）：戲論寂滅，不生不滅
