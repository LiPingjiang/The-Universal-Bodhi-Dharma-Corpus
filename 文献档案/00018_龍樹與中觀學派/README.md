## 概述

龙树（Nāgārjuna，约2世纪）是大乘佛教最重要的哲学家，中观学派（Mādhyamaka，"中道学派"）的创始人。他的核心思想是空性（śūnyatā）：一切现象都没有固有的自性（svabhāva），都是依缘起（pratītyasamutpāda）而存在的。中观学派对后来的大乘佛教哲学影响极为深远，是藏传佛教和东亚佛教的重要思想基础。

## 核心概念

- **龙树**（Nāgārjuna）：中观学派创始人，约2世纪，南印度人
- **中观**（Mādhyamaka）：中道之学，不落两边（有/无、常/断）
- **空性**（śūnyatā）：一切现象无固有自性
- **自性**（svabhāva）：固有的、独立的存在性
- **缘起**（pratītyasamutpāda）：一切现象依条件而生起
- **二谛**（dvi-satya）：世俗谛（saṃvṛti-satya）与胜义谛（paramārtha-satya）
- **戏论**（prapañca）：概念的增殖与执著
- **涅槃**（nirvāṇa）：戏论寂灭，不生不灭
