## 阿毗达磨兴起的四阶段

阿毗达磨（Abhidhamma / Abhidharma，意为「对法」「殊胜法」）是佛教论藏的专称，代表佛教从经教传统向系统哲学的飞跃。其兴起过程可分为四个阶段：

```
        阿毗达磨论藏兴起（Abhidhamma/Abhidharma Rise）
        ├── 第一阶段：经中萌芽（公元前5–前3世纪）
        │       └── 经典中的本母（mātikā）、法数、条件关系
        │           如《经集》中的法相列举，《增支部》中的数目分类
        ├── 第二阶段：独立活动（公元前3–前2世纪）
        │       └── 论母（abhidhamma-kathā）从经中独立
        │           论师以「论议」方式解释经义，形成口头传统
        ├── 第三阶段：完整成书（公元前2–前1世纪）
        │       ├── 巴利七论（南传上座部）
        │       │       法集论、分别论、界论、人施设论、论事、双论、发趣论
        │       └── 六足一身论（说一切有部）
        │               集异门足论、法蕴足论、施设足论、识身足论、
        │               界身足论、品类足论、发智论
        └── 第四阶段：神圣化与注释化（公元1–5世纪）
                ├── 阿毗达磨被尊为「佛口所说」
                ├── 觉音《法集论注》《分别论注》等注释
                └── 世友《品类足论》、迦旃延尼子《发智论》等北传系统完成
```

### 第一阶段：经中萌芽

在原始佛教经典（如四阿含、四尼柯耶）中，已出现阿毗达磨的萌芽形态：

- **本母（mātikā）**：纲目表、分类框架，如《增支部》按数目（一法、二法……十一法）组织经文
- **法数（dharma-saṅkhyā）**：法的列举与计数，如五蕴、十二处、十八界
- **条件关系（paccaya / pratyaya）**：因缘条件的分析，如十二因缘、四缘说

这些元素最初是经文中的「记忆口诀」和「教学框架」，后来逐渐独立发展为系统的论书。

### 第二阶段：独立活动

公元前3世纪前后，佛教论师开始以「论议」（kathā）的方式系统解释经义，形成独立的学术传统：

- **论母（abhidhamma-kathā）**：专门讨论法的性质、分类、关系的口头论议
- **论师（vādin）**：专门从事论议的学者，如大迦旃延（Mahākaccāna / Mahākātyāyana）
- **对法问（abhidhamma-pañha）**：以问答形式讨论法相的文献

这一阶段的阿毗达磨尚未形成固定文本，而是以口头传承和论议实践的方式流传。

### 第三阶段：完整成书

公元前2世纪至前1世纪，阿毗达磨论书开始系统化、文本化：

**南传系统（巴利语）**：形成「阿毗达磨七论」（Abhidhamma-piṭaka），成为南传上座部三藏之一。

**北传系统（梵语）**：说一切有部形成「六足一身论」体系，为后来《大毗婆沙论》的编纂奠定基础。

### 第四阶段：神圣化与注释化

公元1世纪以后，阿毗达磨被赋予神圣地位，同时进入高度注释化的阶段：

- **神圣化**：南传认为阿毗达磨是佛陀在忉利天为母说法的内容；北传则认为阿毗达磨是佛弟子对佛语的系统整理
- **注释化**：觉音（Buddhaghosa）在南传、世友（Vasumitra）等在北传，对阿毗达磨论书进行系统注释
- **哲学化**：阿毗达磨从「经义解释」发展为独立的哲学体系，与经藏、律藏并列为三藏之一

## 关键概念速查

| 概念 | 梵/巴利语 | 简释 |
|------|-----------|------|
| 阿毗达磨 | Abhidhamma / Abhidharma | 「对法」「殊胜法」，佛教论藏专称，系统分析法的性质与关系 |
| 本母 | mātikā | 纲目表、分类框架，阿毗达磨的基础组织形式 |
| 胜义法 | paramattha-dhamma | 终极真实的法，与世俗法（sammuti-dhamma）相对 |
| 三世实有 | tri-kāla-sat | 过去、现在、未来三世皆实有，说一切有部核心主张 |
| 法体恒有 | dharma-svabhāva-sat | 法的自体（本质）恒常存在，不因时间而改变 |
