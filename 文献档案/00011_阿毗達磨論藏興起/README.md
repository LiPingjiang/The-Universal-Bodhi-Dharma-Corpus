## 阿毗達磨興起的四階段

阿毗達磨（Abhidhamma / Abhidharma，意爲「對法」「殊勝法」）是佛教論藏的專稱，代表佛教從經教傳統向系統哲學的飛躍。其興起過程可分爲四個階段：

```
        阿毗达磨论藏兴起（Abhidhamma/Abhidharma Rise）
        ├── 第一阶段：经中萌芽（公元前5–前3世纪）
        │       └── 经典中的本母（mātikā）、法数、条件关系
        │           如《经集》中的法相列举，《增支部》中的数目分类
        ├── 第二阶段：独立活动（公元前3–前2世纪）
        │       └── 论母（abhidhamma-kathā）从经中独立
        │           论师以「论议」方式解释经义，形成口头传统
        ├── 第三阶段：完整成书（公元前2–前1世纪）
        │       ├── 巴利七论（南传上座部）
        │       │       法集论、分别论、界论、人施设论、论事、双论、发趣论
        │       └── 六足一身论（说一切有部）
        │               集异门足论、法蕴足论、施设足论、识身足论、
        │               界身足论、品类足论、发智论
        └── 第四阶段：神圣化与注释化（公元1–5世纪）
                ├── 阿毗达磨被尊为「佛口所说」
                ├── 觉音《法集论注》《分别论注》等注释
                └── 世友《品类足论》、迦旃延尼子《发智论》等北传系统完成
```

### 第一階段：經中萌芽

在原始佛教經典（如四阿含、四尼柯耶）中，已出現阿毗達磨的萌芽形態：

- **本母（mātikā）**：綱目表、分類框架，如《增支部》按數目（一法、二法……十一法）組織經文
- **法數（dharma-saṅkhyā）**：法的列舉與計數，如五蘊、十二處、十八界
- **條件關係（paccaya / pratyaya）**：因緣條件的分析，如十二因緣、四緣說

這些元素最初是經文中的「記憶口訣」和「教學框架」，後來逐漸獨立發展爲系統的論書。

### 第二階段：獨立活動

公元前3世紀前後，佛教論師開始以「論議」（kathā）的方式系統解釋經義，形成獨立的學術傳統：

- **論母（abhidhamma-kathā）**：專門討論法的性質、分類、關係的口頭論議
- **論師（vādin）**：專門從事論議的學者，如大迦旃延（Mahākaccāna / Mahākātyāyana）
- **對法問（abhidhamma-pañha）**：以問答形式討論法相的文獻

這一階段的阿毗達磨尚未形成固定文本，而是以口頭傳承和論議實踐的方式流傳。

### 第三階段：完整成書

公元前2世紀至前1世紀，阿毗達磨論書開始系統化、文本化：

**南傳系統（巴利語）**：形成「阿毗達磨七論」（Abhidhamma-piṭaka），成爲南傳上座部三藏之一。

**北傳系統（梵語）**：說一切有部形成「六足一身論」體系，爲後來《大毗婆沙論》的編纂奠定基礎。

### 第四階段：神聖化與註釋化

公元1世紀以後，阿毗達磨被賦予神聖地位，同時進入高度註釋化的階段：

- **神聖化**：南傳認爲阿毗達磨是佛陀在忉利天爲母說法的內容；北傳則認爲阿毗達磨是佛弟子對佛語的系統整理
- **註釋化**：覺音（Buddhaghosa）在南傳、世友（Vasumitra）等在北傳，對阿毗達磨論書進行系統註釋
- **哲學化**：阿毗達磨從「經義解釋」發展爲獨立的哲學體系，與經藏、律藏並列爲三藏之一

## 體例說明

- **原文**：巴利語/梵語拉丁轉寫落盤，附甲乙雙軌漢譯對照（甲譯白話意譯 + 乙譯學術直譯），術語腳註鏈至 GLOSSARY
- **學術解讀**：文獻來源、版本考訂、南北傳異說、思想史定位、現代學術參考
- **白話解讀**：全程現代中文熟詞，術語沉入腳註，面向無佛學背景讀者

|------|-----------|------|
