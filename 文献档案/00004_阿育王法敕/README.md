## 關於本目錄

本目錄收錄**阿育王法敕**（Edicts of Aśoka）的全部原始文獻與解讀資料。阿育王法敕是佛教史上現存最早的實物文獻，是孔雀王朝第三代君主阿育王（Aśoka，約前269–前231年在位）在帝國各地巖壁和石柱上刻寫的詔令，宣揚不殺生、宗教寬容、社會公益等"法"（Dharma）的理念。這些石刻距今約兩千三百年，是研究早期佛教和孔雀王朝最珍貴的第一手實物證據。

本目錄中每部文獻均提供**三件套**：
- **原文** — 婆羅米文拉丁轉寫、希臘文、阿拉姆文等原始文本，含甲譯（白話意譯）與乙譯（嚴謹對照）
- **學術解讀** — 文獻學、歷史學、考古學角度的專業分析
- **白話解讀** — 面向普通讀者的通俗講解，術語均作白話化翻譯並附腳註

## 歷史背景

阿育王（Aśoka，約公元前304—前232年），孔雀王朝第三代君主，是印度歷史上最偉大的帝王之一，也是佛教史上最重要的護法者。約公元前261年，阿育王親征羯陵伽（Kaliṅga），戰爭造成十餘萬人死亡。目睹戰爭慘烈，阿育王深感悔恨，隨後皈依佛教，以"法"（Dharma）治國，開始在帝國各地刻寫詔令。

## 關鍵概念

- **天愛喜見王**（Devānāṃpriya Priyadarśin）：阿育王法敕中的自稱，意爲"諸天所愛、慈祥和善的王"
- **法**（Dhamma/Dharma）：阿育王詔書中的"法"不完全等於佛教教義，而是更寬泛的"正道、善行、道德準則"——孝順父母、不殺生、誠實、對一切宗教寬容等
- **婆羅米文**（Brāhmī）：帝國東部和中部使用的文字，自左向右書寫，是現代印度諸文字的共同祖先
- **佉盧文**（Kharoṣṭhī）：帝國西北部使用的文字，自右向左書寫，源自阿拉姆文

## 歷史意義

1. **最早的實物文獻**：佛教史上現存最早的實物文獻，早於任何現存佛教寫本
2. **歷史年代錨點**：法敕中提及的希臘化諸王爲阿育王在位年代提供了精確的歷史錨點
3. **第三次結集的證據**：結集詔令是第三次結集的重要歷史證據
4. **世界遺產**：多處阿育王石柱和法敕遺址已被列入聯合國教科文組織世界遺產名錄
