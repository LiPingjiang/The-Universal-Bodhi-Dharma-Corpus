## 关于本目录

本目录收录与**佛教第二次结集**（巴利语：Dutiya Saṅgīti，梵语：Dutiya Saṅgīti）直接相关的原始文献与解读资料。第二次结集是佛陀入灭约一百年后，由长老耶舍（Yasa Kākaṇḍakaputta）发起、在毗舍离（Vesālī）举行的经典结集，是佛教史上第一次因戒律争议而召开的大规模僧团会议，也是导致上座部与大众部根本分裂的直接导火索。此次结集裁定"十事"全部非法，确立了戒律的严格标准，同时也开启了佛教部派分裂的历史进程。

本目录中每部文献均提供**三件套**：
- **原文** — 巴利语或汉文原始文本节录，含甲译（白话意译）与乙译（严谨对照）
- **学术解读** — 文献学、历史学、思想史角度的专业分析
- **白话解读** — 面向普通读者的通俗讲解，佛教术语均作白话化翻译并附脚注

### 1. 律藏·毗舍离犍度（Cullavagga XII）

> 巴利律藏《小品》第十二章，记载第二次结集最权威的原始资料。全犍度叙述结构清晰：起因（耶舍目睹十事）→ 奔走召集 → 七百人结集 → 逐条裁定十事 → 结集圆满。是现存关于这一历史事件最详尽、最权威的第一手文献。

### 2. 十事非法

> "十事"（dasa vatthūni）是毗舍离跋耆族比丘奉行的十项被保守派长老认为违律的做法，是导致第二次结集召开、进而引发上座部与大众部分裂的直接导火索。七百长老逐一裁定十事均属"非法"（adhamma）。

## 关键人物

- **耶舍**（Yasa Kākaṇḍakaputta）：发起结集的长老，来自拘舍弥（Kosambī），是第二次结集的核心推动者
- **离婆多**（Revata）：德高望重的长老，被推举主持裁定十事，其权威性是结集合法性的关键
- **萨婆迦弥**（Sabbakāmī）：据传是当时在世最年长的比丘，曾亲从阿难受戒，其证词具有历史连接意义
- **跋耆族比丘**（Vajjiputtaka bhikkhū）：毗舍离的比丘群体，十事的奉行者，结集后另立大众部

## 关键概念

- **十事**（dasa vatthūni）：毗舍离比丘奉行的十项争议做法，详见[十事非法_原文.md](十事非法_原文.md)
- **七百人结集**（Sattasatikā）：正统派召集七百比丘裁定十事的结集，即第二次结集
- **大结集**（Mahāsaṅgīti）：毗耆离比丘不服裁决后另行召集的万人大会，大众部由此得名
- **根本分裂**（mūlabheda）：上座部与大众部的分裂，被视为佛教部派史的起点
