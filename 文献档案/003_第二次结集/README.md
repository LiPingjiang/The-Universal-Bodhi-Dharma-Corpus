# 003 第二次结集（毗舍离结集）文献档案

> 节点ID：003 · 历史事件：第二次结集（Dutiyasaṅgīti）
> 时间：约佛灭后百年（约公元前 383 年或前 350 年，学界有争议）
> 地点：毗舍离（Vesālī，今印度比哈尔邦）
> 别称：七百人结集（Sattasatikā）

---

## 事件概述

第二次结集是佛教史上第一次因戒律争议而召开的大规模僧团会议，也是导致佛教部派分裂的直接导火索。事件起因于毗舍离（Vesālī）的跋耆族比丘（Vajjiputtaka bhikkhū）公开奉行十项被保守派长老认为违律的做法，史称"十事"（dasa vatthūni）。

长老耶舍（Yasa Kākaṇḍakaputta）目睹毗舍离比丘向在家众索取金银，认为此举严重违律，遂奔走各方，召集七百位长老比丘在毗舍离举行结集，逐一裁定十事均属"非法"（adhamma）。

然而，毗舍离比丘不服裁决，另行召集万人大会，史称"大结集"（Mahāsaṅgīti）。这一分裂被后世视为上座部（Sthavira/Theravāda）与大众部（Mahāsāṅghika）分裂的起点，尽管学界对分裂的确切时间与机制仍有争议。

## 关键人物

- **耶舍**（Yasa Kākaṇḍakaputta）：发起结集的长老，来自拘舍弥（Kosambī），是第二次结集的核心推动者
- **离婆多**（Revata）：德高望重的长老，被推举主持裁定十事，其权威性是结集合法性的关键
- **萨婆迦弥**（Sabbakāmī）：据传是当时在世最年长的比丘，曾亲从阿难受戒，其证词具有历史连接意义
- **跋耆族比丘**（Vajjiputtaka bhikkhū）：毗舍离的比丘群体，十事的奉行者，结集后另立大众部

## 关键概念

- **十事**（dasa vatthūni）：毗舍离比丘奉行的十项争议做法，详见[十事非法_原文.md](十事非法_原文.md)
- **七百人结集**（Sattasatikā）：正统派召集七百比丘裁定十事的结集，即第二次结集
- **大结集**（Mahāsaṅgīti）：毗舍离比丘不服裁决后另行召集的万人大会，大众部由此得名
- **根本分裂**（mūlabheda）：上座部与大众部的分裂，被视为佛教部派史的起点
