# 因明大疏 · 學術解讀

## 一、文獻定位

《因明入正理论疏》，通称《因明大疏》，为唐代唯识学大师窥基（632–682）所撰，是玄奘译《因明入正理论》最权威、最详尽的注释书。全书凡三卷，系统阐释陈那《因明入正理论》的因明逻辑体系，包括现量、比量、宗、因、喻等核心概念，以及因明论式的构造规则、过失辨析等。该书是研究唐代因明学、中国逻辑思想史及佛教认识论的关键文本，与窥基的《大乘法苑义林章·因明义林》共同构成唐代因明学的最高成就。

## 三、思想史分析

### 3.1 歷史背景

因明（hetu-vidyā）是印度佛教的逻辑学和认识论，由陈那（Dignāga，约480–540）集大成，其代表作《因明入正理论》系统阐述了因明的基本原理。玄奘在印度游学期间，深入学习了陈那因明，回国后于贞观二十二年（648）翻译了《因明入正理论》。

然而，因明作为印度特有的逻辑体系，其概念和规则对中国学者来说极为陌生。玄奘在译场中亲自讲解因明，窥基作为玄奘的嫡传弟子，亲闻其师讲解，并将这些内容记录下来，写成《因明大疏》。该书不仅解释了《因明入正理论》的文本，还补充了大量玄奘口传的内容，是研究玄奘因明学的第一手资料。

### 3.2 核心論題

《因明大疏》作为《因明入正理论》的注释书，系统阐述了因明学的全部核心内容：

1. **现量与比量**：详细解释现量（直接经验）和比量（逻辑推理）的性质、种类和条件。现量分为五根现量、意识现量、自证现量、瑜伽现量四种；比量分为为自比量（自己推理）和为他比量（向他人证明）两种。
2. **宗、因、喻三支**：系统阐述因明论式的三个组成部分：宗（命题）、因（理由）、喻（例证）。窥基详细解释了每一支的构造规则、有效条件和过失类型。
3. **因三相**：详细分析因的三种属性：遍是宗法性（因必须遍及宗的主词）、同品定有性（因必须在同品中存在）、异品遍无性（因必须在异品中不存在）。这是因明逻辑的核心规则。
4. **过失论**：系统列举和解释因明论式中可能出现的各种过失，包括宗过失（如自语相违、自教相违等）、因过失（如四不成、六不定、四相违等）、喻过失（如所立法不成、能立法不成等）。
5. **因明与外道逻辑的辨析**：窥基在注释中经常对比因明与印度外道（如正理派、胜论派）的逻辑体系，阐明因明的独特之处。

### 3.3 學術評價

- **吕澂**评价《因明大疏》为"中国因明学的奠基之作"，认为没有此书，因明学在中国无法传播和发展。
- **日本学者**如宇井伯寿、武邑尚邦等，将《因明大疏》视为研究陈那因明中国化的核心文献，尤其重视窥基对因三相的解释。
- **现代逻辑学者**关注因明与西方形式逻辑的异同，认为因明在论证理论方面有独到之处，特别是其"过失论"对论证谬误的系统分析。
- 当代研究还关注该书的文献学价值，因为窥基在注释中引用了大量现已失传的印度逻辑学著作，保存了珍贵的思想史资料。

## 四、術語對照

| 漢譯術語 | 梵文/巴利原文 | 現代語義 |
|---------|-------------|---------|
| 因明 | hetu-vidyā | 印度佛教逻辑学与认识论，包括现量、比量、因三相、过失论等 |
| 現量 | pratyakṣa | 直接经验，不通过概念和推理的直接认知 |
| 比量 | anumāna | 逻辑推理，通过已知推未知 |
| 宗 | pratijñā | 因明论式的命题，即需要证明的结论 |
| 因 | hetu | 因明论式的理由，即支持命题的根据 |
| 喻 | dṛṣṭānta | 因明论式的例证，分为同喻（正面例证）和异喻（反面例证） |
| 因三相 | tri-rūpa-liṅga | 因的三种属性：遍是宗法性、同品定有性、异品遍无性 |
| 三支作法 | 无对应梵文 | 因明论式的三个组成部分：宗、因、喻 |
