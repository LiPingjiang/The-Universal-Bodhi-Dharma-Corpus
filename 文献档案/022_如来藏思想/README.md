## 概述

如来藏思想（Tathāgatagarbha，"如来的胎藏"）是大乘佛教的重要思想传统，主张一切众生都具有如来藏（tathāgatagarbha），即成佛的潜能或佛性（buddha-dhātu）。如来藏思想与唯识学派和中观学派有复杂的互动关系，是东亚佛教（特别是禅宗、华严宗、天台宗）和藏传佛教的重要思想基础。

主要文献包括《如来藏经》（Tathāgatagarbha-sūtra）、《胜鬘经》（Śrīmālādevī-siṃhanāda-sūtra）、《大乘起信论》（Mahāyāna-śraddhotpāda-śāstra）等。

## 核心概念

- **如来藏**（tathāgatagarbha）："如来的胎藏"，一切众生成佛的潜能
- **佛性**（buddha-dhātu/buddha-gotra）：成佛的种性或本性
- **一心**（eka-citta）：《大乘起信论》的核心概念，真如与生灭的统一
- **真如**（tathatā）：如实的本性，不生不灭
- **生灭门**（utpāda-nirodha-dvāra）：现象生灭的层面
- **真如门**（tathatā-dvāra）：真如不变的层面
- **本觉**（ādi-bodhi）：本来具有的觉性
- **始觉**（ādi-bodhi-utpāda）：修行中觉悟的开始
