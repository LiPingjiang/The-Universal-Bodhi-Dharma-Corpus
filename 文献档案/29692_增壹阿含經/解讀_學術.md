# 增壹阿含經 學術解讀

## 一、文獻學基本信息

**主要譯者**：曇摩難提（Dharmanandi），出生地不詳，前秦時代來華，以口誦梵文底本著稱；筆受由竺佛念擔任。譯出後，僧伽提婆（Saṃghadeva）認為此譯有誤，於東晉太元年間（約385–398 CE之間）在廬山重新校訂，形成今本。  
**譯時**：初譯於前秦建元二十年（384 CE）；重訂本於東晉太元年間完成。  
**底本語言**：梵文，底本部派歸屬有爭議（見下文）。  
**大正藏**：T02, No. 125，第二冊，頁549–834。  
**卷帙**：五十一卷，五十二品（含序品），四七三經以上。  

### 版本源流特殊性

《增壹阿含》在四部阿含中版本情況最為複雜。此經只有一個漢譯本，沒有梵文原典留存，巴利《增支部》（Aṅguttara Nikāya）是其最接近的對應文本，但兩者之間差異相當大，遠大於《長阿含》對《長部》、《中阿含》對《中部》的差異程度。

在犍陀羅出土的梵文殘片中，有部分被鑒定為《增壹阿含》類文本，但數量極少，無法對底本全貌作出重建。

## 二、在佛教史上的地位與影響

《增壹阿含》在漢傳佛教中扮演了幾個獨特的角色：

**1. 大乘色彩**：四部阿含中，《增壹阿含》是唯一在內容上明顯包含大乘佛教思想傾向的文本。其中提到「大乘」（mahāyāna）一詞，有菩薩思想的萌芽，部分學者（如印順法師）認為這反映了翻譯時代（4世紀）的大乘佛教影響已滲入到底本的最終編定中，或是在漢譯過程中有所增補。

**2. 比丘尼教團史料**：《大愛道品》保存了比丘尼教團建立的完整敘述，包括摩訶波闍波提（Mahāpajāpatī Gotamī）請求出家、佛陀的初次拒絕、阿難的仲介和「八敬法」的制定，是研究早期佛教女性主義議題不可缺少的原始文獻。

**3. 數字分類法的教育功能**：按法數一、二、三……排列的方式，是古代印度口誦傳統的記憶技術——數字框架便於背誦和傳授。這種編排方式使佛法的核心要點得到系統化整理，對後來阿毗達磨（論藏）的分類傳統有直接的奠基作用。

**4. 社會史資料**：《增壹阿含》保存了大量關於早期佛教僧團社會生活的細節：弟子的性格和才能、在家信衆（優婆塞、優婆夷）的名稱和故事、與婆羅門教的對話，是研究公元前5世紀至公元4世紀佛教社會的重要史料。

## 三、與南傳巴利尼柯耶的對應關係

《增壹阿含經》與巴利《增支部》（Aṅguttara Nikāya，AN）對應，但兩者的對應率是四部阿含中最低的：

- 《增支部》收錄約九千五百五十條（按各種計算方式）；《增壹阿含》按嚴格計算約有四七三到五零零篇。
- 很多在巴利《增支部》中有完整對應的主題，在漢譯《增壹阿含》中或缺失，或形式不同。
- 《增壹阿含》中有若干長篇敘事（如卷四十九至五十一的大型故事），在巴利《增支部》中完全沒有對應。

部分重要對應如下：

| 漢譯《增壹阿含》 | 巴利《增支部》 |
|--------------|-------------|
| 弟子品（各科第一弟子） | AN 1.188–267 |
| 壹入道品（四念處） | AN 1.21 等 |
| 四諦相關諸品 | AN 4.1–220 等 |
| 五根、五力相關品 | AN 5.1–29 等 |
| 七覺支相關品 | AN 7.67 等 |
| 八正道品 | AN 8.81 等 |

赤沼智善的《漢巴四部四尼柯耶對照録》（1929）提供了最早的系統對比表。近年無著比丘（Bhikkhu Analayo）也發表了多篇專題論文，比較兩傳的具體差異。

## 四、主要教義內容概述

### 1. 三十七菩提分法的系統化
《增壹阿含》是四部阿含中對三十七菩提分法（四念處、四正勤、四神足、五根、五力、七覺支、八正道）最系統的集中呈現之地。數字分類體例天然地讓這些修行框架各自獨立成品，便於修學者按次第研習。

### 2. 在家修行與居士佛教
《增壹阿含》對在家信衆（清信士、清信女）的記載遠比其他阿含豐富。其中保存了大量優婆塞（upāsaka，男性在家信徒）和優婆夷（upāsikā，女性在家信徒）的名字、特長和故事，以及他們在社區中承擔的角色，顯示早期佛教並非只是出家人的修行，在家眾同樣有完整的修行體系和宗教身份。

### 3. 大乘萌芽與菩薩思想
序品和若干品次中提到「大乘」、菩薩行等概念。印順法師在《原始佛教聖典之集成》中詳細分析，認為這些大乘色彩是在文本的最後編定階段（約公元2–3世紀）被添加進去的，反映了大衆部（Mahāsāṃghika）傳誦傳統對大乘思想的早期接受。

### 4. 業報論的豐富化
《增壹阿含》中關於業報的敘述，比其他三部阿含更加豐富和具體，包含大量業報故事（說明某種行為導致某種果報的敘事）。這些故事是後來各種業報文學（如《雜寶藏經》）的重要先驅。

## 五、學術研究現狀與爭議

### 底本部派歸屬：最大的爭議

《增壹阿含》的底本究竟屬於哪個部派，是漢譯阿含研究中爭議最大的問題：

- **大衆部說**：印順法師在《原始佛教聖典之集成》中有力論證底本屬大衆部（Mahāsāṃghika），主要依據是文本中的大乘色彩、菩薩思想，以及某些與銅鍱部律藏不同的制度描述。這一說法目前被學界廣泛接受。
- **法藏部說**：部分日本學者（如平川彰）曾提出底本可能屬法藏部。
- **混合說**：也有學者認為現存《增壹阿含》是多個部派文本的混合體，在漢譯過程中有相當程度的增補和改動。

### 大乘內容的真實性問題

序品中明確出現「大乘」一詞，在四部阿含中屬獨特現象。學界爭議的是：這是底本原有的，還是翻譯過程中（曇摩難提背誦時或竺佛念筆受時）的增補？此問題直接關係到大乘佛教起源的歷史判斷。

### 八敬法的歷史性問題

《大愛道品》中的「八敬法」（比丘尼需向比丘禮拜、遵從等規定）在近代學術界引發廣泛爭論：這些規定是佛陀真正制定的，還是後代男性僧團強加的？Ute Hüsken、Anālayo 等學者從文本比較和律典對比的角度，論證八敬法可能是部分後來添加的，在不同部派版本之間也有差異。

### 竺佛念翻譯的可靠性

竺佛念在四部阿含翻譯中參與了《長阿含》和《增壹阿含》，其翻譯風格偏向意譯，引入了較多漢語習慣表達，有時距原文甚遠。曾有學者指出《增壹阿含》中某些句子可能是意譯而非直譯，給文獻語義的精確重建帶來困難。
