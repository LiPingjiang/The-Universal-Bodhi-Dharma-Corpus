# 大乘起信論探玄

## 學術解讀

### 一、文獻學信息

《大乘起信論探玄》是唐代淨影慧遠（523-592）所撰。慧遠，俗姓李，敦煌人，爲隋代地論宗南道系高僧，世稱"淨影寺慧遠"或"小慧遠"（以別於廬山慧遠）。此疏以"探玄"爲名，取"探賾索隱，鉤深致遠"之意，是《大乘起信論》早期註疏中較爲重要的一部。

淨影慧遠是地論宗南道系的代表人物，地論宗以《十地經論》爲根本，探討"佛性""阿梨耶識"等問題，與《大乘起信論》的如來藏思想有密切關聯。慧遠一生著述宏富，有《大乘義章》《十地經論義記》等傳世。此疏作於隋代，是現存較早的《起信論》註疏之一。

### 二、歷史背景

隋代是中國佛教從南北分裂走向統一的重要時期。北方地論宗與南方攝論宗在"佛性""阿梨耶識"等問題上存在分歧。淨影慧遠作爲地論宗南道系的代表，主張"阿梨耶識"即是"真如"（清淨心），與地論宗北道系主張"阿梨耶識"是"無明"（妄心）形成對立。《起信論》的"一心二門"思想恰好爲這一爭論提供了新的視角，慧遠此疏正是在這一學術背景下產生。

### 三、思想史定位

慧遠此疏的特色在於以地論宗的"阿梨耶識"理論來解釋《起信論》的"一心"。他認爲，《起信論》的"一心"即是地論宗所說的"阿梨耶識"，但"阿梨耶識"有"真""妄"兩面："真"的一面即是"心真如門"，"妄"的一面即是"心生滅門"。慧遠進一步將"無明"解釋爲"阿梨耶識"的"不覺"狀態，將"始覺"解釋爲"阿梨耶識"迴歸"本覺"的過程。這一解釋對後來地論宗與《起信論》的關係有重要影響。

## 白話解讀

### 核心內容

《大乘起信論探玄》是隋代高僧淨影慧遠對《大乘起信論》的解釋。慧遠大師是地論宗的專家，地論宗專門研究《十地經論》和"佛性"問題。

慧遠大師用"阿梨耶識"來解釋《起信論》的"一心"：阿梨耶識是我們最深層的意識，它有兩個面——"真"的面（清淨）和"妄"的面（染污）。"心真如門"就是阿梨耶識的"真面"，"心生滅門"就是阿梨耶識的"妄面"。

慧遠大師還解釋了"無明"：無明就是阿梨耶識"睡着了"——它本來清淨，但暫時迷失了。修行的過程，就是"叫醒"阿梨耶識，讓它從"不覺"回到"本覺"。

### 現代啓示

1. **深層意識**：地論宗對"阿梨耶識"的分析，可以看作是對人類深層意識的探索。現代人通過心理學瞭解自己的潛意識，與古人通過佛學瞭解"阿梨耶識"有相似之處。

2. **本具與迷失**：慧遠說佛性"本具"但"迷失"，啓示我們：很多能力和智慧不是從外面獲得的，而是本來就有的，只是暫時被遮蔽了。

3. **覺醒的過程**：從"不覺"到"始覺"再到"本覺"的過程，可以看作是一個自我覺醒的旅程。

### 閱讀建議

- 慧遠的疏解較爲古樸，建議配合《大乘義章》一同閱讀
- 可瞭解地論宗南北道之爭的背景，理解慧遠"阿梨耶識爲真如"的立場
- 注意地論宗與攝論宗、唯識宗在"阿梨耶識"問題上的差異
