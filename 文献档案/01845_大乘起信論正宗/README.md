# 大乘起信論正宗

## 學術解讀

### 一、文獻學信息

《大乘起信論正宗》是唐代窺基（632-682）所撰。窺基，俗姓尉遲，長安（今陝西西安）人，爲唐代唯識宗創始人玄奘的大弟子，世稱"慈恩大師"，被尊爲唯識宗實際奠基人。此疏以"正宗"爲名，取"顯正摧邪，宗歸一心"之意，是唯識宗對《大乘起信論》進行系統批判和重新詮釋的重要著作。

窺基一生隨玄奘譯經，精通唯識、因明，著有《成唯識論述記》《瑜伽師地論略纂》《因明入正理論疏》等，被合稱爲"慈恩宗"的根本典籍。此疏作於玄奘圓寂後，是窺基以唯識宗義理獨立詮釋《起信論》的代表作。

### 二、歷史背景

玄奘從印度帶回的唯識學，與當時中國流行的《起信論》如來藏思想存在根本差異。唯識宗主張"阿賴耶識緣起"，強調"轉識成智"的漸修次第；而《起信論》主張"心性本淨"，強調"本覺"與"始覺"的不二。玄奘在世時，對《起信論》持保留態度；窺基繼承師說，以"正宗"爲名撰寫此疏，旨在以唯識宗的正統義理來"校正"《起信論》的"偏邪"。

### 三、思想史定位

窺基此疏的核心立場是"以唯識正《起信》"。他認爲，《起信論》的"一心二門"雖然善巧，但容易讓人誤解爲"真心"與"妄心"是兩個獨立的存在，從而陷入"真妄二元"的誤區。窺基以唯識宗的"八識"理論重新解釋："一心"即是"第八阿賴耶識"，"真如"不是離開八識的另一種"心"，而是第八識的"清淨分"。這一解釋雖然被華嚴宗、天台宗批評爲"將《起信論》唯識化"，但體現了唐代佛教義學的深度辯論，對後來理解唯識與如來藏的關係有重要參考價值。

## 白話解讀

### 核心內容

《大乘起信論正宗》是唐代唯識宗大師窺基對《大乘起信論》的批判性詮釋。窺基大師是玄奘大師的大弟子，唯識學的權威。

窺基大師說：《起信論》講"一心二門"很好，但容易讓人誤解。有人以爲"心真如門"是一個心，"心生滅門"是另一個心，這樣就變成了"兩個心"。其實，"一心"就是唯識宗說的"阿賴耶識"——我們最深層的意識。

"真如"不是離開阿賴耶識的另一個東西，而是阿賴耶識的"清淨面"。就像一塊水晶，本身透明（清淨），但放在不同顏色的布上就會顯出不同顏色（染污）。水晶和透明不是兩個東西，而是同一個東西的兩個描述。

### 現代啓示

1. **避免二元對立**：窺基批評"真妄二元"，啓示我們：看問題不要陷入非此即彼的二元思維。清淨與染污不是兩個獨立的東西，而是同一個心的不同狀態。

2. **批判性學習**：窺基對《起信論》持批判態度，但不是全盤否定。這種"批判性繼承"的態度，對今天的學習和研究都有啓示。

3. **系統思維**：唯識宗的"八識"理論是一個複雜的系統。窺基用系統思維來解釋《起信論》，啓示我們：複雜問題需要系統的分析方法。

### 閱讀建議

- 窺基的疏解需要唯識學基礎，建議先讀《成唯識論》或《八識規矩頌》
- 可對比良賁、慧沼以唯識解《起信論》的方式，理解唯識宗解釋傳統的延續
- 注意窺基"以唯識正《起信》"的批判立場，理解唐代佛教義學的辯論
