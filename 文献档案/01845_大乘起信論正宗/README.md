## 学术解读

### 一、文献学信息

《大乘起信论正宗》是唐代窥基（632-682）所撰。窥基，俗姓尉迟，长安（今陕西西安）人，为唐代唯识宗创始人玄奘的大弟子，世称"慈恩大师"，被尊为唯识宗实际奠基人。此疏以"正宗"为名，取"显正摧邪，宗归一心"之意，是唯识宗对《大乘起信论》进行系统批判和重新诠释的重要著作。

窥基一生随玄奘译经，精通唯识、因明，著有《成唯识论述记》《瑜伽师地论略纂》《因明入正理论疏》等，被合称为"慈恩宗"的根本典籍。此疏作于玄奘圆寂后，是窥基以唯识宗义理独立诠释《起信论》的代表作。

### 二、历史背景

玄奘从印度带回的唯识学，与当时中国流行的《起信论》如来藏思想存在根本差异。唯识宗主张"阿赖耶识缘起"，强调"转识成智"的渐修次第；而《起信论》主张"心性本净"，强调"本觉"与"始觉"的不二。玄奘在世时，对《起信论》持保留态度；窥基继承师说，以"正宗"为名撰写此疏，旨在以唯识宗的正统义理来"校正"《起信论》的"偏邪"。

### 三、思想史定位

窥基此疏的核心立场是"以唯识正《起信》"。他认为，《起信论》的"一心二门"虽然善巧，但容易让人误解为"真心"与"妄心"是两个独立的存在，从而陷入"真妄二元"的误区。窥基以唯识宗的"八识"理论重新解释："一心"即是"第八阿赖耶识"，"真如"不是离开八识的另一种"心"，而是第八识的"清净分"。这一解释虽然被华严宗、天台宗批评为"将《起信论》唯识化"，但体现了唐代佛教义学的深度辩论，对后来理解唯识与如来藏的关系有重要参考价值。

## 白话解读

### 核心内容

《大乘起信论正宗》是唐代唯识宗大师窥基对《大乘起信论》的批判性诠释。窥基大师是玄奘大师的大弟子，唯识学的权威。

窥基大师说：《起信论》讲"一心二门"很好，但容易让人误解。有人以为"心真如门"是一个心，"心生灭门"是另一个心，这样就变成了"两个心"。其实，"一心"就是唯识宗说的"阿赖耶识"——我们最深层的意识。

"真如"不是离开阿赖耶识的另一个东西，而是阿赖耶识的"清净面"。就像一块水晶，本身透明（清净），但放在不同颜色的布上就会显出不同颜色（染污）。水晶和透明不是两个东西，而是同一个东西的两个描述。

### 现代启示

1. **避免二元对立**：窥基批评"真妄二元"，启示我们：看问题不要陷入非此即彼的二元思维。清净与染污不是两个独立的东西，而是同一个心的不同状态。

2. **批判性学习**：窥基对《起信论》持批判态度，但不是全盘否定。这种"批判性继承"的态度，对今天的学习和研究都有启示。

3. **系统思维**：唯识宗的"八识"理论是一个复杂的系统。窥基用系统思维来解释《起信论》，启示我们：复杂问题需要系统的分析方法。

### 阅读建议

- 窥基的疏解需要唯识学基础，建议先读《成唯识论》或《八识规矩颂》
- 可对比良贲、慧沼以唯识解《起信论》的方式，理解唯识宗解释传统的延续
- 注意窥基"以唯识正《起信》"的批判立场，理解唐代佛教义学的辩论
