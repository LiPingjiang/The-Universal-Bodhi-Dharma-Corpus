## 學術解讀

### 一、文獻學信息

《大乘起信論筆削記》是唐代宗密（780-841）所撰。宗密，俗姓何，號圭峯，世稱"圭峯大師"，爲華嚴宗第五祖，同時精通禪宗，是唐代"教禪合一"思潮的代表人物。此疏以"筆削"爲名，取孔子"筆削春秋"之意，表明宗密對《起信論》及前人註疏有所裁正、刪削。

宗密一生致力於融合華嚴宗與禪宗，著有《禪源諸詮集都序》《華嚴經疏》《圓覺經大疏》等。此疏是其晚年作品，綜合法藏《義記》、元曉《義疏》等前人成果，加以自己的判釋，形成集大成的《起信論》註疏。

### 二、歷史背景

中唐時期，佛教內部宗派林立，教宗（華嚴、天台）與禪宗之間時有爭論。宗密出身禪宗（荷澤神會一系），後學華嚴於澄觀大師，成爲融通教禪的關鍵人物。此疏作於會昌法難之前，當時佛教雖盛，但內部整合的需求日益迫切。宗密以"筆削"的方式重新整理《起信論》的解釋傳統，既尊重前人，又有所創新，體現了其"和會"的學術立場。

### 三、思想史定位

宗密此疏的最大特色在於以"教禪一致"的立場來解釋《起信論》。他認爲，《起信論》的"一心二門"即是禪宗"即心即佛"的教理基礎："心真如門"即是"本心"，"心生滅門"即是"妄念"。宗密進一步將荷澤禪的"知"概念融入其中，認爲"心真如"的"自知"即是禪宗所證的"靈知"。這一解釋對後來宋明理學中的"心學"一派有間接影響。

### 四、術語對照表

| 術語 | 來源/釋義 |
|------|----------|
| 筆削 | 取自孔子筆削《春秋》，指裁正、刪削 |
| 一心二門 | 《起信論》核心框架 |
| 教禪一致 | 教宗（華嚴）與禪宗的融合 |
| 靈知 | 宗密對"心"的特稱，指本心的覺知能力 |
| 荷澤宗 | 禪宗六祖慧能弟子神會開創的宗派 |
| 真心 | 宗密術語，指"靈知"即真心，非妄心 |

## 白話解讀

### 核心內容

《大乘起信論筆削記》是圭峯宗密大師對《大乘起信論》的集大成註疏。"筆削"就是像孔子修訂《春秋》一樣，對前人的解釋進行整理和修正。

宗密大師的獨特之處在於：他既是禪宗傳人，又是華嚴宗祖師。所以他用禪宗的語言來解釋《起信論》："心真如門"就是禪宗說的"本心"——清淨、光明、能知能覺；"心生滅門"就是"妄念"——來來去去的念頭。宗密特別強調這個"知"：心不只是靜態的本體，它本身就有"知道"的能力，這個"知"就是佛性。

### 現代啓示

1. **知與覺**：宗密強調"靈知"，啓示我們：修行不只是靜坐，而是培養一種清醒的覺知。在日常生活中保持覺知，就是修行。

2. **融合思維**：宗密融通教禪，啓示我們：不要把自己侷限在某個"門派"裏，各種智慧傳統都有值得學習的地方。

3. **批判繼承**："筆削"的精神啓示我們：學習前人的東西，不是全盤接受，而是要有自己的判斷和選擇。

### 閱讀建議

- 宗密的註疏較爲系統，適合有一定基礎的讀者
- 可配合《禪源諸詮集都序》一同閱讀，理解其"教禪一致"的整體思想
- 注意宗密對"知"的強調，這是理解其思想的關鍵
