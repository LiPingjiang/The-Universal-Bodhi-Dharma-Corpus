## 学术解读

### 一、文献学信息

《大乘起信论笔削记》是唐代宗密（780-841）所撰。宗密，俗姓何，号圭峰，世称"圭峰大师"，为华严宗第五祖，同时精通禅宗，是唐代"教禅合一"思潮的代表人物。此疏以"笔削"为名，取孔子"笔削春秋"之意，表明宗密对《起信论》及前人注疏有所裁正、删削。

宗密一生致力于融合华严宗与禅宗，著有《禅源诸诠集都序》《华严经疏》《圆觉经大疏》等。此疏是其晚年作品，综合法藏《义记》、元晓《义疏》等前人成果，加以自己的判释，形成集大成的《起信论》注疏。

### 二、历史背景

中唐时期，佛教内部宗派林立，教宗（华严、天台）与禅宗之间时有争论。宗密出身禅宗（荷泽神会一系），后学华严于澄观大师，成为融通教禅的关键人物。此疏作于会昌法难之前，当时佛教虽盛，但内部整合的需求日益迫切。宗密以"笔削"的方式重新整理《起信论》的解释传统，既尊重前人，又有所创新，体现了其"和会"的学术立场。

### 三、思想史定位

宗密此疏的最大特色在于以"教禅一致"的立场来解释《起信论》。他认为，《起信论》的"一心二门"即是禅宗"即心即佛"的教理基础："心真如门"即是"本心"，"心生灭门"即是"妄念"。宗密进一步将荷泽禅的"知"概念融入其中，认为"心真如"的"自知"即是禅宗所证的"灵知"。这一解释对后来宋明理学中的"心学"一派有间接影响。

## 白话解读

### 核心内容

《大乘起信论笔削记》是圭峰宗密大师对《大乘起信论》的集大成注疏。"笔削"就是像孔子修订《春秋》一样，对前人的解释进行整理和修正。

宗密大师的独特之处在于：他既是禅宗传人，又是华严宗祖师。所以他用禅宗的语言来解释《起信论》："心真如门"就是禅宗说的"本心"——清净、光明、能知能觉；"心生灭门"就是"妄念"——来来去去的念头。宗密特别强调这个"知"：心不只是静态的本体，它本身就有"知道"的能力，这个"知"就是佛性。

### 现代启示

1. **知与觉**：宗密强调"灵知"，启示我们：修行不只是静坐，而是培养一种清醒的觉知。在日常生活中保持觉知，就是修行。

2. **融合思维**：宗密融通教禅，启示我们：不要把自己局限在某个"门派"里，各种智慧传统都有值得学习的地方。

3. **批判继承**："笔削"的精神启示我们：学习前人的东西，不是全盘接受，而是要有自己的判断和选择。

### 阅读建议

- 宗密的注疏较为系统，适合有一定基础的读者
- 可配合《禅源诸诠集都序》一同阅读，理解其"教禅一致"的整体思想
- 注意宗密对"知"的强调，这是理解其思想的关键
