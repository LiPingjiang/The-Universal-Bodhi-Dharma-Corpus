# 大乘起信論顯宗

## 學術解讀

### 一、文獻學信息

《大乘起信論顯宗》是唐代善無畏（637-735）所撰。善無畏，俗姓李，中天竺（今印度中部）人，爲唐代密宗（真言宗）的傳入者之一，與金剛智、不空並稱"開元三大士"，被尊爲中國密宗的祖師。此疏以"顯宗"爲名，取"顯明宗要，揭示密意"之意，是密宗系統解釋《大乘起信論》的重要著作。

善無畏於唐玄宗開元四年（716）來華，傳授密宗胎藏界法門，譯出《大日經》等密宗根本經典。他精通顯密二教，認爲密宗的"即身成佛"與《起信論》的"本覺"思想相通。此疏作於其中土弘法期間，是其以密宗立場重新詮釋《起信論》的代表作。

### 二、歷史背景

唐代開元年間，密宗正式傳入中國，形成與顯宗（天台、華嚴、禪等）並立的格局。善無畏作爲印度密宗大師，需要將密宗的深奧義理與中國人熟悉的佛教概念相聯繫。《起信論》的"一心二門""本覺"思想，恰好爲密宗的"即身成佛""六大無礙"等概念提供了教理基礎。此疏正是在這一"顯密融合"的背景下產生。

### 三、思想史定位

善無畏此疏的最大特色在於以密宗的"六大"（地、水、火、風、空、識）和"四曼"（大曼陀羅、三昧耶曼陀羅、法曼陀羅、羯磨曼陀羅）來解釋《起信論》的"一心"。他認爲，《起信論》的"一心"即是密宗的"心王大日"——宇宙的本體、萬物的根源。"心真如門"對應"胎藏界"（理性、本覺），"心生滅門"對應"金剛界"（智性、始覺）。善無畏進一步將"修行"解釋爲"三密相應"（身密、口密、意密），認爲通過密宗的儀軌修行，可以即身成就"心真如"。這一解釋對後來密宗與如來藏思想的融合有重要影響。

## 白話解讀

### 核心內容

《大乘起信論顯宗》是唐代密宗大師善無畏對《大乘起信論》的解釋。善無畏是印度來的高僧，把密宗傳到中國。

善無畏大師用密宗的概念來解釋《起信論》：《起信論》的"一心"就是密宗的"大日如來"——宇宙的本體、一切的根本。"心真如門"對應"胎藏界"——像胎兒在母胎中一樣，本具一切但尚未顯現；"心生滅門"對應"金剛界"——像金剛一樣堅固的智慧，能破除一切障礙。

善無畏大師還說：修行不需要等很久，通過"三密相應"（身體做手印、口唸真言、心意觀想），可以"即身成佛"——這輩子就證悟。

### 現代啓示

1. **身心合一**：密宗"三密相應"強調身體、語言、意識的統一。現代人常常身心分離（如邊喫飯邊看手機），"三密"啓示我們：專注當下，身心合一。

2. **即身成佛**："即身成佛"不是說不需要努力，而是強調修行的當下性。不要等待未來，就在此刻修行。

3. **宇宙一體**：密宗"六大"思想認爲人體與宇宙是由同樣的元素構成的。這啓示我們：人與自然是統一的，保護環境就是保護自己。

### 閱讀建議

- 善無畏的疏解需要密宗基礎知識，建議先了解密宗的基本概念
- 可配合《大日經疏》一同閱讀，理解密宗的哲學體系
- 注意密宗與顯宗在修行方法上的差異，理解"顯密融合"的意圖
