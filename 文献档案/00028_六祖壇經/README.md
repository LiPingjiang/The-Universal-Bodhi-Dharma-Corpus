## 目錄簡介

本目錄收錄《六祖壇經》的核心文獻，這是中國佛教史上唯一被尊稱爲"經"的本土著作，記錄了禪宗六祖慧能（638—713年）的生平、開悟經歷和核心教法。

**梵文對應**：Liù-zǔ tán-jīng（無梵文原典，純中國本土經典）  
**作者**：慧能（638—713年），由弟子法海等記錄整理  
**成書時代**：約公元8世紀  
**核心主題**：頓悟、無念、無相、無住、自性是佛

## 關鍵概念

- **頓悟**（dùn wù）：一念之間徹底覺悟，不需要漸進修行
- **無念**（wú niàn）：不執著於念頭，而非沒有念頭
- **無相**（wú xiāng）：不執著於外在形相
- **無住**（wú zhù）：心不停留在任何地方，自由流動
- **自性**（zì xìng）：每個人本有的佛性，清淨光明
