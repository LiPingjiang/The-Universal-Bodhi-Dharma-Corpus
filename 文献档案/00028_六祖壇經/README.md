## 目录简介

本目录收录《六祖坛经》的核心文献，这是中国佛教史上唯一被尊称为"经"的本土著作，记录了禅宗六祖慧能（638—713年）的生平、开悟经历和核心教法。

**梵文对应**：Liù-zǔ tán-jīng（无梵文原典，纯中国本土经典）  
**作者**：慧能（638—713年），由弟子法海等记录整理  
**成书时代**：约公元8世纪  
**核心主题**：顿悟、无念、无相、无住、自性是佛

## 关键概念

- **顿悟**（dùn wù）：一念之间彻底觉悟，不需要渐进修行
- **无念**（wú niàn）：不执著于念头，而非没有念头
- **无相**（wú xiāng）：不执著于外在形相
- **无住**（wú zhù）：心不停留在任何地方，自由流动
- **自性**（zì xìng）：每个人本有的佛性，清净光明
