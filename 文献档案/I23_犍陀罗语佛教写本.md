## 名称还原与解释

**犍陀罗**（Gandhāra）：古代地名——大约对应今天巴基斯坦白沙瓦谷地+阿富汗东部。这里是佛教从印度北传入中亚（然后丝绸之路→中国）的关键门户。

**犍陀罗语**（Gāndhārī/Gandhari Prakrit）：不是梵语也不是巴利语——而是古代犍陀罗地区人民实际说的"方言"（俗语/普拉克里特语的西北方言）。佛陀曾允许弟子"用各自的语言传教"——犍陀罗语佛经就是这一传统的产物。

## 原始内容落盘

### 发现史

| 年代 | 事件 |
|------|------|
| 1994 | 英国图书馆从阿富汗走私古物市场获得第一批桦树皮卷轴 |
| 1996–至今 | 陆续出现更多批次——来自不同收藏（Schøyen Collection, Bajaur Collection等） |
| 2000s | 华盛顿大学 Richard Salomon 团队开始系统解读 |

这些写本的物理形态：卷起的桦树皮条——极度脆弱——2000年后展开时很多已碎裂。

### 已辨识的内容

| 类型 | 具体文本 |
|------|----------|
| 经 | 法句经犍陀罗语版（与巴利本差异显著）、犀角经、本生故事 |
| 律 | 法藏部律的片段 |
| 论 | 早期阿毗达磨式分类文本 |
| 偈颂 | 不见于巴利/汉传的独立偈颂 |
| 注释 | 对佛教术语的犍陀罗语解释 |
| 杂项 | 大乘经典的早期版本？（争议中）——可能包括般若经的某些片段 |

### 学术意义

1. **最古老的佛教实物文本**：碳-14测年显示部分写本可追溯到前1世纪——这意味着它们比任何现存的巴利写本（最早约15世纪）或梵文写本（最早约2世纪）都要古老得多。
2. **第三种语言传统**：在巴利文和梵文之外，证明了早期佛教确实存在第三种（甚至更多种）语言的传教传统——佛教从来就不是"只用一种语言"的。
3. **法藏部的实证**：部分写本明确属于法藏部——这让我们第一次有了这个部派的实物文本（之前只通过汉译四分律等间接了解）。
4. **大乘起源问题**：如果某些犍陀罗写本中确实包含大乘经典的早期形式——这对"大乘佛教何时/何地起源"这个大问题有重大影响。
5. **丝绸之路佛教传播**：犍陀罗是佛教进入中亚和中国的"中转站"——这些写本证明了在"翻译成汉文"之前，佛经已经经历了从巴利/梵文→犍陀罗语→中亚语言的多层传递过程。

### 法句经犍陀罗语版——具体比较

巴利法句经有423偈/26品；犍陀罗语法句经的结构不同——品的排列不同，某些偈颂有文字差异。这证明"法句经"并不是一本固定的书——而是各部派都有自己版本的"佛教名言选集"。

---

## 大白话总结

犍陀罗语佛教写本——是我们今天能用手摸到的最古老的佛教"书"。

在1994年之前，我们从来没有见过任何2000年前的佛经实物——一切都是后来的抄本。然后突然之间——从阿富汗战乱中流出了一批桦树皮卷轴——打开一看——居然是公元前1世纪的佛经！而且不是巴利文也不是梵文——是一种我们之前只在钱币和铭文上见过的"犍陀罗语"。

这些写本告诉我们一个重要的事实：早期佛教的多样性远超我们的想象。不同地区的佛教徒用不同语言传教、不同的方式编排经典——这种多样性后来在"标准化"过程（巴利化/梵文化）中被抹平了。犍陀罗语写本让我们瞥见了被抹平之前的原始多样性。
