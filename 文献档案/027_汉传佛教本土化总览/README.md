## 目录简介

本目录收录汉传佛教本土化进程的核心文献，涵盖佛教传入中国后与儒道文化融合、形成独特宗派体系的历史与思想。

**梵文名称**：Cīna-bauddha-dharma（中国佛法）  
**时代范围**：东汉至宋代（约公元1世纪—13世纪）  
**核心主题**：格义佛教、判教体系、宗派分化、儒佛道三教融合

## 关键概念

- **格义**（gé yì）：用儒道概念比附佛教概念的翻译方法
- **判教**（pàn jiào）：对佛陀一生教法进行分类排序的体系
- **五时八教**：天台宗智顗的判教体系
- **五教十宗**：华严宗法藏的判教体系
- **禅净双修**：禅宗与净土宗的融合修行
