## 序

如來說法，四十九年，隨機應化，言語無量。或長或短，或深或淺，或以偈頌，或以散文，或對天人，或對凡夫，皆為一事——令諸眾生，離苦得樂，究竟解脫。

法既滅後，弟子結集，口耳相傳，輾轉書寫，歷千百載，流布寰宇。巴利、梵文、漢藏諸語，各承一脈；南傳、北傳、密乘諸宗，各守一燈。雖文字有異，所指無二。

今人處末法之世，典籍浩如煙海，欲覓一徑，往往茫然。本藏之作，意在為此：遍搜諸語原典，逐部落盤存真；既錄原文，復以今語疏解；既存術語，復以熟詞裝新意——令古聖之言，不隔於今人；令深奧之義，可達於普眾。

覺海無涯，此藏一葦。願覽者隨文入觀，自得其益。

## 閱讀說明

每部獨立文獻備有三種讀法，讀者可按需取用：

**原文**——巴利、梵、漢、藏原典落盤，附漢譯對照，存真第一。

**解讀·學術**——術語精確，含文獻考證、版本異同、思想史定位，適合有一定基礎的讀者深入研究。

**解讀·白話**——通篇以現代中文日常熟詞講解，術語悉數沉入腳注，流暢如讀一篇現代文章，適合初入門者及一般讀者。
