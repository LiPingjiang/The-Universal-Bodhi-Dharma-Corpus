## 项目简介

本项目是一部**跨语言、跨年代的佛教文献全集**，目标是将人类历史上所有重要的佛教典籍——从最早的口传法藏到近现代研究著作——系统地整理、落盘、解读，并以现代中文读者能够真正读懂的方式呈现。

项目名称"普世菩提法藏"取意：

- **普世**——覆盖巴利、梵、汉、藏、英及各小语种，不限传统、不限地域。
- **菩提**（Bodhi）——觉悟、智慧，佛教一切修行的指向。
- **法藏**——一切佛法教义的总汇。

## 每部经的文件体例

每一部独立文献对应**三份 Markdown + Excel 一行**：

| 文件 | 内容 |
|---|---|
| `XX_原文.md` | 原始语言落盘（巴利/梵/汉/藏原文）+ 汉译对照 |
| `XX_解释_传统.md` | 传统佛教术语体系解读，含名称还原、体系定位、术语小考 |
| `XX_解释_现代.md` | 全程用现代中文日常熟词讲解，术语沉入脚注，流畅如现代文章 |

**"用熟词装新意"** 是本项目现代解读的核心原则：正文不出现令人陌生的音译术语，一切专名以大白话表达，原始术语以脚注形式保留，供深入研究者参考。

## 多级索引体系

顶层总表（`佛教佛法文献古籍大全.xlsx`）按七个历史分期收录 181 条文献，每条可向下展开为独立的多级 Excel 索引，采用**灵活分层**结构（父节点ID串联），节点分"分类节点"与"独立文献"两类。

七个历史分期：

| 分期 | 年代 | 内容 |
|---|---|---|
| I | 约前6–前3世纪 | 原始佛教·口传与早期结集 |
| II | 约前3–前1世纪 | 部派佛教·各部派典籍 |
| III | 约前1–3世纪 | 早期大乘·般若等经典 |
| IV | 约3–7世纪 | 大乘论书·中观唯识 |
| V | 约7–12世纪 | 密乘·金刚乘典籍 |
| VI | 约5–20世纪 | 汉传·藏传本土化典籍 |
| VII | 近现代 | 学术研究·现代整理 |

## 佛国史主线

`佛国史/` 文件夹是独立于文献档案的**横向思想演化史**，记录佛教知识体系的构建、迭代与流变：从佛陀的几条核心洞见，到部派分裂、大乘兴起、密乘发展、传入中土与西藏、近现代复兴，以及术语在不同语言文化中的翻译、误读与再诠释。
