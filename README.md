## 序

如來說法，四十九年，隨機應化，言語無量。或長或短，或深或淺，或以偈頌，或以散文，或對天人，或對凡夫，皆為一事——令諸眾生，離苦得樂，究竟解脫。

法既滅後，弟子結集，口耳相傳，輾轉書寫，歷千百載，流布寰宇。巴利、梵文、漢藏諸語，各承一脈；南傳、北傳、密乘諸宗，各守一燈。雖文字有異，所指無二。

今人處末法之世，典籍浩如煙海，欲覓一徑，往往茫然。本藏之作，意在為此：遍搜諸語原典，逐部落盤存真；既錄原文，復以今語疏解；既存術語，復以熟詞裝新意——令古聖之言，不隔於今人；令深奧之義，可達於普眾。

覺海無涯，此藏一葦。願覽者隨文入觀，自得其益。

## 全庫概覽

本藏收錄 **29,765 部** 佛教文獻，統一編號 `00001`–`29765`，按佛教歷史與文獻邏輯順序排列。涵蓋南傳巴利三藏、漢傳大藏經、藏傳甘珠爾/丹珠爾、日本韓國越南佛教典籍，以及近現代中、日、歐美佛學研究成果。

### 三件套體例

每部文獻目錄內統一使用繁體裸名三件套：

- **原文.md** — 原典文本（巴利/梵/漢/藏對照或單語），含經典資訊、結構大綱、核心段落摘錄
- **解讀_學術.md** — 學術解讀：文獻學資訊、歷史地位、教義分析、研究現狀
- **解讀_白話.md** — 白話解讀：現代白話通俗講解、核心思想、現代啟示

部分文獻另有 `索引卡.md`（文獻信息卡）和 `README.md`（導覽）。

龍樹立中觀、無著世親建唯識，佛教哲學登峰造極；如來藏融攝兩系，因明鑄造理性利器。

- 00018 龍樹與中觀學派

### 第五期　密乘·金剛乘典籍（約7–12世紀）

真言陀羅尼匯為大流，兩部曼荼羅立正純密教，無上瑜伽續以氣脈明點即身成佛。

- 00024 密乘金剛乘總覽
- 00025 大日經與金剛頂經
- 00026 時輪續與無上瑜伽

### 第六期　漢傳·藏傳本土化典籍（約5–20世紀）

佛法東流，漢地成八宗、禪淨雙修；高原立四派、顯密圓融。

- 00028 六祖壇經

西學東漸，佛典入學院；文獻學、比較哲學、認知科學多維交照——古老傳統在現代語境中重獲生命。
