## 一、现状诊断

### 1.1 目录组织三种模式

| 模式 | 范围 | 结构 | 问题 |
|------|------|------|------|
| 嵌套多文献 | `001_口传法藏` | 期目录→8个分类子目录→每子目录平铺多文献三件套 | 文献无独立目录，1123个文件散在8个子目录 |
| 扁平多文献 | `002`–`009` | 期目录→多文献三件套直接平铺 | 一个目录混放6-7个文献 |
| 一文献一目录 | I-IX期、X期 | 期目录→每文献一个子目录→三件套 | 结构清晰，但文件命名不统一 |

### 1.2 文件命名六套体系

| 体系 | 示例 | 范围 | 估计数量 |
|------|------|------|---------|
| 繁体裸名 | `原文.md` `解讀_學術.md` `解讀_白話.md` | X501-X28125（P5/P6替换后） | 28000+ |

| 期级解读 | `解讀_學術.md` `解讀_白話.md`（期目录级别） | 002-009 | ~18 |

| 类型 | 位置 | 数量 | 说明 |

### 1.4 X151-X200编号冲突

X151-X200每个编号下有4个散落文件，分属两个完全不同的文献：

```
X151_大智度論_原文.md          ← 繁体前缀，文献A
X151_大智度論_解讀_學術.md     ← 繁体前缀，文献A
X151_大智度論_解讀_白話.md     ← 繁体前缀，文献A
X151_四分律行事钞.md           ← 简体裸名，文献B（与X151_四分律行事钞/目录对应）
```

同时，`文献档案/X151_四分律行事钞/` 目录已存在（文献B的目录）。所以文献A（大智度論等50个）无目录，文献B（四分律行事钞等50个）有目录但散落了一个同名的md索引卡。

## 二、设计目标

1. **一文献一目录**：全库统一，每个文献拥有独立目录
2. **三件套统一命名**：`原文.md` + `解讀_學術.md` + `解讀_白話.md`（繁体裸名）
3. **索引卡归位**：每个文献的索引卡移入对应目录，命名 `索引卡.md`
4. **根目录整洁**：文献目录全部归入 `文献档案/`，根目录仅保留项目级文件
5. **零删除**：所有调整通过重命名/移动实现，重复文件以"备份"形式保留

### 4.5 三件套统一命名

每个文献目录内统一为繁体裸名：

| 标准文件名 | 来源 |
|-----------|------|
| `原文.md` | 优先用已有裸名文件；若无，将 `{名称}_原文.md` 重命名 |
| `解讀_學術.md` | 优先用已有裸名文件；若无，将 `{名称}_解讀_學術.md` 或 `{名称}_解读_学术.md` 重命名 |
| `解讀_白話.md` | 优先用已有裸名文件；若无，将 `{名称}_解讀_白話.md` 或 `{名称}_解读_白话.md` 重命名 |

简→繁映射：`解读_学术` → `解讀_學術`，`解读_白话` → `解讀_白話`。

VII06-VII212的简体裸名（`原文.md` + `解读_学术.md` + `解读_白话.md`）也需将后两个重命名为繁体。

### 4.6 重复文件处理

目录内同时存在前缀版和裸名版时：

1. 保留裸名版作为标准文件
2. 前缀版重命名为 `{名称}_原文_备份.md` 等
3. 不删除任何文件

### 4.7 002-009期文献拆分

将002-009期目录内平铺的多文献三件套，按文献名拆分为独立子目录。以 `_原文.md`、`_解读_学术.md`、`_解读_白话.md` 为后缀识别文献名。

### 4.8 001_口传法藏子目录内文献拆分

同理，将 `001_口传法藏/01_修道基础/` 等子目录内的多文献三件套拆分为独立子目录。
